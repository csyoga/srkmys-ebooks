‘ಕೇನೋಪನಿಷತ್​’ ಎಂಬ ಹೆಸರಿನಿಂದ ಪ್ರಸಿದ್ಧವಾದ ತಲವಕಾರೋಪ-- ನಿಷತ್ತು, ಸಾಮವೇದದ ತಲವಕಾರಬ್ರಾಹ್ಮಣಕ್ಕೆ ಸೇರಿದ್ದು. ತಲವಕಾರಕ್ಕೆ ಜೈಮಿ ನೀಯಬ್ರಾಹ್ಮಣವೆಂಬ ಹೆಸರೂ ಉಂಟು. ಈ ಉಪನಿಷತ್ತಿನ ಮೊದಲನೆಯ ಮಂತ್ರವು ‘ಕೇನ’ ಎಂಬ ಪದದಿಂದ ಪ್ರಾರಂಭವಾಗುವುದರಿಂದ ಇದಕ್ಕೆ ‘ಕೇನೋ ಪನಿಷತ್​’ ಎಂಬ ಹೆಸರು ಬಂದಿದೆ. ಶ್ರೀಶಂಕರಾಚಾರ್ಯರು ತಮ್ಮ ಅವತರಣಿಕೆ ಯಲ್ಲಿ ಈ ಉಪನಿಷತ್ತು (ತಲವಕಾರಬ್ರಾಹ್ಮಣದ) ಒಂಬತ್ತನೆಯ ಅಧ್ಯಾಯ ವಾಗಿದೆಯೆಂದು ಹೇಳುತ್ತಾರೆ. ಡಾ. ಬರ್ನೆಲ್ (Dr. A. S. Geden) ಮೊದಲಾದ ಪಾಶ್ಚಾತ್ಯ ಪಂಡಿತರ ಅಭಿಪ್ರಾಯದ ಪ್ರಕಾರ ಇದು ಅದಕ್ಕಿಂತಲೂ ಪ್ರಾಚೀನ ವಾದದ್ದು, ಮತ್ತು ಹಳೆಯ ಗದ್ಯಪದ್ಯದ ರೂಪದಲ್ಲಿರುವ ಬೃಹದಾರಣ್ಯಕ, ಛಾಂದೋಗ್ಯ, ತೈತ್ತಿರೀಯ, ಐತರೇಯ, ಕೌಷೀತಕಿ ಈ ಉಪನಿಷತ್ತುಗಳ ಗುಂಪಿಗೆ ಸೇರಿದ್ದು.

ಈ ಉಪನಿಷತ್ತಿಗೆ ಶ್ರೀಶಂಕರಾಚಾರ್ಯರದೆಂದು ಪ್ರಸಿದ್ಧವಾದ ಎರಡು ಭಾಷ್ಯ ಗಳಿರುತ್ತವೆ : ಒಂದಕ್ಕೆ ಪದಭಾಷ್ಯವೆಂದೂ ಇನ್ನೊಂದಕ್ಕೆ ವಾಕ್ಯಭಾಷ್ಯವೆಂದೂ ಹೆಸರುಂಟು. ವಾಕ್ಯಭಾಷ್ಯವು ಪದಭಾಷ್ಯಕಾರರದೇ ಎಂದು ಖಂಡಿತವಾಗಿ ನಿಷ್ಕರ್ ಷಿಸಿ ಹೇಳಲು ಕೆಲವು ಆಕ್ಷೇಪಣೆಗಳುಂಟು. ಈ ಮಾಲೆಯಲ್ಲಿ ಪ್ರಕಟವಾಗಿರುವ ಇತರ ಉಪನಿಷತ್ತುಗಳಂತೆ ಇಲ್ಲಿಯೂ ಕೂಡ ಪ್ರತಿಪದಾರ್ಥ ತಾತ್ಪರ್ಯ- ಟಿಪ್ಪಣಿಗಳು ಶಾಂಕರಭಾಷ್ಯವನ್ನೇ (ಎಂದರೆ ಪದಭಾಷ್ಯವನ್ನೇ) ಅವಲಂಬಿಸಿವೆ.

ನಾಲ್ಕು ಖಂಡಗಳನ್ನು ಒಳಕೊಂಡಿರುವ ಈ ಉಪನಿಷತ್ತು, ಶ್ರೀಶಂಕರಾ ಚಾರ್ಯರು ತಮ್ಮ ಭಾಷ್ಯಾವತರಣಿಕೆಯಲ್ಲಿ ಹೇಳುವಂತೆ, ಪರಬ್ರಹ್ಮವಿಷಯ ವಾದದ್ದು. ಪರಬ್ರಹ್ಮವು ಅತೀಂದ್ರಿಯವಸ್ತುವೆಂದೂ, ಜಡವಾದ ಇಂದ್ರಿಯಗಳ ಚೇತನತ್ವವು ಅದರಿಂದಲೇ ಉಂಟಾಗಿದೆಯೆಂದೂ, ಅದರ ಸ್ವರೂಪವನ್ನು ಪ್ರಮಾಣಗಳಿಂದ ಇತರರಿಗೆ ಉಪದೇಶಿಸುವುದು ಅಸಾಧ್ಯವೆಂದೂ ಮೊದಲನೆಯ ಖಂಡವು ತಿಳಿಸುತ್ತದೆ. ‘ಬ್ರಹ್ಮವನ್ನು ಹೇಗೆ ಅರಿಯಬಹುದು?’ಎಂಬುದಕ್ಕೆ ಎರಡ ನೆಯ ಖಂಡವು ಉತ್ತರವನ್ನು ಕೊಡುತ್ತದೆ ; ‘ತಿಳಿದುಕೊಂಡಿಲ್ಲವೆಂದೂ ಎಣಿಸಿರು ವುದಿಲ್ಲ. ತಿಳಿದೂ ಇರುತ್ತೇನೆ’ ಎಂಬ ಜ್ಞಾನವು ಯಾವನಿಗೆ ಇರುತ್ತದೆಯೊ ಅವನೇ ತಿಳಿದಿರುತ್ತಾನೆ. ಬ್ರಹ್ಮವು ಇಂದ್ರಿಯಗಳಿಗೆ ವಿಷಯವಾಗದಿದ್ದರೂ ಪ್ರತ್ಯ ಗಾತ್ಮನೆಂದು ಅರಿಯಬಹುದು. ಆದರೆ ‘ಚೆನ್ನಾಗಿ ಅರಿತಿದ್ದೇನೆ’ ಎಂದು ಹೇಳ ಲಾಗುವುದಿಲ್ಲ. ಮೂರನೆಯ ಖಂಡದಲ್ಲಿ ಒಂದು ಆಖ್ಯಾಯಿಕೆಯು ಬರುತ್ತದೆ. ದೇವತೆಗಳ ಗರ್ವಭಂಗಕ್ಕಾಗಿ ಬ್ರಹ್ಮವು ಯಕ್ಷರೂಪದಿಂದ ಅಗ್ನಿ ವಾಯುಗಳಿಗೆ ಕಾಣಿಸಿಕೊಳ್ಳಲು ಅವರಿಗೆ ಅದೇನೆಂದು ತಿಳಿಯಲಿಲ್ಲ. ಇಂದ್ರನಿಗೆ ಮಾತ್ರ ಉಮಾ- ಹೈಮವತಿಯ ದರ್ಶನವಾದ ಮೇಲೆ ಅದು ಬ್ರಹ್ಮವೆಂದು ತಿಳಿಯಿತು. ಬ್ರಹ್ಮ ವಿದ್ಯೆಯ ಸ್ತುತಿಗಾಗಿಯೂ, ಬ್ರಹ್ಮದ ಅಸ್ತಿತ್ವವನ್ನು ಪ್ರತಿಪಾದಿಸುವುದಕ್ಕಾಗಿಯೂ ಈ ಆಖ್ಯಾಯಿಕೆಯನ್ನು ಹೇಳಿದೆ. ಈ ಖಂಡಕ್ಕೂ ಮುಂದಿನ ಖಂಡಕ್ಕೂ ಸಂಬಂಧ ವುಂಟು. ನಾಲ್ಕನೆಯ ಖಂಡವು ಅತ್ಯಂತ ಮುಖ್ಯವೆಂದು ಹೇಳಬಹುದು. ಉಮಾ- ಹೈಮವತಿಯು ಉಪಮಾನಗಳ ಮೂಲಕ ಬ್ರಹ್ಮವಿದ್ಯೆಯನ್ನು ಇಂದ್ರನಿಗೆ ಉಪ ದೇಶಿಸುತ್ತಾಳೆ. ಬ್ರಹ್ಮವು ಮಿಂಚಿನ ಬಳ್ಳಿಯಂತೆ ಹೊಳೆದು ಮಾಯವಾಗುತ್ತದೆ. ಬೃಹದಾರಣ್ಯಕೋಪನಿಷತ್ತಿನಲ್ಲಿ (೨. ೩. ೬) ಬ್ರಹ್ಮದ ರೂಪವೂ ‘ಸಕೃದ್ವಿದ್ಯು ತ್ತಮ್​’ ಎಂದು ಹೇಳಿದೆ. ಮುಂದಿನ ಮಂತ್ರದಲ್ಲಿರುವ ‘ಮಾನಸಿಕ’ ಉಪ ಮಾನವು ದುರ್ವಿಜ್ಞೇಯವಾಗಿರುತ್ತದೆ. ಪ್ರಾಚ್ಯ ಭಾಷ್ಯಕಾರರು ಮತ್ತು ಪಾಶ್ಚಾತ್ಯ ಅನುವಾದಕರು ಇದಕ್ಕೆ ನಾನಾ ವಿಧವಾಗಿ ಅರ್ಥವನ್ನು ಕಲ್ಪಿಸುತ್ತಾರೆ. ಬ್ರಹ್ಮವು ಮನಸ್ಸಿನ ಪ್ರತ್ಯಯಗಳೊಂದಿಗೆ ತೋರಿಸಿಕೊಳ್ಳುವ ಸ್ವಭಾವವುಳ್ಳದ್ದೆಂದು ಶ್ರೀ ಶಂಕರಾಚಾರ್ಯರು ಹೇಳಿರುತ್ತಾರೆ. ಮನಸ್ಸಿನಿಂದ ಬ್ರಹ್ಮವನ್ನು ವಿಷಯೀಕರಿಸಲು ಸಾಧ್ಯವಿಲ್ಲವೆಂದು ರಂಗರಾಮಾನುಜರ ಮತ. ಮನಸ್ಸು ವಿಷಯಗಳ ಕಡೆಗೆ ಹೋಗುವುದರಿಂದಲೂ, ನಮ್ಮಲ್ಲಿರುವ ಯಾವುದೊ ಒಂದು ವಸ್ತುವು ವಿಷಯ ಗಳನ್ನು ಅರಿಯುವುದರಿಂದಲೂ ಮತ್ತು ಪ್ರತ್ಯಯಗಳನ್ನು ಒಟ್ಟುಗೂಡಿಸಿ ಸ್ಮರಿಸು ವುದರಿಂದಲೂ ಬ್ರಹ್ಮದ ಅಸ್ತಿತ್ವವು ಸಿದ್ಧವಾಯಿತೆಂದು ಮ್ಯಾಕ್ಸ್​ಮುಲ್ಲರ್ ಹೇಳಿರುತ್ತಾರೆ. ಏನೇ ಆಗಲಿ ಮನಸ್ಸು ಬ್ರಹ್ಮದ ಬಳಿಗೆ ಹೋಗುತ್ತದೆ ಎಂಬುದು ಸ್ಪಷ್ಟವಾಗಿದೆ. ಕೊನೆಗೆ ಬ್ರಹ್ಮವನ್ನು ‘ತದ್ವನ’ ಅಥವಾ ‘ವನ’ ಎಂದು ಉಪಾಸಿಸ ಬೇಕೆಂದು ಹೇಳಿದೆ. ಎಲ್ಲರೂ ಪ್ರತ್ಯಗಾತ್ಮವಾದ ಬ್ರಹ್ಮವನ್ನು ಪ್ರೀತಿಸುವುದರಿಂದ ಬಯಸುವುದರಿಂದ, ಅದಕ್ಕೆ ಈ ಹೆಸರು ಬಂದಿದೆ. ಈ ಉಪಾಸನೆಯೇ ಕೇನೋಪ ನಿಷತ್ತಿನ ಪರಮರಹಸ್ಯವೆಂದು ಹೇಳಬಹುದು. ಛಾಂದೋಗ್ಯೋಪನಿಷತ್ತಿನಲ್ಲಿ (೩. ೧೪. ೧) ಬ್ರಹ್ಮವನ್ನು ‘ತಜ್ಜಲಾನ್​’ ಎಂದು ಕರೆದಿರುವುದನ್ನು ಈ ಸಂದರ್ಭದಲ್ಲಿ ನೆನಪಿಗೆ ತಂದುಕೊಳ್ಳಬಹುದು.

ನಮ್ಮ ಇಂದ್ರಿಯಗಳಿಗೆ ವಿಷಯಗಳನ್ನು ಪ್ರಕಾಶಗೊಳಿಸುವ ಸಾಮರ್ಥ್ಯ ಆತ್ಮ ಜ್ಯೋತಿಯಿಂದಲೇ ಆಗಿದೆ ; ಪ್ರತ್ಯಾಕ್ಷಾದಿಪ್ರಮಾಣಗಳಿಂದ ತಿಳಿಸಿಕೊಡಲಾಗದ ಬ್ರಹ್ಮವನ್ನು ಆಗಮದಿಂದ ತಿಳಿಯಬಹುದು ; ಬ್ರಹ್ಮವು ನಮ್ಮ ಪ್ರತ್ಯಗಾತ್ಮವಾದು ದರಿಂದ ಅದನ್ನು ‘ತದ್ವನ’ ಎಂದು ಉಪಾಸಿಸತಕ್ಕದ್ದು ; ತಪೋದಮ ಕರ್ಮಗಳು ಅದಕ್ಕೆ ಪ್ರತಿಷ್ಠೆ. ಇದು ಈ ಉಪನಿಷತ್ತಿನ ಸಾರಾಂಶ.

ಮಮ = ನನ್ನ, ಅಂಗಾನಿ = ಅವಯವಗಳು, ಅಥೋ = ಮತ್ತು, ವಾಕ್​ = ಮಾತು, ಪ್ರಾಣಃ = ಪ್ರಾಣ, ಚಕ್ಷುಃ = ಕಣ್ಣು, ಶ್ರೋತ್ರಂ = ಕಿವಿ, ಬಲಂ = ಬಲ, ಚ = ಮತ್ತು, ಸರ್ವಾಣಿ =ಎಲ್ಲ, ಇಂದ್ರಿಯಾಣಿ = ಇಂದ್ರಿಯಗಳು, ಆಪ್ಯಾಯಂತು = ಪುಷ್ಟಿಯನ್ನು ಹೊಂದಲಿ ; ಸರ್ವಂ = ಎಲ್ಲವು, ಉಪನಿಷದಂ = ಉಪನಿಷತ್ತಿನ, ಬ್ರಹ್ಮ = ಬ್ರಹ್ಮವೇ ; ಅಹಂ = ನಾನು, ಬ್ರಹ್ಮ = ಬ್ರಹ್ಮವನ್ನು, ಮಾ ನಿರಾಕುರ್ಯಾಂ = ನಿರಾಕರಿಸದಂತಿರಲಿ ; ಬ್ರಹ್ಮ = ಬ್ರಹ್ಮವು, ಮಾ = ನನ್ನನ್ನು, ಮಾ ನಿರಾಕರೋತ್ = ನಿರಾಕರಿಸದಂತಿರಲಿ ; ಅನಿರಾಕರಣಂ = ಅನಿರಾಕರಣವು, ಅಸ್ತು =ಇರಲಿ ; ಮೇ = ನನ್ನ ಎಂದರೆ ನನ್ನಿಂದ, ಅನಿರಾಕರಣಂ = ಅನಿರಾಕರಣವು, ಅಸ್ತು = ಇರಲಿ ; ಉಪನಿಷತ್ಸು = ಉಪನಿಷತ್ತುಗಳಲ್ಲಿ, ಯೇ ಧರ್ಮಾಃ (ಸಂತಿ) = ಯಾವ ಧರ್ಮಗಳು ಉಕ್ತವಾಗಿವೆಯೋ, ತೇ = ಅವು, ತದಾತ್ಮನಿ = ಆ ಆತ್ಮನಲ್ಲಿ, ನಿರತೇ ಮಯಿ = ನಿರತನಾದ ನನ್ನಲ್ಲಿ, ಸಂತು = ನೆಲಸಿರಲಿ ; ತೇ ಮಯಿ ಸಂತು = ಅವು ನನ್ನಲ್ಲಿ ನೆಲಸಿರಲಿ.

ನನ್ನ ಅವಯವಗಳೂ ಮಾತೂ ಪ್ರಾಣವೂ ಕಣ್ಣು ಕಿವಿಗಳೂ ಬಲವೂ ಮತ್ತು ಎಲ್ಲ ಇಂದ್ರಿಯಗಳೂ ಪುಷ್ಟಿಯನ್ನು ಹೊಂದಲಿ ; ಎಲ್ಲವೂ ಉಪನಿಷತ್ತುಗಳಲ್ಲಿ ಉಕ್ತವಾಗಿರುವ ಬ್ರಹ್ಮವೇ ; ನಾನು ಬ್ರಹ್ಮವನ್ನು ನಿರಾಕರಿಸದಂತಿರಲಿ ; ಬ್ರಹ್ಮವು ನನ್ನನ್ನು ನಿರಾಕರಿಸದಂತಿರಲಿ ; ನಿರಾಕರಣವೆಂಬುದೇ ಇಲ್ಲಿದಿರಲಿ ; ನನ್ನಿಂದಾ ದರೂ ನಿಕಾರಣೆಯಿಲ್ಲದಿರಲಿ ; ಉಪನಿಷತ್ತುಗಳಲ್ಲಿ ಉಕ್ತವಾಗಿರುವ ಧರ್ಮಗಳು ಆತ್ಮನಿರತನಾದ ನನ್ನಲ್ಲಿ ನೆಲಸಿರಲಿ ; ಅವು ನನ್ನಲ್ಲಿ ನೆಲಸಿರಲಿ.

ಓಂ ಶಾಂತಿಃ ಶಾಂತಿಃ ಶಾಂತಿಃ
ಕೇನೋಪನಿಷತ್​
ಮೊದಲನೆಯ ಖಂಡ
ಕೇನೇಷಿತಂ ಪತತಿ ಪ್ರೇಷಿತಂ ಮನಃ
ಕೇನ ಪ್ರಾಣಃ ಪ್ರಥಮಃ ಪ್ರೈತಿ ಯುಕ್ತಃ ।
ಕೇನೇಷಿತಾಂ ವಾಚಮಿಮಾಂ ವದಂತಿ
ಚಕ್ಷುಃ ಶ್ರೋತ್ರಂ ಕ ಉ ದೇವೋ ಯುನಕ್ತಿ ॥ ೧ ॥
ಕೇನ ಇಷಿತಂ (ಸತ್​) = ಯಾರ ಇಷ್ಟದಂತೆ, ಪ್ರೇಷಿತಂ = ಪ್ರೇರಿತವಾಗಿ, ಮನಃ = ಮನಸ್ಸು (ಸ್ವವಿಷಯಂ ಪ್ರತಿ = ತನ್ನ ವಿಷಯದ ಕಡೆಗೆ) ಪತತಿ = ಹೋಗುತ್ತದೆ ? ಕೇನ = ಯಾರಿಂದ, ಯುಕ್ತಃ = ಪ್ರೇರಿತವಾಗಿ, ಪ್ರಥಮಃ = ಪ್ರಥಮ, ಪ್ರಾಣಃ = ಪ್ರಾಣವು (ಸ್ವವ್ಯಾಪಾರಂ ಪ್ರತಿ = ತನ್ನ ಕೆಲಸದಲ್ಲಿ ) ಪ್ರೈತಿ = ತೊಡಗುತ್ತದೆ ? ಕೇನ ಇಷಿತಾಂ = ಯಾರ ಇಷ್ಟದಂತೆ, ಇಮಾಂ = ಈ, ವಾಚಂ = ಮಾತನ್ನು, ವದಂತಿ = ಆಡುತ್ತಾರೆ ? ಕಃ ಉ ದೇವಃ = ಯಾವ ದೇವನು, ಚಕ್ಷುಃ = ಕಣ್ಣನ್ನು ಶ್ರೋತ್ರಂ = ಕಿವಿಯನ್ನು (ಸ್ವೇಸ್ವೇ ವಿಷಯೇ = ತನ್ನ ವಿಷಯದಲ್ಲಿ) ಯುನಕ್ತಿ = ಪ್ರೇರಿಸುತ್ತಾನೆ ?

೧. ಯಾರ ಇಷ್ಟದಂತೆ ಪ್ರೇರಿತವಾಗಿ ಮನಸ್ಸು (ತನ್ನ ವಿಷಯದ ಕಡೆಗೆ) ಹೋಗುತ್ತದೆ ? ಯಾರಿಂದ ಪ್ರೇರಿತವಾಗಿ ಪ್ರಥಮಪ್ರಾಣವು (ತನ್ನ ಕಾರ್ಯದಲ್ಲಿ) ತೊಡಗುತ್ತದೆ ? ಯಾರ ಇಷ್ಟದಂತೆ ಈ ಮಾತನ್ನು ಜನರು ಆಡುತ್ತಾರೆ ? ಯಾವ ದೇವನು ಕಣ್ಣು ಕಿವಿಗಳನ್ನು (ತಮ್ಮ ತಮ್ಮ ವಿಷಯದಲ್ಲಿ) ಪ್ರೇರಿಸುತ್ತಾನೆ ?

[ಯಾವನೋ ಒಬ್ಬ ಶಿಷ್ಯನು ಪ್ರತ್ಯಗಾತ್ಮವಿಷಯದಲ್ಲಿ ಹೊರತು ಇನ್ನೆಲ್ಲಿಯೂ ಆಶ್ರಯ ವನ್ನು ಕಾಣದೆ ಬ್ರಹ್ಮನಿಷ್ಠನಾದ ಗುರುವಿನ ಬಳಿಗೆ ವಿಧಿಪ್ರಕಾರ ಬಂದು, ಅಭಯವೂ ನಿತ್ಯವೂ ಮಂಗಳವೂ ಅಚಲವೂ ಆದುದನ್ನು ಇಚ್ಛಿಸುವವನಾಗಿ ಈ ಪ್ರಕಾರ ಪ್ರಶ್ನಿಸುವನು.]
ಶ್ರೋತ್ರಸ್ಯ ಶ್ರೋತ್ರಂ ಮನಸೋ ಮನೋ ಯದ್​
ವಾಚೋ ಹ ವಾಚಂ ಸ ಉ ಪ್ರಾಣಸ್ಯ ಪ್ರಾಣಃ ।
ಚಕ್ಷುಷಶ್ಚಕ್ಷುರತಿಮುಚ್ಯ ಧೀರಾಃ
ಪ್ರೇತ್ಯಾಸ್ಮಾಲ್ಲೋಕಾದಮೃತಾ ಭವಂತಿ ॥೨ ॥
ಯತ್ = ಯಾವ ಕಾರಣದಿಂದ, ಶ್ರೋತ್ರಸ್ಯ = ಕಿವಿಗೆ, ಶ್ರೋತ್ರಂ = ಕಿವಿಯೊ, ಮನಸಃ = ಮನಸ್ಸಿಗೆ, ಮನಃ = ಮನಸ್ಸೊ, ವಾಚಃ ಹ = ವಾಕ್ಕಿಗೆ, ವಾಚಂ (ಇಲ್ಲಿ ದ್ವಿತೀಯಾ ವಿಭಕ್ತಿಯನ್ನು ‘ವಾಕ್​’ ಎಂದು ಪ್ರಥಮಾವಿಭಕ್ತಿಯಾಗಿ ತೆಗೆದುಕೊಳ್ಳಬೇಕು ) =ವಾಕ್ಕೊ, ಸಃ ಉ = ಅದು, ಪ್ರಾಣಸ್ಯ = ಪ್ರಾಣಕ್ಕೆ, ಪ್ರಾಣಃ = ಪ್ರಾಣವೊ, ಚಕ್ಷುಷಃ = ಕಣ್ಣಿಗೆ, ಚಕ್ಷುಃ = ಕಣ್ಣೋ, ಅತಿಮುಚ್ಯ = (ಶ್ರೋತ್ರಾದಿ ಇಂದ್ರಿಯಗಳಲ್ಲಿ ಆತ್ಮಭಾವವನ್ನು) ಬಿಟ್ಟು, ಧೀರಾಃ = ವಿವೇಕಿಗಳು, ಅಸ್ಮಾತ್ ಲೋಕಾತ್ =ಈ ಲೋಕದಿಂದ, ಪ್ರೇತ್ಯ = ಹೋಗಿ, ಅಮೃತಾಃ ಭವಂತಿ = ಅಮೃತರಾಗುತ್ತಾರೆ.

೨. (ಆಚಾರ್ಯ) : (ಅದು) ಕಿವಿಗೆ ಕಿವಿ, ಮನಸ್ಸಿಗೆ ಮನಸ್ಸು, ವಾಕ್ಕಿಗೆ ವಾಕ್ಕು, ಪ್ರಾಣಕ್ಕೆ ಪ್ರಾಣವು, ಕಣ್ಣಿಗೆ ಕಣ್ಣು. ಆದ್ದರಿಂದ ವಿವೇಕಿಗಳು (ಶ್ರೋತ್ರಾದಿ ಇಂದ್ರಿಯಗಳಲ್ಲಿ ಆತ್ಮಭಾವವನ್ನು) ಬಿಟ್ಟು ಈ ಲೋಕದಿಂದ ಹೋಗಿ೧ ಅಮೃತ ರಾಗುತ್ತಾರೆ.

[ಮನಸ್ಸು ಪ್ರಾಣ ಇಂದ್ರಿಯ ಮೊದಲಾದವುಗಳಿಗೆ ತಮ್ಮ ವಿಷಯಗಳನ್ನು ಪ್ರಕಾಶ ಗೊಳಿಸುವ ಸಾಮರ್ಥ್ಯ, ಚೈತನ್ಯವೂ ನಿತ್ಯವೂ ಆದ ಆತ್ಮಜ್ಯೋತಿಯಿಂದಲೇ ಉಂಟಾಗುತ್ತದೆ. ಏಕೆಂದರೆ ಇಂದ್ರಿಯಾದಿಗಳಿಗೆ ಸ್ವರೂಪತಃ ಚೈತನ್ಯ ಶಕ್ತಿಯಿಲ್ಲ. ಇದನ್ನು ತಿಳಿಯದ ಅವಿ ವೇಕಿಯು ಇಂದ್ರಿಯಾದಿಗಳ ಗುಂಪನ್ನು ಆತ್ಮವೆಂದು ಭಾವಿಸುತ್ತಾನೆ. ಆದರೆ ಜ್ಞಾನಿಯು ಇಡಿಯ ಕರಣಸಮೂಹದ ಪ್ರವೃತ್ತಿಯು ಆತ್ಮನಿಗಾಗಿಯೇ ಆಗುತ್ತಿರುವುದೆಂದು ತಿಳಿದಿರು ತ್ತಾನೆ.
೧ ಹೆಂಡತಿ ಮಕ್ಕಳು ಸ್ನೇಹಿತರು ನಂಟರು ಇವರಲ್ಲಿ ‘ನಾನು’, ‘ನನ್ನದು’ ಎಂಬ ಭಾವನೆಯ ವ್ಯವಹಾರಲಕ್ಷಣವಾಗಿರುವ (ಈ ಸಂಸಾರದಿಂದ) ಹಿಂತಿರುಗಿ ಎಂದರೆ ಸರ್ವೈ ಷಣಗಳನ್ನೂ ಬಿಟ್ಟು ; ಅಥವಾ ಸತ್ತಮೇಲೆ ಎಂಬ ಅರ್ಥವನ್ನೂ ಮಾಡಬಹುದು.]
ನ ತತ್ರ ಚಕ್ಷುರ್ಗಚ್ಛತಿ ನ ವಾಗ್ಗಚ್ಛತಿ ನೋ ಮನಃ ।
ನ ವಿದ್ಮೋ ನ ವಿಜಾನೀಮೋ ಯಥೈತದನುಶಿಷ್ಯಾತ್ ॥ ೩ ॥
ತತ್ರ = ಅಲ್ಲಿಗೆ, ಚಕ್ಷುಃ = ಕಣ್ಣು, ನ ಗಚ್ಛತಿ = ಹೋಗುವುದಿಲ್ಲ, ವಾಕ್ = ವಾಕ್ಕು, ನ ಗಚ್ಛತಿ = ಹೋಗುವುದಿಲ್ಲ , ನೋ ಮನಃ = ಮನಸ್ಸು ಹೋಗುವುದಿಲ್ಲ ; ನ ವಿದ್ಮಃ = ನಾವು ಅರಿಯೆವು ; ಯಥಾ = ಹೇಗೆ, ಏತತ್ = ಇದನ್ನು, ಅನುಶಿಷ್ಯಾತ್​= ತಿಳಿಸಿಕೊಡ ಬೇಕೊ, ನ ವಿಜಾನೀಮಃ = ಅರಿಯೆವು.

೩. ಅಲ್ಲಿಗೆ೧ ಕಣ್ಣು ಹೋಗುವುದಿಲ್ಲ, ವಾಕ್ಕು ಹೋಗುವುದಿಲ್ಲ. ಮನಸ್ಸು ಹೋಗುವುದಿಲ್ಲ : (ಬ್ರಹ್ಮವು ಹೇಗಿರುವುದೆಂಬುದನ್ನು) ನಾವು ಅರಿಯೆವು ; ಇದನ್ನು ಹೇಗೆ ತಿಳಿಸಿಕೊಡಬೇಕೊ(ಅದನ್ನೂ) ಅರಿಯೆವು೨.

[ಇಂದ್ರಿಯಮನಸ್ಸುಗಳ ಆತ್ಮವೇ ಬ್ರಹ್ಮ ; ಆದುದರಿಂದ ತಮ್ಮ ಸ್ವರೂಪವನ್ನು ತಿಳಿದುಕೊಳ್ಳಲು ಇವುಗಳಿಗೆ ಸಾಧ್ಯವಿಲ್ಲ .
೧ ಬ್ರಹ್ಮವು ಇರುವಲ್ಲಿಗೆ.
೨ ಕರಣಗೋಚರವಾದ ವಸ್ತುವನ್ನು ಜಾತಿಗುಣಕ್ರಿಯಾವಿಶೇಷಣಗಳಿಂದ ಇನ್ನೊಬ್ಬನಿಗೆ ತಿಳಿಸಬಹುದು. ಆದುದರಿಂದ ಜಾತ್ಯಾದಿವಿಶೇಷಣಗಳಿಲ್ಲದ ಬ್ರಹ್ಮವನ್ನು ತಿಳಿಸಿಕೊಡುವುದು ಕಷ್ಟ. ಉಪದೇಶವನ್ನು ಮಾಡಲು ಮತ್ತು ಅದರ ಅರ್ಥವನ್ನು ತಿಳಿದುಕೊಳ್ಳಲು ವಿಶೇಷ ಪ್ರಯತ್ನವನ್ನು ಮಾಡಬೇಕು.]
ಅನ್ಯದೇವ ತದ್ವಿದಿತಾದಥೋ ಅವಿದಿತಾದಧಿ ।
ಇತಿ ಶುಶ್ರುಮ ಪೂರ್ವೇಷಾಂ ಯೇ ನಸ್ತದ್ವ್ಯಾಚಚಕ್ಷಿರೇ ॥ ೪ ॥
ತತ್​= ಅದು, ವಿದಿತಾತ್​ = ತಿಳಿದಿರುವುದಕ್ಕಿಂತ, ಅನ್ಯತ್ ಏವ = ಬೇರೆ, ಅಥೋ = ಮತ್ತು, ಅವಿದಿತಾತ್ = ತಿಳಿಯದಿರುವುದಕ್ಕಿಂತ, ಅಧಿ = ಬೇರೆ ; ಯೇ = ಯಾರು, ನಃ = ನಮಗೆ, ತತ್ = ಅದನ್ನು, ವ್ಯಾಚಚಕ್ಷಿರೇ = ವಿವರಿಸಿದರೊ, (ತೇಷಾಂ = ಅಂತಹ) ಪೂರ್ವೇಷಾಂ = ಪೂರ್ವಾಚಾರ್ಯರಿಂದ, ಇತಿ = ಹೀಗೆ ಶುಶ್ರುಮ = ಕೇಳಿದ್ದೇವೆ.

೪. ಆ (ಬ್ರಹ್ಮವು) ತಿಳಿದಿರುವುದಕ್ಕಿಂತ೧ ಬೇರೆ ಮತ್ತು ತಿಳಿಯದಿರುವುದ ಕ್ಕಿಂತ೨ ಬೇರೆ ; ಯಾರು ನಮಗೆ ಅದನ್ನು ವಿವರಿಸಿದರೊ (ಅಂತಹ) ಪೂರ್ವಾ ಚಾರ್ಯರಿಂದ ಹೀಗೆ ಕೇಳಿದ್ದೇವೆ೩.

[ಪ್ರತ್ಯಕ್ಷಾದಿಪ್ರಮಾಣಗಳಿಂದ ಬ್ರಹ್ಮವನ್ನು ತಿಳಿಸಿಕೊಡಲು ಆಗದಿದ್ದರೂ ಆಗಮದಿಂದ ಉಪದೇಶಿಸಲು ಸಾಧ್ಯವೆಂಬುದನ್ನು ಈ ಮಂತ್ರವು ಸೂಚಿಸಿದೆ.
೧ ಜ್ಞಾನಕ್ಕೆ ವಿಷಯವಾದ ವ್ಯಾಕೃತವಸ್ತುವಿಗಿಂತ.
೨ ವ್ಯಾಕೃತಕ್ಕೆ ಕಾರಣವಾದ ಅವಿದ್ಯಾಲಕ್ಷಣವುಳ್ಳ ಅವ್ಯಾಕೃತಕ್ಕಿಂತ.
೩ ಹೀಗೆ ಸರ್ವಾತ್ಮವೂ ಸರ್ವವಿಶೇಷರಹಿತವೂ ಆದ ಚಿನ್ಮಾತ್ರಜ್ಯೋತಿಯ ಬ್ರಹ್ಮತ್ವ ವನ್ನು ಪ್ರತಿಪಾದಿಸುವ ವಾಕ್ಯವು ಆಚಾರ್ಯೋಪದೇಶ—ಪರಂಪರೆಯಿಂದ ಬಂದಿದೆ. ಬ್ರಹ್ಮ ವನ್ನು ತರ್ಕದಿಂದಾಗಲಿ ಅಧ್ಯಯನದಿಂದಾಗಲಿ ತಪೋಯಜ್ಞಾದಿಗಳಿಂದಾಗಲಿ ತಿಳಿದುಕೊಳ್ಳಲು ಸಾಧ್ಯವಿಲ್ಲ.]
ಯದ್ವಾಚಾsನಭ್ಯುದಿತಂ ಯೇನ ವಾಗಭ್ಯುದ್ಯತೇ ।
ತದೇವ ಬ್ರಹ್ಮ ತ್ವಂ ವಿದ್ಧಿ ನೇದಂ ಯದಿದಮುಪಾಸತೇ ॥ ೫ ॥
ಯತ್ = ಯಾವುದು, ವಾಚಾ =ವಾಕ್ಕಿನಿಂದ, ಅನಭ್ಯುದಿತಂ = ವ್ಯಕ್ತವಾಗುವುದಿಲ್ಲವೊ, ಯೇನ = ಯಾವುದರಿಂದ, ವಾಕ್​= ವಾಕ್ಕು, ಅಭ್ಯುದ್ಯತೇ = ವ್ಯಕ್ತವಾಗುವುದೊ, ತತ್ ಏವ = ಅದನ್ನೇ, ತ್ವಂ = ನೀನು, ಬ್ರಹ್ಮ =ಬ್ರಹ್ಮವೆಂದು, ವಿದ್ಧಿ = ತಿಳಿ, ಯತ್ = ಯಾವುದನ್ನು, ಇದಂ = ಇದು (ಎಂದು), ಉಪಾಸತೇ = ಉಪಾಸನೆ ಮಾಡುತ್ತಾರೆಯೊ, ಇದಂ ನ =ಇದಲ್ಲ.

೫. ಯಾವುದು ವಾಕ್ಕಿನಿಂದ ವ್ಯಕ್ತವಾಗುವುದಿಲ್ಲವೊ, ಯಾವುದರಿಂದ ವಾಕ್ಕು ವ್ಯಕ್ತವಾಗುವುದೊ ಅದನ್ನೇ ನೀನು ಬ್ರಹ್ಮವೆಂದು೧ ತಿಳಿ ; ಯಾವುದನ್ನು ‘ಇದು’ ಎಂದು (ಜನರು) ಉಪಾಸನೆಮಾಡುತ್ತಾರೆಯೊ ಅದಲ್ಲ೨.

[೧ಆತ್ಮಸ್ವರೂಪವೇ ಬ್ರಹ್ಮ ; ಎಲ್ಲಕ್ಕಿಂತಲೂ ಬೃಹತ್ತಾಗಿರುವುದರಿಂದ ಬ್ರಹ್ಮವೆಂದು ಹೆಸರು. ವ್ಯವಹಾರಕ್ಕೆ ವಿಷಯವಲ್ಲದ, ನಿರ್ವಿಶೇಷವೂ ಸಾಮ್ಯವೂ ಆದ ಬ್ರಹ್ಮದಲ್ಲಿ ಉಂಟಾಗುತ್ತಿರುವ ‘ಕರ್ತೃತ್ವ’ ‘ಭೋಕ್ಚತ್ವ’ ಮೊದಲಾದ ಉಪಾಧಿಗಳನ್ನು ತಳ್ಳಿಹಾಕಿ ನಿರ್ವಿ ಶೇಷನಾದ ಆತ್ಮನನ್ನೇ ಬ್ರಹ್ಮವೆಂದು ತಿಳಿಯಬೇಕು.
೨ಯಾವುದನ್ನು ‘ಇದು’ ಎಂದು ಉಪಾಧಿಭೇದಗಳಿಂದ ಕೂಡಿದ ಅನಾತ್ಮರಾದ ಈಶ್ವರಾ ದಿಗಳನ್ನು, ಉಪಾಸಿಸುತ್ತಾರೆಯೊ ಆ ‘ಇದು’ ಬ್ರಹ್ಮವಲ್ಲ]
ಯನ್ಮನಸಾ ನ ಮನುತೇ ಯೇನಾಹುರ್ಮನೋ ಮತಮ್ ।
ತದೇವ ಬ್ರಹ್ಮ ತ್ವಂ ವಿದ್ಧಿ ನೇದಂ ಯದಿದಮುಪಾಸತೇ ॥ ೬ ॥
ಯತ್ = ಯಾವುದನ್ನು, ಮನಸಾ = ಮನಸ್ಸಿನಿಂದ, ನ ಮನುತೇ = ಮನನ ಮಾಡುವುದಿಲ್ಲವೊ. ಯೇನ = ಯಾವುದರಿಂದ, ಮನಃ = ಮನಸ್ಸು, ಮತಂ = ಮನನ ಮಾಡಲ್ಪಡುವುದು (ಎಂದು), ಆಹುಃ = ಹೇಳುತ್ತಾರೆಯೊ, ತತ್ ಏವ= ಅದನ್ನೇ, ತ್ವಂ = ನೀನು, ಬ್ರಹ್ಮ = ಬ್ರಹ್ಮವೆಂದು, ವಿದ್ಧಿ = ತಿಳಿ ; ಯತ್ = ಯಾವುದನ್ನು, ಇದಂ = ಇದು (ಎಂದು), ಉಪಾಸತೇ = ಉಪಾಸನೆಮಾಡುತ್ತಾರೆಯೊ, ಇದಂ ನ = ಇದಲ್ಲ.

೬. ಯಾವುದನ್ನು ಮನಸ್ಸಿನಿಂದ೧ ಮನನಮಾಡಲು ಆಗುವುದಿಲ್ಲವೊ, ಯಾವುದರಿಂದ ಮನಸ್ಸು ಮನನಮಾಡಲ್ಪಟ್ಟಿದೆಯೆಂದು ಹೇಳುವರೊ ಅದನ್ನೇ೨ ನೀನು ಬ್ರಹ್ಮವೆಂದು ತಿಳಿ ; ಯಾವುದನ್ನು ‘ಇದು’ ಎಂದು (ಜನರು) ಉಪಾಸನೆ ಮಾಡುತ್ತಾರೆಯೊ ಅದಲ್ಲ.

[೧ ತನ್ನೊಳಗಿರುವ ಚೈತನ್ಯಜ್ಯೋತಿಯಿಂದ ಪ್ರಕಾಶಿಸಲ್ಪಟ್ಟ ಮನಸ್ಸಿಗೆ ಮನನ ಸಾಮರ್ಥ್ಯ ಉಂಟಾಗಿದೆ.
೨ ಮನಸ್ಸಿನ ಆತ್ಮವಾದ ಪ್ರತ್ಯಕ್​ಚೈತನ್ಯ.]
ಯಚ್ಚಕ್ಷುಷಾ ನ ಪಶ್ಯತಿ ಯೇನ ಚಕ್ಷೂಂಷಿ ಪಶ್ಯತಿ ।
ತದೇವ ಬ್ರಹ್ಮ ತ್ವಂ ವಿದ್ಧಿ ನೇದಂ ಯದಿದಮುಪಾಸತೇ ॥ ೭॥
ಯತ್ = ಯಾವುದನ್ನು, ಚಕ್ಷುಷಾ = ಕಣ್ಣಿನಿಂದ, ನ ಪಶ್ಯತಿ = ನೋಡುವುದಿಲ್ಲವೊ, ಯೇನ = ಯಾವುದರಿಂದ, ಚಕ್ಷೂಂಷಿ = ಕಣ್ಣುಗಳನ್ನು, ಪಶ್ಯತಿ = ನೋಡುತ್ತಾನೆಯೊ, ತತ್ ಏವ ತ್ವಂ ಬ್ರಹ್ಮ ವಿದ್ಧಿ ; ಯತ್ ಇದಮ್ ಉಪಾಸತೇ ಇದಂ ನ.

೭. ಯಾವುದನ್ನು ಕಣ್ಣಿನಿಂದ ನೋಡುವುದಿಲ್ಲವೊ, ಯಾವುದರಿಂದ ಕಣ್ಣು ಗಳನ್ನು ನೋಡುತ್ತಾನೆಯೊ೧ ಅದನ್ನೇ ನೀನು ಬ್ರಹ್ಮವೆಂದು ತಿಳಿ ; ಯಾವುದನ್ನು ‘ಇದು’ ಎಂದು (ಜನರು) ಉಪಾಸನೆ ಮಾಡುತ್ತಾರೆಯೊ ಅದಲ್ಲ.

[೧ ಯಾವುದರ ಮೂಲಕ ಅಂತಃಕರಣದ ವೃತ್ತಿಭೇದದಿಂದ ಭಿನ್ನವಾದ ಚಕ್ಷುರ್ವೃತ್ತಿಗ ಳನ್ನು ನೋಡುತ್ತಾನೆಯೊ ಎಂದರೆ ಆತ್ಮಚೈತನ್ಯಜ್ಯೋತಿಯಿಂದ ವಿಷಯೀಕರಿಸುತ್ತಾನೆಯೊ.]
ಯಚ್ಛ್ರೋತ್ರೇಣ ನ ಶೃಣೋತಿ ಯೇನ ಶ್ರೋತ್ರಮಿದಂ ಶ್ರುತಮ್ ।
ತದೇವ ಬ್ರಹ್ಮ ತ್ವಂ ವಿದ್ಧಿ ನೇದಂ ಯದಿದಮುಪಾಸತೇ ॥ ೮ ॥
ಯತ್ = ಯಾವುದನ್ನು, ಶ್ರೋತ್ರೇಣ = ಕಿವಿಯಿಂದ, ನ ಶೃಣೋತಿ = ಕೇಳುವುದಿಲ್ಲವೊ, ಯೇನ = ಯಾವುದರಿಂದ, ಇದಂ = ಈ, ಶ್ರೋತ್ರಂ = ಕಿವಿಯು, ಶ್ರುತಂ = ಕೇಳಲ್ಪಡು ವುದೊ, ತತ್ ಏವ ತ್ವಂ ಬ್ರಹ್ಮ ವಿದ್ಧಿ ; ಯತ್ ಇದಂ ಉಪಾಸತೇ ಇದಂ ನ.

೮. ಯಾವುದನ್ನು ಕಿವಿಯಿಂದ ಕೇಳುವುದಿಲ್ಲವೊ, ಯಾವುದರಿಂದ ಕಿವಿಯು ಕೇಳಲ್ಪಡುವುದೊ ಅದನ್ನೇ ನೀನು ಬ್ರಹ್ಮವೆಂದು ತಿಳಿ ; ಯಾವುದನ್ನು ‘ಇದು’ ಎಂದು (ಜನರು) ಉಪಾಸನೆ ಮಾಡುತ್ತಾರೆಯೊ ಅದಲ್ಲ.

ಯತ್ ಪ್ರಾಣೇನ ನ ಪ್ರಾಣಿತಿ ಯೇನ ಪ್ರಾಣಃ ಪ್ರಣೀಯತೇ ।
ತದೇವ ಬ್ರಹ್ಮ ತ್ವಂ ವಿದ್ಧಿ ನೇದಂ ಯದಿದಮುಪಾಸತೇ ॥ ೯ ॥
ಯತ್ = ಯಾವುದನ್ನು, ಪ್ರಾಣೇನ = ಪ್ರಾಣದಿಂದ, ನ ಪ್ರಾಣಿತಿ = ಆಘ್ರಾಣಿಸು ವುದಿಲ್ಲವೊ, ಯೇನ = ಯಾವುದರಿಂದ ಪ್ರಾಣಃ = ಪ್ರಾಣವು ಪ್ರಣೀಯತೇ = ಕಳುಹಿಸಲ್ಪ ಡುವುದೋ, ತತ್ ಏವ ತ್ವಂ ಬ್ರಹ್ಮ ವಿದ್ಧಿ ; ಯತ್ ಇದಂ ಉಪಾಸತೇ ಇದಂ ನ.

೯. ಯಾವುದನ್ನು ಪ್ರಾಣದಿಂದ ಆಘ್ರಾಣಿಸುವುದಿಲ್ಲವೊ, ಯಾವುದರಿಂದ ಪ್ರಾಣವು ಕಳುಹಿಸಲ್ಪಡುವುದೊ ಅದನ್ನೇ ನೀನು ಬ್ರಹ್ಮವೆಂದು ತಿಳಿ ; ಯಾವು ದನ್ನು ‘ಇದು’ ಎಂದು (ಜನರು) ಉಪಾಸನೆ ಮಾಡುತ್ತಾರೆಯೋ ಅದಲ್ಲ.

ಎರಡನೆಯ ಖಂಡ
ಯದಿ ಮನ್ಯಸೇ ಸುವೇದೇತಿ ದಭ್ರಮೇವಾಪಿ
ನೂನಂ ತ್ವಂ ವೇತ್ಥ ಬ್ರಹ್ಮಣೋ ರೂಪಮ್ ।
ಯದಸ್ಯ ತ್ವಂ ಯದಸ್ಯ ದೇವೇಷ್ವಥ ನು
ಮೀಮಾಂಸ್ಯಮೇವ ತೇ ಮನ್ಯೇ ವಿದಿತಮ್ ॥ ೧ ॥
ಸುವೇದ ಇತಿ = ಚೆನ್ನಾಗಿ ತಿಳಿದಿರುತ್ತೇನೆ ಎಂದು, ಯದಿ ಮನ್ಯಸೇ = ನೀನು ಅರಿತಿದ್ದರೆ, ಅಸ್ಯ ಬ್ರಹ್ಮಣಃ = ಈ ಬ್ರಹ್ಮದ, ಯತ್ ರೂಪಂ = ಯಾವ ರೂಪವನ್ನು, ತ್ವಂ ವೇತ್ಥ = ನೀನು ತಿಳಿದಿರುತ್ತೀಯೋ, ದೇವೇಷು = ದೇವತೆಗಳಲ್ಲಿ, ಅಸ್ಯ = ಇದರ, ಯತ್ (ರೂಪಂ) = ಯಾವ ರೂಪವನ್ನು, ತ್ವಂ = ನೀನು, (ವೇತ್ಥ= ತಿಳಿದಿರುತ್ತೀಯೊ), (ತತ್ ಸರ್ವಂ = ಅವೆಲ್ಲವೂ), ನೂನಂ =ಖಂಡಿತವಾಗಿ, ದಭ್ರಮ್ ಏವ ಅಪಿ = ಅಲ್ಪವೇ ; ಅಥ ನು = ಆದುದರಿಂದ, ತೇ = ನಿನ್ನ, (ಬ್ರಹ್ಮ= ಬ್ರಹ್ಮವು), ಮೀಮಾಂಸ್ಯಮ್ ಏವ = ವಿಚಾರಿಸ ಲ್ಪಡತಕ್ಕದ್ದು. ಮನ್ಯೇ ವಿದಿತಂ = ತಿಳಿಯಿತು ಎಂದು ಎಣಿಸುತ್ತೇನೆ.

೧. (ಆಚಾರ್ಯ): ‘ನಾನು ಚೆನ್ನಾಗಿ ತಿಳಿದಿರುತ್ತೇನೆ’ ಎಂದು ನೀನು ಅರಿತಿದ್ದರೆ, (ಮನುಷ್ಯ—ಶರೀರದಿಂದ ಪರಿಚ್ಛಿನ್ನವಾದ) ಈ ಬ್ರಹ್ಮದ ಯಾವ ರೂಪವನ್ನು ನೀನು ತಿಳಿದಿರುತ್ತೀಯೊ ಮತ್ತು (ಅಧಿದೈವತೋಪಾಧಿಯಿಂದ ಪರಿಚ್ಛಿನ್ನವಾದ) ಇದರ ಯಾವ (ರೂಪವನ್ನು) ದೇವತೆಗಳಲ್ಲಿ (ತಿಳಿದಿರುತ್ತೀಯೊ ಅವೆಲ್ಲವೂ) ಅಲ್ಪವೇ. ಆದುದರಿಂದ ನಿನ್ನ ಬ್ರಹ್ಮವು ಇನ್ನೂ ವಿಚಾರಿಸಲ್ಪಡತಕ್ಕದ್ದು.

(ಶಿಷ್ಯ): ‘ನನಗೆ ತಿಳಿಯಿತು’ ಎಂದು ಎಣಿಸುತ್ತೇನೆ೧.

[ಯಾವ ವಸ್ತುವು ನಮ್ಮ ಜ್ಞಾನಕ್ಕೆ ವಿಷಯವಾಗಿದೆಯೊ ಅದನ್ನು ಚೆನ್ನಾಗಿ ತಿಳಿದು ಕೊಳ್ಳುವುದಕ್ಕೆ ಸಾಧ್ಯವಾಗುತ್ತದೆ. ಬಾಹ್ಯವಸ್ತುವನ್ನು ತಿಳಿಯುವಂತೆ, ‘ನಾನು ಬ್ರಹ್ಮವನ್ನು ಚೆನ್ನಾಗಿ ತಿಳಿದಿರುತ್ತೇನೆ’ ಎಂದು ಹೇಳಲು ಆಗುವುದಿಲ್ಲ. ಏಕೆಂದರೆ ತಿಳಿಯುವವನಿಗಿಂತ ಬ್ರಹ್ಮವು ಬೇರೆಯಲ್ಲ ; ಹಾಗಾಗಿದ್ದರೆ ಬ್ರಹ್ಮವನ್ನು ಇತರ ವಸ್ತುಗಳಂತೆ ತಿಳಿಯಬಹು ದಾಗಿತ್ತು. ಬೆಂಕಿಯು ತನಗಿಂತ ಬೇರೆಯಾದ ವಸ್ತುವನ್ನು ಸುಡುತ್ತದೆಯೇ ವಿನಾ ತನ್ನನ್ನು ತಾನೆ ಸುಡಲಾರದು. ಶಿಷ್ಯರಲ್ಲಿ ‘ನಾನೇ ಬ್ರಹ್ಮವೆಂದು ನನ್ನನ್ನು ನಾನು ಚೆನ್ನಾಗಿ ಅರಿತು ಕೊಂಡೆನು’ ಎಂಬ ತಪ್ಪು ಭಾವನೆ ಬರದಿರಲೆಂದು ಆಚಾರ್ಯನು ಈ ರೀತಿ ಹೇಳುತ್ತಾನೆ.
೧ ಹೀಗೆಂದು ಆಚಾರ್ಯನು ಹೇಳಲು ಶಿಷ್ಯನು ಏಕಾಂತದಲ್ಲಿ ಕುಳಿತು ಸಮಾಹಿತನಾಗಿ ಆಚಾರ್ಯನು ಹಿಂದೆ ಉಪದೇಶಿಸಿದ ಶಾಸ್ತ್ರವನ್ನು ಅರ್ಥತಃ ವಿಚಾರಮಾಡಿ ತರ್ಕತಃ ನಿರ್ಧರಿಸಿ ಅನುಭವವನ್ನು ಮಾಡಿಕೊಂಡು ಆಚಾರ್ಯನ ಸಮೀಪಕ್ಕೆ ಬಂದು ‘ಈಗ ನನಗೆ ಬ್ರಹ್ಮವು ತಿಳಿಯಿತು ಎಂದು ಎಣಿಸುತ್ತೇನೆ’ ಎಂದನು.]
ನಾಹಂ ಮನ್ಯೇ ಸುವೇದೇತಿ ನೋ ನ ವೇದೇತಿ ವೇದ ಚ ।
ಯೋ ನಸ್ತದ್ವೇದ ತದ್ವೇದ ನೋ ನ ವೇದೇತಿ ವೇದ ಚ ॥ ೨ ॥
ಅಹಂ = ನಾನು, ಸುವೇದ ಇತಿ = ಚೆನ್ನಾಗಿ ತಿಳಿದಿರುತ್ತೇನೆ ಎಂದು, ನ ಮನ್ಯೇ= ಎಣಿಸಿರುವುದಿಲ್ಲ. ನ ವೇದ ಇತಿ = ತಿಳಿದುಕೊಂಡಿಲ್ಲ ಎಂದು, ನೋ = ಎಣಿಸಿರುವುದಿಲ್ಲ, ವೇದ ಚ = ತಿಳಿದೂ ಇರುತ್ತೇನೆ ; ನಃ = ನಮ್ಮಲ್ಲಿ ಯಾವನು, ನೋ ನ ವೇದ ಇತಿ ವೇದ ಚ = ತಿಳಿದುಕೊಂಡಿಲ್ಲವೆಂದು ಎಣಿಸಿರುವುದಿಲ್ಲ, ತಿಳಿದೂ ಇರುತ್ತೇನೆ, (ಇತಿ =ಎಂದು), ತತ್ = ಅದನ್ನು, ವೇದ = ಅರಿಯುತ್ತಾನೆಯೊ, (ಸಃ = ಅವನೇ), ತತ್ = ಅದನ್ನು, ವೇದ = ಅರಿತಿರುತ್ತಾನೆ.

೨. ನಾನು ಚೆನ್ನಾಗಿ ತಿಳಿದಿರುತ್ತೇನೆಂದು ಎಣಿಸಿರುವುದಿಲ್ಲ, ತಿಳಿದುಕೊಂಡಿಲ್ಲ ವೆಂದೂ ಎಣಿಸಿರುವುದಿಲ್ಲ. ತಿಳಿದೂ ಇರುತ್ತೇನೆ೧. ‘ತಿಳಿದುಕೊಂಡಿಲ್ಲವೆಂದೂ ಎಣಿಸಿರುವುದಿಲ್ಲ, ತಿಳಿದೂ ಇರುತ್ತೇನೆ’ ಎಂಬುದನ್ನು ನಮ್ಮಲ್ಲಿ ಯಾವನು ಅರಿ ತಿರುತ್ತಾನೆಯೊ (ಅವನೇ) ಅದನ್ನು ಅರಿತಿರುತ್ತಾನೆ.

[ಆಚಾರ್ಯನು ಶಿಷ್ಯನನ್ನು ಕುರಿತು ‘ಬ್ರಹ್ಮವು ನಿನಗೆ ಹೇಗೆ ತಿಳಿದಿದೆ ?’ ಎಂದು ಕೇಳಿದ ಪ್ರಶ್ನೆಗೆ ಇದು ಶಿಷ್ಯನು ಕೊಟ್ಟ ಉತ್ತರ.
೧ ‘ವೇದ ಚ’ ಎಂಬುದರಲ್ಲಿ ‘ಚ ’—ಶಬ್ದವನ್ನು ಪ್ರಯೋಗಿಸಿರುವುದರಿಂದ ‘ತಿಳಿದೂ ಇಲ್ಲ’ ಎಂದು ಅಧ್ಯಾಹಾರಮಾಡಿಕೊಳ್ಳಬೇಕು.]
ಯಸ್ಯಾಮತಂ ತಸ್ಯ ಮತಂ ಮತಂ ಯಸ್ಯ ನ ವೇದ ಸಃ ।
ಅವಿಜ್ಞಾತಂ ವಿಜಾನತಾಂ ವಿಜ್ಞಾತಮವಿಜಾನತಾಮ್ ॥ ೩ ॥
ಯಸ್ಯ ಅಮತಂ = ಯಾವನಿಗೆ (ಬ್ರಹ್ಮವನ್ನು ಅರಿತಿದ್ದೇನೆಂಬ )ತಿಳಿವಳಿಕೆ ಇಲ್ಲವೊ, ತಸ್ಯ ಮತಂ = ಅವನಿಗೆ ತಿಳಿದಿದೆ ; ಯಸ್ಯ ಮತಂ = ಯಾವನಿಗೆ (ಬ್ರಹ್ಮವನ್ನು ಅರಿತಿದ್ದೇನೆಂಬ) ತಿಳಿವಳಿಕೆ ಇದೆಯೊ, ಸಃ = ಅವನು, ನ ವೇದ = ಅರಿಯನು. ವಿಜಾನತಾಂ = ತಿಳಿದವರಿಗೆ, ಅವಿಜ್ಞಾತಂ = ತಿಳಿಯದೆ ಇರುತ್ತದೆ, ಅವಿಜಾನತಾಂ = ತಿಳಿಯದವರಿಗೆ, ವಿಜ್ಞಾತಂ = ತಿಳಿದಿರುತ್ತದೆ.

೩. ಯಾವನಿಗೆ೧ (ಬ್ರಹ್ಮವನ್ನು ಅರಿತಿದ್ದೇನೆಂಬ) ತಿಳಿವಳಿಕೆ ಇಲ್ಲವೊ ಅವನಿಗೆ ತಿಳಿದಿದೆ ; ಯಾರಿಗೆ (ಬ್ರಹ್ಮವನ್ನು ಅರಿತಿದ್ದೇನೆಂಬ) ತಿಳಿವಳಿಕೆ ಇದೆಯೊ ಅವರಿಗೆ ತಿಳಿದಿಲ್ಲ. ತಿಳಿದವರಿಗೆ೨ ತಿಳಿಯದೆ ಇರುತ್ತದೆ. ತಿಳಿಯದವರಿಗೆ೩ ತಿಳಿದಿರುತ್ತದೆ.

[ಶ್ರುತಿಯು ತನ್ನ ರೂಪದಿಂದಲೇ ಗುರುಶಿಷ್ಯರ ಸಂವಾದದ ವಿಷಯವನ್ನು ಉಪಸಂಹಾರ ಮಾಡುತ್ತದೆ.
೧ ಯಾವ ಬ್ರಹ್ಮಜ್ಞಾನಿಗೆ ಬ್ರಹ್ಮವು ತಿಳಿದುಕೊಳ್ಳತಕ್ಕ ವಿಷಯವಲ್ಲವೆಂಬ ಮತವಿರು ವುದೊ.
೨ ಸಮ್ಯಗ್​ಜ್ಞಾನಿಗಳಿಗೆ ಬ್ರಹ್ಮವು ಜ್ಞಾನಕ್ಕೆ ವಿಷಯವಲ್ಲವೆಂದು ತಿಳಿದಿರುತ್ತದೆ.
೩ ‘ತಿಳಿಯದವರು’ ಎಂದರೆ ಏನನ್ನೂ ತಿಳಿಯದವರು ಎಂದರ್ಥ ಅಲ್ಲ, ಅಸಮ್ಯಗ್ದರ್ ಶಿಗಳು ಎಂದಿಷ್ಟೇ ಅರ್ಥ. ಇವರು ಇಂದ್ರಿಯಮನೋಬುದ್ಧಿಗಳಲ್ಲಿ ಆತ್ಮನನ್ನು ನೋಡುವು ದರಿಂದ ಬ್ರಹ್ಮವು ಇವರ ಜ್ಞಾನಕ್ಕೆ ವಿಷಯವಾಗಿರುತ್ತದೆ.]
ಪ್ರತಿಬೋಧವಿದಿತಂ ಮತಮಮೃತತ್ವಂ ಹಿ ವಿಂದತೇ ॥
ಆತ್ಮನಾ ವಿಂದತೇ ವೀರ್ಯಂ ವಿದ್ಯಯಾ ವಿಂದತೇsಮೃತಮ್ ॥ ೪ ॥
ಪ್ರತಿಬೋಧ—ವಿದಿತಂ = ಪ್ರತಿಬೋಧದಲ್ಲಿಯೂ ತಿಳಿಯಲ್ಪಟ್ಟಿದ್ದು, ಮತಂ = ತಿಳಿಯ ಲ್ಪಟ್ಟಿದೆ, ಹಿ = ಏಕೆಂದರೆ, ಅಮೃತತ್ವಂ = ಅಮೃತತ್ವವನ್ನು, ವಿಂದತೇ = ಹೊಂದುತ್ತಾನೆ. ಆತ್ಮನಾ = ಆತ್ಮನಿಂದ, ವೀರ್ಯಂ = ವೀರ್ಯವನ್ನು, ವಿಂದತೇ = ಹೊಂದುತ್ತಾನೆ. ವಿದ್ಯಯಾ = ವಿದ್ಯೆಯಿಂದ ಅಮೃತಂ = ಅಮೃತವನ್ನು, ವಿಂದತೇ = ಹೊಂದುತ್ತಾನೆ.

೪. (ಯಾವಾಗ ಬ್ರಹ್ಮವು) ಪ್ರತಿಬೋಧನೆಯಲ್ಲಿಯೂ೧ ತಿಳಿಯಲ್ಪಡುತ್ತ ದೆಯೊ (ಆಗ) ತಿಳಿದಂತಾಗುತ್ತದೆ೨ ಏಕೆಂದರೆ ಅಮೃತತ್ವವನ್ನು ಹೊಂದುತ್ತಾನೆ೩, ಆತ್ಮನಿಂದ ವೀರ್ಯವನ್ನು೪ ಹೊಂದುತ್ತಾನೆ, ವಿದ್ಯೆಯಿಂದ೫ ಅಮೃತವನ್ನು ಹೊಂದುತ್ತಾನೆ.

[ಬ್ರಹ್ಮವು ನಿಜವಾಗಿ ತಿಳಿಯದೆ ಇದ್ದರೆ ಲೌಕಿಕರಿಗೂ ಬ್ರಹ್ಮಜ್ಞಾನಿಗಳಿಗೂ ಯಾವ ವಿಶೇಷವೂ ಇಲ್ಲವೆಂದಾಯಿತು ; ‘ತಿಳಿಯುವವರಿಗೆ ತಿಳಿಯದೆ ಇರುತ್ತದೆ’ ಎಂಬುದು ಪರಸ್ಪರ ವಿರುದ್ಧವಾದ ಮಾತು. ಆ ಬ್ರಹ್ಮವನ್ನು ಹೇಗೆ ತಿಳಿದರೆ ಸರಿಯಾಗಿ ತಿಳಿದಂತೆ ಆಗುವುದು ಎಂಬುದನ್ನು ಈ ಮಂತ್ರವು ಉಪದೇಶಿಸುತ್ತದೆ.
೧ ಬುದ್ಧಿಯ ಪ್ರತಿಯೊಂದು ಪ್ರತ್ಯಯಗಳಲ್ಲಿಯೂ.
೨ ಸರ್ವಪ್ರತ್ಯಯಗಳನ್ನು ನೋಡುವವನೂ ಚಿಚ್ಛಕ್ತಿಸ್ವರೂಪನೂ ಆದ ಆತ್ಮನು ಪ್ರತ್ಯಯ ಗಳಿಂದಲೇ ಎಲ್ಲ ಪ್ರತ್ಯಯಗಳಲ್ಲಿಯೂ ಸಮವಾಗಿ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತಾನೆ. ಅಂತರಾತ್ಮನನ್ನು ಅರಿಯುವುದಕ್ಕೆ ಬೇರೆ ದ್ವಾರವಿಲ್ಲ. ಆದುದರಿಂದ ಪ್ರತ್ಯಯಗಳ ಪ್ರತ್ಯಗಾತ್ಮನೆಂದು ಯಾವಾಗ ಬ್ರಹ್ಮವನ್ನು ಅರಿಯುತ್ತೇವೆಯೊ ಆಗ ಅದನ್ನು ಸರಿಯಾಗಿ ತಿಳಿದಂತಾಗುತ್ತದೆ.
೩ ಬೋಧವು ಪ್ರತ್ಯಗಾತ್ಮನ ವಿಷಯವು ಅಥವಾ ಪ್ರತ್ಯಗಾತ್ಮನು ಸರ್ವಪ್ರತ್ಯಯಗಳ ಸಾಕ್ಷಿ ಎಂಬ ಜ್ಞಾನವು ಅಮೃತತ್ವಕ್ಕೆ ಕಾರಣವಾಗಿರುತ್ತದೆ.
೪ ಅನಿತ್ಯಗಳಾದ ಧನ ಸಹಾಯ ಮಂತ್ರ ಔಷಧ ತಪಸ್ಸು ಯೋಗ–ಇವುಗಳಿಂದ ಉತ್ಪನ್ನವಾದ ವೀರ್ಯವು ಮೃತ್ಯುವನ್ನು ಜಯಿಸಲಾರದು. ಆತ್ಮವಿದ್ಯೆಯಿಂದ ಉಂಟಾದ ವೀರ್ಯವು ಮಾತ್ರ ಮೃತ್ಯುವನ್ನು ಜಯಿಸಬಲ್ಲದು. ಇದನ್ನು ಆತ್ಮನಿಂದಲೆ ಪಡೆಯಬೇಕು.
೫ ಆತ್ಮ ವಿಷಯವಾದ ವಿದ್ಯೆಯಿಂದ.]
ಇಹ ಚೇದವೇದೀದಥ ಸತ್ಯಮಸ್ತಿ
ನ ಚೇದಿಹಾವೇದೀನ್ಮಹತೀ ವಿನಷ್ಟಿಃ ।
ಭೂತೇಷು ಭೂತೇಷು ವಿಚಿತ್ಯ ಧೀರಾಃ
ಪ್ರೇತ್ಯಾಸ್ಮಾಲ್ಮೋಕಾದಮೃತಾ ಭವಂತಿ ॥ ೫ ॥
ಇಹ = ಇಲ್ಲಿ, ಅವೇದಿತ್ ಚೇತ್ = ತಿಳಿದುಕೊಂಡರೆ, ಅಥ = ಆಗ, ಸತ್ಯಂ = ಸತ್ಯವು, ಅಸ್ತಿ = ಉಂಟು ; ಇಹ = ಇಲ್ಲಿ, ನ ಅವೇದಿತ್ ಚೇತ್ = ಅರಿತುಕೊಳ್ಳದಿದ್ದರೆ, ಮಹತೀ = ದೊಡ್ಡ, ವಿನಷ್ಟಿಃ = ವಿನಾಶ. ಧೀರಾಃ = ವಿವೇಕಿಗಳು, ಭೂತೇಷು ಭೂತೇಷು = ಸರ್ವಪ್ರಾಣಿಗಳಲ್ಲಿಯೂ, ವಿಚಿತ್ಯ = ಆತ್ಮತತ್ವವನ್ನು ತಿಳಿದುಕೊಂಡು, ಅಸ್ಮಾತ್ ಲೋಕಾತ್ = ಈ ಲೋಕದಿಂದ, ಪ್ರೇತ್ಯ = ಹೋಗಿ, ಅಮೃತಾಃ ಭವಂತಿ = ಅಮೃತರಾಗುತ್ತಾರೆ.

೫. (ಅಧಿಕಾರಿಯಾದ ಮನುಷ್ಯನು) ಇಲ್ಲಿಯೇ ಅರಿತುಕೊಂಡರೆ೧ ಆಗ ಸತ್ಯ ವುಂಟು೨. ಇಲ್ಲಿ ಅರಿತುಕೊಳ್ಳದಿದ್ದರೆ ದೊಡ್ಡ ವಿನಾಶವಾಗುತ್ತದೆ೩. ವಿವೇಕಿಗಳು ಸರ್ವಪ್ರಾಣಿಗಳಲ್ಲಿಯೂ ಆತ್ಮತತ್ವವನ್ನು ಅರಿತುಕೊಂಡು ಈ ಲೋಕದಿಂದ ಹೋಗಿ೪ ಅಮೃತರಾಗುತ್ತಾರೆ೫.

[೧ ಹಿಂದೆ ಹೇಳಿದ ಲಕ್ಷಣವುಳ್ಳ ಆತ್ಮವನ್ನು ಹಿಂದೆ ಹೇಳಿದ ಪ್ರಕಾರದಿಂದ.
೨ ಈ ಮನುಷ್ಯಜನ್ಮದಲ್ಲಿ ಅವಿನಾಶವುಂಟು, ಅರ್ಥವುಂಟು ; ಸದ್ಭಾವವುಂಟು ಅಥವಾ ಪರಮಾರ್ಥವಾದ ಸತ್ಯ ಉಂಟು.
೩ ಜನ್ಮಜರಾಮರಣಾದಿಗಳ ಸಂಸಾರಗತಿ.
೪ ಒಂದನೆಯ ಖಂಡದ ಎರಡನೆಯ ಮಂತ್ರದ ಟಿಪ್ಪಣಿಯನ್ನು ನೋಡಿ.
೫ ಬ್ರಹ್ಮವೇ ಆಗುವರು.]
ಮೂರನೆಯ ಖಂಡ
ಬ್ರಹ್ಮ ಹ ದೇವೇಭ್ಯೋ ವಿಜಿಗ್ಯೇ, ತಸ್ಯ ಹ ಬ್ರಹ್ಮಣೋ
ವಿಜಯೇ ದೇವಾ ಅಮಹೀಯಂತ । ತ ಐಕ್ಷಂತಾಸ್ಮಾಕಮೇವಾಯಂ
ವಿಜಯೋsಸ್ಮಾಕಮೇವಾಯಂ ಮಹಿಮೇತಿ ॥ ೧ ॥
ಬ್ರಹ್ಮ ಹ = ಬ್ರಹ್ಮವು, ದೇವೇಭ್ಯಃ = ದೇವತೆಗಳಿಗೆ, ವಿಜಿಗ್ಯೇ = ಜಯಿಸಿತು. ತಸ್ಯ = ಆ, ಬ್ರಹ್ಮಣಃ ಹ = ಬ್ರಹ್ಮದ, ವಿಜಯೇ = ಜಯದಲ್ಲಿ, ದೇವಾಃ = ದೇವತೆಗಳು, ಅಮಹೀಯಂತ = ಮಹಿಮಾನ್ವಿತರಾದರು. ತೇ = ಅವರು, ಐಕ್ಷಂತ = ಯೋಚನೆ ಮಾಡಿದರು. ಅಯಂ = ಈ, ವಿಜಯಃ = ವಿಜಯವು, ಅಸ್ಮಾಕಮ್ ಏವ = ನಮ್ಮದೇ, ಅಯಂ = ಈ, ಮಹಿಮಾ = ಮಹಿಮೆಯು, ಅಸ್ಮಾಕಮ್ ಏವ = ನಮ್ಮದೇ, ಇತಿ = ಎಂದು.

೧. ಬ್ರಹ್ಮವು ದೇವತೆಗಳಿಗಾಗಿ ಜಯಿಸಿತು. ಆ ಬ್ರಹ್ಮದ ಜಯದಲ್ಲಿ ದೇವತೆ ಗಳು ಮಹಿಮಾನ್ವಿತರಾದರು. ‘ಈ ಜಯವು ನಮ್ಮದೇ, ಈ ಮಹಿಮೆಯು ನಮ್ಮದೆ’ ಎಂದು ಅವರು ಯೋಚನೆ ಮಾಡಿದರು೧.

[ಬ್ರಹ್ಮವು ಅವಿಜ್ಞಾತವಾದುದರಿಂದ ಅದು ಇಲ್ಲವೇ ಇಲ್ಲ ಎಂಬ ಭ್ರಾಂತಿ ಬರದಿರ ಲೆಂದು ಅಥವಾ ಬ್ರಹ್ಮವಿದ್ಯೆಯ ಸ್ತುತಿಗಾಗಿ ಅಥವಾ ನಾಲ್ಕನೆಯ ಖಂಡದಲ್ಲಿ ಬಂದಿರುವ ಉಪನಿಷತ್ತಿನ ವಿಧಿಗಾಗಿ ಈ ಆಖ್ಯಾಯಿಕೆಯನ್ನು ಹೇಳಲಾಗಿದೆ.
೧ ಪ್ರತ್ಯಗಾತ್ಮನೂ ಸರ್ವಜ್ಞನೂ ಪ್ರಾಣಿಗಳ ಸರ್ವಕರ್ಮಗಳಿಗೆ ಫಲವನ್ನು ಕೊಡುವವನೂ ಸರ್ವಶಕ್ತನೂ ಆದ ಈಶ್ವರನು ಜಗತ್ತಿನ ಸ್ಥಿತಿಗಾಗಿ ಕೊಟ್ಟ ವಿಜಯವಿದು ಮತ್ತು ಆ ಮಹಿಮೆ ಅವನದೇ ಎಂದು ಇವರು ತಿಳಿಯಲಿಲ್ಲ.]
ತದ್ಧೈಷಾಂ ವಿಜಜ್ಞೌ, ತೇಭ್ಯೋ ಹ ಪ್ರಾದುರ್ಬಭೂವ,
ತನ್ನ ವ್ಯಜಾನತ ಕಿಮಿದಂ ಯಕ್ಷಮಿತಿ ॥ ೨ ॥
ತತ್ ಹ = ಅದು, ಏಷಾಂ = ಇವರ (ಮಿಥ್ಯೇಕ್ಷಣಂ = ತಪ್ಪು ಭಾವನೆಯನ್ನು), ವಿಜಜ್ಞೌ = ತಿಳಿದುಕೊಂಡಿತು. ತೇಭ್ಯಃ ಹ = ಅವರಿಗೆ, ಪ್ರಾದುರ್ಬಭೂವ = ಕಾಣಿಸಿ ಕೊಂಡಿತು. ಇದಂ = ಈ, ಯಕ್ಷಂ = ಯಕ್ಷವು, ಕಿಮ್ ಇತಿ = ಯಾವುದು ಎಂದು, ತತ್ = ಅದನ್ನು, ನ ವ್ಯಜಾನತ = ತಿಳಿಯಲಿಲ್ಲ.

೨. ಅದು ಇವರ (ತಪ್ಪು ಭಾವನೆಯನ್ನು) ತಿಳಿದುಕೊಂಡಿತು೧ ; ಅವರಿಗೆ ಕಾಣಿಸಿಕೊಂಡಿತು೨. ಈ ಯಕ್ಷವು ಯಾವುದೆಂಬುದನ್ನು (ಅವರು) ತಿಳಿಯಲಿಲ್ಲ.

[೧ ಸರ್ವಪ್ರಾಣಿಗಳ ಕರಣಗಳನ್ನೂ ಪ್ರೇರಿಸುವುದಾದುದರಿಂದ ಬ್ರಹ್ಮವು ಎಲ್ಲವನ್ನೂ ತಿಳಿಯಬಲ್ಲದು.
೨ ಅಸುರರಂತೆ ದೇವತೆಗಳೂ ಮಿಥ್ಯಾಭಿಮಾನದಿಂದ ಸೋತುಹೋಗಬಾರದೆಂದು].
ತೇsಗ್ನಿಮಬ್ರುವನ್ ಜಾತವೇದ ಏತದ್ವಿಜಾನೀಹಿ
ಕಿಮೇತದ್ಯಕ್ಷಮಿತಿ, ತಥೇತಿ ॥ ೩ ॥
ತೇ = ಅವರು, ಅಗ್ನಿಂ = ಅಗ್ನಿಯನ್ನು ಕುರಿತು, ಅಬ್ರುವನ್ = ಹೇಳಿದರು. ಜಾತವೇದಃ = ಹೇ ಜಾತವೇದನೆ, ಕಿಮ್ ಏತತ್ ಯಕ್ಷಂ = ಈ ಯಕ್ಷವು ಯಾವುದು, ಇತಿ = ಎಂದು, ಏತತ್ = ಇದನ್ನು, ವಿಜಾನೀಹಿ = ತಿಳಿದುಕೊ, ತಥಾ ಇತಿ = ಆಗಲಿ ಎಂದು.

೧. ಅವರು ಅಗ್ನಿಯನ್ನು೧ ಕುರಿತು , ‘ಹೇ ಜಾತವೇದ, ಈ ಯಕ್ಷವು ಯಾವುದೆಂಬುದನ್ನು ತಿಳಿದುಕೊ’ ಎಂದು ಹೇಳಿದರು. ‘ಹಾಗೆಯೇ ಆಗಲಿ’ ಎಂದು (ಅಗ್ನಿಯು ಹೇಳಿದನು)

[೧ ಅಗ್ನಿಯು ದೇವತೆಗಳ ಪುರೋಹಿತ ಮತ್ತು ಸರ್ವಜ್ಞ]
ತದಭ್ಯದ್ರವತ್, ತಮಭ್ಯವದತ್ ಕೋsಸೀತಿ, ಅಗ್ನಿರ್ವಾ
ಅಹಮಸ್ಮೀತ್ಯಬ್ರವೀಜ್ಜಾತವೇದಾ ವಾ ಅಹಮಸ್ಮೀತಿ ॥ ೪ ॥
(ಅಗ್ನಿಃ =ಅಗ್ನಿಯು), ತತ್ ಅಭ್ಯದ್ರವತ್ = ಅದರ ಸಮೀಪಕ್ಕೆ ಓಡಿದನು, ತಂ = ಅವನನ್ನು, ಅಭ್ಯವದತ್ = ಪ್ರಶ್ನಿಸಿತು, ಕಃ ಅಸಿ ಇತಿ = ನೀನು ಯಾರು ಎಂದು, ಅಹಂ = ನಾನು, ಅಗ್ನಿಃ ವೈ ಅಸ್ಮಿ ಇತಿ = ಅಗ್ನಿಯೇ ಆಗಿರುವೆನು ಎಂದು, ಅಹಂ = ನಾನು, ಜಾತವೇದಾಃ ವೈ ಅಸ್ಮಿ ಇತಿ = ಜಾತವೇದನೆ ಆಗಿರುವೆನು ಎಂದು, ಅಬ್ರವೀತ್ = ಹೇಳಿದನು.

೪. (ಅಗ್ನಿಯು) ಅದರ ಸಮೀಪಕ್ಕೆ ಓಡಿದನು, ‘ನೀನು ಯಾರು ?’ ಎಂದು ಅವನನ್ನು (ಯಕ್ಷವು) ಪ್ರಶ್ನಿಸಿತು. ‘ನಾನು ಅಗ್ನಿಯೇ ಆಗಿರುವೆನು, ಜಾತವೇದನೆ ಆಗಿರುವೆನು’ ಎಂದು ಹೇಳಿದನು.

[ಅಗ್ನಿಗೆ ತಾನು ಪ್ರಶ್ನಿಸಬೇಕೆಂದು ಇಚ್ಛೆ ಇದ್ದರೂ ಆ ಯಕ್ಷದ ಬಳಿಯಲ್ಲಿ ಏನೂ ಮಾತನಾಡಲಾಗಲಿಲ್ಲ.]
ತಸ್ಮಿಂಸ್ತ್ವಯಿ ಕಿಂ ವೀರ್ಯಮಿತ್ಯಪೀದಂ
ಸರ್ವಂ ದಹೇಯಂ ಯದಿದಂ ಪೃಥಿವ್ಯಾಮಿತಿ ॥ ೫ ॥
(ಬ್ರಹ್ಮ = ಯಕ್ಷರೂಪದಲ್ಲಿರುವ ಬ್ರಹ್ಮವು), ತಸ್ಮಿನ್ ತ್ವಯಿ = ಅಂತಹ ನಿನ್ನಲ್ಲಿ, ಕಿಂ ವೀರ್ಯಂ = ಯಾವ ವೀರ್ಯವುಂಟು, ಇತಿ = ಎಂದು, ಯತ್ ಇದಂ = ಏನೇನು, ಪೃಥಿವ್ಯಾಂ = ಭೂಮಿಯಲ್ಲಿ, (ಅಸ್ತಿ= ಇದೆಯೊ), ಇದಂ ಸರ್ವಮ್ ಅಪಿ = ಇದೆಲ್ಲವನ್ನೂ, ದಹೇಯಂ = ಸುಡಬಲ್ಲೆನು, ಇತಿ = ಎಂದು.

೫. ‘ಅಂತಹ ನಿನ್ನಲ್ಲಿ ಯಾವ ವೀರ್ಯವುಂಟು ?’ ಎಂದು (ಯಕ್ಷವು ಪ್ರಶ್ನಿಸಲು) (ಅಗ್ನಿಯು) ‘ಈ ಭೂಮಿಯಲ್ಲಿ೧ ಏನೇನಿದೆಯೋ ಇದೆಲ್ಲವನ್ನೂ ಸುಡಬಲ್ಲೆನು’ ಎಂದು ಹೇಳಿದನು.

[೧ ‘ಭೂಮಿಯಲ್ಲಿ’ ಎಂದು ಹೇಳಿರುವುದು ಉಪಲಕ್ಷಣ ; ಏಕೆಂದರೆ ಅಂತರಿಕ್ಷ ದಲ್ಲಿರುವುದನ್ನೂ ಅಗ್ನಿಯು ಸುಡಬಲ್ಲದು.]
ತಸ್ಮೈ ತೃಣಂ ನಿದಧಾವೇತದ್ದಹೇತಿ,
ತದುಪಪ್ರೇಯಾಯ ಸರ್ವಜವೇನ,
ತನ್ನ ಶಶಾಕ ದಗ್ಧುಂ, ಸ ತತ ಏವ ನಿವವೃತೇ,
ನೈತದಶಕಂ ವಿಜ್ಞಾತುಂ ಯದೇತದ್ಯಕ್ಷಮಿತಿ ॥ ೬ ॥
ಏತತ್ = ಇದನ್ನು, ದಹ = ಸುಡು, ಇತಿ = ಎಂದು, ತಸ್ಮೈ = ಅವನ ಎದುರಿಗೆ, ತೃಣಂ = ಹುಲ್ಲುಕಡ್ಡಿಯನ್ನು, ನಿದಧೌ = ಇಟ್ಟಿತು. (ಅಗ್ನಿಃ = ಅಗ್ನಿಯು), ಸರ್ವಜವೇನ = ಸರ್ವವೇಗದಿಂದ, ತತ್ ಉಪಪ್ರೇಯಾಯ = ಅದರ ಬಳಿ ಓಡಿದನು ; ತತ್ = ಅದನ್ನು, ದಗ್ಧುಂ = ಸುಡಲು, ನಾ ಶಶಾಕ = ಶಕ್ತನಾಗಲಿಲ್ಲ ; ಸಃ = ಅವನು ತತಃ ಏವ = ಅಲ್ಲಿಂದಲೇ, ನಿವವೃತೇ= ಹಿಂತಿರುಗಿದನು. ಯತ್ ಏತತ್ ಯಕ್ಷಮ್ ಇತಿ = ಇದು ಯಾವ ಯಕ್ಷವು ಎಂದು, ಏತತ್ = ಇದನ್ನು, ವಿಜ್ಞಾತುಂ = ತಿಳಿಯಲು, ನ ಅಶಕಂ = ಶಕ್ತನಾಗಲಿಲ್ಲ.

೬. ‘ಇದನ್ನು ಸುಡು’ ಎಂದು ಅವನ ಎದುರಿಗೆ (ಒಂದು) ಹುಲ್ಲುಕಡ್ಡಿಯನ್ನು (ಯಕ್ಷವು) ಇಟ್ಟಿತು. (ಅಗ್ನಿಯು) ಸರ್ವವೇಗದಿಂದಲೂ ಅದರ ಬಳಿಗೆ ಓಡಿದನು ; ಅದನ್ನು ಸುಡಲು ಶಕ್ತನಾಗಲಿಲ್ಲ. ಅವನು ಅಲ್ಲಿಂದಲೇ ಹಿಂತಿರುಗಿದನು. ‘ಇದು ಯಾವ ಯಕ್ಷವು ಎಂಬುದನ್ನು ತಿಳಿಯಲು ಶಕ್ತನಾಗಲಿಲ್ಲ (ಎಂದು ದೇವತೆಗಳಿಗೆ ಹೇಳಿದನು).

ಅಥ ವಾಯುಮಬ್ರುವನ್ ವಾಯವೇತದ್ವಿಜಾನೀಹಿ
ಕಿಮೇತದ್ಯಕ್ಷಮಿತಿ ; ತಥೇತಿ ॥ ೭ ॥
ಅಥ = ಅನಂತರ, ವಾಯುಂ = ವಾಯುವನ್ನು ಕುರಿತು, ಅಬ್ರುವನ್ = ಹೇಳಿದರು, ವಾಯೋ = ಹೇ ವಾಯು, ಕಿಮ್ ಏತತ್ ಯಕ್ಷಮ್ = ಈ ಯಕ್ಷವು ಯಾವುದು, ಇತಿ = ಎಂದು, ಏತತ್ = ಇದನ್ನು, ವಿಜಾನೀಹಿ = ತಿಳಿದುಕೊ, ತಥಾ ಇತಿ = ಆಗಲಿ ಎಂದು.

೭. ಅನಂತರ ವಾಯುವನ್ನು ಕುರಿತು ‘ಹೇ ವಾಯು, ಈ ಯಕ್ಷವು ಯಾವು ದೆಂಬುದನ್ನು ತಿಳಿದುಕೊ’ ಎಂದು ಹೇಳಿದರು. ‘ಹಾಗೆಯೇ ಆಗಲಿ’ ಎಂದು (ವಾಯುವು ಹೇಳಿದನು).

ತದಭ್ಯದ್ರವತ್, ತಮಭ್ಯವದತ್ ಕೋsಸೀತಿ, ವಾಯುರ್ವಾ
ಅಹಮಸ್ಮೀತ್ಯಬ್ರವೀನ್ಮಾತರಿಶ್ವಾ ವಾ ಅಹಮಸ್ಮೀತಿ ॥ ೮ ॥
(ವಾಯುಃ = ವಾಯುವು ) ತತ್ ಅಭ್ಯದ್ರವತ್, ತಮ್ ಅಭ್ಯವದತ್ ಕಃ ಅಸಿ ಇತಿ. ಅಹಂ = ನಾನು, ವಾಯುಃ ವೈ ಅಸ್ಮಿ ಇತಿ = ವಾಯುವೇ ಆಗಿರುವೆನು ಎಂದು, ಅಹಂ = ನಾನು, ಮಾತರಿಶ್ವಾ ವೈ ಅಸ್ಮಿ ಇತಿ = ಮಾತರಿಶ್ವನೇ ಆಗಿರುವೆನು ಎಂದು, ಅಬ್ರವೀತ್ = ಹೇಳಿದನು.

೮. (ವಾಯುವು) ಅದರ ಸಮೀಪಕ್ಕೆ ಓಡಿದನು. ‘ನೀನು ಯಾರು ?’ ಎಂದು ಅವನನ್ನು (ಯಕ್ಷವು ) ಪ್ರಶ್ನಿಸಿತು. ‘ನಾನು ವಾಯುವೇ೧ ಆಗಿರುವೆನು, ಮಾತರಿ ಶ್ವನೇ೨ ಆಗಿರುವೆನು’ಎಂದು ಹೇಳಿದನು.

[೧ ಚಲಿಸುವುದರಿಂದ ಅಥವಾ ವಾಸನೆಯನ್ನುಂಟುಮಾಡುವುದರಿಂದ ವಾಯುವೆಂದು ಹೆಸರು.
೨ ಅಂತರಿಕ್ಷದಲ್ಲಿ ಚಲಿಸುವುದರಿಂದ ಮಾತರಿಶ್ವನೆಂದು ಹೆಸರು.]
ತಸ್ಮಿಂಸ್ತ್ವಯಿ ಕಿಂ ವೀರ್ಯಮಿತ್ಯಪೀದಂ
ಸರ್ವಮಾದದೀಯ ಯದಿದಂ ಪೃಥಿವ್ಯಾಮಿತಿ ॥ ೯ ॥
ತಸ್ಮಿನ್ ತ್ವಯಿ ಕಿಂ ವೀರ್ಯಮ್ ಇತಿ, ಯತ್ ಇದಂ ಪೃಥಿವ್ಯಾಂ ಇದಂ ಸರ್ವಮ್ ಅಪಿ ಆದದೀಯ = ಎತ್ತಬಲ್ಲೆನು, ಇತಿ = ಎಂದು.

೯. ‘ಅಂತಹ ನಿನ್ನಲ್ಲಿ ಯಾವ ವೀರ್ಯವುಂಟು ?’ ಎಂದು (ಯಕ್ಷವು ಪ್ರಶ್ನಿ ಸಲು), (ವಾಯುವು) ‘ಈ ಭೂಮಿಯಲ್ಲಿ ಏನೇನಿದೆಯೋ ಇದೆಲ್ಲವನ್ನೂ ಎತ್ತ ಬಲ್ಲೆನು’ ಎಂದು ಹೇಳಿದನು.

ತಸ್ಮೈ ತೃಣಂ ನಿದಧಾವೇತದಾದತ್ಸ್ವೇತಿ,
ತದುಪಪ್ರೇಯಾಯ ಸರ್ವಜವೇನ,
ತನ್ನ ಶಶಾಕಾದಾತುಂ, ಸ ತತ ಏವ ನಿವವೃತೇ
ನೈತದಶಕಂ ವಿಜ್ಞಾತುಂ ಯದೇತದ್ಯಕ್ಷಮಿತಿ ॥ ೧ಂ ॥
ಏತತ್ = ಇದನ್ನು, ಆದತ್ಸ್ವ = ಎತ್ತು, ಇತಿ = ಎಂದು, ತಸ್ಮೈ ತೃಣಂ ನಿದಧೌ. (ವಾಯುಃ ) ಸರ್ವಜವೇನ ತತ್ ಉಪಪ್ರೇಯಾಯ ; ತತ್ = ಅದನ್ನು, ಆದಾತುಂ = ಎತ್ತಲು, ನ ಶಶಾಕ = ಶಕ್ತನಾಗಲಿಲ್ಲ. ಸಃ ತತಃ ಏವ ನಿವವೃತೇ. ಯತ್ ಏತತ್ ಯಕ್ಷಮ್ ಇತಿ ಏತತ್ ವಿಜ್ಞಾತುಂ ನ ಅಶಕಮ್.

೧ಂ. ‘ಇದನ್ನು ಎತ್ತು’ ಎಂದು ಅವನ ಎದುರಿಗೆ (ಒಂದು) ಹುಲ್ಲು ಕಡ್ಡಿಯನ್ನು (ಯಕ್ಷವು) ಇಟ್ಟಿತು. (ವಾಯುವು) ಸರ್ವವೇಗದಿಂದಲೂ ಅದರ ಬಳಿಗೆ ಓಡಿದನು. ಅದನ್ನು ಎತ್ತಲು ಶಕ್ತನಾಗಲಿಲ್ಲ. ಅವನು ಅಲ್ಲಿಂದಲೇ ಹಿಂತಿರುಗಿದನು. ‘ಇದು ಯಾವ ಯಕ್ಷವು ಎಂಬುದನ್ನು ತಿಳಿಯಲು ಶಕ್ತನಾಗಲಿಲ್ಲ’ (ಎಂದು ದೇವತೆಗಳಿಗೆ ಹೇಳಿದನು).

ಅಥೇಂದ್ರಮಬ್ರುವನ್ ಮಘವನ್ನೇತದ್ವಿಜಾನೀಹಿ
ಕಿಮೇತದ್ಯಕ್ಷಮಿತಿ ; ತಥೇತಿ । ತದಭ್ಯದ್ರವತ್, ತಸ್ಮಾತ್​
ತಿರೋದಧೇ ॥ ೧೧ ॥
ಅಥ = ಅನಂತರ, ಇಂದ್ರಂ = ಇಂದ್ರನನ್ನು ಕುರಿತು, ಅಬ್ರುವನ್ = ಹೇಳಿದರು ; ಮಘವನ್ = ಹೇ ಮಘವಂತನೆ, ಕಿಮ್ ಏತತ್ ಯಕ್ಷಂ = ಈ ಯಕ್ಷವು ಯಾವುದು, ಇತಿ = ಎಂದು, ಏತತ್ = ಇದನ್ನು, ವಿಜಾನೀಹಿ = ತಿಳಿದುಕೊ, ತಥಾ ಇತಿ = ಆಗಲಿ ಎಂದು. ತತ್ ಅಭ್ಯದ್ರವತ್ = ಅದರ ಸಮೀಪಕ್ಕೆ ಓಡಿದನು. ತಸ್ಮಾತ್ = ಅವನಿಂದ, ತಿರೋದಧೇ = ಮರೆಯಾಯಿತು.

೧೧. ಅನಂತರ ಇಂದ್ರನನ್ನು ಕುರಿತು ‘ಹೇ ಮಘವಂತನೆ೧, ಈ ಯಕ್ಷವು ಯಾವುದೆಂಬುದನ್ನು ತಿಳಿದುಕೊ’ ಎಂದು ಹೇಳಿದರು. ‘ಹಾಗೆಯೇ ಆಗಲಿ’ ಎಂದು (ಇಂದ್ರನು ಹೇಳಿದನು). (ಇಂದ್ರನು) ಅದರ ಸಮೀಪಕ್ಕೆ ಓಡಿದನು. ಅದು ಅವನಿಂದ ಮರೆಯಾಯಿತು.

[೧ ಬಲವುಳ್ಳವನು ಎಂದರ್ಥ.]
ಸ ತಸ್ಮಿನ್ನೇವಾಕಾಶೇ ಸ್ತ್ರಿಯಮಾಜಗಾಮ ಬಹು
ಶೋಭಮಾನಾಮುಮಾಂ ಹೈಮವತೀಮ್ ।
ತಾಂ ಹೋವಾಚ ಕಿಮೇತದ್ಯಕ್ಷಮಿತಿ ॥ ೧೨ ॥
ಸಃ = ಅವನು, ತಸ್ಮಿನ್ ಏವ ಆಕಾಶೇ = ಆ ಆಕಾಶದಲ್ಲಿಯೇ, ಹೈಮವತೀಂ = ಹೈಮವ ತಿಯೂ, ಬಹುಶೋಭಮಾನಾಂ = ಬಹುಶೋಭಮಾನಳೂ, ಸ್ತ್ರಿಯಂ = ಸ್ತ್ರೀರೂಪಿಯೂ (ಆದ ), ಉಮಾಂ = ಉಮೆಯನ್ನು, ಆಜಗಾಮ = ಹೊಂದಿದನು. ತಾಂ ಹ = ಅವಳನ್ನು, ಉವಾಚ =ಹೇಳಿದನು ; ಕಿಮ್ ಏತತ್ ಯಕ್ಷಂ = ಈ ಯಕ್ಷವು ಯಾವುದು, ಇತಿ = ಎಂದು.

೧೨. ಅವನು ಆ ಆಕಾಶದಲ್ಲಿಯೇ೧ (ಇದ್ದ) ಹೈಮವತಿಯೂ೨ ಬಹುಶೋಭ ಮಾನಳೂ೩ ಸ್ತ್ರೀರೂಪಿಯೂ ಆದ ಉಮೆಯ ಸಮೀಪಕ್ಕೆ ಬಂದನು. ‘ಈ ಯಕ್ಷವು ಯಾವುದು ?’ ಎಂದು ಅವಳನ್ನು ಕೇಳಿದನು.

[ಇಂದ್ರನಿಗೆ ಯಕ್ಷದಲ್ಲಿದ್ದ ಭಕ್ತಿಯನ್ನು ನೋಡಿ ಬ್ರಹ್ಮವಿದ್ಯೆಯು ಉಮಾರೂಪದಿಂದ ಕಾಣಿಸಿಕೊಂಡಿತು.
೧ ಯಕ್ಷವು ಮರೆಯಾದ ಆಕಾಶಪ್ರದೇಶದಲ್ಲಿಯೇ.
೨ ಚಿನ್ನದ ಆಭರಣಗಳನ್ನು ಧರಿಸಿದವಳಂತೆ ಶೋಭಿಸುತ್ತಿರುವ ಅಥವಾ ಹಿಮವಂತನ ಮಗಳಾದ.
೩ ಶೋಭಿಸುವ ವಿದ್ಯೆಗಳಲ್ಲಿ ಬ್ರಹ್ಮವಿದ್ಯೆಯ ಅತ್ಯಂತ ಶೋಭೆಯುಳ್ಳದ್ದು.]
ನಾಲ್ಕನೆಯ ಖಂಡ
ಸಾ ಬ್ರಹ್ಮೇತಿ ಹೋವಾಚ, ಬ್ರಹ್ಮಣೋ ವಾ ಏತದ್ವಿಜಯೇ
ಮಹೀಯಧ್ವಮಿತಿ । ತತೋ ಹೈವ ವಿದಾಂಚಕಾರ ಬ್ರಹ್ಮೇತಿ ॥ ೧ ॥
ಸಾ = ಅವಳು, ಉವಾಚ ಹ = ಹೇಳಿದಳು ; ಬ್ರಹ್ಮ ಇತಿ = ಬ್ರಹ್ಮ ಎಂದು; ಬ್ರಹ್ಮಣಃ ವೈ = ಬ್ರಹ್ಮದ, ವಿಜಯೇ = ಜಯದಲ್ಲಿ, ಏತತ್ ಮಹೀಯಧ್ವಂ = ಹೀಗೆ ಮಹಿಮೆಯನ್ನು ಹೊಂದಿದಿರಿ, ಇತಿ = ಎಂದು. ತತಃ ಹ ಏವ = ಅದರಿಂದಲೇ, ಬ್ರಹ್ಮ ಇತಿ = ಬ್ರಹ್ಮ ಎಂದು, ವಿದಾಂಚಕಾರ = ತಿಳಿದುಕೊಂಡನು.

೧. ಉಮೆಯು ಹೇಳಿದಳು : ‘(ಅದು ) ಬ್ರಹ್ಮ ; ಬ್ರಹ್ಮದ ಜಯದಲ್ಲಿಯೇ ಹೀಗೆ ಮಹಿಮೆಯನ್ನು ಹೊಂದಿದಿರಿ. ಅದರಿಂದಲೆ೧ (ಇಂದ್ರನು) ಅದು ಬ್ರಹ್ಮ ವೆಂದು ತಿಳಿದುಕೊಂಡನು.

[೧ ಉಮೆಯ ವಾಕ್ಯದಿಂದಲೇ.]
ತಸ್ಮಾದ್ವಾ ಏತೇ ದೇವಾ ಅತಿತರಾಮಿವಾನ್ಯಾನ್ ದೇವಾನ್​
ಯದಗ್ನಿರ್ವಾಯುರಿಂದ್ರಸ್ತೇ ಹ್ಯೇನನ್ನೇದಿಷ್ಠಂ ಪಸ್ಪೃಶುಸ್ತೇ
ಹ್ಯೇನತ್ ಪ್ರಥಮೋ ವಿದಾಂಚಕಾರ ಬ್ರಹ್ಮೇತಿ ॥ ೨ ॥
ಯತ್ = ಯಾವ ಕಾರಣದಿಂದ, ಅಗ್ನಿಃ ವಾಯುಃ ಇಂದ್ರಃ ಇತಿ ತೇ ಹಿ = ಅಗ್ನಿ ವಾಯು ಇಂದ್ರ—ಇವರು, ಏನತ್ = ಇದನ್ನು, ನೇದಿಷ್ಠಂ = ಹತ್ತಿರದಿಂದ, ಪಸ್ಪೃಶುಃ = ಮುಟ್ಟಿದರೊ, ತೇ ಹಿ = ಅವರು, ಏನತ್ = ಇದನ್ನು, ಬ್ರಹ್ಮ ಇತಿ = ಬ್ರಹ್ಮವೆಂದು, ಪ್ರಥಮಃ (ಪ್ರಥಮಾಃ ಸಂತಃ) =ಮೊದಲನೆಯವರಾಗಿ, ವಿದಾಂಚಕಾರ (ವಿದಾಂಚಕ್ರುಃ) = ತಿಳಿದುಕೊಂಡರೊ, ತಸ್ಮಾತ್ ವೈ = ಆದುದರಿಂದಲೆ, ಏತೇ ದೇವಾಃ = ಈ ದೇವತೆಗಳು, ಅನ್ಯಾನ್ ದೇವಾನ್ = ಇತರ ದೇವತೆಗಳನ್ನು, ಅತಿತರಾಮ್ ಇವ = ಮೀರಿರುತ್ತಾರೆ.

೨. ಆದುದರಿಂದ ಅಗ್ನಿ ವಾಯು ಇಂದ್ರ ಎಂಬ ಈ ದೇವತೆಗಳು ಇತರ ದೇವತೆಗಳನ್ನು ಮೀರಿರುತ್ತಾರೆ ; ಏಕೆಂದರೆ ಅವರು ಇದನ್ನು ಅತ್ಯಂತ ಹತ್ತಿರ ದಿಂದ ೧ ಮುಟ್ಟಿದರು ಮತ್ತು ಇದನ್ನು ಬ್ರಹ್ಮವೆಂದು ಮೊದಲನೆಯವರಾಗಿ ತಿಳಿದುಕೊಂಡರು.

[೧ ಸಂಭಾಷಣೆ ದರ್ಶನಾದಿಗಳ ಮೂಲಕ.]
ತಸ್ಮಾದ್ವಾ ಇಂದ್ರೋsತಿತರಾಮಿವಾನ್ಯಾನ್ ದೇವಾನ್ ಸ
ಹ್ಯೇನನ್ನೇದಿಷ್ಠಂ ಪಸ್ಪರ್ಶ ಸ ಹ್ಯೇನತ್ ಪ್ರಥಮೋ
ವಿದಾಂಚಕಾರ ಬ್ರಹ್ಮೇತಿ ॥ ೩ ॥
ಸಃ ಹಿ = ಅವನು, ಏನತ್ ನೇದಿಷ್ಠಂ ಪಸ್ಪರ್ಶ = ಇದನ್ನು ಹತ್ತಿರದಿಂದ ಮುಟ್ಟಿದನು. ಸಃ ಹಿ = ಅವನು, ಏನತ್ ಬ್ರಹ್ಮ ಇತಿ ಪ್ರಥಮಃ ವಿದಾಂಚಕಾರ = ಇದನ್ನು ಬ್ರಹ್ಮವೆಂದು ಮೊದಲನೆಯವನಾಗಿ ತಿಳಿದುಕೊಂಡನು ; ತಸ್ಮಾತ್ ವೈ = ಆದುದರಿಂದಲೇ, ಇಂದ್ರಃ = ಇಂದ್ರನು, ಅನ್ಯಾನ್ ದೇವಾನ್ = ಇತರ ದೇವತೆಗಳನ್ನು, ಅತಿತರಾಮ್ ಇವ = ಮೀರಿರು ತ್ತಾನೆ.

೩. ಆದುದರಿಂದ ಇಂದ್ರನು ಇತರ ದೇವತೆಗಳನ್ನು ಮೀರಿರುತ್ತಾನೆ ; ಏಕೆಂದರೆ ಅವನು ಇದನ್ನು ಅತ್ಯಂತ ಹತ್ತಿರದಿಂದ ಮುಟ್ಟಿದನು ಮತ್ತು ಇದನ್ನು ಬ್ರಹ್ಮ ವೆಂದು ಮೊದಲನೆಯವನಾಗಿ ತಿಳಿದುಕೊಂಡನು.

ತಸ್ಯೈಷ ಆದೇಶೋ ಯದೇತದ್ವಿದ್ಯುತೋ ವ್ಯದ್ಯುತದಾ
ಇತೀನ್ನ್ಯಮೀಮಿಷದಾ ಇತ್ಯಧಿದೈವತಮ್ ॥ ೪ ॥
ತಸ್ಯ = ಅದರ (ವಿಷಯದಲ್ಲಿ), ಏಷಃ =ಇದು, ಆದೇಶಃ = ಆದೇಶ, ಯತ್ ಏತತ್ = ಯಾವ ಈ, ವಿದ್ಯುತಃ = ಮಿಂಚಿನ ವ್ಯದ್ಯುತತ್ ( ವಿದ್ಯೋತನಂ), ಆ = ಹೊಳಪಿನಂತೆ, ಇತ್ = ಮತ್ತು, ನ್ಯಮೀಮಿಷತ್ ಆ ಇತಿ =ಕಣ್ಣು ಮಿಟಿಕಿಸಿದಂತೆ ಎಂದು, ಇತಿ ಅಧಿದೈವತಂ = ಇದು ಅಧಿದೈವತ.

೪. ಬ್ರಹ್ಮದ ವಿಷಯದಲ್ಲಿ ಈ ಆದೇಶವುಂಟು೧ : ‘ಮಿಂಚಿನ ಹೊಳಪಿನಂತೆ ಕಣ್ಣು ಮಿಟಿಕಿಸಿದಂತೆ (ಬ್ರಹ್ಮವು ಕ್ಷಿಪ್ರವಾಗಿ ತೋರಿಕೊಂಡು ಮಾಯವಾಗು ತ್ತದೆ)’. ಇದು ಅಧಿದೈವತ೨.

[೧ ಬ್ರಹ್ಮದ ಲಕ್ಷಣವನ್ನು ಯಾವ ಉಪಮೆಯಿಂದಲೂ ತಿಳಿಸಲು ಸಾಧ್ಯವಿಲ್ಲ. ಅಂತಹ ನಿರುಪಮವಾದ ಬ್ರಹ್ಮವನ್ನು ಉಪಮೆಯ ಮೂಲಕ ಉಪದೇಶಿಸುವುದಕ್ಕೆ ಆದೇಶವೆಂದು ಹೆಸರು.
೨ ಈ ಉಪಮಾನವು ದೇವತಾವಿಷಯವಾದುದರಿಂದ ಅಧಿದೈವತವೆಂದು ಹೇಳಿದೆ.]
ಅಥಾಧ್ಯಾತ್ಮಂ ಯದೇತದ್ಗಚ್ಛತೀವ ಚ ಮನೋsನೇನ
ಚೈತದುಪಸ್ಮರತ್ಯಭೀಕ್ಷ್ಣಂ ಸಂಕಲ್ಪಃ ॥ ೫ ॥
ಅಥ = ಈಗ, ಅಧ್ಯಾತ್ಮಂ = ಅಧ್ಯಾತ್ಮ ; ಯತ್ ಮನಃ = ಯಾವ ಈ ಮನಸ್ಸು, ಏತದ್ = ಇದರ ಸಮೀಪಕ್ಕೆ, ಗಚ್ಛತಿ ಇವ = ಹೋಗುವಂತೆ, ಚ = ಮತ್ತು, (ಸಾಧಕಃ = ಸಾಧಕನು), ಅನೇನ = ಇದರ ಮೂಲಕ, ಏತತ್ = ಇದನ್ನು, ಅಭೀಕ್ಷ್ಣಂ = ಚೆನ್ನಾಗಿ, ಉಪಸ್ಮರತಿ = ಸ್ಮರಿಸುತ್ತಾನೆ, ಚ = ಮತ್ತು, (ಮನಸಃ = ಮನಸ್ಸಿನ), ಸಂಕಲ್ಪಃ = ಸಂಕಲ್ಪವು, (ಬ್ರಹ್ಮವಿಷಯಃ ಏವ = ಬ್ರಹ್ಮವಿಷಯವಾದದ್ದು).

೫. ಈಗ ಅಧ್ಯಾತ್ಮ೧ : ‘ಮನಸ್ಸು ಬ್ರಹ್ಮದ ಸಮೀಪಕ್ಕೆ (ಶೀಘ್ರವಾಗಿ) ಹೋಗುವಂತೆ೨, (ಸಾಧಕನು) ಮನಸ್ಸಿನಿಂದ ಇದನ್ನು ಚೆನ್ನಾಗಿ ಸ್ಮರಿಸಿದಂತೆ೩ ಮತ್ತು (ಮನಸ್ಸು ಬ್ರಹ್ಮದ ವಿಷಯದಲ್ಲಿ ಪುನಃ ಪುನಃ) ಸಂಕಲ್ಪಿಸಿದಂತೆ೪.

[ಹಿಂದಿನ ಆದೇಶದಲ್ಲಿ ಬ್ರಹ್ಮವು ವಿದ್ಯುನ್ನಿಮೇಷಣಗಳಂತೆ ದ್ರುತಪ್ರಕಾಶನಸ್ವಭಾವ ವುಳ್ಳದ್ದು ಎಂದು ಹೇಳಿದ್ದಾಯಿತು. ಇಲ್ಲಿಯೂ ಕೂಡ ಬ್ರಹ್ಮವು ಮನಸ್ಸಿನ ಪ್ರತ್ಯಯಗಳು ಉಂಟಾಗುವ ಕಾಲದಲ್ಲಿಯೇ ತೋರಿಕೊಳ್ಳುವ ಧರ್ಮವುಳ್ಳದ್ದು ಎಂದು ಹೇಳಿದೆ.
೧ ಈಗ ಪ್ರತ್ಯಗಾತ್ಮನ ವಿಷಯವಾದ ಆದೇಶ.
೨ ಬ್ರಹ್ಮವನ್ನು ಮುಟ್ಟುವಂತೆ ವಿಷಯೀಕರಿಸುವಂತೆ.
೩ ಬ್ರಹ್ಮವು ಹತ್ತಿರದಲ್ಲಿ ಇರುವಂತೆ.
೪ ಮನಸ್ಸೇ ಉಪಾಧಿಯಾಗಿ ಉಳ್ಳ ಬ್ರಹ್ಮವು ಮನಸ್ಸಿನ ಸಂಕಲ್ಪಸ್ಮೃತ್ಯಾದಿ ಪ್ರತ್ಯಯ ಗಳಿಂದ ವಿಷಯೀಕರಿಸಲ್ಪಟ್ಟಂತೆ ಅಭಿವ್ಯಕ್ತವಾದಂತೆ.]
ತದ್ಧ ತದ್ವನಂ ನಾಮ ತದ್ವನಮಿತ್ಯುಪಾಸಿತವ್ಯಮ್ ;
ಸ ಯ ಏತದೇವಂ ವೇದಾಭಿ ಹೈನಂ ಸರ್ವಾಣಿ
ಭೂತಾನಿ ಸಂವಾಂಛಂತಿ ॥ ೬ ॥
ತತ್ ಹ = ಅದು, ತದ್ವನಂ ನಾಮ =ತದ್ವನ ಎನಿಸಿರುವುದು. ತದ್ವನಮ್ ಇತಿ = ತದ್ವನ ಎಂದು, ಉಪಾಸಿತವ್ಯಂ = ಉಪಾಸಿಸಲ್ಪಡತಕ್ಕದ್ದು ; ಸಃ ಯಃ = ಯಾವ ಅವನು, ಏತತ್ = ಇದನ್ನು, ಏವಂ = ಹೀಗೆ, ವೇದ = ತಿಳಿಯುತ್ತಾನೆಯೊ, ಏನಂ = ಅವನನ್ನು, ಸರ್ವಾಣಿ = ಎಲ್ಲ, ಭೂತಾನಿ ಹ = ಪ್ರಾಣಿಗಳೂ, ಅಭಿಸಂವಾಂಛಂತಿ = ಇಚ್ಛಿಸುವುವು.

೬.ಅದಕ್ಕೆ ‘ತದ್ವನ’೧ ಎಂದು ಹೆಸರು ; (ಅದನ್ನು) ತದ್ವನವೆಂದು ಉಪಾಸಿಸ ಬೇಕು. ಯಾವನು ಇದನ್ನು ಹೀಗೆಂದು ತಿಳಿಯುವನೊ ಅವನನ್ನು ಎಲ್ಲ ಪ್ರಾಣಿ ಗಳೂ ಇಚ್ಛಿಸುವುವು೨.

[೧ ತದ್ವನ ಎಂದರೆ ಅದರ ವನ (ತಸ್ಯ ವನಂ ತದ್ವನಂ) ಬ್ರಹ್ಮವು ಎಲ್ಲ ಪ್ರಾಣಿಗಳ ಆತ್ಮವಾಗಿರುವುದರಿಂದ ವನವು ಎಂದರೆ ವನನೀಯವಾದುದು, ಭಜಿಸಲ್ಪಡತಕ್ಕದ್ದು (ವನಂ ವನನೀಯಂ ಸಂಭಜನೀಯಂ).
೨ ಬ್ರಹ್ಮವನ್ನು ತದ್ವನವೆಂದು ಉಪಾಸನೆ ಮಾಡುವುದರ ಫಲ.]
ಉಪನಿಷದಂ ಭೋ ಬ್ರೂಹೀತ್ಯುಕ್ತಾ ತ
ಉಪನಿಷದ್ಬ್ರಾಹ್ಮೀಂ ವಾವ ತ ಉಪನಿಷದಮಬ್ರೂಮೇತಿ ॥ ೭ ॥
ಭೋ = ಸ್ವಾಮಿ, ಉಪನಿಷದಂ = ಉಪನಿಷತ್ತನ್ನು, ಬ್ರೂಹಿ = ಉಪದೇಶಿಸಿರಿ; ತೇ = ನಿನಗೆ, ಉಪನಿಷತ್ = ಉಪನಿಷತ್ತು, ಉಕ್ತಾ = ಹೇಳಲ್ಪಟ್ಟಿತು. ತೇ = ನಿನಗೆ, ಬ್ರಾಹ್ಮೀಂ ವಾವ ಉಪನಿಷದಂ = ಬ್ರಾಹ್ಮೀ ಉಪನಿಷತ್ತನ್ನೇ, ಅಬ್ರೂಮ = ಹೇಳಿದ್ದೇವೆ, ಇತಿ = ಎಂದು.

೭. (ಶಿಷ್ಯ) : ‘ಸ್ವಾಮಿ, ಉಪನಿಷತ್ತನ್ನು೧ ಉಪದೇಶಿಸಿರಿ.

(ಆಚಾರ್ಯ): ‘ನಿನಗೆ ಉಪನಿಷತ್ತು ಹೇಳಲ್ಪಟ್ಟಿತು. ಬ್ರಾಹ್ಮೀ೨ ಉಪನಿಷತ್ತನ್ನೇ ನಿನಗೆ ಹೇಳಿರುತ್ತೇವೆ೩.’

[೧ ಚಿಂತಿಸಲ್ಪಡತಕ್ಕ ರಹಸ್ಯ.
೨ ಬ್ರಹ್ಮದ ಅಥವಾ ಪರಮಾತ್ಮನ ವಿಷಯವಾದದ್ದು.
೩ ಶಿಷ್ಯನ ಪ್ರಶ್ನೆಯಲ್ಲಿ ಪುನರುಕ್ತಿಯ ದೋಷವಿಲ್ಲ. “ಬ್ರಹ್ಮವಿದ್ಯೆಗೆ ಏನಾದರೂ ಶೇಷವಿದ್ದರೆ ಅಥವಾ ಸಹಕಾರಿಸಾಧನವಿದ್ದರೆ ತಿಳಿಸಿ, ಇಲ್ಲದಿದ್ದರೆ , ‘ಇಲ್ಲ’ ಎಂದು ಹೇಳಿ” ಎಂಬುದು ಅವನ ಪ್ರಶ್ನೆ. ‘ನಿನಗೆ ಉಪನಿಷತ್ತು ಹೇಳಲ್ಪಟ್ಟಿತು’ ಎಂದು ಗುರು ಕೊಟ್ಟ ಉತ್ತರ ಅವಧಾರಣೆ ಮಾಡುವುದಕ್ಕಾಗಿ.]
ತಸ್ಯೈ ತಪೋ ದಮಃ ಕರ್ಮೇತಿ ಪ್ರತಿಷ್ಠಾ
ವೇದಾಃ ಸರ್ವಾಂಗಾನಿ ಸತ್ಯಮಾಯತನಮ್ ॥೮ ॥

ತಪಃ = ತಪಸ್ಸು, ದಮಃ = ದಮ, ಕರ್ಮ = ಕರ್ಮ, ಇತಿ = ಎಂಬ ಇವು, ತಸ್ಯೈ = ಅದಕ್ಕೆ, ಪ್ರತಿಷ್ಠಾ = ಪ್ರತಿಷ್ಠೆ ; ವೇದಾಃ = ವೇದಗಳು, ಸರ್ವಾಂಗಾನಿ = ಎಲ್ಲ ಅಂಗಗಳು, ಸತ್ಯಂ = ಸತ್ಯವು, ಆಯತನಂ =ಆಶ್ರಯ.

೮. ತಪಸ್ಸು೧ ದಮ೨ ಕರ್ಮ೩ ಎಂಬ ಇವು ಅದಕ್ಕೆ೪ ಪ್ರತಿಷ್ಠೆ೫ ; (ಹಾಗೆಯೇ) ವೇದಗಳು (ಅದರ) ಎಲ್ಲ೬ ಅಂಗಗಳು೭ ; ಸತ್ಯವು೮ (ಅದರ) ಆಶ್ರಯ.

[ಬ್ರಹ್ಮವಿದ್ಯೆಯನ್ನು ಪಡೆದುಕೊಳ್ಳುವುದಕ್ಕೆ ತಪೋದಮಕರ್ಮಗಳು ಸಾಧನಗಳು. ಇವು ಗಳಿಂದ ಸಂಸ್ಕಾರವನ್ನು ಪಡೆದವನಿಗೆ ಸತ್ತ್ವಶುದ್ಧಿಯ ಮೂಲಕ ತತ್ತ್ವಜ್ಞಾನ ಉಂಟಾಗುತ್ತದೆ.
೧ ಶರೀರೇಂದ್ರಿಯ ಮನಸ್ಸುಗಳ ಸಮಾಧಾನ.
೨ ಉಪಶಮ.
೩ ಅಗ್ನಿಹೋತ್ರಾದಿ ಕರ್ಮಗಳು.
೪ ಬ್ರಹ್ಮವಿದ್ಯೆಗೆ.
೫ ಪಾದಗಳು.
೬ ಶಿಕ್ಷೆಯೇ ಮೊದಲಾದವು.
೭ ತಪೋದಮಕರ್ಮಗಳಂತೆ ವೇದಗಳೂ ಅದರ ಆರು ಅಂಗಗಳೂ ಪ್ರತಿಷ್ಠೆ. ಭಾಷ್ಯಕಾ ರರು ಹೇಳಿರುವಂತೆ ವೇದಗಳು ಕರ್ಮಜ್ಞಾನಗಳನ್ನು ಪ್ರಕಾಶಿಸುವುದರಿಂದಲೂ, ಅಂಗಗಳು ಅವುಗಳನ್ನು ರಕ್ಷಿಸುವುದರಿಂದಲೂ ಅವು ಬ್ರಹ್ಮವಿದ್ಯೆಗೆ ಪ್ರತಿಷ್ಠೆಯಾಗಿರುವುವು. ಅಥವಾ ಪ್ರತಿಷ್ಠಾ ಶಬ್ದವನ್ನು ಪ್ರಯೋಗಿಸಿರುವುದು ಪಾದರೂಪಕ ಕಲ್ಪನೆಯೆಂದು ತಿಳಿದರೆ, ‘ವೇದ ಗಳು ಶಿರವೇ ಮೊದಲಾದ ಉಳಿದ ಅಂಗಗಳು’ ಎಂದು ಅರ್ಥಮಾಡಬಹುದು. ಈ ಪಕ್ಷದಲ್ಲಿ ವೇದಗಳನ್ನು ಹೇಳಿದ್ದರಿಂದ ಶಿಕ್ಷಾ ಮೊದಲಾದ ಆರು ಅಂಗಗಳನ್ನೂ ಹೇಳಿದಂತೆ ಆಯಿತೆಂದು ಗ್ರಹಿಸಬೇಕು.
೮. ವಾಕ್ಕು ಶರೀರ ಮನಸ್ಸು ಇವುಗಳಲ್ಲಿ ಕುಟಿಲತೆಯಿಲ್ಲದಿರುವುದು.]
ಯೋ ವಾ ಏತಾಮೇವಂ ವೇದಾಪಹತ್ಯ ಪಾಪ್ಮಾನಮನಂತೇ
ಸ್ವರ್ಗೇ ಲೋಕೇ ಜ್ಯೇಯೇ ಪ್ರತಿತಿಷ್ಠತಿ ಪ್ರತಿತಿಷ್ಠತಿ ॥ ೯ ॥
ಯಃ ವೈ = ಯಾವನು, ಏತಾಂ = ಇದನ್ನು, ಏವಂ = ಹೀಗೆ, ವೇದ = ತಿಳಿಯುತ್ತಾನೆಯೊ, (ಸಃ= ಅವನು), ಪಾಪ್ಮಾನಂ = ಪಾಪವನ್ನು, ಅಪಹತ್ಯ = ಕಳೆದುಕೊಂಡು, ಅನಂತೇ = ಅನಂತವಾದ, ಜ್ಯೇಯೇ = ಸರ್ವಮಹತ್ತಮವಾದ, ಸ್ವರ್ಗೇ ಲೋಕೇ =ಸ್ವರ್ಗ ಲೋಕದಲ್ಲಿ, ಪ್ರತಿತಿಷ್ಠತಿ = ಪ್ರತಿಷ್ಠಿತನಾಗುತ್ತಾನೆ.

೯. ಯಾವನು ಇದನ್ನು೧ ಹೀಗೆ ತಿಳಿಯುತ್ತಾನೆಯೊ (ಅವನು) ಪಾಪವನ್ನು೨ ಕಳೆದುಕೊಂಡು ಅನಂತವೂ ಸರ್ವಮಹತ್ತಮವೂ ಆದ ಸ್ವರ್ಗಲೋಕದಲ್ಲಿ೩ ಪ್ರತಿಷ್ಠಿತನಾಗುತ್ತಾನೆ೪. ಪ್ರತಿಷ್ಠಿತನಾಗುತ್ತಾನೆ೫.

[೧ ಈ ಉಪನಿಷತ್ತಿನ ಮೊದಲಿನಿಂದ ಕೊನೆಯವರೆಗೆ ಹೇಳಿರುವ ಬ್ರಹ್ಮವಿದ್ಯೆಯನ್ನು.
೨ ಅವಿದ್ಯಾ-ಕಾಮ-ಕರ್ಮವೆಂಬ ಸಂಸಾರಬೀಜವನ್ನು ನಾಶಮಾಡಿಕೊಂಡು.
೩ ಸುಖಾತ್ಮಕವಾದ ಬ್ರಹ್ಮದಲ್ಲಿ.
೪ ಸಂಸಾರವನ್ನು ಹೊಂದುವುದಿಲ್ಲ.
೫ ದ್ವಿರುಕ್ತಿಯು ಉಪನಿಷತ್ತಿನ ಸಮಾಪ್ತಿಯನ್ನು ಸೂಚಿಸುತ್ತದೆ.]
