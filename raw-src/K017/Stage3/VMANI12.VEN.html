ಬಯಕೆಯುಂಟಾಗುತ್ತದೆ. ಬಯಕೆಯಿಂದ ಮನುಷ್ಯನು ಕರ್ಮದಲ್ಲಿ ಪ್ರವರ್ತಿಸು ತ್ತಾನೆ.
ತತಃ ಸ್ವರೂಪ-ವಿಭ್ರಂಶೋ ವಿಭ್ರಷ್ಟಸ್ತು ಪತತ್ಯಧಃ ।
ಪತಿತಸ್ಯ ವಿನಾ ನಾಶಂ ಪುನರ್ನಾರೋಹ ಈಕ್ಷ್ಯತೇ ।
ಸಂಕಲ್ಪಂ ವರ್ಜಯೇತ್ ತಸ್ಮಾತ್ ಸರ್ವಾನರ್ಥಸ್ಯ ಕಾರಣಮ್ ॥ ೩೨೬ ॥
್ಣಋಸ್ಶ್ಯ್¯ತತ್ಣಃಋಸ್ಬಋ¯ = ಅದರ ದೆಸೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವರೂಪ-ವಿಭ್ರಂಶಃ = ಸ್ವರೂಪಚ್ಯುತಿಯು ; ್ಣಋಸ್ಶ್ಯ್¯ವಿಭ್ರಷ್ಟ್ಣಃಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ತ್ಣುಋಸ್ಬಋ¯ = ಭ್ರಷ್ಟನಾದವನು ್ಣಋಸ್ಶ್ಯ್¯ಅಧ್ಣಃಋಸ್ಬಋ¯ = ಕೆಳಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಪತತ್ಣಿಋಸ್ಬಋ¯ = ಬೀಳುತ್ತಾನೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಪತಿತಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಪತಿತನಾದವ ನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ನಾಶಂ ವಿನ್ಣಾಋಸ್ಬಋ¯ = ನಾಶವೇ ಹೊರತು ್ಣಋಸ್ಶ್ಯ್¯ಪುನ್ಣಃಋಸ್ಬಋ¯ = ಮತ್ತ್ಣೆಋಸ್ಶ್ಯ್¯ ಆರೋಹಃ =ಅಭಿವೃದ್ಧಿ‡ಯು ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಈಕ್ಷ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಕಂಡುಬರುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಮಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಆದುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವ-ಅನರ್ಥಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಎಲ್ಲ ಅನರ್ಥಗಳಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಕಾರಣ್ಣಂಋಸ್ಬಋ¯ = ಕಾರಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಕಲ್ಪ್ಣಂಋಸ್ಬಋ¯ = ವಿಷಯಚಿಂತನೆಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ವರ್ಜ ಯೇತ್ಣ್​ಋಸ್ಬಋ¯ = ಬಿಡಬೇಕು.

೩೨೬. ಅದರಿಂದ ಸ್ವರೂಪ-ಚ್ಯುತಿಯು ಉಂಟಾಗುತ್ತದೆ. ಸ್ವರೂಪದಿಂದ ಜಾರಿದವನು ಕೆಳಕ್ಕೆ ಬೀಳುತ್ತಾನೆ. ಕೆಳಕ್ಕೆ ಬಿದ್ದವನು ನಾಶವಾಗುವನೇ ಹೊರತು ಪುನಃ ಅಭಿವೃದ್ಧಿಯನ್ನು ಪಡೆಯುವುದಿಲ್ಲ. ಆದುದರಿಂದ ಎಲ್ಲ ಅನರ್ಥಗಳಿಗೂ ಕಾರಣವಾದ ವಿಷಯಚಿಂತೆಯನ್ನು ಬಿಡಬೇಕು.

[ ಗೀತಾ ೨. ೬೨-೬೩ ನೊಡಿ]
ಅತಃ ಪ್ರಮಾದಾನ್ನ ಪರೋsಸ್ತಿ ಮೃತ್ಯು-
ವಿRವೇಕಿನೋ ಬ್ರಹ್ಮವಿದಃ ಸಮಾಧೌ ।
ಸಮಾಹಿತಃ ಸಿದ್ಧಿಮುಪೈತಿ ಸಮ್ಯಕ್​
ಸಮಾಹಿತಾತ್ಮಾ ಭವ ಸಾವಧಾನಃ ॥ ೩೨೭ ॥
್ಣಋಸ್ಶ್ಯ್¯ಅತಃ = ಆದುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಿವೇಕಿನ್ಣಃಋಸ್ಬಋ¯ = ವಿವೇಕಿಯಾದ್ಣಋಸ್ಶ್ಯ್¯ ಬ್ರಹ್ಮವಿದ್ಣಃಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಜ್ಞಾನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಮಾಧ್ಣೌಋಸ್ಬಋ¯ = ಸಮಾಧಿಯಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಮಾದಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಎಚ್ಚರಿಕೆ ತಪ್ಪುವುದಕ್ಕಿಂತ ್ಣಋಸ್ಶ್ಯ್¯ಪರ್ಣಃಋಸ್ಬಋ¯ = ಬೇರೊಂದು ್ಣಋಸ್ಶ್ಯ್¯ಮೃತ್ಯುಃ = ಮೃತ್ಯುವು ನ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ತಿ = ಇಲ್ಲವು ; ್ಣಋಸ್ಶ್ಯ್¯ಸಮ್ಯಕ್ಣ್​ಋಸ್ಬಋ¯ = ಚೆನ್ನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸಮಾಹಿತ್ಣಃಋಸ್ಬಋ¯ = ಸಮಾಹಿತನಾದವನೇ ್ಣಋಸ್ಶ್ಯ್¯ಸಿದ್ಧ್ಣಿಂಋಸ್ಬಋ¯ = ಸಿದ್ಧಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಉಪೈತ್ಣಿಋಸ್ಬಋ¯ = ಹೊಂದುತ್ತಾನೆ ; [ಆದುದ ರಿಂದ] ್ಣಋಸ್ಶ್ಯ್¯ಸಾವಧಾನಃ = ಎಚ್ಚರಿಕೆಯುಳ್ಳವನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸಮಾಹಿತಾತ್ಮಾ ಭವ್ಣಋಸ್ಬಋ¯ = ಸಮಾಹಿತವಾದ ಚಿತ್ತವುಳ್ಳವನಾಗು.

೩೨೭. ಆದುದರಿಂದ ವಿವೇಕಿಯಾದ ಬ್ರಹ್ಮಜ್ಞಾನಿಗೆ ಸಮಾಧಿಯಲ್ಲಿ ಎಚ್ಚರಿಕೆ ತಪ್ಪುವುದಕ್ಕಿಂತ ಬೇರೊಂದು ಮೃತ್ಯುವಿರುವುದಿಲ್ಲ. ಚೆನ್ನಾಗಿ ಸಮಾಹಿತನಾದ ವನೇ ಸಿದ್ಧಿಯನ್ನು ಹೊಂದುತ್ತಾನೆ. ಆದುದರಿಂದ ಎಚ್ಚರಿಕೆಯಿಂದ ಮನಸ್ಸನ್ನು ಸಮಾಧಿಯಲ್ಲಿಡು.

ಜೀವತೋ ಯಸ್ಯ ಕೈವಲ್ಯಂ ವಿದೇಹೇ ಸ ಚ ಕೇವಲಃ ।
ಯತ್ಕಿಂಚಿತ್ಪಶ್ಯತೋ ಭೇದಂ ಭಯಂ ಬ್ರೂತೇ ಯಜುಃಶ್ರುತಿಃ ॥ ೩೨೮ ॥
್ಣಋಸ್ಶ್ಯ್¯ಜೀವತಃ = ಬದುಕಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಯಾವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಕೈವಲ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಕೈವಲ್ಯವು [ಆಗಿರುವದೊ] ್ಣಋಸ್ಶ್ಯ್¯ವಿದೇಹೇ ಚ = ದೇಹಾನಂತರದಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಸಃ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ಕೇವಲ್ಣಃಋಸ್ಬಋ¯ = ಕೇವಲನು ; ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣ್​ಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂಚಿತ್ ಭೇದ್ಣಂಋಸ್ಬಋ¯ = ಸ್ಪಲ್ಪ ಭೇದವನ್ನ್ಣೂಋಸ್ಶ್ಯ್¯ ಪಶ್ಯತ್ಣಃಋಸ್ಬಋ¯ = ನೋಡುವವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಭಯ್ಣಂಋಸ್ಬಋ¯ = ಭಯವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಯಜುಃಶ್ರುತಿಃ ಬ್ರೂತ್ಣೇಋಸ್ಬಋ¯ = ಯಜುರ್ವೇದ-ಶ್ರುತಿಯು ಹೇಳುತ್ತದೆ.

೩೨೮. ಬದುಕಿರುವಾಗ ಯಾವನಿಗೆ ಕೈವಲ್ಯವುಂಟಾಗಿರುವುದೊ ಅವನಿಗೇ ದೇಹಾನಂತರದಲ್ಲಿಯೂ ಕೈವಲ್ಯವುಂಟಾಗುತ್ತದೆ. ಸ್ಪಲ್ಪ ಭೇದವನ್ನು ನೋಡು ವವನಿಗೂ ಭಯವಿದೆ–ಎಂದು ಯಜುರ್ವೇದದ ಶ್ರುತಿಯು ಹೇಳುತ್ತದೆ.೧

[೧ ‘ಯಾವಾಗ ಇವನು ಈ ಬ್ರಹ್ಮದಲ್ಲಿ ಒಂದಿಷ್ಟು ಭೇದವನ್ನು ಮಾಡುತ್ತಾನೆಯೊ ಅದರಿಂದ ಅವನಿಗೆ ಭಯವಾಗುತ್ತದೆ’ ್ಣಋಸ್ಶ್ಯ್¯ಯದಾ ಹ್ಯೇವೈಷ ಏತಸ್ಮಿನ್ನುದರಮಂತರ್ಣಂಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಕುರುತೇ ಅಥ ತಸ್ಯ ಭಯಂ ಭವತ್ಣಿಋಸ್ಬಋ¯ (ತೈತ್ತಿರೀಯ ಉ. ೨. ೭)]
ಯದಾ ಕದಾ ವಾsಪಿ ವಿಪಶ್ಚಿದೇಷ
ಬ್ರಹ್ಮಣ್ಯನಂತೇsಪ್ಯಣುಮಾತ್ರಭೇದಮ್ ।
ಪಶ್ಯತ್ಯಥಾಮುಷ್ಯ ಭಯಂ ತದೈವ
ಯದ್ವೀಕ್ಷಿತಂ ಭಿನ್ನತಯಾ ಪ್ರಮಾದಾತ್ ॥ ೩೨೯ ॥
್ಣಋಸ್ಶ್ಯ್¯ಯದಾ ಕದಾ ವಾ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಯಾವಾಗಲೇ ಆಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಏಷಃ ವಿಪಶ್ಚಿತ್ಣ್​ಋಸ್ಬಋ¯ = ಈ ಜ್ಞಾನಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಅನಂತ್ಣೇಋಸ್ಬಋ¯ = ಅನಂತವಾದ್ಣಋಸ್ಶ್ಯ್¯ ಬ್ರಹ್ಮಣ್ಣಿಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಣುಮಾತ್ರಭೇದಮ್ ಅಪಿ = ಸ್ವಲ್ಪ ಭೇದವನ್ನಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಪಶ್ಯತಿ = ನೋಡಿದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಥ = ಅನಂತರ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಮಾದಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಪ್ರಮಾದದ ಮೂಲಕ ್ಣಋಸ್ಶ್ಯ್¯ಭಿನ್ನತಯ್ಣಾಋಸ್ಬಋ¯ = ಭಿನ್ನವಾಗ್ಣಿಋಸ್ಶ್ಯ್¯ ಯತ್ ವೀಕ್ಷಿತ್ಣಂಋಸ್ಬಋ¯ = ಯಾವುದು ನೋಡಲ್ಪಟ್ಟಿತೊ [ಅದು] ್ಣಋಸ್ಶ್ಯ್¯ತದಾ ಏವ್ಣಋಸ್ಬಋ¯ = ಆಗಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಅಮುಷ್ಯ್ಣಋಸ್ಬಋ¯ = ಇವನಿಗ್ಣೆಋಸ್ಶ್ಯ್¯ ಭಯ್ಣಂಋಸ್ಬಋ¯ = ಭಯವನ್ನು [ಉಂಟುಮಾಡುತ್ತದೆ].

೩೨೯. ಯಾವಾಗಲೇ ಆಗಲಿ ಈ ಜ್ಞಾನಿಯು ಅನಂತವಾದ ಬ್ರಹ್ಮದಲ್ಲಿ ಸ್ವಲ್ಪ ಭೇದವನ್ನಾದರೂ ನೋಡಿದರೆ, ಪ್ರಮಾದದಿಂದ ಭಿನ್ನವಾಗಿ ಕಂಡ ಅದು ಆಗಲೇ ಆ ಭೇದದರ್ಶಿಗೆ ಭಯವನ್ನುಂಟುಮಾಡುತ್ತದೆ.

ಶ್ರುತಿ-ಸ್ಮೃತಿ-ನ್ಯಾಯ-ಶತೈರ್ನಿಷಿದ್ಧೇ
ದೃಶ್ಯೇsತ್ರ ಯಃ ಸ್ವಾತ್ಮಮತಿಂ ಕರೋತಿ ।
ಉಪೈತಿ ದುಃಖೋಪರಿ ದುಃಖಜಾತಂ
ನಿಷಿದ್ಧಕರ್ತಾ ಸ ಮಲಿಮ್ಲುಚೋ ಯಥಾ ॥ ೩೩ಂ ॥
್ಣಋಸ್ಶ್ಯ್¯ಶ್ರುತಿ-ಸ್ಮೃತಿ-ನ್ಯಾಯ-ಶತ್ಣೈಃಋಸ್ಬಋ¯ = ಶ್ರುತಿ ಸ್ಮೃತಿ ನೂರಾರು ಯುಕ್ತಿಗಳು–ಇವುಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನಿಷಿದ್ಧ್ಣೇಋಸ್ಬಋ¯ = ನಿಷೇಧಿಸಲ್ಪಟ್ಟ ್ಣಋಸ್ಶ್ಯ್¯ಅತ್ರ ದೃಶ್ಯೇ = ಈ ದೃಶ್ಯಪ್ರಪಂಚದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಯಃ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ-ಆತ್ಮ-ಮತಿಂ = ನಾನು ಎಂಬ ಬುದ್ಧಿಯನ್ನ್ಣುಋಸ್ಶ್ಯ್¯ ಕರೋತ್ಣಿಋಸ್ಬಋ¯ = ಮಾಡುತ್ತಾನೆಯೊ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ನಿಷಿದ್ಧಕರ್ತಾ್ಣಋಸ್ಬಋ¯ = ಮಾಡಬಾರದ್ದನ್ನು ಮಾಡುವ ್ಣಋಸ್ಶ್ಯ್¯ಮಲಿಮ್ಲುಚಃ ಯಥ್ಣಾಋಸ್ಬಋ¯ = ಕಳ್ಳನು ಹೇಗೋ ಹಾಗೆ, ್ಣಋಸ್ಶ್ಯ್¯ದುಃಖ-ಉಪರಿ ದುಃಖಜಾತ್ಣಂಋಸ್ಬಋ¯ = ದುಃಖಗಳ ಮೇಲೆ ದುಃಖಸಮೂಹ ವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಉಪೈತ್ಣಿಋಸ್ಬಋ¯ = ಹೊಂದುತ್ತಾನೆ.

೩೩ಂ. ಶ್ರುತಿ, ಸ್ಮೃತಿ ಮತ್ತು ನೂರಾರು ಯುಕ್ತಿಗಳು ನಿಷೇಧಿಸಿರುವ ಈ ದೃಶ್ಯಜಗತ್ತಿನಲ್ಲಿ ಯಾವನು ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು ಮಾಡುತ್ತಾನೆಯೊ ಅವನು–ಮಾಡಬಾರದ್ದನ್ನು ಮಾಡುವ ಕಳ್ಳನಂತೆ–ದುಃಖಗಳ ಮೇಲೆ ದುಃಖ ವನ್ನು ಹೊಂದುತ್ತಾನೆ.

ಸತ್ಯಾಭಿಸಂಧಾನರತೋ ವಿಮುಕ್ತೋ
ಮಹತ್ವಮಾತ್ಮೀಯಮುಪೈತಿ ನಿತ್ಯಮ್ ।
ಮಿಥ್ಯಾಭಿಸಂಧಾನರತಸ್ತು ನಶ್ಯೇದ್
ದೃಷ್ಟಂ ತದೇತದಚೋರ-ಚೋರಯೋಃ ॥ ೩೩೧ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸತ್ಯ-ಅಭಿಸಂಧಾನ-ರತ್ಣಃಋಸ್ಬಋ¯ = ಸತ್ಯಬ್ರಹ್ಮವನ್ನು ಅನುಸಂಧಾನಮಾಡುತ್ತಿರುವವನು ್ಣಋಸ್ಶ್ಯ್¯ವಿಮುಕ್ತಃ = ಮುಕ್ತನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮೀಯಂ = ತನ್ನ ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಶಾಶ್ವತವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮಹತ್ತ್ವ್ಣಂಋಸ್ಬಋ¯ = ಮಹಿಮೆಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಉಪೈತ್ಣಿಋಸ್ಬಋ¯ = ಹೊಂದುತ್ತಾನೆ ; ್ಣಋಸ್ಶ್ಯ್¯ತ್ಣುಋಸ್ಬಋ¯ = ಆದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಮಿಥ್ಯಾ-ಅಭಿಸಂಧಾನ-ರತ್ಣಃಋಸ್ಬಋ¯ = ಮಿಥ್ಯಾವಸ್ತುವನ್ನೇ ಅನುಸಂಧಾನಮಾಡುತ್ತಿರುವವನು ್ಣಋಸ್ಶ್ಯ್¯ನಶ್ಯೇತ್ಣ್​ಋಸ್ಬಋ¯ = ನಾಶವಾಗುತ್ತಾನೆ; ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣ್​ಋಸ್ಬಋ¯ = ಏಕೆಂದರೆ, ್ಣಋಸ್ಶ್ಯ್¯ತತ್ ಏತತ್ಣ್​ಋಸ್ಬಋ¯ = ಆ ಇದು ್ಣಋಸ್ಶ್ಯ್¯ಅಚೋರ-ಚೋರಯ್ಣೋಃಋಸ್ಬಋ¯ = ಚೋರನು ಚೋರನಲ್ಲದವನು–ಇವರ ವಿಷಯದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ದೃಷ್ಟ್ಣಂಋಸ್ಬಋ¯ = ನೋಡಲ್ಪಟ್ಟಿದೆ.

೩೩೧. ಯಾವನು ಸತ್ಯಬ್ರಹ್ಮವನ್ನೇ ಅನುಸಂಧಾನಮಾಡುತ್ತಿರುವನೊ ಅವನು (ಸಂಸಾರದಿಂದ) ಬಿಡುಗಡೆಯನ್ನು ಹೊಂದಿದವನಾಗಿ ತನ್ನ ಶಾಶ್ವತವಾದ ಮಹಿಮೆ ಯನ್ನು ಹೊಂದುತ್ತಾನೆ ; ಆದರೆ ಯಾವನು ಮಿಥ್ಯಾರೂಪವಾಗಿರುವ (ದೇಹಾದಿ ಗಳನ್ನೇ ಆತ್ಮವಸ್ತುವೆಂದು) ಅನುಸಂಧಾನಮಾಡುತ್ತಾನೆಯೊ ಅವನು ನಾಶವಾಗು ತ್ತಾನೆ. ಆ ಇದು ಚೋರನು ಚೋರನಲ್ಲದವನು–ಇದರ ವಿಷಯದಲ್ಲಿ ನೋಡ ಲ್ಪಟ್ಟಿದೆ.೧

[೧. ಹಿಂದಿನ ಕಾಲದಲ್ಲಿ ಒಂದು ಕೊಡಲಿಯನ್ನು ಕಾಯಿಸಿ ಅದನ್ನು ಆಪಾದಿತನು ಮುಟ್ಟುವ ಹಾಗೆ ಮಾಡುತ್ತಿದ್ದರು ; ಅವನ ಕೈಸುಡದಿದ್ದರೆ ಅವನು ನಿರಪರಾಧಿಯೆಂದೂ ಸುಟ್ಟರೆ ಅಪರಾಧಿಯೆಂದೂ ನಿರ್ಧರಿಸುತ್ತಿದ್ದರು. ಇದು ಛಾಂದೋಗ್ಯಪನಿಷತ್ತಿನ ೬. ೧೬ರಲ್ಲಿ ಬರುವ ಅಗ್ನಿಪರೀಕ್ಷೆಯನ್ನು ಸೂಚಿಸುತ್ತದೆ.]

ಣಿಲ೧೮ಂಃ೪ಣಬಶ್ಲಸ್ರಿವ್ ಪ್ ಯತಿರಸದನುಸಂಧಿಂ ಬಂಧಹೇತುಂ ವಿಹಾಯ
ಸ್ವಯಮಯಮಹಮಸ್ಮೀತ್ಯಾತ್ಮದೃಷ್ಟ್ಯೈವ ತಿಷ್ಠೇತ್ ।
ಸುಖಯತಿ ನನು ನಿಷ್ಠಾ ಬ್ರಹ್ಮಣಿ ಸ್ವಾನುಭೂತ್ಯಾ
ಹರತಿ ಪರಮವಿದ್ಯಾಕಾರ್ಯದುಃಖಂ ಪ್ರತೀತಮ್ ॥ ೩೩೨ ॥

್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣಿಃಋಸ್ಬಋ¯ = ಯತ್ನಶೀಲನು ್ಣಋಸ್ಶ್ಯ್¯ಬಂಧಹೇತ್ಣುಂಋಸ್ಬಋ¯ = ಬಂಧಕ್ಕೆ ಕಾರಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್​- ಅನು ಸಂಧ್ಣಿಂಋಸ್ಬಋ¯ = ಅಸತ್ತಿನ ಅನುಸಂಧಾನವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ವಿಹಾಯ್ಣಋಸ್ಬಋ¯ = ತೊರೆದು ್ಣಋಸ್ಶ್ಯ್¯ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಇವನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಮ್ ಅಹಂ =ನಾನೇ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ಆಗಿರುವೆನು ್ಣಋಸ್ಶ್ಯ್¯ಇತಿ = ಎಂದು ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮದೃಷ್ಟ್ಯಾ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಏವ್ಣಋಸ್ಬಋ¯ = ಆತ್ಮದೃಷ್ಟಿಯಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ತಿಷ್ಠೇತ್ಣ್​ಋಸ್ಬಋ¯ = ನಿಲ್ಲಬೇಕು ; ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಾನುಭೂತ್ಯ್ಣಾಋಸ್ಬಋ¯ = ಆತ್ಮಾನುಭವದಿಂದ್ಣಋಸ್ಶ್ಯ್¯ ಬ್ರಹ್ಮಣ್ಣಿಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮದಲ್ಲ್ಣಿಋಸ್ಶ್ಯ್¯ ನಿಷ್ಣಾಋಸ್ಬಋ¯್ಠ = ನಿಷ್ಠೆಯು ್ಣಋಸ್ಶ್ಯ್¯ಸುಖಯತಿ = ಸುಖವನ್ನು ಕೊಡುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತೀತ್ಣಂಋಸ್ಬಋ¯ = ತೋರಿಬರುತ್ತಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಅವಿದ್ಯಾ-ಕಾರ್ಯ-ದುಃಖ್ಣಂಋಸ್ಬಋ¯ =ಅವಿದ್ಯಾಕಾರ್ಯವಾದ ದುಃಖವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಪರಂ ಹರತ್ಣಿಋಸ್ಬಋ¯ = ಅತ್ಯಂತವಾಗಿ ಹೋಗಲಾಡಿಸುತ್ತದೆ.

೩೩೨. ಪ್ರಯತ್ನಶೀಲನಾದವನು ಬಂಧಕ್ಕೆ ಕಾರಣವಾದ ಅಸದ್ವಸ್ತುವಿನ ಅನು ಸಂಧಾನವನ್ನು ತೊರೆದು, ‘ನಾನೇ ಇವನಾಗಿರುತ್ತೇನೆ’ ಎಂದು ಆತ್ಮದೃಷ್ಟಿಯಿಂ ದಲೇ ನಿಲ್ಲಬೇಕು. ಆತ್ಮಾನುಭವದಿಂದ ಬ್ರಹ್ಮದಲ್ಲಿಯೇ ನಿಷ್ಠೆಯನ್ನು ಹೊಂದು ವುದು ಸುಖಗೊಳಿಸುತ್ತದೆ ಮತ್ತು ಅವಿದ್ಯೆಯ ಕಾರ್ಯವಾಗಿ ತೋರಿಬರುತ್ತಿರುವ ದುಃಖವನ್ನೂ ಆತ್ಯಂತಿಕವಾಗಿ ಹೋಗಲಾಡಿಸುತ್ತದೆ.

ಬಾಹ್ಯಾನುಸಂಧಿಃ ಪರಿವರ್ಧಯೇತ್ ಫಲಂ
ದುರ್ವಾಸನಾಮೇವ ತತಸ್ತತೋsಧಿಕಾಮ್ ।
ಜ್ಞಾತ್ವಾ ವಿವೇಕೈಃ ಪರಿಹೃತ್ಯ ಬಾಹ್ಯಂ
ಸ್ವಾತ್ಮಾನುಸಂಧಿಂ ವಿದಧೀತ ನಿತ್ಯಮ್ ॥ ೩೩೩ ॥
್ಣಋಸ್ಶ್ಯ್¯ಬಾಹ್ಯ-ಅನುಸಂಧಿಃ = ವಿಷಯಚಿಂತನೆಯು ್ಣಋಸ್ಶ್ಯ್¯ತತಃ ತತ್ಣಃಋಸ್ಬಋ¯ = ಪುನಃ ಪುನಃ ್ಣಋಸ್ಶ್ಯ್¯ಅಧಿಕ್ಣಾಂಋಸ್ಬಋ¯ = ಅಧಿಕವಾಗುವ ್ಣಋಸ್ಶ್ಯ್¯ದುರ್ವಾಸನ್ಣಾಂಋಸ್ಬಋ¯ =- ಮಲಿನವಾಸನೆಯೆಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಫಲಮ್ ಏವ್ಣಋಸ್ಬಋ¯ = ಫಲವನ್ನೇ ್ಣಋಸ್ಶ್ಯ್¯ಪರಿವರ್ಧಯೇತ್ಣ್​ಋಸ್ಬಋ¯ = ಬೆಳೆಯಿಸುತ್ತದೆ ; [ಯತಿಯು] ್ಣಋಸ್ಶ್ಯ್¯ಜ್ಞಾತ್ವ್ಣಾಋಸ್ಬಋ¯ = [ಇದನ್ನು] ಅರಿತು ಕೊಂಡು ್ಣಋಸ್ಶ್ಯ್¯ವಿವೇಕ್ಣೈಃಋಸ್ಬಋ¯ = ವಿವೇಚನೆಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಬಾಹ್ಯ್ಣಂಋಸ್ಬಋ¯ = ವಿಷಯಚಿಂತನೆಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಪರಿಹೃತ್ಯ್ಣಋಸ್ಬಋ¯ = ತೊರೆದು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ-ಆತ್ಮ-ಅನುಸಂಧಿಂ = ಆತ್ಮಚಿಂತನೆಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ವಿದಧೀತ್ಣಋಸ್ಬಋ¯ = ಮಾಡಬೇಕು.

೩೩೩. ವಿಷಯಚಿಂತನೆಯು ಪುನಃಪುನಃ ಹೆಚ್ಚಾಗುವ ಮಲಿನವಾಸನೆಯೆಂಬ ಫಲವನ್ನೇ ಬೆಳೆಯಿಸುತ್ತದೆ. (ಯತಿಯು) ಇದನ್ನು೧ ಅರಿತುಕೊಂಡು ವಿವೇಚನೆ ಯಿಂದ ವಿಷಯಚಿಂತನೆಯನ್ನು ತೊರೆದು ಆತ್ಮಚಿಂತನೆಯನ್ನೇ ಯಾವಾಗಲೂ ಮಾಡಬೇಕು.

[೧ ‘ವಾಸನಾವೃದ್ಧಿಯಿಂದ ಕರ್ಮವೂ ಕರ್ಮವೃದ್ಧಿಯಿಂದ ವಾಸನೆಯೂ ಬೆಳೆಯು ವುದು’ (೩೧೨).]
ಬಾಹ್ಯೇ ನಿರುದ್ಧೇ ಮನಸಃ ಪ್ರಸನ್ನತಾ
ಮನಃಪ್ರಸಾದೇ ಪರಮಾತ್ಮದರ್ಶನಮ್ ।
ತಸ್ಮಿನ್ ಸುದೃಷ್ಟೇ ಭವಬಂಧನಾಶೋ
ಬಹಿರ್ನಿರೋಧಃ ಪದವೀ ವಿಮುಕ್ತೇಃ ॥ ೩೩೪ ॥
್ಣಋಸ್ಶ್ಯ್¯ಬಾಹ್ಯೇ ನಿರುದ್ಧೇ [ಸತಿ] = ವಿಷಯಚಿಂತನೆಯು ನಿರುದ್ಧವಾದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಮನಸಃ = ಮನಸ್ಸಿನ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಸನ್ನತ್ಣಾಋಸ್ಬಋ¯ = ಪ್ರಸನ್ನತೆಯು ; ಮನಃಪ್ರಸಾದ್ಣೇಋಸ್ಬಋ¯ =ಮನಸ್ಸು ಪ್ರಸನ್ನವಾದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮಾತ್ಮ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ದರ್ಶನಂ= ಪರಮಾತ್ಮಸಾಕ್ಷಾತ್ಕಾರವು ; ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಮಿನ್ ಸುದೃಷ್ಟ್ಣೇಋಸ್ಬಋ¯ = ಪರಮಾತ್ಮನನ್ನು ಚೆನ್ನಾಗಿ ಸಾಕ್ಷಾತ್ಕರಿಸಿದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಭವಬಂಧನಾಶ್ಣಃಋಸ್ಬಋ¯ = ಭವಬಂಧವು ನಾಶವು ; [ಆದುದರಿಂದ] ಬಹಿರ್ನಿ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ರೋಧಃ = ವಿಷಯಚಿಂತಾನಿರೋಧವೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿಮುಕ್ತೇಃ = ಮುಕ್ತಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಪದವ್ಣೀಋಸ್ಬಋ¯ = ಮಾರ್ಗವು.

೩೩೪. ವಿಷಯಚಿಂತನೆಯ ನಿರೋಧದಿಂದ ಮನಸ್ಸಿನ ಪ್ರಸನ್ನತೆಯೂ, ಮನ ಸ್ಸಿನ ಪ್ರಸನ್ನತೆಯಿಂದ೧ ಪರಮಾತ್ಮನ ಸಾಕ್ಷಾತ್ಕಾರವೂ ಉಂಟಾಗುತ್ತದೆ. ಆ ಪರಮಾತ್ಮನನ್ನು ಚೆನ್ನಾಗಿ ಸಾಕ್ಷಾತ್ಕರಿಸಿದರೆ ಸಂಸಾರಬಂಧವು ನಾಶವಾಗುತ್ತದೆ. ಆದುದರಿಂದ ವಿಷಯಚಿಂತನೆಯ ನಿರೋಧವೇ ಮುಕ್ತಿಗೆ ಮಾರ್ಗವು.

[೧ ‘ಚಿತ್ತದ ಪ್ರಸನ್ನತೆಯುಂಟಾದರೆ ಇವನ ಸಮಸ್ತದುಃಖಗಳೂ ನಾಶವಾಗುತ್ತವೆ; ಏಕೆಂದರೆ ಪ್ರಸನ್ನಚಿತ್ತವುಳ್ಳ ಮನುಷ್ಯನ ಬುದ್ಧಿಯು ಶೀಘ್ರವಾಗಿ ಸ್ಥಿರವಾಗುವುದು’ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಸಾದೇ ಸರ್ವದುಃಖಾನಾಂ ಹಾನಿರಸ್ಯೋಪಜಾಯತೇ । ಪ್ರಸನ್ನಚೇತಸೋಹ್ಯಾಶ್ಣುಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಬುದ್ಧಿಃ ಪರ್ಯವತಿಷ್ಠತೇ ॥ (ಗೀತಾ ೨. ೬೫) .]
ಕಃ ಪಂಡಿತಃ ಸನ್ ಸದಸದ್ವಿವೇಕೀ
ಶ್ರುತಿಪ್ರಮಾಣಃ ಪರಮಾರ್ಥದರ್ಶೀ ।
ಜಾನನ್ ಹಿ ಕುರ್ಯಾದಸತೋsವಲಂಬಂ
ಸ್ವಪಾತಹೇತೋಃ ಶಿಶುವನ್ಮುಮುಕ್ಷುಃ ॥ ೩೩೫ ॥
್ಣಋಸ್ಶ್ಯ್¯ಕ್ಣಃಋಸ್ಬಋ¯ = ಯಾವ ್ಣಋಸ್ಶ್ಯ್¯ಮುಮುಕ್ಷ್ಣುಃಋಸ್ಬಋ¯ = ಮುಮುಕ್ಷುವು ್ಣಋಸ್ಶ್ಯ್¯ಪಂಡಿತಃ ಸನ್ಣ್​ಋಸ್ಬಋ¯ = ಪಂಡಿತನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸತ್​-ಅಸತ್​-ವಿವೇಕ್ಣೀಋಸ್ಬಋ¯ = ಇದು ಸತ್, ಇದು ಅಸತ್ ಎಂಬ ವಿವೇಚನೆಯುಳ್ಳವನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಶ್ರುತಿಪ್ರಮಾಣ್ಣಃಋಸ್ಬಋ¯ =ಶ್ರುತಿಯೇ ಪ್ರಮಾಣವೆಂದು ನಂಬಿದವನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮಾರ್ಥದರ್ಶೀ = ಪರಮಾರ್ಥದರ್ಶಿಯಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಜಾನನ್ ಹ್ಣಿಋಸ್ಬಋ¯ = ತಿಳಿದಿದ್ದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಶಿಶುವತ್ಣ್​ಋಸ್ಬಋ¯ = ಮಗುವಿನಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಪಾತಹೇತೋಃ = ತನ್ನ ಪತನಕ್ಕೆ ಕಾರಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅಸತಃ = ಅಸದ್ವಸ್ತುವಿನ ್ಣಋಸ್ಶ್ಯ್¯ಅವಲಂಬ್ಣಃಋಸ್ಬಋ¯ = ಆಶ್ರಯವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಕುರ್ಯಾತ್ = ಮಾಡಿಯಾನು ?

೩೩೫. ಪಂಡಿತನೂ ಸದಸದ್ವಿವೇಕಿಯೂ ಶ್ರುತಿಪ್ರಮಾಣವನ್ನು ನಂಬಿರುವ ವನೂ ಪರಮಾರ್ಥದರ್ಶಿಯೂ ಆದ ಯಾವ ಮುಮುಕ್ಷುವು ತಾನೇ ತಿಳಿದೂ ತಿಳಿದೂ–ಮಗುವಿನಂತೆ–ತನ್ನ ಪತನಕ್ಕೆ ಕಾರಣವಾದ ಅಸದ್ವಸ್ತುವನ್ನು ಆಶ್ರಯಿ ಸಿಯಾನು ?

ಣಿ೪ಣಬ೧೩ಶ್ಲಸ್ನ್ಇವ್ಶ್ಶ್ ಪ್ ದೇಹಾದಿ-ಸಂಸಕ್ತಿಮತೋ ನ ಮುಕ್ತಿ-
ಮುRಕ್ತಸ್ಯ ದೇಹಾದ್ಯಭಿಮತ್ಯಭಾವಃ ।
ಸುಪ್ತಸ್ಯ ನೋ ಜಾಗರಣಂ ನ ಜಾಗ್ರತಃ
ಸ್ವಪ್ನಸ್ತಯೋರ್ಭಿನ್ನ-ಗುಣಾಶ್ರಯತ್ವಾತ್ ॥ ೩೩೬ ॥

್ಣಋಸ್ಶ್ಯ್¯ದೇಹಾದಿ-ಸಂಸಕ್ತಿಮತ್ಣಃಋಸ್ಬಋ¯ = ಶರೀರಾದಿಗಳಲ್ಲಿ ಅಭಿಮಾನವುಳ್ಳವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಮುಕ್ತಿಃ ನ = ಮುಕ್ತಿಯಿಲ್ಲ; ್ಣಋಸ್ಶ್ಯ್¯ಮುಕ್ತಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಮುಕ್ತನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹಾದಿ-ಅಭಿಮತಿ-ಅಭಾವಃ = ಶರೀರಾದಿಗಳಲ್ಲಿ ಅಭಿಮಾನವಿರುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಸುಪ್ತಸ್ಯ್ಣಋಸ್ಬಋ¯ = ನಿದ್ರಿಸುತ್ತಿರುವವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ನ ಉ ಜಾಗರಣ್ಣಂಋಸ್ಬಋ¯ = ಎಚ್ಚರವಿಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ಜಾಗ್ರತ್ಣಃಋಸ್ಬಋ¯ = ಎಚ್ಚರದಲ್ಲಿರುವವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಪ್ನಃ ನ = ನಿದ್ರೆಯಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ತಯ್ಣೋಃಋಸ್ಬಋ¯ = ಈ ಅವಸ್ಥೆಗಳೆರಡೂ ್ಣಋಸ್ಶ್ಯ್¯ಭಿನ್ನ-ಗುಣ-ಆಶ್ರಯತ್ವಾತ್ = ಪರಸ್ಪರ ವಿರುದ್ಧವಾದ ಗುಣ ಗಳನ್ನು ಆಶ್ರಯಿಸಿರುವುದರಿಂದ.

೩೩೬. ಶರೀರಾದಿಗಳಲ್ಲಿ ಅಭಿಮಾನವುಳ್ಳವನಿಗೆ ಮುಕ್ತಿಯಿಲ್ಲ, ಮುಕ್ತನಾದವ ನಿಗೆ ಶರೀರಾದಿಗಳಲ್ಲಿ ಅಭಿಮಾನವಿರುವುದಿಲ್ಲ. ನಿದ್ರಿಸುತ್ತಿರುವವನಿಗೆ ಎಚ್ಚರ ವಿಲ್ಲ, ಎಚ್ಚರದಲ್ಲಿರುವವನಿಗೆ ನಿದ್ರೆಯಿಲ್ಲ; ಏಕೆಂದರೆ ಈ ಅವಸ್ಥೆಗಳು ಪರಸ್ಪರ ವಿರುದ್ಧಗುಣಗಳನ್ನು ಆಶ್ರಯಿಸಿರುತ್ತವೆ.

೫೯. ಆತ್ಮನಿಷ್ಠೆಯ ವಿಧಾನ
ಅಂತರ್ಬಹಿಃ ಸ್ವಂ ಸ್ಥಿರಜಂಗಮೇಷು
ಜ್ಞಾತ್ವಾssತ್ಮನಾssಧಾರತಯಾ ವಿಲೋಕ್ಯ ।
ತ್ಯಕ್ತಾಖಿಲೋಪಾಧಿರಖಂಡರೂಪಃ
ಪೂರ್ಣಾತ್ಮನಾ ಯಃ ಸ್ಥಿತ ಏಷ ಮುಕ್ತಃ ॥ ೩೩೭ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸ್ಥಿರ-ಜಂಗಮೇಷ್ಣುಋಸ್ಬಋ¯ = ಸ್ಥಾವರ-ಜಂಗಮಗಳಲ್ಲ್ಣಿಋಸ್ಶ್ಯ್¯ ಅಂತ್ಣಃಋಸ್ಬಋ¯ = ಒಳಗೂ ್ಣಋಸ್ಶ್ಯ್¯ಬಹಿಃ = ಹೊರಗೂ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ್ಣಂಋಸ್ಬಋ¯ = ತನ್ನ ಆತ್ಮನನ್ನೇ ್ಣಋಸ್ಶ್ಯ್¯ಜ್ಞಾತ್ವ್ಣಾಋಸ್ಬಋ¯ = ತಿಳಿದು,್ಣಋಸ್ಶ್ಯ್¯ ಆತ್ಮನ್ಣಾಋಸ್ಬಋ¯ = ತನ್ನ ಮೂಲಕವೇ ್ಣಋಸ್ಶ್ಯ್¯ಆಧಾರತಯ್ಣಾಋಸ್ಬಋ¯ = ಆಧಾರಭಾವದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿಲೋಕ್ಯ = ತಿಳಿದ ್ಣಋಸ್ಶ್ಯ್¯ತ್ಯಕ್ತ-ಅಖಿಲ-ಉಪಾಧಿಃ = ಸಮಸ್ತ ಉಪಾಧಿಗಳನ್ನು ಬಿಟ್ಟವನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಖಂಡರೂಪ್ಣಃಋಸ್ಬಋ¯ = ಪರಿಪೂರ್ಣ ಸ್ವರೂಪನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಪೂರ್ಣಾತ್ಮನ್ಣಾಋಸ್ಬಋ¯ = ಸರ್ವಾತ್ಮಸ್ವರೂಪದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಯಃ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಥಿತ್ಣಃಋಸ್ಬಋ¯ = ಇರುವನೊ ್ಣಋಸ್ಶ್ಯ್¯ಏಷಃ = ಇವನೇ ್ಣಋಸ್ಶ್ಯ್¯ಮುಕ್ತಃ = ಮುಕ್ತನು.

೩೩೭. ಸ್ಥಾವರ-ಜಂಗಮಗಳಲ್ಲೆಲ್ಲ ಒಳಗೂ ಹೊರಗೂ ತನ್ನ ಆತ್ಮನನ್ನೇ ತಿಳಿದುಕೊಂಡು, ಆತ್ಮನನ್ನೇ ಅವುಗಳ ಆಧಾರವೆಂದು ಅರಿತುಕೊಂಡು, ಸಮಸ್ತ ಉಪಾಧಿಗಳನ್ನು ಬಿಟ್ಟವನಾಗಿ, ಪರಿಪೂರ್ಣಸ್ವರೂಪನಾಗಿಯೂ ಸರ್ವಾತ್ಮ ಸ್ವರೂಪದಿಂದಲೂ ಯಾವನು ಇರುವನೊ ಅವನೇ ಮುಕ್ತನು.

ಸರ್ವಾತ್ಮನಾ ಬಂಧವಿಮುಕ್ತಿಹೇತುಃ
ಸರ್ವಾತ್ಮಭಾವಾನ್ನ ಪರೋsಸ್ತಿ ಕಶ್ಚಿತ್ ।
ದೃಶ್ಯಾಗ್ರಹೇ ಸತ್ಯುಪಪದ್ಯತೇsಸೌ
ಸರ್ವಾತ್ಮಭಾವೋsಸ್ಯ ಸದಾssತ್ಮನಿಷ್ಠಯಾ ॥ ೩೩೮ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾತ್ಮನ್ಣಾಋಸ್ಬಋ¯ = ಹೇಗೆ ನೋಡಿದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾತ್ಮಭಾವಾತ್ = ಸರ್ವಾತ್ಮಭಾವಕ್ಕಿಂತ ್ಣಋಸ್ಶ್ಯ್¯ಪರ್ಣಃಋಸ್ಬಋ¯ = ಬೇರೆಯಾದ ್ಣಋಸ್ಶ್ಯ್¯ಕಶ್ಚಿತ್ಣ್​ಋಸ್ಬಋ¯ = ಯಾವುದೊಂದ್ಣೂಋಸ್ಶ್ಯ್¯ ಬಂಧವಿಮುಕ್ತಿಹೇತುಃ ನ್ಣಋಸ್ಬಋ¯ = ಬಂಧ ವಿಮುಕ್ತಿಗೆ ಕಾರಣವಾಗಲಾರದು ; ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಇವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಸೌ ಸರ್ವಾತ್ಮಭಾವಃ = ಈ ಸರ್ವಾತ್ಮಭಾವವು ್ಣಋಸ್ಶ್ಯ್¯ಸದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನಿಷ್ಠಯ್ಣಾಋಸ್ಬಋ¯ = ಆತ್ಮನಿಷ್ಠೆಯ ಮೂಲಕ ್ಣಋಸ್ಶ್ಯ್¯ದೃಶ್ಯ-ಅಗ್ರಹೇ ಸತಿ = ದೃಶ್ಯವನ್ನು ಗ್ರಹಿಸದಿದ್ದಾಗ ್ಣಋಸ್ಶ್ಯ್¯ಉಪಪದ್ಯತೇ = ಉಂಟಾಗುತ್ತದೆ.

೩೩೮. ಸಂಸಾರಬಂಧದಿಂದ ಮುಕ್ತನಾಗುವುದಕ್ಕೆ ಸರ್ವಾತ್ಮಭಾವಕ್ಕಿಂತ ಬೇರೆ ಯಾವ ಕಾರಣವೂ ಸರ್ವಪ್ರಕಾರದಿಂದಲೂ ಇರುವುದಿಲ್ಲ. ಇವನಿಗೆ ಈ ಸರ್ವಾತ್ಮ ಭಾವವು ಸದಾ ಆತ್ಮನಿಷ್ಠೆಯ ಮೂಲಕ ದೃಶ್ಯಜಗತ್ತನ್ನು ಗ್ರಹಿಸದಿದ್ದಾಗ ಉಂಟಾ ಗುತ್ತದೆ.

ದೃಶ್ಯಸ್ಯಾಗ್ರಹಣಂ ಕಥಂ ನು ಘಟತೇ ದೇಹಾತ್ಮನಾ ತಿಷ್ಠತೋ
ಬಾಹ್ಯಾರ್ಥಾನುಭವ-ಪ್ರಸಕ್ತ-ಮನಸಸ್ತತ್ತತ್ಕ್ರಿ್ರಯಾಂ ಕುರ್ವತಃ ।
ಸಂನ್ಯಸ್ತಾಖಿಲ-ಧರ್ಮ-ಕರ್ಮ-ವಿಷಯೈರ್ನಿತ್ಯಾತ್ಮನಿಷ್ಠಾಪರೈ
ಸ್ತತ್ತ್ವಜ್ಞೈಃ ಕರಣೀಯಮಾತ್ಮನಿ ಸದಾನಂದೇಚ್ಛುಭಿರ್ಯತ್ನತಃ ॥ ೩೩೯ ॥
್ಣಋಸ್ಶ್ಯ್¯ದೇಹಾತ್ಮನಾ = ದೇಹಾತ್ಮಭಾವದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತಿಷ್ಠತ್ಣಃಋಸ್ಬಋ¯ = ಇರುತ್ತಿರುವವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಬಾಹ್ಯ- ್ಣಋಸ್ಶ್ಯ್¯ಅರ್ಥ-ಅನುಭವ-ಪ್ರಸಕ್ತ-ಮನಸಃ = ವಿಷಯಾನುಭವದಲ್ಲಿ ನೆಲೆಗೊಂಡ ಮನಸ್ಸುಳ್ಳ ವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ತತ್​-ತತ್ ಕ್ರಿಯ್ಣಾಂಋಸ್ಬಋ¯ = ಆಯಾ ಕರ್ಮವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಕುರ್ವತ್ಣಃಋಸ್ಬಋ¯ = ಮಾಡುತ್ತಿರುವವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದೃಶ್ಯಸ್ಯ್ಣಋಸ್ಬಋ¯ = ದೃಶ್ಯಪ್ರಪಂಚದ ್ಣಋಸ್ಶ್ಯ್¯ಅಗ್ರಹಣ್ಣಂಋಸ್ಬಋ¯ = ಪರಿಗ್ರಹಿಸದಿರೋಣವು ್ಣಋಸ್ಶ್ಯ್¯ಕಥಂ ನು ಘಟತ್ಣೇಋಸ್ಬಋ¯ = ಹೇಗೆ ಸಾಧ್ಯವಾಗುತ್ತದೆ ? ್ಣಋಸ್ಶ್ಯ್¯ಸಂನ್ಯಸ್ತ-ಅಖಿಲ-ಧರ್ಮ-ಕರ್ಮ-ವಿಷಯೈಃ = ಸರ್ವಧರ್ಮ ಗಳನ್ನೂ ಕರ್ಮಗಳನ್ನೂ ವಿಷಯಗಳನ್ನೂ ಬಿಟ್ಟಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ-ಆತ್ಮ-ನಿಷ್ಠಾ-ಪರೈಃ = ನಿತ್ಯಾತ್ಮ-ನಿಷ್ಠೆಯಲ್ಲಿ ಆಸಕ್ತರಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನಿ = ತನ್ನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸದಾನಂದ-ಇಚ್ಛುಭಿಃ = ನಿತ್ಯಾ ನಂದವಅನ್ನು ಬಯಸುವ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ತ್ವಜ್ಞೈಃ = ಜ್ಞಾನಿಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ನತಃ = ಪ್ರಯತ್ನಪೂರ್ವಕವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಕರಣೀಯ್ಣಂಋಸ್ಬಋ¯ = [ದೃಶ್ಯದ ಅಗ್ರಹಣವು] ಮಾಡಲ್ಪಡತಕ್ಕದ್ದು.

೩೩೯. ದೇಹಾತ್ಮಭಾವದಿಂದ ಕೂಡಿ, ವಿಷಯಾನುಭವದಲ್ಲಿ ನೆಲೆಗೊಂಡ ಮನಸ್ಸುಳ್ಳವನಾಗಿ, ಆಯಾ ಕರ್ಮವನ್ನು ಮಾಡುತ್ತಿರುವವನಿಗೆ೧ ದೃಶ್ಯಜಗತ್ತನ್ನು ಪರಿಗ್ರಹಿಸದಿರುವಿಕೆಯು ಹೇಗೆ ಸಾಧ್ಯವಾಗುತ್ತದೆ ? ಸಮಸ್ತ ಧರ್ಮಗಳನ್ನೂ೨ ಕರ್ಮಗಳನ್ನೂ೩ ವಿಷಯಗಳನ್ನೂ೪ ತೊರೆದು ನಿತ್ಯಾತ್ಮನಿಷ್ಠೆಯಲ್ಲಿ ಆಸಕ್ತರಾಗಿ ತಮ್ಮಲ್ಲಿ ಸದಾನಂದವನ್ನು ಬಯಸುವ ತತ್ತ್ವಜ್ಞರಿಂದಲೇ (ಈ ದೃಶ್ಯಜಗತ್ತಿನ ಅಪರಿಗ್ರಹವು) ಪ್ರಯತ್ನಪೂರ್ವಕವಾಗಿ ಮಾಡಲ್ಪಡತಕ್ಕದ್ದು.

[೧ ವಿಷಯಲಾಭಕ್ಕೋಸ್ಕರ.
೨ ವೈದಿಕ-ಧರ್ಮಗಳನ್ನು.
೩ ಲೌಕಿಕ-ಕರ್ಮಗಳನ್ನು.
೪ ಶಬ್ದಾದಿ-ವಿಷಯಗಳನ್ನು.]
ಸರ್ವಾತ್ಮಸಿದ್ಧೆಯೇ ಭಿಕ್ಷೋಃ ಕೃತ-ಶ್ರವಣ-ಕರ್ಮಣಃ ।
ಸಮಾಧಿಂ ವಿದಧಾತ್ಯೇಷಾ ಶಾಂತೋ ದಾಂತ ಇತಿ ಶ್ರುತಿಃ ॥ ೩೪ಂ ॥
್ಣಋಸ್ಶ್ಯ್¯ಕೃತ-ಶ್ರವಣ-ಕರ್ಮಣಃ = ವೇದಾಂತಶ್ರವಣವನ್ನು ಮಾಡಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಭಿಕ್ಷ್ಣೋಃಋಸ್ಬಋ¯ = ಭಿಕ್ಷುವಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾತ್ಮ-ಸಿದ್ಧಯ್ಣೇಋಸ್ಬಋ¯ = ಸರ್ವಾತ್ಮಭಾವದ ಸಿದ್ಧಿಗೋಸ್ಕರ ್ಣಋಸ್ಶ್ಯ್¯ಶಾಂತಃ ದಾಂತ್ಣಃಋಸ್ಬಋ¯ = ಶಾಂತನು, ದಾಂತನು ್ಣಋಸ್ಶ್ಯ್¯ಇತಿ = ಎಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಏಷಾ ಶ್ರುತ್ಣಿಃಋಸ್ಬಋ¯ = ಈ ಶ್ರುತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಸಮಾಧ್ಣಿಂಋಸ್ಬಋ¯ = ಸಮಾಧಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ವಿದಧಾತ್ಣಿಋಸ್ಬಋ¯ = ವಿಧಿಸುತ್ತದೆ.

೩೪ಂ. ವೇದಾಂತವಾಕ್ಯದ ಶ್ರವಣವನ್ನು ಮಾಡಿರುವ ಭಿಕ್ಷುವಿಗೆ ಸರ್ವಾತ್ಮ ಭಾವದ ಸಿದ್ಧಿಗೋಸ್ಕರ ‘ಶಾಂತನೂ ದಾಂತನೂ’೧ ಎಂಬ ಈ ಶ್ರುತಿಯು ಸಮಾಧಿಯನ್ನು ವಿಧಿಸುತ್ತದೆ.

[೧ ‘ಆದುದರಿಂದ ಹೀಗೆ ಅರಿತುಕೊಂಡಿರುವವನು ಶಾಂತನೂ ದಾಂತನೂ ಉಪ ರತನೂ ತಿತಿಕ್ಷುವೂ ಸಮಾಹಿತನೂ ಆಗಿ ತನ್ನ ಶರೀರದಲ್ಲಿಯೇ ಆತ್ಮನನ್ನು ನೋಡು ತ್ತಾನೆ, ಸರ್ವವನ್ನೂ ಆತ್ಮನನ್ನಾಗಿ ನೋಡುತ್ತಾನೆ’ ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಮಾದೇವಂವಿಚ್ಛಾಂತೋದಾಂತ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಉಪರತಸ್ತಿತಿಕ್ಷುಃ ಸಮಾಹಿತೋ ಭೂತ್ವಾssತ್ಮನ್ಯೇವಾತ್ಮಾನಂ ಪಶ್ಯತಿ, ಸರ್ವಮಾತ್ಮಾನಂ ಪಶ್ಯತ್ಣಿಋಸ್ಬಋ¯ (ಬೃಹದಾರಣ್ಯಕ ಉ ೪.೪. ೨೩).]
ಆರೂಢಶಕ್ತೇರಹಮೋ ವಿನಾಶಃ
ಕರ್ತುಂ ನ ಶಕ್ಯಃ ಸಹಸಾsಪಿ ಪಂಡಿತೈಃ ।
ಯೇ ನಿರ್ವಿಕಲ್ಪಾಖ್ಯ-ಸಮಾಧಿ-ನಿಶ್ಚಲಾ-
ಸ್ತಾನಂತರಾsನಂತ-ಭವಾ ಹಿ ವಾಸನಾಃ ॥ ೩೪೧ ॥
್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣೇಋಸ್ಬಋ¯ = ಯಾರು ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕಲ್ಪಾಖ್ಯ-ಸಮಾಧಿ-ನಿಶ್ಚಲ್ಣಾಃಋಸ್ಬಋ¯ = ನಿರ್ವಿಕಲ್ಪವೆಂಬ ಸಮಾಧಿ ಯಲ್ಲಿ ನೆಲೆಗೊಂಡಿರುವರೊ ್ಣಋಸ್ಶ್ಯ್¯ತಾನ್ ಅಂತರ್ಣಾಋಸ್ಬಋ¯ = ಅದನ್ನು ಬಿಟ್ಟರೆ ್ಣಋಸ್ಶ್ಯ್¯ಆರೂಢಶಕ್ತ್ಣೇಃಋಸ್ಬಋ¯ = ಪ್ರಬಲವಾದ ಶಕ್ತಿಯುಳ್ಳ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ಣಃಋಸ್ಬಋ¯ = ಅಹಂಕಾರದ ್ಣಋಸ್ಶ್ಯ್¯ವಿನಾಶ್ಣಃಋಸ್ಬಋ¯ = ನಾಶವು ್ಣಋಸ್ಶ್ಯ್¯ಪಂಡಿತೈಃ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಜ್ಞಾನಿಗಳಿಂದಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಸಹಸ್ಣಾಋಸ್ಬಋ¯ = ಕೂಡಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ತು್ಣಂಋಸ್ಬಋ¯ = ಮಾಡಲು ್ಣಋಸ್ಶ್ಯ್¯ನ ಶಕ್ಯ್ಣಃಋಸ್ಬಋ¯ = ಶಕ್ಯವಲ್ಲ; ್ಣಋಸ್ಶ್ಯ್¯ಹ್ಣಿಋಸ್ಬಋ¯ = ಏಕೆಂದರೆ ್ಣಋಸ್ಶ್ಯ್¯ವಾಸನ್ಣಾಃಋಸ್ಬಋ¯ = ಸಂಸ್ಕಾರಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಅನಂತ-ಭವ್ಣಾಃಋಸ್ಬಋ¯ = ಅನೇಕ ಜನ್ಮಗಳಿಂದ ಸಂಪಾದಿಸಲ್ಪಟ್ಟವು.

೩೪೧. ಯಾರು ನಿರ್ವಿಕಲ್ಪವೆಂಬ ಸಮಾಧಿಯಲ್ಲಿ ನೆಲೆಗೊಂಡಿರುವರೊ ಅವರನ್ನು ಬಿಟ್ಟರೆ ಇತರರು ಜ್ಞಾನಿಗಳಾಗಿದ್ದರೂ ಪ್ರಬಲವಾದ ಶಕ್ತಿಯುಳ್ಳ ಅಹಂಕಾರವನ್ನು ಕೂಡಲೇ ನಾಶಮಾಡಲು ಅವರಿಂದ ಆಗುವುದಿಲ್ಲ ; ಏಕೆಂದರೆ ಸಂಸ್ಕಾರಗಳು ಅನೇಕ ಜನ್ಮಗಳಿಂದ ಸಂಪಾದಿಸಲ್ಪಟ್ಟು ಬಂದಿರುತ್ತವೆ.

ಅಹಂಬುದ್ಧ್ಯೈವ ಮೋಹಿನ್ಯಾ ಯೋಜಯಿತ್ವಾssವೃತೇರ್ಬಲಾತ್ ।
ವಿಕ್ಷೇಪಶಕ್ತಿಃ ಪುರುಷಂ ವಿಕ್ಷೇಪಯತಿ ತದ್ಗುಣೈಃ ॥ ೩೪೨ ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿಕ್ಷೇಪಶಕ್ತಿಃ = ವಿಕ್ಷೇಪಶಕ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಆವೃತೇಃ ಬಲಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಆವರಣಶಕ್ತಿಯ ಬಲದ ಮೂಲಕ ್ಣಋಸ್ಶ್ಯ್¯ಪುರುಷ್ಣಂಋಸ್ಬಋ¯ = ಮನುಷ್ಯನನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂಬುದ್ಧ್ಯಾ ಏವ್ಣಋಸ್ಬಋ¯ಅ = ಅಹಂಬುದ್ಧಿಯೆಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಮೋಹಿನ್ಯ್ಣಾಋಸ್ಬಋ¯ = ಮೋಹಿನಿಯೊಡನೆ ್ಣಋಸ್ಶ್ಯ್¯ಯೋಜಯಿತ್ವಾ = ಸೇರಿಸಿ ್ಣಋಸ್ಶ್ಯ್¯ತದ್ಗುಣ್ಣೈಃಋಸ್ಬಋ¯ = ಅದರ ಗುಣಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಕ್ಷೇಪಯತ್ಣಿಋಸ್ಬಋ¯ = ವಿಕ್ಷೇಪಗೊಳಿಸುತ್ತದೆ.

೩೪೨. ವಿಕ್ಷೇಪಶಕ್ತಿಯು ಆವರಣಶಕ್ತಿಯ ಬಲದ ಮೂಲಕ ಮನುಷ್ಯನನ್ನು ಅಹಂಬುದ್ಧಿಯೆಂಬ ಮೋಹಿನಿಯೊಂದಿಗೆ ಸೇರಿಸಿ ಅದರ ಗುಣಗಳಿಂದ೧ ಅವ ನನ್ನು ವಿಕ್ಷೇಪಗೊಳಿಸುತ್ತದೆ.

[೧ ಕರ್ತೃತ್ವವೇ ಮೊದಲಾದ.]

ಣಿಲ೧೮ಂಃ೪ಣಬಶ್ಲಸ್ರಿವ್ ಪ್ ವಿಕ್ಷೇಪಶಕ್ತಿ-ವಿಜಯೋ ವಿಷಮೋ ವಿಧಾತುಂ
ನಿಃಶೇಷಮಾವರಣಶಕ್ತಿ-ನಿವೃತ್ತ್ಯಭಾವೇ ।
ದೃಗ್​-ದೃಶ್ಯಯೋಃ ಸ್ಫುಟಪಯೋಜಲವದ್ವಿಭಾಗೇ
ನಶ್ಯೇತ್ ತದಾವರಣಮಾತ್ಮನಿ ಚ ಸ್ವಭಾವಾತ್ ।
ನಿಃಸಂಶಯೇನ ಭವತಿ ಪ್ರತಿಬಂಧಶೂನ್ಯೋ
ವಿಕ್ಷೇಪಣಂ ನ ಹಿ ತದಾ ಯದಿ ಚೇನ್ಮೃಷಾರ್ಥೇ ।। ೩೪೩ ॥

್ಣಋಸ್ಶ್ಯ್¯ನಿಃಶೇಷಂ = ನಿಃಶೇಷವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಆವರಣಶಕ್ತಿ-ನಿವೃತ್ತಿ-ಅಭಾವ್ಣೇಋಸ್ಬಋ¯ = ಆವರಣ ಶಕ್ತಿಯ ಅಭಾವವಾಗದಿದ್ದರೆ ್ಣಋಸ್ಶ್ಯ್¯ವಿಕ್ಷೇಪಶಕ್ತಿ-ವಿಜಯ್ಣಃಋಸ್ಬಋ¯ = ವಿಕ್ಷೇಪಶಕ್ತಿಯ ವಿಜಯವು ್ಣಋಸ್ಶ್ಯ್¯ವಿಧಾತ್ಣುಂಋಸ್ಬಋ¯ = ಸಾಧಿಸಲು ್ಣಋಸ್ಶ್ಯ್¯ವಿಷಮಃ = ಕಷ್ಟ ; ್ಣಋಸ್ಶ್ಯ್¯ದೃಗ್ -ದೃಶ್ಯಯ್ಣೋಃಋಸ್ಬಋ¯ = ದೃಗ್​-ದೃಶ್ಯಗಳ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಫುಟ-ಪಯೋಜಲ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ವತ್ಣ್​ಋಸ್ಬಋ¯ = ಸ್ಫುಟವಾದ ಹಾಲುನೀರುಗಳಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ವಿಭಾಗೇ [ಸತಿ] = ವಿವೇಚನೆಯಿರುತ್ತಿರಲು ್ಣಋಸ್ಶ್ಯ್¯ತದ್ಣಾಋಸ್ಬಋ¯ = ಆಗ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನ್ಣಿಋಸ್ಬಋ¯ = ಆತ್ಮನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಭಾವಾತ್ಣ್​ಋಸ್ಬಋ¯ = ತಾನಾಗಿಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಆವರಣ್ಣಂಋಸ್ಬಋ¯ = ಆವರಣವು ್ಣಋಸ್ಶ್ಯ್¯ನಶ್ಯೇತ್ಣ್​ಋಸ್ಬಋ¯ = ನಾಶವಾಗುವುದು ; ್ಣಋಸ್ಶ್ಯ್¯ಮೃಷಾರ್ಥೇ್ಣಋಸ್ಬಋ¯ = ಮಿಥ್ಯೆಯಾದ ಅನಾತ್ಮನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ವಿಕ್ಷೇಪಣಂ = ವಿಕ್ಷೇಪವು ್ಣಋಸ್ಶ್ಯ್¯ನ ಹಿ ಚೇತ್ ಯದ್ಣಿಋಸ್ಬಋ¯ = ಆಗದಿದ್ದರೆ ್ಣಋಸ್ಶ್ಯ್¯ತದಾ = ಆಗ ್ಣಋಸ್ಶ್ಯ್¯ನಿಸ್ಸಂಶಯೇನ್ಣಋಸ್ಬಋ¯ = ನಿಸ್ಸಂಶಯವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತಿಬಂಧಶೂನ್ಯಃ ಭವತ್ಣಿಋಸ್ಬಋ¯ = ಪ್ರತಿಬಂಧವಿಲ್ಲದೆ ಆಗುವುದು.

೩೪೩. ಆವರಣಶಕ್ತಿಯನ್ನು ನಿಃಶೇಷವಾಗಿ ತೊಲಗಿಸಿಕೊಳ್ಳದಿದ್ದರೆ ವಿಕ್ಷೇಪ ಶಕ್ತಿಯನ್ನು ಜಯಸಿಕೊಳ್ಳುವುದು ಕಷ್ಟ. ದೃಗ್ರೂಪನಾದ ಆತ್ಮ, ದೃಶ್ಯವಾದ ಜಗತ್ತು ಇವರೆಡನ್ನೂ–(ಹಂಸವು) ಹಾಲುನೀರುಗಳನ್ನು ಸ್ಫುಟವಾಗಿ ವಿಂಗಡಿಸುವಂತೆ– ವಿಚೇಚನಮಾಡಿದಾಗ ಆ ಆವರಣವು ಆತ್ಮನಲ್ಲಿ ತಾನಾಗಿಯೇ ನಾಶವಾಗುವುದು. ಮಿಥ್ಯಾಭೂತವಾದ ಅನಾತ್ಮನಲ್ಲಿ (ಮನಸ್ಸು) ವಿಕ್ಷೇಪಹೊಂದದಿದ್ದರೆ (ವಿಕ್ಷೇಪ ಶಕ್ತಿಯನ್ನು ಗೆಲ್ಲುವುದು) ನಿಃಸಂಶಯವಾಗಿ ಪ್ರತಿಬಂಧವಿಲ್ಲದೆ ಆಗುವುದು.

ಸಮ್ಯಗ್​ವಿವೇಕಃ ಸ್ಫುಟಬೋಧಜನ್ಯೋ
ವಿಭಜ್ಯ ದೃಗ್​-ದೃಶ್ಯ-ಪದಾರ್ಥ-ತತ್ತ್ವಮ್ ।
ಛಿನತ್ತಿ ಮಾಯಾಕೃತ-ಮೋಹಬಂಧಂ
ಯಸ್ಮಾದ್ವಿಮುಕ್ತಸ್ಯ ಪುನರ್ನ ಸಂಸೃತಿಃ ॥ ೩೪೪ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸ್ಫುಟಬೋಧಜನ್ಯಃ = ಸ್ಫುಟವಾದ ಜ್ಞಾನದಿಂದ ಉಂಟಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸಮ್ಯಗ್​-ವಿವೇಕ್ಣಃಋಸ್ಬಋ¯ = ಸಮ್ಯಗ್​ವಿವೇಕವು ್ಣಋಸ್ಶ್ಯ್¯ದೃಗ್​-ದೃಶ್ಯ-ಪದಾರ್ಥ-ತತ್ತ್ವ್ಣಂಋಸ್ಬಋ¯ = ದೃಕ್ ಮತ್ತು ದೃಶ್ಯ ಎಂಬ ಪದಾರ್ಥಗಳ ತತ್ತ್ವವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ವಿಭಜ್ಯ್ಣಋಸ್ಬಋ¯ = ವಿಂಗಡಿಸ್ಣಿಋಸ್ಶ್ಯ್¯ ಮಾಯಾಕೃತ-ಮೋಹಬಂಧ್ಣಂಋಸ್ಬಋ¯ = ಮಾಯೆಯಿಂದ ಆದ ಮೋಹವೆಂಬ ಬಂಧವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಛಿನತ್ಣಿಋಸ್ಬಋ¯್ತ = ಕತ್ತರಿಸುತ್ತದೆ ; ಯಸ್ಮಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಯಾವುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಮುಕ್ತಸ್ಯ್ಣಋಸ್ಬಋ¯ =ಮುಕ್ತನಾದವನಿಗ್ಣೆಋಸ್ಶ್ಯ್¯ ಪುನ್ಣಃಋಸ್ಬಋ¯ = ಮತ್ತೆ ್ಣಋಸ್ಶ್ಯ್¯ನ ಸಂಸೃತ್ಣಿಃಋಸ್ಬಋ¯ = ಸಂಸಾರವಿಲ್ಲವೊ.

೩೪೪. ಸ್ಫುಟವಾದ ಜ್ಞಾನದಿಂದ ಉತ್ಪನ್ನವಾದ೧ ಸಮ್ಯಗ್​ವಿವೇಕವು೨ ದೃಗ್ರೂಪ ನಾದ ಆತ್ಮ, ದೃಶ್ಯರೂಪವಾದ ಪ್ರಪಂಚ–ಎಂಬ ಪದಾರ್ಥಗಳ ತತ್ತ್ವವನ್ನು ವಿಂಗಡಿಸಿ, ಮಾಯಾಕೃತವಾದ ಮೋಹವೆಂಬ ಬಂಧವನ್ನು ಕತ್ತರಿಸುತ್ತದೆ ; ಅದ ರಿಂದ ಮುಕ್ತನಾದವನಿಗೆ ಪುನಃ ಸಂಸಾರವಿಲ್ಲ.

[೧ ಶಾಸ್ತ್ರ, ಆಚಾರ್ಯ–ಇವರ ಪ್ರಸಾದದಿಂದಲೂ ಮಹಾವಾಕ್ಯಾರ್ಥದ ಅನುಭವ ದಿಂದಲೂ ಉತ್ಪನ್ನವಾದ.
೨ ಬ್ರಹ್ಮವು ಸತ್ಯ, ಜಗತ್ತು ಮಿಥ್ಯೆ ಎಂಬ ರೂಪದ ಸರಿಯಾದ ವಿವೇಕ.]
ಪರಾವರೈಕತ್ವ-ವಿವೇಕವಹ್ನಿ-
ದRಹತ್ಯವಿದ್ಯಾಗಹನಂ ಹ್ಯಶೇಷಮ್ ।
 ಕಿಂ ಸ್ಯಾತ್ ಪುನಃ ಸಂಸರಣಸ್ಯ ಬೀಜ-
ಮದ್ವೈತಭಾವಂ ಸಮುಪೇಯುಷೋsಸ್ಯ ॥ ೩೪೫ ॥
್ಣಋಸ್ಶ್ಯ್¯ಪರ-ಅವರ-ಏಕತ್ವ-ವಿವೇಕ-ವಹ್ನ್ಣಿಃಋಸ್ಬಋ¯ = ಜೀವ-ಪರಮಾತ್ಮರಿಬ್ಬರೂ ಒಂದೇ ಎಂಬ ವಿವೇಕಾಗ್ನಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಅವಿದ್ಯಾಗಹನಂ = ಅವಿದ್ಯೆಯೆಂಬ ಕಾಡನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅಶೇಷ್ಣಂಋಸ್ಬಋ¯ = ನಿಃಶೇಷವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ದಹತಿ = ಸುಡುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವೈತಭಾವ್ಣಂಋಸ್ಬಋ¯ = ಅದ್ವೈತಭಾವವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಸಮ್​-ಉಪೇಯುಷ್ಣಃಋಸ್ಬಋ¯ = ಚೆನ್ನಾಗಿ ಪಡೆದಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಯ = ಇವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಪುನ್ಣಃಋಸ್ಬಋ¯ = ಮತ್ತೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸರಣಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಸಂಸಾರಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಬೀಜ್ಣಂಋಸ್ಬಋ¯ = ಕಾರಣವು ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂ ಸ್ಯಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಯಾವುದು ಇದ್ದೀತು ?

೩೪೫. ಜೀವ ಈಶ್ವರ–ಇವರಿಬ್ಬರೂ ಒಂದೇ ಎಂಬ ವಿವೇಕಾಗ್ನಿಯು ಅವಿದ್ಯೆ ಯೆಂಬ ಕಾಡನ್ನು ನಿಃಶೇಷವಾಗಿ ಸುಡುತ್ತದೆ. ಅದ್ವೈತಭಾವವನ್ನೇ ಚೆನ್ನಾಗಿ ಪಡೆ ದಿರುವ ಜ್ಞಾನಿಗೆ ಪುನಃ ಸಂಸಾರಕ್ಕೆ ಬರಲು ಯಾವ ಕಾರಣವು ಇದ್ದೀತು ?

ಆವರಣಸ್ಯ ನಿವೃತ್ತಿರ್ಭವತಿ ಹಿ ಸಮ್ಯಕ್ಪದಾರ್ಥ-ದರ್ಶನತಃ ।
ಮಿಥ್ಯಾಜ್ಞಾನ-ವಿನಾಶಸ್ತದ್ವದ್ವಿಕ್ಷೇಪ-ಜನಿತ-ದುಃಖನಿವೃತ್ತಿಃ ॥ ೩೪೬ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸಮ್ಯಕ್​-ಪದಾರ್ಥ-ದರ್ಶನತಃ = ವಸ್ತವನ್ನು ಚೆನ್ನಾಗಿ ಸಾಕ್ಷಾತ್ಕರಿಸುವುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಆವರಣಸ್ಯ= ಆವರಣದ ್ಣಋಸ್ಶ್ಯ್¯ನಿವೃತ್ತಿಃ = ನಿವೃತ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಭವತಿ = ಉಂಟಾಗುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ಮಿಥ್ಯಾ- ್ಣಋಸ್ಶ್ಯ್¯ಜ್ಞಾನ-ವಿನಾಶಃ = ಮಿಥ್ಯಾಜ್ಞಾನದ ನಾಶವು [ಆಗುತ್ತದೆ], ್ಣಋಸ್ಶ್ಯ್¯ತದ್ವತ್ಣ್​ಋಸ್ಬಋ¯ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿಕ್ಷೇಪ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಜನಿತ-ದುಃಖನಿವೃತ್ತಿಃ = ವಿಕ್ಷೇಪದಿಂದ ಆದ ದುಃಖದ ನಿವೃತ್ತಿಯೂ [ಆಗುತ್ತದೆ].

೩೪೬. ಪರಮಾರ್ಥವಸ್ತುವನ್ನು ಚೆನ್ನಾಗಿ ಸಾಕ್ಷಾತ್ಕರಿಸುವುದರಿಂದ ಆವರಣವು ಹೋಗುತ್ತದೆ, ಮಿಥ್ಯಾಜ್ಞಾನವೂ ಹೋಗುತ್ತದೆ ; ಹಾಗೆಯೇ ವಿಕ್ಷೇಪದಿಂದ ಉಂಟಾದ ದುಃಖವೂ ಹೋಗುತ್ತದೆ.

ಏತತ್ ತ್ರಿತಯಂ ದೃಷ್ಟಂ ಸಮ್ಯಗ್​-ರಜ್ಜುಸ್ವರೂಪ-ವಿಜ್ಞಾನಾತ್ ।
ತಸ್ಮಾದ್ವಸ್ತು-ಸತತ್ತ್ವಂ ಜ್ಞಾತವ್ಯಂ ಬಂಧಮುಕ್ತಯೇ ವಿದುಷಾ ॥ ೩೪೭ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸಮ್ಯಕ್​-ರಜ್ಜುಸ್ವರೂಪ-ವಿಜ್ಞಾನಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಚೆನ್ನಾಗಿ ಆದ ಹಗ್ಗದ ಸ್ವರೂಪ ಜ್ಞಾನದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಏತತ್ ತ್ರಿತಯಂ = ಈ ಮೂರು ್ಣಋಸ್ಶ್ಯ್¯ದೃಷ್ಟ್ಣಂಋಸ್ಬಋ¯ =ನೋಡಲ್ಪಟ್ಟಿದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಮಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಆದುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಸ್ತು-ಸತತ್ತ್ವಂ = ವಸ್ತುವಿನ ಯಾಥಾತ್ಮ್ಯವು ್ಣಋಸ್ಶ್ಯ್¯ಬಂಧಮುಕ್ತಯ್ಣೇಋಸ್ಬಋ¯ = ಬಂಧನಿವೃತ್ತಿಗಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ವಿದುಷಾ = ಹೀಗೆ ಅರಿತಿರುವವನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಜ್ಞಾತವ್ಯಂ = ತಿಳಿಯಲ್ಪಡತಕ್ಕದ್ದು.

೩೪೭. ಹಗ್ಗದ ಸ್ವರೂಪವನ್ನು ಚೆನ್ನಾಗಿ ಅರಿತುಕೊಂಡದ್ದರಿಂದ ಈ ಮೂರೂ೧ ಕಂಡುಬರುತ್ತವೆ. ಆದುದರಿಂದ ಮುಮುಕ್ಷುವು ಸಂಸಾರಬಂಧ ನಿವೃತ್ತಿಗಾಗಿ ವಸ್ತುವಿನ ಯಾಥಾತ್ಮ್ಯವನ್ನು ಅರಿತುಕೊಳ್ಳಬೇಕು.

ಕ್ಷಣೇsನ್ಯಥಾ ಭಾವಿತಯಾ ಹ್ಯಮೀಷಾ-
ಮಸತ್ತ್ವಮಾತ್ಮಾ ತು ಕದಾಪಿ ನಾನ್ಯಥಾ ॥ ೩೪೯ ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಗ್ನಿಯೋಗಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಅಗ್ನಿಸಂಬಂಧದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅಯಃ ಇವ = ಕಬ್ಬಿಣವು ಹೇಗೋ ಹಾಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಧೀಃ =ಬುದ್ಧಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಸತ್​’-ಸಮನ್ವಯಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಸದ್ವಸ್ತುವಿನ ಸಂಬಂಧದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಮಾತ್ರಾದಿ ರೂಪೇಣ್ಣಋಸ್ಬಋ¯ = ಪ್ರಮಾತೃ ಮೊದಲಾದ ರೂಪದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಜೃಂಭತ್ಣೇಋಸ್ಬಋ¯ = ತೋರಿಕೊಳ್ಳುತ್ತದೆ; ್ಣಋಸ್ಶ್ಯ್¯ಯತಃ = ಯಾವ ಕಾರಣದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತತ್​-ಕಾರ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಬುದ್ಧಿಯ ಕಾರ್ಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಏತತ್ಣ್​ಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ತ್ರಿತಯ್ಣಂಋಸ್ಬಋ¯ =ಈ ಮೂರು ್ಣಋಸ್ಶ್ಯ್¯ಭ್ರಮ-ಸ್ವಪ್ನ-ಮನೋರಥೇಷು = ಭ್ರಾಂತಿ-ಸ್ವಪ್ನ-ಕೋರಿಕೆ ಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಮೃಷ್ಣಾಋಸ್ಬಋ¯ = ಹುಸಿಯಾಗ್ಣಿಋಸ್ಶ್ಯ್¯ ದೃಷ್ಟಂ = ನೋಡಲ್ಪಟ್ಟಿದೆಯೊ [ಆ ಕಾರಣದಿಂದ] ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂಮುಖ್ಣಾಃಋಸ್ಬಋ¯ = ಅಹಂಕಾರವೇ ಮೊದಲಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹಾವಸಾನ್ಣಾಃಋಸ್ಬಋ¯ = ದೇಹದವರೆಗಿನ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಕೃತೇಃ ವಿಕಾರಾಃ = ಪ್ರಕೃತಿಯ ವಿಕಾರಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವೇ ವಿಷಯಾಃ ಚ = ಮತ್ತು ಸಮಸ್ತ ವಿಷಯಗಳೂ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಷಣ್ಣೇಋಸ್ಬಋ¯ = ಕ್ಷಣಕಾಲದಲ್ಲಿಯ್ಣೆಋಸ್ಶ್ಯ್¯ ಅನ್ಯಥಾಭಾವಿತಯಾ = ಬೇರೆಯಾಗಿ ಪಅರಿ ಣಮಿಸುವುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅಮೀಷ್ಣಾಂಋಸ್ಬಋ¯ = ಇವುಗಳಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್ತ್ವ್ಣಂಋಸ್ಬಋ¯ = ಮಿಥ್ಯಾಸ್ವರೂಪವು [ಸಿದ್ಧ ವಾಗಿದೆ] ; ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮಾ ತ್ಣುಋಸ್ಬಋ¯ = ಆತ್ಮನಾದರೋ ್ಣಋಸ್ಶ್ಯ್¯ಕದಾ ಅಪಿ= ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ಅನ್ಯಥ್ಣಾಋಸ್ಬಋ¯ = ಪರಿಣಮಿಸುವುದಿಲ್ಲ.

೩೪೮-೩೪೯. ಅಗ್ನಿಯ ಸಂಬಂಧದಿಂದ ಕಬ್ಬಿಣವು ಕಾದು ಕೆಂಪಾಗುವಂತೆ ಸದ್ವಸ್ತುವಾದ ಆತ್ಮನ ಸಂಬಂಧದಿಂದ ಬುದ್ಧಿಯು ಪ್ರಮಾತೃ (ಪ್ರಮಾಣ ಪ್ರಮೇಯ) ಮೊದಲಾದ ರೂಪದಿಂದ ತೋರಿಕೊಳ್ಳುತ್ತದೆ. ಹೇಗೆ ಬುದ್ಧಿಯ ಕಾರ್ಯವಾದ ಈ ಪ್ರಮಾತೃವೇ ಮೊದಲಾದ ತ್ರಿತಯವು ಭ್ರಾಂತಿ ಸ್ವಪ್ನ ಮನೋ ರಥ–ಇವುಗಳಲ್ಲಿ ಹುಸಿಯಾಗಿರುತ್ತದೆಯೊ ಹಾಗೆಯೇ ಅಹಂಕಾರದಿಂದ ಪ್ರಾರಂಭಿಸಿ ದೇಹದವರೆಗಿನ ಪ್ರಕೃತಿಯ ವಿಕಾರಗಳೂ ಸಮಸ್ತ ವಿಷಯಗಳೂ ಪ್ರತಿ ಕ್ಷಣವೂ ಬೇರೆಯಾಗಿ ಪರಿಣಮಿಸುತ್ತಿರುವುದರಿಂದ ಮಿಥ್ಯೆಯಾಗಿರುವುವು. ಆತ್ಮನಾದರೋ ಯಾವಾಗಲೂ ಪರಿಣಮಿಸುವುದಿಲ್ಲ.

ನಿತ್ಯಾದ್ವಯಾಖಂಡ-ಚಿದೇಕರೂಪೋ
ಬುದ್ಧ್ಯಾದಿಸಾಕ್ಷೀ ಸದಸದ್ವಿಲಕ್ಷಣಃ ।
ಅಹಂಪದ-ಪ್ರತ್ಯಯ-ಲಕ್ಷಿತಾರ್ಥಃ
ಪ್ರತ್ಯಕ್​-ಸದಾನಂದಘನಃ ಪರಾತ್ಮಾ ॥ ೩೫ಂ ॥
್ಣಋಸ್ಶ್ಯ್¯ಪರಾತ್ಮಾ = ಪರಮಾತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ-ಅದ್ವಯ-ಅಖಂಡ-ಚಿದೇಕರೂಪ್ಣಃಋಸ್ಬಋ¯ = ನಿತ್ಯನೂ ಅದ್ವಯನೂ ಅಖಂಡನೂ ಚಿದ್ರೂಪನೂ ್ಣಋಸ್ಶ್ಯ್¯ಬುದ್ಧ್ಯಾದಿಸಾಕ್ಷೀ = ಬುದ್ಧಿ ಮೊದಲಾದು ವುಗಳಿಗೆ ಸಾಕ್ಷಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಸತ್​-ಅಸತ್​-ವಿಲಕ್ಷಣ್ಣಃಋಸ್ಬಋ¯ = ಸ್ಥೂಲಸೂಕ್ಷ್ಮಗಳಿಗಿಂತ ವಿಲಕ್ಷಣನೂ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂಪದ-ಪ್ರತ್ಯಯ-ಲಕ್ಷಿತಾರ್ಥಃ = ಅಹಂಪದ, ಅಹಂಪ್ರತ್ಯಯ ಇವುಗಳಿಂದ ಲಕ್ಷಿತ ನಾದವನೂ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತ್ಯಕ್ಣ್​ಋಸ್ಬಋ¯ = ಪ್ರತ್ಯಕ್​ಸ್ವರೂಪನೂ ್ಣಋಸ್ಶ್ಯ್¯ಸದಾನಂದಘನ್ಣಃಋಸ್ಬಋ¯ = ಸದಾನಂದ ಘನನೂ [ಆಗಿರುತ್ತಾನೆ].

೩೫ಂ. ಪರಮಾತ್ಮನು ನಿತ್ಯನೂ ಅದ್ವಯನೂ ಅಖಂಡನೂ ಚಿದ್ರೂಪನೂ ಬುದ್ಧ್ಯಾದಿಗಳಿಗೆ ಸಾಕ್ಷಿಯೂ ಸ್ಥೂಲ-ಸೂಕ್ಷ್ಮಗಳಿಗಿಂತ ವಿಲಕ್ಷಣನೂ ಅಹಂಪದ- ಪ್ರತ್ಯಯಗಳಿಂದ ಲಕ್ಷಿತನಾದವನೂ ಪ್ರತ್ಯಕ್​ಸ್ವರೂಪನೂ ಸದಾನಂದಘನನೂ ಆಗಿರುತ್ತಾನೆ.

ಇತ್ಥಂ ವಿಪಶ್ಚಿತ್ ಸದಸದ್ವಿಭಜ್ಯ
ನಿಶ್ಚಿತ್ಯ ತತ್ತ್ವಂ ನಿಜಬೋಧದೃಷ್ಟ್ಯಾ ।
ಜ್ಞಾತ್ವಾ ಸ್ವಮಾತ್ಮಾನಮಖಂಡಬೋಧಂ
ತೇಭ್ಯೋ ವಿಮುಕ್ತಃ ಸ್ವಯಮೇವ ಶಾಮ್ಯತಿ ॥ ೩೫೧ ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿಪಶ್ಚಿತ್ಣ್​ಋಸ್ಬಋ¯ = ಜ್ಞಾನಿಯಾದವನು ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಥ್ಣಂಋಸ್ಬಋ¯ = ಹೀಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸತ್​-ಅಸತ್ ವಿಭಜ್ಯ್ಣಋಸ್ಬಋ¯ = ಸದ್ವಸ್ತ ವನ್ನೂ ಅಸದ್ವಸ್ತುವನ್ನೂ ವಿಂಗಡಿಸಿ ್ಣಋಸ್ಶ್ಯ್¯ನಿಜಬೋಧದೃಷ್ಟ್ಯಾ್ಣಋಸ್ಬಋ¯ = ಸ್ವಾನುಭವ-ದೃಷ್ಟಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ತ್ವಂ = ವಸ್ತುಯಾಥಾತ್ಮ್ಯವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನಿಶ್ಚಿತ್ಯ್ಣಋಸ್ಬಋ¯ = ಗೊತ್ತುಮಾಡಿಕೊಂಡು, ್ಣಋಸ್ಶ್ಯ್¯ಅಖಂಡ- ್ಣಋಸ್ಶ್ಯ್¯ಬೋಧ್ಣಂಋಸ್ಬಋ¯ = ಅಖಂಡಜ್ಞಾನಸ್ವರೂಪನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಮ್ ಆತ್ಮಾನ್ಣಂಋಸ್ಬಋ¯ = ತನ್ನ ಆತ್ಮವನ್ನ್ಣೇಋಸ್ಶ್ಯ್¯ ಜ್ಞಾತ್ವ್ಣಾಋಸ್ಬಋ¯ = ಅರಿತುಕೊಂಡು ್ಣಋಸ್ಶ್ಯ್¯ತೇಭ್ಯಃ = ಆ ಅನಾತ್ಮ ವಸ್ತುಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಮುಕ್ತಃ = ಮುಕ್ತನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಮ್ ಏವ = ತಾನಾಗಿಯ್ಣೇಋಸ್ಶ್ಯ್¯ ಶಾಮ್ಯತ್ಣಿಋಸ್ಬಋ¯ = ಶಾಂತನಾಗುತ್ತಾನೆ.

೩೫೧. ಜ್ಞಾನಿಯಾದ ಮುಮುಕ್ಷುವು ಹೀಗೆ ಸದ್ವಸ್ತುವನ್ನೂ೧ ಅಸದ್ವಸ್ತುವನ್ನೂ೨ ವಿಂಗಡಿಸಿ, (ವಿಚಾರದಿಂದ ಉತ್ಪನ್ನವಾದ) ಸ್ವಾನುಭವದೃಷ್ಟಿಯಿಂದ ವಸ್ತು ಯಾಥಾತ್ಮ್ಯವನ್ನು ಗೊತ್ತುಮಾಡಿಕೊಂಡು, ಅಖಂಡಜ್ಞಾನ-ಸ್ವರೂಪನಾದ ತನ್ನ ಆತ್ಮನನ್ನೇ ಅರಿತುಕೊಂಡು, ಆ ಅನಾತ್ಮವಸ್ತುಗಳಿಂದ ಮುಕ್ತನಾಗಿ ತಾನಾಗಿಯೇ ಶಾಂತನಾಗುತ್ತಾನೆ.

[೧ ಬ್ರಹ್ಮವನ್ನೂ.
೨ ಅವಿದ್ಯೆ ಮತ್ತು ಅದರ ಕಾರ್ಯಗಳು–ಇವುಗಳನ್ನೂ.]
೬೧. ಸಮಾಧಿಯ ನಿರೂಪಣ
ಅಜ್ಞಾನ-ಹೃದಯ-ಗ್ರಂಥೇರ್ನಿಶ್ಶೇಷ-ವಿಲಯಸ್ತದಾ ।
ಸಮಾಧಿನಾsವಿಕಲ್ಪೇನ ಯದಾsದ್ವೈತಾತ್ಮ-ದರ್ಶನಮ್ ॥ ೩೫೨ ॥
್ಣಋಸ್ಶ್ಯ್¯ಅವಿಕಲ್ಪೇನ = ನಿರ್ವಿಕಲ್ಪವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸಮಾಧಿನಾ = ಸಮಾಧಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಯದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವೈತಾತ್ಮ ದರ್ಶನ್ಣಂಋಸ್ಬಋ¯ = ಅದ್ವಯನಾದ ಪರಮಾತ್ಮನ ಸಾಕ್ಷಾತ್ಕಾರವು [ಭವತಿ = ಆಗುವುದೊ] ತದ್ಣಾಋಸ್ಬಋ¯ = ಆಗಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಅಜ್ಞಾನ-ಹೃದಯ-ಗ್ರಂಥೇಃ = ಅಜ್ಞಾನವೆಂಬ ಹೃದಯ ಗ್ರಂಥಿಯ ್ಣಋಸ್ಶ್ಯ್¯ನಿಶ್ಶೇಷ-ವಿಲಯಃ = ಸಂಪೂರ್ಣನಾಶವು [ಆಗುತ್ತದೆ].

೩೫೨. ಯಾವಾಗ ನಿರ್ವಿಕಲ್ಪಸಮಾಧಿಯಿಂದ ಅದ್ವಯನಾದ ಪರಮಾತ್ಮನ ಸಾಕ್ಷಾತ್ಕಾರವಾಗುವುದೋ ಆಗಲೇ ಅಜ್ಞಾನವೆಂಬ ಹೃದಯಗ್ರಂಥಿಯು ಸಂಪೂರ್ಣವಾಗಿ ನಾಶವಾಗುವುದು.

ತ್ವಮಹಮಿದಮಿತೀಯಂ ಕಲ್ಪನಾ ಬುದ್ಧಿದೋಷಾತ್​
ಪ್ರಭವತಿ ಪರಮಾತ್ಮನ್ಯದ್ವಯೇ ನಿರ್ವಿಶೇಷೇ ।
ಪ್ರವಿಲಸತಿ ಸಮಾಧಾವಸ್ಯ ಸರ್ವೋ ವಿಕಲ್ಪೋ
ವಿಲಯನಮುಪಗಚ್ಛೇದ್ವಸ್ತುತತ್ತ್ವಾವಧೃತ್ಯಾ ॥ ೩೫೩ ॥
್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಯ್ಣೇಋಸ್ಬಋ¯ = ಅದ್ವಿತೀಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಶೇಷ್ಣೇಋಸ್ಬಋ¯ = ವಿಶೇಷಶೂನ್ಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮಾತ್ಮನ್ಣಿಋಸ್ಬಋ¯ = ಪರಮಾತ್ಮನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಬುದ್ಧಿದೋಷಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಬುದ್ಧಿಯ ದೋಷದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತ್ವ್ಣಂಋಸ್ಬಋ¯ = ನೀನು ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಇದ್ಣಂಋಸ್ಬಋ¯ = ಇದು ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಎಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಇಯಂ ಕಲ್ಪನ್ಣಾಋಸ್ಬಋ¯ = ಈ ಕಲ್ಪನೆಯು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಭವತ್ಣಿಋಸ್ಬಋ¯ = ಉಂಟಾಗುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಇವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಮಾಧ್ಣೌಋಸ್ಬಋ¯ =ಸಮಾಧಿಯಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರವಿಲಸತಿ [ಸತಿ] = [ಪರಮಾರ್ಥವು] ಪ್ರಕಾಶಿಸುತ್ತಿರುವಾಗ ್ಣಋಸ್ಶ್ಯ್¯ವಸ್ತುತತ್ತ್ವ-ಅವಧೃತ್ಯ್ಣಾಋಸ್ಬಋ¯ = ವಸ್ತುಯಾಥಾತ್ಮ್ಯದ ಅವಧಾರಣೆಯ ಮೂಲಕ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಃ ವಿಕಲ್ಪ್ಣಃಋಸ್ಬಋ¯ = ವಿಕಲ್ಪಗಳೆಲ್ಲವೂ ್ಣಋಸ್ಶ್ಯ್¯ವಿಲಯನಮ್ ಉಪ- ಗಚ್ಛೇತ್ಣ್​ಋಸ್ಬಋ¯ = ವಿನಾಶವನ್ನು ಹೊಂದುತ್ತವೆ.

೩೫೩. ಅದ್ವಿತೀಯನೂ ವಿಶೇಷಶೂನ್ಯನೂ ಆದ ಪರಮಾತ್ಮನಲ್ಲಿ ‘ನೀನು’ ‘ನಾನು’ ‘ಇದು’–ಎಂಬ ಕಲ್ಪನೆಯು ಬುದ್ಧಿದೋಷದಿಂದ ಉಂಟಾಗುತ್ತದೆ. ಇವನಿಗೆ ಸಮಾಧಿಯಲ್ಲಿ (ಪರಮಾರ್ಥತತ್ತ್ವವು) ಪ್ರಕಾಶಿಸುತ್ತಿರುವಾಗ ವಸ್ತು ಯಾಥಾತ್ಮ್ಯದ ಅವಧಾರಣೆಯ ಮೂಲಕ ವಿಕಲ್ಪಗಳೆಲ್ಲವೂ ವಿನಾಶವನ್ನು ಹೊಂದುತ್ತವೆ.

ಣಿಲ೧೮ಂಃ೪ಣಬಶ್ಲಸ್ರಿವ್ ಪ್ ಶಾಂತೋ ದಾಂತಃ ಪರಮುಪರತಃ ಕ್ಷಾಂತಿಯುಕ್ತಃ ಸಮಾಧಿಂ
ಕುರ್ವನ್ನಿತ್ಯಂ ಕಲಯತಿ ಯತಿಃ ಸ್ವಸ್ಯ ಸರ್ವಾತ್ಮಭಾವಮ್ ।
 ತೇನಾವಿದ್ಯಾ-ತಿಮಿರ-ಜನಿತಾನ್ ಸಾಧು ದಗ್ಧ್ವಾ ವಿಕಲ್ಪಾನ್
ಬ್ರಹ್ಮಾಕೃತ್ಯಾ ನಿವಸತಿ ಸುಖಂ ನಿಷ್ಕ್ರಿಯೋ ನಿರ್ವಿಕಲ್ಪಃ ॥ ೩೫೪ ॥

್ಣಋಸ್ಶ್ಯ್¯ಶಾಂತ್ಣಃಋಸ್ಬಋ¯ = ಶಾಂತನಾದ ್ಣಋಸ್ಶ್ಯ್¯ದಾಂತ್ಣಃಋಸ್ಬಋ¯ = ದಾಂತನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮ್ ಉಪರತ್ಣಃಋಸ್ಬಋ¯ = ಅತ್ಯಂತವಾಗಿ ಉಪರತನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಷಾಂತಿಯುಕ್ತ್ಣಃಋಸ್ಬಋ¯ = ದ್ವಂದ್ವಸಹಿಷ್ಣುವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣಿಃಋಸ್ಬಋ¯ = ಯತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಸಮಾಧಿಂ ಕುರ್ವನ್ಣ್​ಋಸ್ಬಋ¯ = ಸಮಾಧಿಯನ್ನು ಮಾಡುತ್ತ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಸ್ಯ್ಣಋಸ್ಬಋ¯ = ತನ್ನ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾತ್ಮಭಾವ್ಣಂಋಸ್ಬಋ¯ = ಸರ್ವಾತ್ಮಭಾವವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಕಲಯತ್ಣಿಋಸ್ಬಋ¯ = ಅನುಭವಿಸುತ್ತಾನೆ ; ್ಣಋಸ್ಶ್ಯ್¯ತೇನ = ಆ ಸಮಾಧಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅವಿದ್ಯಾ-ತಿಮಿರ-ಜನಿತಾನ್ಣ್​ಋಸ್ಬಋ¯ = ಅವಿದ್ಯೆಯೆಂಬ ಅಂಧಕಾರದಿಂದ ಉಂಟಾದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಕಲ್ಪಾನ್ಣ್​ಋಸ್ಬಋ¯ = ವಿಕಲ್ಪಗಳನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಸಾಧುದಗ್ಧ್ವಾ್ಣಋಸ್ಬಋ¯ = ಚೆನ್ನಾಗಿ ಸುಟ್ಟು ್ಣಋಸ್ಶ್ಯ್¯ನಿಷ್ಕ್ರಿಯ್ಣಂಋಸ್ಬಋ¯ = ನಿಷ್ಕ್ರಿಯನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕಲ್ಪ್ಣಃಋಸ್ಬಋ¯ = ನಿರ್ವಿಕಲ್ಪನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ-ಆಕೃತ್ಯ್ಣಾಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಾಕಾರದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸುಖಂ ನಿವಸತ್ಣಿಋಸ್ಬಋ¯ = ಸುಖವಾಗಿರುತ್ತಾನೆ.

೩೫೪. ಶಾಂತನೂ೧ ದಾಂತನೂ೨ ಅತ್ಯಂತವಾಗಿ ಉಪರತನೂ೩ ದ್ವಂದ್ವ ಸಹಿಷ್ಣುವೂ ಆದ ಯತಿಯು ಯಾವಾಗಲೂ ಸಮಾಧಿಯನ್ನು ಮಾಡುತ್ತ, ತನ್ನ ಸರ್ವಾತ್ಮಭಾವವನ್ನು ಅನುಭವಿಸುತ್ತಾನೆ. ಅವಿದ್ಯೆಯೆಂಬ ಅಂಧಕಾರದಿಂದ ಉಂಟಾದ ವಿಕಲ್ಪಗಳನ್ನು ಆ ಸಮಾಧಿಯಿಂದ ಚೆನ್ನಾಗಿ ಸುಟ್ಟು, ನಿಷ್ಕ್ರಿಯನೂ ನಿರ್ವಿಕಲ್ಪನೂ ಆಗಿ ಬ್ರಹ್ಮಾಕಾರದಿಂದ ಸುಖವಾಗಿರುತ್ತಾನೆ.
