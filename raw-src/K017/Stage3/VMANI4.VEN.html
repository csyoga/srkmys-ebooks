ಪ್ರಾಣಾದಿ ಪಂಚಾಭ್ರಮುಖಾನಿ ಪಂಚ ।
ಬುದ್ಧ್ಯಾದ್ಯವಿದ್ಯಾಪಿ ಚ ಕಾಮ-ಕರ್ಮಣೀ
ಪುರ್ಯಷ್ಟಕಂ ಸೂಕ್ಷ್ಮಶರೀರಮಾಹುಃ ॥ ೯೬ ॥
್ಣಋಸ್ಶ್ಯ್¯ವಾಗಾದಿ ಪಂಚ್ಣಋಸ್ಬಋ¯ = ವಾಗಾದಿ ಪಂಚಕ ್ಣಋಸ್ಶ್ಯ್¯ಶ್ರವಣಾದಿ ಪಂಚ = ಶ್ರವಣಾದಿ ಪಂಚಕ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಣಾದಿ ಪಂಚ = ಪ್ರಾಣಾದಿ ಪಂಚಕ ್ಣಋಸ್ಶ್ಯ್¯ಅಭ್ರಮುಖಾನಿ ಪಂಚ = ಆಕಾಶಾದಿ ಭೂತ ಪಂಚಕ್ಣಋಸ್ಶ್ಯ್¯ ಬುದ್ಧ್ಯಾದಿ= ಬುದ್ಧಿಯೇ ಮೊದಲಾದುವು ್ಣಋಸ್ಶ್ಯ್¯ಅವಿದ್ಯ್ಣಾಋಸ್ಬಋ¯ = ಅವಿದ್ಯಾ ್ಣಋಸ್ಶ್ಯ್¯ಅಪಿ ಚ = ಮತ್ತು ್ಣಋಸ್ಶ್ಯ್¯ಕಾಮ-ಕರ್ಮಣ್ಣೀಋಸ್ಬಋ¯ = ಕಾಮ ಮತ್ತು ಕರ್ಮ–ಪುರ್ಯಷ್ಟಕಂ = ಈ ಎಂಟು ಪುರಗಳನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಸೂಕ್ಷ್ಮಶರೀರ್ಣಂಋಸ್ಬಋ¯ = ಸೂಕ್ಷ್ಮಶರೀರವೆಂದು ್ಣಋಸ್ಶ್ಯ್¯ಆಹ್ಣುಃಋಸ್ಬಋ¯ = ಹೇಳುತ್ತಾರೆ.

೯೬. ವಾಕ್ಕು ಮೊದಲಾದ ಐದು ಕರ್ಮೇಂದ್ರಿಯಗಳು, ಶ್ರವಣವೇ ಮೊದಲಾದ ಐದು ಜ್ಞಾನೇಂದ್ರಿಯಗಳು, ಪ್ರಾಣವೇ ಮೊದಲಾದ ಪಂಚ ಪ್ರಾಣಗಳು, ಆಕಾಶವೇ ಮೊದಲಾದ ಸೂಕ್ಷ್ಮಭೂತಗಳು, ಬುದ್ಧಿಯೇ ಮೊದಲಾದ ನಾಲ್ಕು, ಅವಿದ್ಯೆ, ಕಾಮ, ಕರ್ಮ–ಈ ಎಂಟು ‘ಪುರಗಳನ್ನು’ ಸೂಕ್ಷ್ಮಶರೀರವೆಂದು ಹೇಳುತ್ತಾರೆ.

[ಭಾಷ್ಯಕಾರರು ತಮ್ಮ ಸೂತ್ರಭಾಷ್ಯದಲ್ಲಿ (೨. ೪. ೭) ಈ ಸ್ಮೃತಿವಾಕ್ಯವನ್ನು ಉದ್ಧರಿಸಿರುತ್ತಾರೆ : ‘ಪುರ್ಯಷ್ಟಕರೂಪವಾದ ಪ್ರಾಣಾದಿ ಸೂಕ್ಷ್ಮಶರೀರದಿಂದ ಅವನು ಕೂಡಿರುತ್ತಾನೆ ; ಅದರಿಂದ ಬದ್ಧನಾಗಿರುವುದರಿಂದ ಅವನಿಗೆ ಬಂಧನ, ಅದರಿಂದ ಮಕ್ತನಾದಾಗ ಅವನಿಗೆ ಮೋಕ್ಷ’ ್ಣಋಸ್ಶ್ಯ್¯ಪುರ್ಯಷ್ಟಕೇನ ಲಿಂಗೇನ ಪ್ರಾಣಾದ್ಯೇನ ಸ ಯುಜ್ಯತೇ। ತೇನ ಬದ್ಧಸ್ಯ ವೈ ಬಂಧೋ ಮೋಕ್ಷೋ ಮುಕ್ತಸ್ಯ ತೇನ ಚ ॥]
೨೪. ಸೂಕ್ಷ್ಮಶರೀರ
ಇದಂ ಶರೀರಂ ಶೃಣು ಸೂಕ್ಷ್ಮಸಂಜ್ಞಿತಂ
ಲಿಂಗಂ ತ್ವಪಂಚೀಕೃತಭೂತ-ಸಂಭವಮ್ ।
ಸವಾಸನಂ ಕರ್ಮಫಲಾನುಭಾವಕಂ
ಸ್ವಾಜ್ಞಾನತೋsನಾದಿರುಪಾಧಿರಾತ್ಮನಃ ॥ ೯೭ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸೂಕ್ಷ್ಮಸಂಜ್ಞಿತ್ಣಂಋಸ್ಬಋ¯ = ಸೂಕ್ಷ್ಮವೆಂಬ ಹೆಸರುಳ್ಳ ್ಣಋಸ್ಶ್ಯ್¯ಅಪಂಚೀಕೃತಭೂತ-ಸಂಭವ್ಣಂಋಸ್ಬಋ¯ = ಪಂಚೀಕೃತವಾಗದ ಭೂತಗಳಿಂದ ಉತ್ಪನ್ನವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸವಾಸನಂ = ವಾಸನೆಯಿಂದ ಕೂಡಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ ಕರ್ಮಫಲಾನುಭಾವಕಂ = ಕರ್ಮಫಲಗಳ ಅನುಭವವನ್ನುಂಟುಮಾಡುವ ್ಣಋಸ್ಶ್ಯ್¯ಇದಂ ಲಿಂಗಂ ಶರೀರಂ ತ್ಣುಋಸ್ಬಋ¯ = ಈ ಲಿಂಗಶರೀರದ ವಿಷಯವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಶೃಣು = ಕೇಳು [ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಇದು] ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ-ಅಜ್ಞಾನತಃ = ತನ್ನ ಸ್ವರೂಪವನ್ನು ತಿಳಿಯದಿರುವುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನಃ = ಆತ್ಮನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಅನಾದಿಃ = ಅನಾದಿಯಾದ ್ಣಋಸ್ಶ್ಯ್¯ಉಪಾಧಿಃ = ಉಪಾಧಿಯು.

೯೭. ಸೂಕ್ಷ್ಮಶರೀರವೆಂಬ ಹೆಸರುಳ್ಳ, ಪಂಚೀಕೃತವಾಗದ ಭೂತಗಳಿಂದ ಉತ್ಪನ್ನವಾದ, ವಾಸನೆಯಿಂದ೧ ಕೂಡಿರುವ ಮತ್ತು ಕರ್ಮಫಲಗಳ ಅನುಭವ ವನ್ನುಂಟುಮಾಡುವ೨ ಈ ಲಿಂಗಶರೀರದ ವಿಷಯವನ್ನು ಕೇಳು ; ತನ್ನ ಸ್ವರೂಪ ವನ್ನು ತಿಳಿಯದೆ ಇರುವುದರಿಂದ ಇದು ಆತ್ಮನಿಗೆ ಅನಾದಿಯಾದ ಉಪಾಧಿ ಯಾಗಿದೆ.

[ಇಲ್ಲಿ ಸೂಕ್ಷ್ಮಶರೀರದ ಅಥವಾ ಲಿಂಗದೇಹದ ಸ್ವರೂಪವನ್ನು ಹೇಳಿದೆ.
೧ ಅನುಭವದಿಂದ ಉತ್ಪನ್ನವಾದ ಸಂಸ್ಕಾರಗಳಿಂದ.
೨ ಕರ್ಮವು ಅದೃಷ್ಟರೂಪದಿಂದ ಲಿಂಗದೇಹದಲ್ಲಿ ಲಯವಾಗಿ ಸುಖದುಃಖರೂಪ ವಾದ ಫಲವನ್ನುಂಟುಮಾಡುತ್ತದೆ.]
ಸ್ವಪ್ನೋ ಭವತ್ಯಸ್ಯ ವಿಭಕ್ತ್ಯವಸ್ಥಾ
ಸ್ವಮಾತ್ರಶೇಷೇಣ ವಿಭಾತಿ ಯತ್ರ ।
ಸ್ವಪ್ನೇ ತು ಬುದ್ಧಿಃ ಸ್ವಯಮೇವ ಜಾಗ್ರತ್​-
ಕಾಲೀನ-ನಾನಾವಿಧ-ವಾಸನಾಭಿಃ ॥ ೯೮ ॥
ಕರ್ತ್ರಾದಿಭಾವಂ ಪ್ರತಿಪದ್ಯ ರಾಜತೇ
ಯತ್ರ ಸ್ಯಯಂಜ್ಯೋತಿರಯಂ ಪರಾತ್ಮಾ ।
ಧೀಮಾತ್ರಕೋಪಾಧಿರಶೇಷಸಾಕ್ಷೀ
ನ ಲಿಪ್ಯತೇ ತತ್ಕೃತ-ಕರ್ಮಲೇಶೈಃ ।
ಯಸ್ಮಾದಸಂಗಸ್ತತ ಏವ ಕರ್ಮಭಿ-
ನR ಲಿಪ್ಯತೇ ಕಿಂಚಿದುಪಾಧಿನಾ ಕೃತೈಃ ॥ ೯೯ ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಇವನ ್ಣಋಸ್ಶ್ಯ್¯ವಿಭಕ್ತಿ-ಅವಸ್ಥಾ = ಬೇರೆ ಸ್ಥಿತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಪ್ನಃ ಭವತ್ಣಿಋಸ್ಬಋ¯ = ಸ್ವಪ್ನವೆನಿಸುತ್ತದೆ ಯತ್ರ = ಎಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಮಾತ್ರಶೇಷೇಣ್ಣಋಸ್ಬಋ¯ = ತಾನೊಬ್ಬನೇ ಶೇಷವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ವಿಭಾತ್ಣಿಋಸ್ಬಋ¯ = ಪ್ರಕಾಶಿಸುತ್ತಾನೆಯೊ,್ಣಋಸ್ಶ್ಯ್¯ ಸ್ವಪ್ನೇ ತ್ಣುಋಸ್ಬಋ¯ = ಸ್ವಪ್ನದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಬುದ್ಧ್ಣಿಃಋಸ್ಬಋ¯ = ಅಂತಃಕರಣವು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಮೇವ್ಣಋಸ್ಬಋ¯ = ತಾನೊಂದೇ ್ಣಋಸ್ಶ್ಯ್¯ಜಾಗ್ರತ್ ಕಾಲೀನ-ನಾನಾವಿಧ-ವಾಸನಾಭ್ಣಿಃಋಸ್ಬಋ¯ = ಜಾಗ್ರದವಸ್ಥೆಯ ನಾನಾ ವಿಧವಾದ [ಅನುಭವಗಳ] ವಾಸನೆಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ತ್ರಾದಿಭಾವಂ = ಕರ್ತೃವೇ ಮೊದಲಾದ ಭಾವವನ್ನು ಹೊಂದಿ ್ಣಋಸ್ಶ್ಯ್¯ರಾಜತೇ = ಪ್ರಕಾಶಿಸುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ರ್ಣಋಸ್ಬಋ¯ = ಎಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಯಂ ಪರಾತ್ಮ್ಣಾಋಸ್ಬಋ¯ = ಈ ಪರಮಾತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಂಜ್ಯೋತ್ಣಿಃಋಸ್ಬಋ¯ = ಸ್ವಯಂಜ್ಯೋತಿಯು ; ್ಣಋಸ್ಶ್ಯ್¯ಧೀಮಾತ್ರಕ-ಉಪಾಧ್ಣಿಃಋಸ್ಬಋ¯ = ಕೇವಲ ಅಂತಃಕರಣವೆಂಬ ಉಪಾಧಿಯನ್ನು ಹೊಂದಿದವನಾದ, ್ಣಋಸ್ಶ್ಯ್¯ಅಶೇಷ ಸಾಕ್ಷೀ = ಎಲ್ಲಕ್ಕೂ ಸಾಕ್ಷಿಯಾದ [ಈ ಆತ್ಮನು] ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಕೃತ-ಕರ್ಮ-ಲೇಶೈಃ = ಅಂತಃಕರಣದಿಂದಾದ ಕರ್ಮಲೇಶದಿಂದಲೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ಲಿಪ್ಯತೇ = ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಮಾತ್ = ಯಾವ ಕಾರಣದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅಸಂಗಃ = ಸಂಗರಹಿತನೊ ್ಣಋಸ್ಶ್ಯ್¯ತತಃ ಏವ = ಆ ಕಾರಣದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಉಪಾಧಿನಾ = ಉಪಾಧಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಕೃತ್ಣೈಃಋಸ್ಬಋ¯ = ಮಾಡಲ್ಪಟ್ಟ ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ಮಭ್ಣಿಃಋಸ್ಬಋ¯ = ಕರ್ಮಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂಚಿತ್ಣ್​ಋಸ್ಬಋ¯ = ಸ್ವಲ್ಪವಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ಲಿಪ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ.

೯೮-೯೯. ಈ (ಸೂಕ್ಷ್ಮಶರೀರಾಭಿಮಾನಿಯಾದ) ಜೀವನಿಗೆ ಸ್ವಪ್ನವು ಜಾಗ್ರತ್ತಿ ಗಿಂತ ಭಿನ್ನವಾದ ಅವಸ್ಥೆ ; ಅಲ್ಲಿ ಅವನು ತಾನೊಬ್ಬನೇ ಉಳಿದುಕೊಂಡು೧ ಪ್ರಕಾಶಿಸುತ್ತಾನೆ. ಸ್ವಪ್ನದಲ್ಲಿ ಅಂತಃಕರಣವು ತಾನೊಂದೇ ಜಾಗ್ರದವಸ್ಥೆಯ ನಾನಾ ವಿಧವಾದ ಅನುಭವಗಳ ವಾಸನೆಗಳಿಂದ ಕರ್ತೃವೇ ಮೊದಲಾದ ಭಾವವನ್ನು ಹೊಂದಿ ಪ್ರಕಾಶಿಸುತ್ತದೆ ; ಆದರೆ ಅಲ್ಲಿ ಈ ಪರಮಾತ್ಮನು ಸ್ವಯಂಜ್ಯೋತಿಯಾಗಿ ರುತ್ತಾನೆ.೨ ಕೇವಲ ಅಂತಃಕರಣವೆಂಬ ಉಪಾಧಿಯನ್ನು ಹೊಂದಿದವನಾಗಿ ಅಶೇಷ ಸಾಕ್ಷಿಯಾಗಿರುವ ಆತ್ಮನು ಅಂತಃಕರಣದಿಂದಾದ ಕರ್ಮಲೇಶದಿಂದಲೂ ಲಿಪ್ತ ನಾಗುವುದಿಲ್ಲ. ಇವನು ಅಸಂಗನಾಗಿರುವುದರಿಂದಲೇ ಉಪಾಧಿಯಿಂದಾದ ಕರ್ಮ ಗಳಿಂದ ಸ್ವಲ್ಪವೂ ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ.

[ಇಲ್ಲಿ ಸೂಕ್ಷ್ಮಶರೀರದ ವಿಶೇಷಾವಸ್ಥೆಯನ್ನು ಹೇಳಿದೆ.
೧ ಬಾಹ್ಯೇಂದ್ರಿಯಗಳ ಸಹಾಯವಿಲ್ಲದೆ.
೨ ‘ಸ್ವಪ್ನದಲ್ಲಿ ಈ ಪುರುಷನು ಸ್ವಯಂಜ್ಯೋತಿಯಾಗಿರುತ್ತಾನೆ’ ್ಣಋಸ್ಶ್ಯ್¯ಅತ್ರಾಯಂ ಪುರುಷ್ಣಃಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಂಜ್ಯೋತಿರ್ಭವತಿ (ಬೃಹದಾರಣ್ಯಕ ಉ. ೪. ೧೪)]
ಸರ್ವವ್ಯಾಪೃತಿ-ಕರಣಂ ಲಿಂಗಮಿದಂ ಸ್ಯಾಚ್ಚಿದಾತ್ಮನಃ ಪುಂಸಃ ।
ವಾಸ್ಯಾದಿಕಮಿವ ತಕ್ಷ್ಣಸ್ತೇನೈವಾತ್ಮಾ ಭವತ್ಯಸಂಗೋsಯಮ್ ॥ ೧ಂಂ ॥
್ಣಋಸ್ಶ್ಯ್¯ತಕ್ಷ್ಣ್ಣಃಋಸ್ಬಋ¯ = ಬಡಗಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ವಾಸ್ಯಾದಿಕಮ್ ಇವ್ಣಋಸ್ಬಋ¯ = ಉಳಿ ಮೊದಲಾದುವುಗಳಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಚಿದಾತ್ಮನ್ಣಃಋಸ್ಬಋ¯ = ಚಿತ್ಸ್ವರೂಪನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪುಂಸ್ಣಃಋಸ್ಬಋ¯ = ಮನುಷ್ಯನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಇದಂ ಲಿಂಗಂ = ಈ ಸೂಕ್ಷ್ಮಶರೀರವು ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವವ್ಯಾಪೃತಿಕರಣಂ ಸ್ಯಾತ್ಣ್​ಋಸ್ಬಋ¯ = ಸಮಸ್ತ ಪ್ರವೃತ್ತಿಗೆ kಕರಣವಾಗಿರುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ತೇನ ಏವ = ಆದುದರಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಅಯಮ್ ಆತ್ಮಾ = ಈ ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ಅಸಂಗಃ ಭವತಿ = ನಿರ್ಲಿಪ್ತ ನಾಗಿರುತ್ತಾನೆ.

೧ಂಂ. ಚಿತ್ಸ್ವರೂಪನಾದ ಮನುಷ್ಯನಿಗೆ ಈ ಸೂಕ್ಷ್ಮಶರೀರವು–ಬಡಗಿಗೆ ಉಳಿಯೇ ಮೊದಲಾದ ಕರಣಗಳಂತೆ–ಸಮಸ್ತ ಪ್ರವೃತ್ತಿಗೂ ಕರಣವಾಗಿರುತ್ತದೆ. ಆದುದರಿಂದಲೇ ಈ ಆತ್ಮನು ನಿರ್ಲಿಪ್ತನಾಗಿರುತ್ತಾನೆ.

[ಬಡಗಿಗೆ ಉಳಿ ಮೊದಲಾದ ಕರಣಗಳಿದ್ದರೆ ದುಃಖಾದಿ ಅನುಭವಗಳು ಉಂಟಾಗು ತ್ತವೆ. ಕರಣರಹಿತವಾದ ಬಡಗಿಗೆ ಯಾವ ಆಯಾಸವೂ ಇಲ್ಲದಿರುವುದರಿಂದ ಸುಖ ವುಂಟಾಗುತ್ತದೆ. ಹಾಗೆಯೇ ಆತ್ಮನು ಸ್ವಪ್ನಜಾಗ್ರತ್ತುಗಳಲ್ಲಿ ಕರ್ತೃವಾಗಿ ದುಃಖಿಯಾ ಗಿರುತ್ತಾನೆ. ಅವನು ತನ್ನ ಶ್ರಮಪರಿಹಾರಕ್ಕಾಗಿ ಕಾರ್ಯಕರಣಸಂಘಾತವನ್ನು ತ್ಯಜಿಸಿ ತನ್ನ ಆತ್ಮನಾದ ಪರಬ್ರಹ್ಮವನ್ನು ಪ್ರವೇಶಿಸಿ ಸುಷುಪ್ತ್ಯವಸ್ಥೆಯಲ್ಲಿ ಅಕರ್ತೃವೂ ಸುಖಿಯೂ ಆಗುತ್ತಾನೆ. ವೇದಾಂತಸೂತ್ರಗಳ ್ಣಋಸ್ಶ್ಯ್¯ತಕ್ಷಾಧಿಕರಣ್ಣಋಸ್ಬಋ¯ವನ್ನು ನೋಡಿ (೨ . ೩. ೪ಂ)]
ಅಂಧತ್ವ-ಮಂದತ್ವ-ಪಟುತ್ವ-ಧರ್ಮಾಃ
ಸೌಗುಣ್ಯ-ವೈಗುಣ್ಯ-ವಶಾದ್ಧಿ ಚಕ್ಷುಷಃ ।
ಬಾಧಿರ್ಯ-ಮೂಕತ್ವ-ಮುಖಾಸ್ತಥೈವ
ಶ್ರೋತ್ರಾದಿಧರ್ಮಾ ನ ತು ವೇತ್ತುರಾತ್ಮನಃ ॥ ೧ಂ೧ ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಂಧತ್ವ-ಮಂದತ್ವ-ಪಟತ್ವ-ಧರ್ಮಾಃ = ಕುರುಡು ಮಂದದೃಷ್ಟಿ ತೀಕ್ಷ ್ಣದೃಷ್ಟಿ– ಇವೇ ಮೊದಲಾದ ಧರ್ಮಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಚಕ್ಷುಷ್ಣಃಋಸ್ಬಋ¯ = ಕಣ್ಣಿನ ್ಣಋಸ್ಶ್ಯ್¯ಸೌಗುಣ್ಯ-ವೈಗುಣ್ಯ-ವಶಾತ್ ಹ್ಣಿಋಸ್ಬಋ¯ =ಸೌಗುಣ್ಯ ವೈಗುಣ್ಯಇವುಗಳಿಂದಲೇ [ಉಂಟಾಗುತ್ತವೆ] ; ್ಣಋಸ್ಶ್ಯ್¯ತಥಾ ಏವ್ಣಋಸ್ಬಋ¯ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಬಾಧಿರ್ಯ-ಮೂಕತ್ವ-ಮುಖಾಃ = ಕಿವುಡು ಮೂಕತನ ಮೊದಲಾದುವು ್ಣಋಸ್ಶ್ಯ್¯ಶ್ರೋತ್ರಾದ್ಣಿಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಧರ್ಮಾಃ = ಕಿವಿಯೇ ಮೊದಲಾದುವುಗಳ ಧರ್ಮಗಳು ್ಣಋಸ್ಶ್ಯ್¯ವೇತ್ತ್ಣುಃಋಸ್ಬಋ¯ = ಜ್ಞಾತೃವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನ್ಣಃಋಸ್ಬಋ¯ = ಆತ್ಮನ ್ಣಋಸ್ಶ್ಯ್¯ನ ತ್ಣುಋಸ್ಬಋ¯ = ಧರ್ಮಗಳಲ್ಲ.

೧ಂ೧. ಕಣ್ಣಿನ ಸೌಗುಣ್ಯದಿಂದ ತೀಕ್ಷ್ಣತ್ವವೂ ವೈಗುಣ್ಯದಿಂದ ಕುರುಡೂ ಮಂದ ತ್ವವೂ ಉಂಟಾಗುತ್ತದೆ. ಹೀಗೆಯೇ ಕಿವುಡು ಮೂಕತನ ಮೊದಲಾದುವು ಕಿವಿಯೇ ಮೊದಲಾದುವುಗಳ ಧರ್ಮವೇ ಹೊರತು ಜ್ಞಾತೃವಾದ ಆತ್ಮನ ಧರ್ಮಗಳಲ್ಲ.೧

[೧ ಸಾಕ್ಷಿಯಾದ ಚಿದಾತ್ಮನಿಗೆ ಈ ಧರ್ಮಗಳಿರುವುದಿಲ್ಲ. ‘ಸರ್ವಲೋಕದ ಕಣ್ಣಾದ ಸೂರ್ಯನು ಹೇಗೆ ಕಣ್ಣಿಗೆ ಸಂಬಂಧಪಟ್ಟ ಮತ್ತು ಬಾಹ್ಯವಾದ ದೋಷಗಳಿಂದ ಲಿಪ್ತ ನಾಗುವುದಿಲ್ಲವೊ ಹಾಗೆಯೇ ಏಕನಾದ ಸರ್ವಪ್ರಾಣಿಗಳ ಅಂತರಾತ್ಮನು ಲೋಕದ ದುಃಖ ದಿಂದ ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ.’ ್ಣಋಸ್ಶ್ಯ್¯ಸೂರ್ಯೋ ಯಥಾ ಸರ್ವಲೋಕಸ್ಯ ಚಕ್ಷುರ್ನ ಲಿಪ್ಯತ್ಣೇಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಚಾಕ್ಷುಷೈರ್ಬಾಹ್ಯದೋಷೈಃ । ಏಕಸ್ತಥಾ ಸರ್ವಭೂತಾಂತರಾತ್ಮಾ ನ ಲಿಪ್ಯತೇ ಲೋಕ ದುಃಖೇನ ಬಾಹ್ಯಃ ॥ (ಕಠ. ಉ. ೨. ೫. ೧೧).]
೨೫. ಪ್ರಾಣದ ಧರ್ಮಗಳು
ಉಚ್ಛ್ವಾಸ-ನಿಃಶ್ವಾಸ-ವಿಜೃಂಭಣ-ಕ್ಷುತ್
ಪ್ರಸ್ಯಂದನಾದ್ಯುತ್ಕ್ರಮಣಾದಿಕಾಃ ಕ್ರಿಯಾಃ ।
ಪ್ರಾಣಾದಿ-ಧರ್ಮಾಣಿ ವದಂತಿ ತಜ್ಞಾಃ
ಪ್ರಾಣಸ್ಯ ಧರ್ಮಾವಶನಾಪಿಪಾಸೇ ॥ ೧ಂ೨ ॥
್ಣಋಸ್ಶ್ಯ್¯ಉಚ್ಛ್ವಾಸ-ನಿಃಶ್ವಾಸ --ವಿಜೃಂಭಣ-ಪ್ರಸ್ಯಂದನಾದಿ-ಉತ್ಕ್ರಮಣಾದಿಕಾಃ ಕ್ರಿಯಾಃ = ಉಚ್ಛ್ವಾಸ ನಿಃಶ್ವಾಸ ಆಕಳಿಕೆ ಸೀನು ಸ್ರಾವ ಶರೀರವನ್ನು ಬಿಡುವುದು ಮೊದಲಾದ ಕರ್ಮಗಳನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ತಜ್ಞಾಃ = ಪ್ರಾಣವಿದರು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಣಾದಿಧರ್ಮಾಣಿ = ಪ್ರಾಣವೇ ಮೊದಲಾದು ವುಗಳ ಧರ್ಮಗಳೆಂದು ್ಣಋಸ್ಶ್ಯ್¯ವದಂತಿ = ಹೇಳುತ್ತಾರೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಅಶನಾ-ಪಿಪಾಸ್ಣೇಋಸ್ಬಋ¯ = ಹಸಿವು ಬಾಯಾರಿಕೆ ಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಣಸ್ಯ = ಪ್ರಾಣದ ್ಣಋಸ್ಶ್ಯ್¯ಧರ್ಮೌ = ಧರ್ಮಗಳು.

೧ಂ೨. ಉಚ್ಛ್ವಾಸ ನಿಃಶ್ವಾಸ ಆಕಳಿಕೆ ಸೀನು ಸ್ರಾವ ಶರೀರತ್ಯಾಗ–ಇವೇ ಮೊದಲಾದ ಕ್ರಿಯೆಗಳನ್ನು ಪ್ರಾಣವಿದರು ಪ್ರಾಣವೇ ಮೊದಲಾದುವುಗಳ ಧರ್ಮ ಗಳೆಂದು ಹೇಳುತ್ತಾರೆ. ಹಸಿವುಬಾಯಾರಿಕೆಗಳೂ ಪ್ರಾಣದ ಧರ್ಮಗಳು.

೨೬. ಅಹಂಕಾರ
ಅಂತಃಕರಣಮೇತೇಷು ಚಕ್ಷುರಾದಿಷು ವರ್ಷ್ಮಣಿ ।
ಅಹಮಿತ್ಯಭಿಮಾನೇನ ತಿಷ್ಠತ್ಯಾಭಾಸ-ತೇಜಸಾ ॥ ೧ಂ೩ ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಂತಃಕರಣ್ಣಂಋಸ್ಬಋ¯ = ಅಂತಃಕರಣವು ್ಣಋಸ್ಶ್ಯ್¯ಏತೇಷು ಚಕ್ಷುರಾದಿಷ್ಣುಋಸ್ಬಋ¯ = ಈ ಚಕ್ಷುಸ್ಸೇ ಮೊದ ಲಾದುವುಗಳಲ್ಲಿಯೂ, ್ಣಋಸ್ಶ್ಯ್¯ವರ್ಷ್ಮಣಿ [ಚ] = [ಮತ್ತು] ಶರೀರದಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಇತಿ = ‘ನಾನು’ ಎಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಅಭಿಮಾನೇನ = ಅಭಿಮಾನದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಆಭಾಸ-ತೇಜಸಾ =ಆತ್ಮನ ಪ್ರತಿ ಚ್ಛಾಯೆಯೊಂದಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ತಿಷ್ಠತಿ = ಇರುತ್ತದೆ.

೧ಂ೩. ಅಂತಃಕರಣವು ಈ ಚಕ್ಷುಸ್ಸೇ ಮೊದಲಾದುವುಗಳಲ್ಲಿಯೂ೧ ಮತ್ತು ಶರೀರದಲ್ಲಿಯೂ ‘ನಾನು’ ಎಂಬ ಅಭಿಮಾನವನ್ನು ಪಡೆದು೨ ಆತ್ಮನ ಪ್ರತಿ ಚ್ಛಾಯೆಯೊಂದಿಗೆ೩ ಕೂಡಿ ಇರುತ್ತದೆ.

[೧ ಜ್ಞಾನೇಂದ್ರಿಯ-ಕರ್ಮೇಂದ್ರಿಯಗಳಲ್ಲಿ.
೨ ‘ನಾನು ನೋಡುತ್ತೇನೆ ’ಇತ್ಯಾದಿ.
೩ ಚಿತ್​-ವಸ್ತುವಿನ ಸಾನ್ನಿಧ್ಯದಲ್ಲಿ ಅದರ ಪ್ರಕಾಶದಂತಿರುವ ತೋರ್ಕೆಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಚಿದಾ ಭಾಸ್ಣಋಸ್ಬಋ¯ಅ ಎಂದು ಹೇಳುತ್ತಾರೆ. ಇದು ನಿಜವಾದ ಚೈತನ್ಯವಲ್ಲ.]
ಅಹಂಕಾರಃ ಸ ವಿಜ್ಞೇಯಃ ಕರ್ತಾ ಭೋಕ್ತಾಭಿಮಾನ್ಯಯಮ್ ।
ಸತ್ತ್ವಾದಿಗುಣಯೋಗೇನ ಚಾವಸ್ಥಾತ್ರಯಮಶ್ನುತೇ ॥ ೧ಂ೪ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅದೇ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂಕಾರಃ = ಅಹಂಕಾರವೆಂದು ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞೇಯಃ = ತಿಳಿಯತಕ್ಕದ್ದು; ್ಣಋಸ್ಶ್ಯ್¯ಅಯಂ= ಇದ್ಣುಋಸ್ಶ್ಯ್¯ ಕರ್ತಾ್ಣಋಸ್ಬಋ¯ = ಕರ್ತೃವೆಂದೂ ್ಣಋಸ್ಶ್ಯ್¯ಭೋಕ್ತಾ = ಭೋಕ್ಚವೆಂದೂ ್ಣಋಸ್ಶ್ಯ್¯ಅಭಿಮಾನ್ಣೀಋಸ್ಬಋ¯ = ಅಭಿಮಾನಿಯೆಂದೂ [ತಿಳಿಯತಕ್ಕದ್ದು] ; ಸತ್ತ್ವಾದಿಗುಣಯೋಗೇನ್ಣಋಸ್ಬಋ¯ = ಸತ್ತ್ವವೇ ಮೊದ ಲಾದ ಗುಣಗಳ ಸಂಬಂಧದಿಂದ್ಣಋಸ್ಶ್ಯ್¯ಅ ಅವಸ್ಥಾತ್ರಯಂ ಚ = ಅವಸ್ಥಾತ್ರಯವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅಶ್ನುತ್ಣೇಋಸ್ಬಋ¯ = ಹೊಂದುತ್ತದೆ.

೧ಂ೪. ಯಾವುದು ಶರೀರದಲ್ಲಿ ಅಭಿಮಾನವನ್ನು ಪಡೆದು ಕರ್ತೃವೂ ಭೋಕ್ತೃವೂ ಆಗುತ್ತದೆಯೊ ಮತ್ತು ಸತ್ತ್ವವೇ ಮೊದಲಾದ ಗುಣಗಳ ಸಂಬಂಧ ದಿಂದ (ಜಾಗ್ರತ್​-ಸ್ವಪ್ನ-ಸುಷುಪ್ತಿಗಳೆಂಬ) ಅವಸ್ಥಾತ್ರಯವನ್ನು೧ ಪಡೆಯುತ್ತ ದೆಯೊ ಅದನ್ನೇ ಅಹಂಕಾರವೆಂದು ತಿಳಿಯಬೇಕು.

[೧ ರಜೋಗುಣಸಂಬಂಧದಿಂದ ಜಾಗ್ರದವಸ್ಥೆಯೂ ಸತ್ತ್ವಸಂಬಂಧದಿಂದ ಸ್ವಪ್ನವೂ ತಮಃ ಸಂಬಂಧದಿಂದ ಸುಷುಷ್ತಿಯೂ ಉಂಟಾಗುತ್ತವೆ.]
ವಿಷಯಾಣಾಮಾನುಕೂಲ್ಯೇ ಸುಖೀ ದುಃಖೀ ವಿಪರ್ಯಯೇ ।
ಸುಖಂ ದುಃಖಂ ಚ ತದ್ಧರ್ಮಃ ಸದಾನಂದಸ್ಯ ನಾತ್ಮನಃ ॥ ೧ಂ೫ ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿಷಯಾಣಾಂ = ವಿಷಯಗಳ ್ಣಋಸ್ಶ್ಯ್¯ಆನುಕೂಲ್ಯ್ಣೇಋಸ್ಬಋ¯ = ಅನುಕೂಲತೆಯಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸುಖೀ = ಸುಖಿಯೆಂದ್ಣೂಋಸ್ಶ್ಯ್¯ ವಿಪರ್ಯಯ್ಣೇಋಸ್ಬಋ¯ = ಪ್ರತಿಕೂಲತೆಯಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ದುಃಖ್ಣೀಋಸ್ಬಋ¯ = ದುಃಖೀಯೆಂದೂ [ವ್ಯವಹರಿಸಲ್ಪಡುತ್ತದೆ] ; ್ಣಋಸ್ಶ್ಯ್¯ಸುಖಂ ದುಃಖಂ ಚ = ಸುಖವೂ ದುಃಖವೂ ್ಣಋಸ್ಶ್ಯ್¯ತತ್​-ಧರ್ಮಃ = ಅದರ ಧರ್ಮವು ್ಣಋಸ್ಶ್ಯ್¯ಸದಾನಂದಸ್ಯ = ನಿತ್ಯವೂ ಆನಂದಸ್ವರೂಪನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನ್ಣಃಋಸ್ಬಋ¯ = ಆತ್ಮನ ್ಣಋಸ್ಶ್ಯ್¯ನ = ಧರ್ಮವಲ್ಲ.

೧ಂ೫. ಇಂದ್ರಿಯವಿಷಯಗಳು ಅನುಕೂಲವಾಗಿರುವಾಗ ಅದು ಸುಖಿಯಾಗು ತ್ತದೆ. ಅವು ವಿಪರೀತವಾದಾಗ ದುಃಖಿಯಾಗುತ್ತದೆ. ಹೀಗೆ ಸುಖದುಃಖಗಳು ಅಹಂಕಾರದ ಧರ್ಮಗಳಾಗಿರುತ್ತವೆಯೇ ಹೊರತು ನಿತ್ಯಾನಂದ-ಸ್ವರೂಪನಾದ ಆತ್ಮನ ಧರ್ಮಗಳಲ್ಲ.

೨೭. ಆತ್ಮನೇ ಪ್ರಿಯತಮನು
ಆತ್ಮಾರ್ಥತ್ವೇನ ಹಿ ಪ್ರೇಯಾನ್ ವಿಷಯೋ ನ ಸ್ವತಃ ಪ್ರಿಯಃ ।
ಸ್ವತ ಏವ ಹಿ ಸರ್ವೇಷಾಮಾತ್ಮಾ ಪ್ರಿಯತಮೋ ಯತಃ ।
ತತ ಆತ್ಮಾ ಸದಾನಂದೋ ನಾಸ್ಯ ದುಃಖಂ ಕದಾಚನ ॥ ೧ಂ೬ ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿಷಯಃ = ವಿಷಯವು ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮಾರ್ಥತ್ವೇನ ಹ್ಣಿಋಸ್ಬಋ¯ = ಆತ್ಮನ ಪ್ರಯೋಜನಕ್ಕಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರೇಯಾನ್ಣಋಸ್ಬಋ¯್ =ಪ್ರಿಯವಾಗಿರುವುದು, ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವತ್ಣಃಋಸ್ಬಋ¯ = ತಾನೇ ್ಣಋಸ್ಶ್ಯ್¯ನ ಪ್ರಿಯಃ = ಪ್ರಿಯವಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣಃಋಸ್ಬಋ¯ = ಯಾವ ಕಾರಣದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮ್ಣಾಋಸ್ಬಋ¯ =ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವತಃ ಏವ್ಣಋಸ್ಬಋ¯ = ತಾನೇ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವೇಷ್ಣಾಂಋಸ್ಬಋ¯ = ಎಲ್ಲರಿಗೂ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಿಯತಮ್ಣಃಋಸ್ಬಋ¯ =ಪ್ರಿಯತಮನೊ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಣಃಋಸ್ಬಋ¯ = ಆ ಕಾರಣದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮ್ಣಾಋಸ್ಬಋ¯ = ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ಸದಾನಂದ್ಣಃಋಸ್ಬಋ¯ = ಸದಾನಂದನು.್ಣಋಸ್ಶ್ಯ್¯ ಅಸ್ಯ= ಇವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಕದಾಚನ್ಣಋಸ್ಬಋ¯ಅ = ಎಂದಿಗೂ ್ಣಋಸ್ಶ್ಯ್¯ದುಃಖ್ಣಂಋಸ್ಬಋ¯ = ದುಃಖವು ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ = ಇಲ್ಲ.

೧ಂ೬. ಇಂದ್ರಿಯವಿಷಯವು ಆತ್ಮನ ಪ್ರಯೋಜನಕ್ಕಾಗಿ ಪ್ರಿಯವಾಗಿರುವುದು, ತನ್ನಷ್ಟಕ್ಕೆ ತಾನೇ ಪ್ರಿಯವಲ್ಲ ; ಏಕೆಂದರೆ ಆತ್ಮನು ಎಲ್ಲರಿಗೂ ತಾನೇ ಪ್ರಿಯತಮ ನಾಗಿರುತ್ತಾನೆ. ಆದುದರಿಂದ ಆತ್ಮನು ನಿತ್ಯಾನಂದ-ಸ್ವರೂಪನಾಗಿರುತ್ತಾನೆ. ಇವ ನಿಗೆ ಎಂದಿಗೂ ದುಃಖವಿಲ್ಲ.

[ಜಗತ್ತಿನಲ್ಲಿ ಯಾವ ವಸ್ತುವೂ ತನ್ನಷ್ಟಕ್ಕೆ ತಾನೇ ಪ್ರಿಯವಾಗಿರುವುದಿಲ್ಲ. ಆತ್ಮನ ಪ್ರಯೋಜನಕ್ಕಾಗಿ ಪ್ರಿಯವಾಗಿರುತ್ತದೆ ಎಂದು ಯಾಜ್ಞವಲ್ಕ್ಯನು ಬಹುವಿಧವಾಗಿ ಮೈತ್ರೇಯಿಗೆ ಉಪದೇಶಿಸುತ್ತಾನೆ. ಆತ್ಮಪ್ರೀತಿಯು ಮುಖ್ಯವೆಂದೂ ಆತ್ಮಪ್ರೀತಿಗೆ ಸಾಧನವಾಗಿರುವ ಇತರ ವಸ್ತುಗಳಲ್ಲಿರುವ ಪ್ರೀತಿಯು ಗೌಣವೆಂದೂ ಅರಿಯಬೇಕು. ಬೃಹದಾರಣ್ಯಕ ಉ. ೨. ೪. ೫ ನೋಡಿ.]
ಯತ್ಸುಷುಪ್ತೌ ನಿರ್ವಿಷಯ ಆತ್ಮಾನಂದೋsನುಭೂಯತೇ ।
ಶ್ರುತಿಃ ಪ್ರತ್ಯಕ್ಷಮೈತಿಹ್ಯಮನುಮಾನಂ ಚ ಜಾಗ್ರತಿ ॥ ೧ಂ೭ ॥
್ಣಋಸ್ಶ್ಯ್¯ಸುಷುಪ್ತೌ = ಸುಷುಪ್ತಿಯಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಷಯಃ = ವಿಷಯರಹಿತವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮಾನಂದಃ = ಆತ್ಮಾನಂದವು ್ಣಋಸ್ಶ್ಯ್¯ಅನುಭೂಯತೇ ಯತ್ಣ್​ಋಸ್ಬಋ¯ = ಅನುಭವಿಸಲ್ಪಡುತ್ತದೆ ಎಂಬುದನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಶ್ರುತ್ಣಿಃಋಸ್ಬಋ¯ =ಶ್ರುತಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತ್ಯಕ್ಷ್ಣಂಋಸ್ಬಋ¯ =ಪ್ರತ್ಯಕ್ಷ ್ಣಋಸ್ಶ್ಯ್¯ಐತಿಹ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಆಪ್ತ್ಣೋಋಸ್ಶ್ಯ್¯ಕ್ತಿ ಅನುಮಾನಂ ಚ್ಣಋಸ್ಬಋ¯ಅ = ಮತ್ತು ಅನುಮಾನ–[ಇವು] ಜಾಗ್ರತಿ = ಸಿದ್ಧಪಡಿಸುತ್ತವೆ.

೧ಂ೭. ಸುಷುಪ್ತಿಯಲ್ಲಿ ಯಾವ ವಿಷಯದ ಸಹಾಯವೂ ಇಲ್ಲದೆ ಆತ್ಮಾ ನಂದವು ಅನುಭವಿಸಲ್ಪಡುತ್ತದೆ. ಶ್ರುತಿಯೂ ಪ್ರತ್ಯಕ್ಷವೂ ಆಪ್ತೋಕ್ತಿಯೂ ಅನು ಮಾನವೂ ಇದನ್ನು ಸಿದ್ಧಪಡಿಸುತ್ತವೆ.

[ಸುಷುಪ್ತಿಯಲ್ಲಿ ಉಂಟಾಗುವ ಅನುಭವವು ಪ್ರತ್ಯಕ್ಷವು. ಜೀವನ್ಮುಕ್ತರಿಂದ ಪರಂ ಪರೆಯಾಗಿ ಬಂದ ಮಾತು ಐತಿಹ್ಯ. ‘ಆತ್ಮನು ಪರಮಾನಂದಸ್ವರೂಪನು, ಏಕೆಂದರೆ ಅವನು ಪರಮಪ್ರೇಮಕ್ಕೆ ಆಸ್ಪದನಾಗಿರುವನು’ ಎಂಬುದು ಅನುಮಾನ. ‘ಇದು ಇವನ ಪರಮಗತಿ, ಇದು ಇವನ ಪರಮಸಂಪತ್ತು. ಇದು ಇವನ ಪರಮಲೋಕ. ಇದು ಇವನ ಪರಮಾನಂದ’ ್ಣಋಸ್ಶ್ಯ್¯ಏಷಾsಸ್ಯ ಪರಮಾ ಗತಿಃ, ಏಷಾsಸ್ಯ ಪರಮಾ ಸಂಪತ್, ಏಷೋsಸ್ಯ ಪರಮೋ ಲೋಕಃ, ಏಷೋsಸ್ಯ ಪರಮ ಆನಂದ್ಣಃಋಸ್ಬಋ¯ (ಬೃಹದಾರಣ್ಯಕ ಉ. ೪. ೩. ೨೨) ಎಂದು ಅನೇಕ ಶ್ರುತಿಗಳು ನಾವು ಸುಷುಪ್ತಿಯಲ್ಲಿ ಅನುಭವಿಸುವ ಪರಮಾನಂದವನ್ನು ಕುರಿತು ಹೇಳುತ್ತವೆ.]
೨೮. ಮಾಯೆಯ ನಿರೂಪಣ
ಅವ್ಯಕ್ತನಾಮ್ನೀ ಪರಮೇಶಶಕ್ತಿ-
ರನಾದ್ಯವಿದ್ಯಾ ತ್ರಿಗುಣಾತ್ಮಿಕಾ ಪರಾ ।
ಕಾರ್ಯಾನುಮೇಯಾ ಸುಧಿಯೈವ ಮಾಯಾ
ಯಯಾ ಜಗತ್ಸರ್ವಮಿದಂ ಪ್ರಸೂಯತೇ ॥ ೧ಂ೮ ॥
್ಣಋಸ್ಶ್ಯ್¯ಯಯ್ಣಾಋಸ್ಬಋ¯ = ಯಾವುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಇದಂ ಸರ್ವಂ ಜಗತ್ಣ್​ಋಸ್ಬಋ¯ =ಈ ಜಗತ್ತೆಲ್ಲ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಸೂಯತ್ಣೇಋಸ್ಬಋ¯ = ಉತ್ಪನ್ನವಾಗುತ್ತದೆಯೊ [ಆ] ್ಣಋಸ್ಶ್ಯ್¯ಅವ್ಯಕ್ತನಾಮ್ನ್ಣೀಋಸ್ಬಋ¯ = ಅವ್ಯಕ್ತವೆಂಬ ಹೆಸರುಳ್ಳ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮೇಶ ಶಕ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಈಶ್ವರಶಕ್ತಿಯಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅನಾದ್ಯವಿದ್ಯ್ಣಾಋಸ್ಬಋ¯ = ಅನಾದಿಕಾಲದಿಂದ ಅವಿದ್ಯೆಯಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ತ್ರಿಗುಣಾತ್ಮಿಕ್ಣಾಋಸ್ಬಋ¯ = ತ್ರಿಗುಣಸ್ವರೂಪವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪರ್ಣಾಋಸ್ಬಋ¯ = ಶ್ರೇಷ್ಠವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮಾಯ್ಣಾಋಸ್ಬಋ¯ = ಮಾಯೆಯ ್ಣಋಸ್ಶ್ಯ್¯ಸುಧಿಯಾ ಏವ್ಣಋಸ್ಬಋ¯ = ಧೀಮಂತನಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಕಾರ್ಯಾನುಮೇಯ್ಣಾಋಸ್ಬಋ¯ = ಕಾರ್ಯದಿಂದಲೇ ಅನುಮಾನಿಸಲು ಯೋಗ್ಯವಾಗಿದೆ.

೧ಂ೮. ಅವಿದ್ಯೆಗೆ ಅಥವಾ ಮಾಯೆಗೆ ಅವ್ಯಕ್ತವೆಂಬ ಹೆಸರಿದೆ.೧ ಇದೇ ಅನಾದಿ, ಪರಮೇಶ್ವರನ ಶಕ್ತಿ,೨ ಸತ್ತ್ವ-ರಜಸ್ತಮೋಗುಣಗಳನ್ನೊಳಕೊಂಡಿದೆ. ತನ್ನ ಕಾರ್ಯವರ್ಗಕ್ಕಿಂತ ಶ್ರೇಷ್ಠವಾದದ್ದು.೩ ಇದನ್ನು ಧೀಮಂತನಾದವನು ಕಾರ್ಯ ದಿಂದ ಅನುಮಾನಿಸಬೇಕು.೪ ಇದರಿಂದ ಈ ಜಗತ್ತೆಲ್ಲ ಉತ್ಪನ್ನವಾಗಿದೆ.೫

[ಇಲ್ಲಿ ಮಾಯೆಯ ಸ್ವರೂಪವನ್ನೂ ಕಾರ್ಯಗಳನ್ನೂ ಹೇಳಿದೆ.
೧ ‘ಮಹತ್ತಿಗಿಂತ ಅವ್ಯಕ್ತವು ಶ್ರೇಷ್ಠವಾದುದು’ ್ಣಋಸ್ಶ್ಯ್¯ಮಹತಃ ಪರಮವ್ಯಕ್ತಮ್ಣ್​ಋಸ್ಬಋ¯ (ಕಠ ಉ. ೧. ೩. ೧೧)
೨ ‘ಪ್ರಕೃತಿಯನ್ನು ಮಾಯೆಯೆಂದು ಅರಿತುಕೊ ; ಮಹೇಶ್ವರನು ಮಾಯೆಯ ಈಶನು’ ್ಣಋಸ್ಶ್ಯ್¯ಮಾಯಾಂ ತು ಪ್ರಕೃತಿಂ ವಿದ್ಯಾನ್ಮಾಯಿನಂ ಚ ಮಹೇಶ್ವರಮ್ಣ್​ಋಸ್ಬಋ¯ (ಶ್ವೇತಾಶ್ವತರ ಉ. ೪. ೧ಂ)
೩ ಏಕೆಂದರೆ ಕಾರಣವು ಯಾವಾಗಲೂ ತನ್ನ ಕಾರ್ಯಕ್ಕಿಂತ ಶ್ರೇಷ್ಠವಾದದ್ದು.
೪ ಏಕೆಂದರೆ ಇದನ್ನು ಪ್ರತ್ಯಕ್ಷದಿಂದ ಅರಿತುಕೊಳ್ಳಲು ಆಗುವುದಿಲ್ಲ.
೫ ‘ಅವ್ಯಕ್ತದಿಂದ ಎಲ್ಲಾ ಚರಾಚರವಸ್ತುಗಳೂ ಹುಟ್ಟುತ್ತವೆ’ ್ಣಋಸ್ಶ್ಯ್¯ಅವ್ಯಕ್ತಾದ್​ವ್ಯಕ್ತಯಃ ಸರ್ವಾಃ ಪ್ರಭವಂತ್ಣಿಋಸ್ಬಋ¯ (ಗೀತಾ. ೧೫. ೧೮ ).]
ಸನ್ನಾಪ್ಯಸನ್ನಾಪ್ಯುಭಯಾತ್ಮಿಕಾ ನೋ
ಭಿನ್ನಾಪ್ಯಭಿನ್ನಾಪ್ಯುಭಯಾತ್ಮಿಕಾ ನೋ ।
ಸಾಂಗಾಪ್ಯನಂಗಾಪ್ಯುಭಯಾತ್ಮಿಕಾ ನೋ
ಮಹಾದ್ಭುತಾsನಿರ್ವಚನೀಯರೂಪಾ ॥ ೧ಂ೯ ॥
[ಈ ಮಾಯೆಯು] ್ಣಋಸ್ಶ್ಯ್¯ಸತ್ ನ್ಣಋಸ್ಬಋ¯ = ಸತ್ ಅಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಮತ್ತು ್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್ ನ = ಅಸತ್ ಅಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಮತ್ತು ್ಣಋಸ್ಶ್ಯ್¯ಉಭಯಾತ್ಮಿಕಾ ನ ಉ್ಣಋಸ್ಬಋ¯ = ಈ ಎರಡೂ ಅಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಭಿನ್ನಾ ಅಪಿ ಅಭಿನ್ನಾ ಅಪಿ ಉಭಯಾತ್ಮಿಕಾ ನ ಉ್ಣಋಸ್ಬಋ¯ = ಭಿನ್ನವೂ ಅಲ್ಲ, ಅಭಿನ್ನವೂ ಅಲ್ಲ, ಈ ಎರಡೂ ಅಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಸಾಂಗಾ ಅಪಿ ಅನಂಗಾ ಅಪಿ ಉಭಯಾತ್ಮಿಕಾ ನ ಉ್ಣಋಸ್ಬಋ¯ = ಸಾವಯವವೂ ಅಲ್ಲ, ನಿರವಯ ವವೂ ಅಲ್ಲ, ಈ ಎರಡೂ ಅಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಮಹಾದ್ಭುತ್ಣಾಋಸ್ಬಋ¯ = ಅತ್ಯಂತ ಅದ್ಭುತವಾದುದು, ್ಣಋಸ್ಶ್ಯ್¯ಅನಿರ್ವಚನೀಯರೂಪ್ಣಾಋಸ್ಬಋ¯ = ಸ್ವರೂಪವನ್ನು ಹೇಳಲು ಅಶಕ್ಯವಾದುದು.

೧ಂ೯. (ಪರಮೇಶ್ವರನ ಶಕ್ತಿಯಾದ ಮಾಯೆಯು) ಸದ್ರೂಪವಲ್ಲ,೧ ಅಸದ್ರೂ ಪವಲ್ಲ,೨ ಸತ್​-ಅಸತ್​-ರೂಪವೂ೩. ಅಲ್ಲ ; (ಪರಮೇಶ್ವರನಿಗಿಂತ) ಭಿನ್ನವೂ೪ ಅಲ್ಲ, ಅಭಿನ್ನವೂ೫ ಅಲ್ಲ, ಭಿನ್ನ-ಅಭಿನ್ನವೂ ಅಲ್ಲ ; (ಅದು) ಸಾವಯವವೂ೬ ಅಲ್ಲ, ನಿರವಯವವೂ೭ ಅಲ್ಲ. ಸಾವಯವ-ನಿರವಯವವೂ ಅಲ್ಲ ; (ಅದು) ಅತ್ಯಂತ ಅದ್ಭುತವಾಗಿದೆ, ಅನಿರ್ವಚನೀಯರೂಪವಾಗಿದೆ.೮

[೧ ಸದ್ರೂಪವಾಗಿದ್ದರೆ ಜ್ಞಾನವು ಅದನ್ನು ನಾಶಮಾಡಲು ಶಕ್ತವಾಗುತ್ತಿರಲಿಲ್ಲ.
೨ ಅಸದ್ರೂಪವಾಗಿದ್ದರೆ ಅದರಿಂದ ಯಾವ ಕಾರ್ಯವೂ ಉಂಟಾಗುತ್ತಿರಲಿಲ್ಲ.
೩ ಒಂದೇ ವಸ್ತುವಿಗೆ ವಿರುದ್ಧಧರ್ಮಗಳಿರುವುದಿಲ್ಲ.
೪ ಭಿನ್ನವಾಗಿದ್ದರೆ ಅದ್ವೈತಕ್ಕೆ ಹಾನಿಯಾಗುತ್ತಿತ್ತು.
೫ ಅಗ್ನಿ ಮತ್ತು ಅದರ ದಾಹಕಶಕ್ತಿ ಒಂದೇ ಆಗಿರಲಾರದು. ಹಾಗೆಯೇ ಪರಮೇ ಶ್ವರನಿಗಿಂತ ಭಿನ್ನವಾಗಿದೆ.
೬ ಏಕೆಂದರೆ ಇದರಲ್ಲಿ ಯಾವ ಅವಯವವೂ ಕಂಡುಬರುತ್ತಿಲ್ಲ, ಮತ್ತು ಅವಯವ ವುಳ್ಳದ್ದಕ್ಕೆ ಜನ್ಮವಿರುತ್ತದೆ.
೭ ತ್ರಿಗುಣಗಳಿರುವುದರಿಂದ, ನಿರವಯವವಾಗಿದ್ದರೆ ಪರಿಣಾಮ ಸಂಭವಿಸು ತ್ತಿರಲಿಲ್ಲ.
೮ ಯಾವ ಲಕ್ಷಣವನ್ನು ಹೇಳಲೂ ಅಸಾಧ್ಯವಾದುದರಿಂದ ಅನಿರ್ವಚನೀಯವು.]
ಶುದ್ಧಾದ್ವಯಬ್ರಹ್ಮ-ವಿಬೋಧ-ನಾಶ್ಯಾ
ಸರ್ಪಭ್ರಮೋ ರಜ್ಜುವಿವೇಕತೋ ಯಥಾ ।
ರಜಸ್ತಮಃಸತ್ತ್ವಮಿತಿ ಪ್ರಸಿದ್ಧಾ
ಗುಣಾಸ್ತದೀಯಾಃ ಪ್ರಥಿತೈಃ ಸ್ವಕಾರ್ಯೈಃ ॥ ೧೧ಂ ॥
್ಣಋಸ್ಶ್ಯ್¯ರಜ್ಜು-ವಿವೇಕತ್ಣಃಋಸ್ಬಋ¯ = ಹಗ್ಗವೆಂಬ ವಿವೇಕದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ಪಭ್ರಮ್ಣಃಋಸ್ಬಋ¯ = ಸರ್ಪಭ್ರಾಂತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಯಥಾ [ನಶ್ಯತಿ] = ಹೇಗೆ [ನಾಶವಾಗುವುದೊ] [ತಥಾ ಮಾಯ್ಣಾಋಸ್ಬಋ¯ = ಹಾಗೆಯೇ ಮಾಯೆಯು] ಶುದ್ಧ-ಅದ್ವಯಬ್ರಹ್ಮ-ವಿಬೋಧ-ನಾಶ್ಯಾ = ಶುದ್ಧವೂ ಅದ್ವೀತಿಯವೂ ಆದ ಪರಬ್ರಹ್ಮದ ಜ್ಞಾನದಿಂದ ನಾಶವಾಗತಕ್ಕದ್ದು ; ್ಣಋಸ್ಶ್ಯ್¯ತದೀಯ್ಣಾಃಋಸ್ಬಋ¯ = ಆ ಮಾಯೆಯ ಸಂಬಂಧಿಗಳಾದ ್ಣಋಸ್ಶ್ಯ್¯ರಜಃ-ತಮಃ-ಸತ್ತ್ವಮಿತ್ಣಿಋಸ್ಬಋ¯ = ರಜಸ್ಸು ತಮಸ್ಸು ಸತ್ತ್ವ ಎಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಗುಣ್ಣಾಃಋಸ್ಬಋ¯ =ಗುಣಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಥಿತೈಃ = ವ್ಯಕ್ತವಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಕಾರ್ಯೈ್ಣಃಋಸ್ಬಋ¯ = ತಮ್ಮ ಕಾರ್ಯಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಸಿದ್ಧಾಃ =ಪ್ರಸಿದ್ಧವಾಗಿವೆ.

೧೧ಂ. ‘ಇದು ಹಾವು ’ಎಂಬ ಭ್ರಾಂತಿಯು‘ಇದು ಹಗ್ಗ’ಎಂಬ ವಿವೇಕದಿಂದ ಹೇಗೆ ನಾಶವಾಗುವುದೊ ಹಾಗೆ ಶುದ್ಧವೂ ಅದ್ವಿತೀಯವೂ ಆದ ಪರಬ್ರಹ್ಮದ ಜ್ಞಾನದಿಂದ ಈ (ಮಾಯೆಯು) ನಾಶವಾಗುವುದು. ಸತ್ತ್ವ ರಜಸ್ಸು ತಮಸ್ಸು ಎಂಬ ಆ ಮಾಯೆಯ ಸಂಬಂಧಿಗಳಾದ ಗುಣಗಳು ವ್ಯಕ್ತವಾಗಿರುವ ತಮ್ಮ ಕಾರ್ಯ ಗಳಿಂದಲೇ ಪ್ರಸಿದ್ಧವಾಗಿವೆ.೧

[೧ ಗುಣಗಳು ನಮಗೆ ಪ್ರತ್ಯಕ್ಷವಲ್ಲದುದರಿಂದ ವ್ಯಕ್ತವಾಗಿರುವ ಕಾರ್ಯಗಳ ಮೂಲಕವೇ ಅವುಗಳನ್ನು ಅರಿತುಕೊಳ್ಳಬೇಕು.]
೨೯. ವಿಕ್ಷೇಪಶಕ್ತಿ–ರಜೋಗುಣ
ವಿಕ್ಷೇಪಶಕ್ತೀ ರಜಸಃ ಕ್ರಿಯಾತ್ಮಿಕಾ
ಯತಃ ಪ್ರವೃತ್ತಿಃ ಪ್ರಸೃತಾ ಪುರಾಣೀ ।
ರಾಗಾದಯೋsಸ್ಯಾkಃ ಪ್ರಭವಂತಿ ನಿತ್ಯಂ
ದುಃಖಾದಯೋ ಯೇ ಮನಸೋ ವಿಕಾರಾಃ ॥ ೧೧೧ ॥
್ಣಋಸ್ಶ್ಯ್¯ಕ್ರಿಯಾತ್ಮಿಕಾ = ಕ್ರಿಯಾತ್ಮಕವಾದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಕ್ಷೇಪಶಕ್ತ್ಣಿಃಋಸ್ಬಋ¯ =ವಿಕ್ಷೇಪಶಕ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ರಜಸಃ = ರಜಸ್ಸಿಗೆ ಸೇರಿದ್ದು. ್ಣಋಸ್ಶ್ಯ್¯ಯತಃ = ಯಾವುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪುರಾಣ್ಣೀಋಸ್ಬಋ¯ =ಪುರಾತನವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರವೃತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಪ್ರವೃತ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಸೃತ್ಣಾಋಸ್ಬಋ¯ = ಹರಡಿಕೊಂಡಿರುವುದೋ ; ್ಣಋಸ್ಶ್ಯ್¯ಮನಸಃ = ಮನಸ್ಸಿನ ್ಣಋಸ್ಶ್ಯ್¯ರಾಗಾದಯ್ಣಃಋಸ್ಬಋ¯ = ರಾಗವೇ ಮೊದಲಾದ್ಣಋಸ್ಶ್ಯ್¯ಅ ದುಃಖಾದಯ್ಣಃಋಸ್ಬಋ¯ = ದುಃಖವೇ ಮೊದಲಾದ ್ಣಋಸ್ಶ್ಯ್¯ಯೇ ವಿಕಾರ್ಣಾಃಋಸ್ಬಋ¯ = ಯಾವ ವಿಕಾರಗಳೊ [ಅವು] ಅಸ್ಯಾಃ = ಇದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಭವಂತ್ಣಿಋಸ್ಬಋ¯ = ಉಂಟಾಗುತ್ತಿರುತ್ತವೆ.

೧೧೧. ಕ್ರಿಯಾತ್ಮಕವಾದ ವಿಕ್ಷೇಪಶಕ್ತಿಯು೧ ರಜೋಗುಣಕ್ಕೆ ಸೇರಿದ್ದು. ಇದ ರಿಂದ ಬಹುಕಾಲದಿಂದಲೂ ಸಮಸ್ತ ಪ್ರವರ್ತನೆಯು ಹರಡಿಕೊಂಡಿದೆ. ಮನಸ್ಸಿಗೆ ಸೇರಿದ ರಾಗವೇ ಮೊದಲಾದುವೂ ದುಃಖವೇ ಮೊದಲಾದುವೂ ಈ ವಿಕ್ಷೇಪ ಶಕ್ತಿಯಿಂದ ಯಾವಾಗಲೂ ಉಂಟಾಗುತ್ತಿರುವುವು.

[೧ ಆವರಣಶಕ್ತಿಯು ವಸ್ತುವಿನ ಸ್ವರೂಪವನ್ನು ಮುಚ್ಚಿದಾಗ ವಿಕ್ಷೇಪಶಕ್ತಿಯು ಒಂದು ಹೊಸ ರೂಪವನ್ನು ಸೃಷ್ಟಿಸುತ್ತದೆ.]
ಕಾಮಃ ಕ್ರೋಧೋ ಲೋಭ-ದಂಭಾದ್ಯಸೂಯಾs-
 ಹಂಕಾರೇರ್ಷ್ಯಾ-ಮತ್ಸರಾದ್ಯಾಸ್ತು ಘೋರಾಃ ।
ಧರ್ಮಾ ಏತೇ ರಾಜಸಾಃ ಪುಂಪ್ರವೃತ್ತಿ-
ಯRಸ್ಮಾದೇಷಾ ತದ್ರಜೋ ಬಂಧಹೇತುಃ ॥ ೧೧೨ ॥
್ಣಋಸ್ಶ್ಯ್¯ಕಾಮ್ಣಃಋಸ್ಬಋ¯ = ಕಾಮ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ರೋಧ್ಣಃಋಸ್ಬಋ¯ = ಕ್ರೋಧ ್ಣಋಸ್ಶ್ಯ್¯ಲೋಭ-ದಂಭಾದಿ- ಅಸೂಯ- ಅಹಂಕಾರ- ಈರ್ಷ್ಯಾ-ಮತ್ಸರಾದ್ಯಾಃ ತ್ಣುಋಸ್ಬಋ¯ = ಲೋಭ ದಂಭ ಮೊದಲಾದ ಅಸೂಯೆ, ಅಹಂಕಾರ, ಈರ್ಷ್ಯೆ, ಮತ್ಸರ ಮೊದಲಾದ ್ಣಋಸ್ಶ್ಯ್¯ಏತೇ ಘೋರಾಃ ಧರ್ಮಾಃ = ಈ ಘೋರವಾದ ಧರ್ಮ ಗಳು ್ಣಋಸ್ಶ್ಯ್¯ರಾಜಸ್ಣಾಃಋಸ್ಬಋ¯ = ರಜೋಗುಣಕ್ಕೆ ಸೇರಿದುವು. ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಮಾತ್ = ಯಾವ ರಜೋಗುಣದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಏಷ್ಣಾಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಪುಂಪ್ರವೃತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಈ ಪುರುಷಪ್ರವೃತ್ತಿಯು [ಉಂಟಾಗಿದೆಯೊ) ; ತತ್​= ಆದುದ ರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ರಜ್ಣಃಋಸ್ಬಋ¯ = ರಜಸ್ಸು ್ಣಋಸ್ಶ್ಯ್¯ಬಂಧಹೇತ್ಣುಃಋಸ್ಬಋ¯ =ಬಂಧಕ್ಕೆ ಕಾರಣ.

೧೧೨. ಕಾಮ ಕ್ರೋಧ ಲೋಭ ದಂಭ ಅಸೂಯೆ ಅಹಂಕಾರ ಈರ್ಷ್ಯೆ ಮತ್ಸರ, ಇವೇ ಮೊದಲಾದ ಘೋರಧರ್ಮಗಳು–ಯಾವುದರಿಂದ ಈ ಪುರುಷ ಪ್ರವೃ ತ್ತಿಯು ಉಂಟಾಗಿರುವುದೊ–ಆ ರಜೋಗುಣಕ್ಕೆ ಸಂಬಂಧಪಟ್ಟವು. ಆದುದ ರಿಂದ ರಜೋಗುಣವು ಬಂಧಕ್ಕೆ ಕಾರಣವಾಗಿದೆ.

ವಿRಕ್ಷೇಪಶಕ್ತೇಃ ಪ್ರವಣಸ್ಯ ಹೇತುಃ ॥ ೧೧೩ ॥
್ಣಋಸ್ಶ್ಯ್¯ಯಯ್ಣಾಋಸ್ಬಋ¯ = ಯಾವುದರಿಂದ್ಣಋಸ್ಶ್ಯ್¯ ವಸ್ತ್ಣುಋಸ್ಬಋ¯ = ವಸ್ತುವು ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ಯಥ್ಣಾಋಸ್ಬಋ¯ = ಬೇರೆ ಬಗೆಯಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಅವಭಾಸತೇ = ತೋರುತ್ತದೆಯ್ಣೊಋಸ್ಶ್ಯ್¯ ಏಷಾ = ಇದೇ ್ಣಋಸ್ಶ್ಯ್¯ತಮೋಗುಣಸ್ಯ = ತಮೋಗುಣದ ್ಣಋಸ್ಶ್ಯ್¯ಆವೃತಿಃ ನಾಮ್ಣಋಸ್ಬಋ¯ = ಆವರಣ ಎಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಶಕ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಶಕ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಸಾ ಏಷ್ಣಾಋಸ್ಬಋ¯ = ಆ ಇದ್ಣೇಋಸ್ಶ್ಯ್¯ ಪುರುಷಸ್ಣಋಸ್ಬಋ¯್ಯ = ಮನಷ್ಯನ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸೃತೇಃ = ಸಂಸಾರಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ನಿದಾನಂ = ಕಾರಣವು. [ಮತ್ತು] ವಿಕ್ಷೇಪಶಕ್ತ್ಣೇಃಋಸ್ಬಋ¯ = ವಿಕ್ಷೇಪಶಕ್ತಿಯ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರವಣಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಕಾರ್ಯೋನ್ಮುಖತ್ವಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಹೇತ್ಣುಃಋಸ್ಬಋ¯ = ಕಾರಣವು.

೧೧೩. ಯಾವುದರಿಂದ ವಸ್ತುವು ಬೇರೊಂದು ಬಗೆಯಾಗಿ ತೋರುತ್ತದೆಯೊ ಆ ಇದೇ ತಮೋಗುಣದ ಆವರಣವೆಂಬ ಪ್ರಸಿದ್ಧವಾದ ಶಕ್ತಿಯು. ಇದೇ ಮನು ಷ್ಯನ ಸಂಸಾರಕ್ಕೂ ವಿಕ್ಷೇಪಶಕ್ತಿಯ ಕಾರ್ಯೋನ್ಮುಖತ್ವಕ್ಕೂ ಕಾರಣವಾಗಿದೆ.

ಣಿಲ೬೩ಪಿ೪ಣಬಲವ್ಸ್ರಿವ್ ಪ್ ಪ್ರಜ್ಞಾವಾನಪಿ ಪಂಡಿತೋsಪಿ ಚತುರೋs-
ಪ್ಯತ್ಯಂತಸೂಕ್ಷ್ಮಾರ್ಥದೃಗ್​
ವ್ಯಾಲೀಢಸ್ತಮಸಾ ನ ವೇತ್ತಿ ಬಹುಧಾ
ಸಂಬೋಧಿತೋsಪಿ ಸ್ಫುಟಮ್ ।
ಭ್ರಾಂತ್ಯಾssರೋಪಿತಮೇವ ಸಾಧು ಕಲಯ-
ತ್ಯಾಲಂಬತೇ ತದ್ಗುಣಾನ್​
ಹಂತಾಸೌ ಪ್ರಬಲಾ ದುರಂತತಮಸಃ
ಶಕ್ತಿರ್ಮಹತ್ಯಾವೃತಿಃ ॥ ೧೧೪ ॥

್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಜ್ಞಾವಾನ್ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಮೇಧಾವಿಯಾಗಿದ್ದರ್ಣೂಋಸ್ಶ್ಯ್¯ ಪಂಡಿತಃ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಶಾಸ್ತ್ರಜ್ಞನಾಗಿದ್ದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಚತುರಃ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಚತುರನಾಗಿದ್ದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಅತ್ಯಂತ-ಸೂಕ್ಷ್ಮ-ಅರ್ಥದೃಕ್ಣ್​ಋಸ್ಬಋ¯ = ಅತ್ಯಂತ ಸೂಕ್ಷ್ಮ ವಾದ ವಿಷಯಗಳನ್ನು ನೋಡಿರುವವನಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ತಮಸಾ = ತಮಸ್ಸಿನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವ್ಯಾಲೀಢಃ = ಮುಚ್ಚಲ್ಪಟ್ಟವನ್ಣುಋಸ್ಶ್ಯ್¯ ಬಹುಧಾ =ಅನೇಕ ವಿಧವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಫುಟಂ ಸಂಬೋಧಿತ್ಣಃಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಅಪಿ = ಸ್ಫುಟ ವಾಗಿ ಉಪದೇಶಿಸಲ್ಪಟ್ಟರೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ವೇತ್ತಿ = ಅರಿಯುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಭ್ರಾಂತ್ಯಾ = ಭ್ರಾಂತಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಆರೋಪಿತಮ್ ಏವ = ಆರೋಪಿಸಲ್ಪಟ್ಟಿರುವುದನ್ನೇ ್ಣಋಸ್ಶ್ಯ್¯ಸಾಧು ಕಲಯತ್ಣಿಋಸ್ಬಋ¯ = ಸರಿಯೆಂದು ಎಣಿಸುತ್ತಾನೆ. ್ಣಋಸ್ಶ್ಯ್¯ತದ್ಗುಣಾನ್ಣ್​ಋಸ್ಬಋ¯ = ಅದರ ಕಾರ್ಯಗಳನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಆಲಂಬತೇ = ಆಶ್ರಯಿಸುತ್ತಾನೆ; ್ಣಋಸ್ಶ್ಯ್¯ಹಂತ = ಆಶ್ಚರ್ಯ ! ್ಣಋಸ್ಶ್ಯ್¯ದುರಂತತಮಸಃ = ನಾಶಮಾಡಲು ದುಃಸಾಧ್ಯವಾದ ತಮಸ್ಸಿನ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಣೌಋಸ್ಬಋ¯ = ಈ ್ಣಋಸ್ಶ್ಯ್¯ಮಹತ್ಣೀಋಸ್ಬಋ¯ = ಮಹತ್ತಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆವೃತಿಃ ಶಕ್ತಿಃ = ಆವರಣ ಶಕ್ತಿಯ್ಣುಋಸ್ಶ್ಯ್¯ ಪ್ರಬಲ್ಣಾಋಸ್ಬಋ¯ = ಪ್ರಬಲವಾದುದು.

೧೧೪. ಮನುಷ್ಯನು ಮೇಧಾವಿಯಾಗಿದ್ದರೂ೧ ಶಾಸ್ತ್ರಜ್ಞನಾಗಿದ್ದರೂ೨ (ಲೌಕಿಕ- ವ್ಯವಹಾರದಲ್ಲಿ) ಚತುರನಾಗಿದ್ದರೂ ಅತ್ಯಂತ ಸೂಕ್ಷ್ಮವಾದ ವಿಷಯಗಳನ್ನು೩ ಅರಿತಿರುವವನಾಗಿದ್ದರೂ ತಮಸ್ಸಿನಿಂದ ಮುಚ್ಚಲ್ಪಟ್ಟರೆ,೪ ಅನೇಕ ವಿಧದಿಂದ ಸ್ಫುಟವಾಗಿ ಬೋಧಿಸಲ್ಪಟ್ಟರೂ (ತತ್ತ್ವವನ್ನು) ಅರಿಯಲಾರನು ;೫ ಭ್ರಾಂತಿ ಯಿಂದ ತಾನು ಆರೋಪಿಸಿರುವುದನ್ನೇ೬ ಸರಿಯೆಂದು ಎಣಿಸುತ್ತಾನೆ. ಅದರ ಕಾರ್ಯಗಳನ್ನೇ ಆಶ್ರಯಿಸುತ್ತಾನೆ. ಆಶ್ಚರ್ಯ ! ದುರಂತವಾದ ತಮಸ್ಸಿನ ಈ ಮಹಾ ಆವರಣಶಕ್ತಿಯು ಎಷ್ಟು ಪ್ರಬಲವಾದುದು !

[೧ ‘ಮೇಧಾಶಕ್ತಿಯಿಂದಲೂ ಬಹುಶ್ರವಣದಿಂದಲೂ ಲಭ್ಯನಲ್ಲ’ ನ ಮೇಧಯಾ ನ ಬಹುನಾ ಶ್ರುತೇನ್ಣಋಸ್ಬಋ¯ (ಕಠ ಉ ೧. ೨. ೨೩)

೩ ಮನಸ್ಸು ಬುದ್ಧಿ ಮೊದಲಾದುವುಗಳನ್ನು.
೪ ಆವರಣಶಕ್ತಿಯಿಂದ ಆಚ್ಛಾದಿತನಾದರೆ.
೫ ದೃಷ್ಟಿಯು ಪಟುವಾಗಿದ್ದರೂ ಕಗ್ಗತ್ತಲಿನಲ್ಲಿ ಏನನ್ನು ನೋಡಲೂ ಸಾಧ್ಯವಿಲ್ಲ.
೬ ಸ್ಥೂಲವಾದ ಶರೀರವೇ ಮೊದಲಾದುವುಗಳನ್ನು.]
ಅಭಾವನಾ ವಾ ವಿಪರೀತಭಾವನಾss-
ಸಂಭಾವನಾ ವಿಪ್ರತಿಪತ್ತಿರಸ್ಯಾಃ ।
ಸಂಸರ್ಗಯುಕ್ತಂ ನ ವಿಮುಂಚತಿ ಧ್ರುವಂ
ವಿಕ್ಷೇಪಶಕ್ತಿಃ ಕ್ಷಪಯತ್ಯಜಸ್ರಮ್ ॥ ೧೧೫ ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಭಾವನಾ = ಅಭಾವನೆಯು ್ಣಋಸ್ಶ್ಯ್¯ವಿಪರೀತಭಾವನಾ ವಾ = ವಿಪರೀತಭಾವನೆಯು ್ಣಋಸ್ಶ್ಯ್¯ಅಸಂಭಾವನಾ= ಅಸಂಭಾವನೆಯ್ಣುಋಸ್ಶ್ಯ್¯ ವಿಪ್ರತಿಪತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಸಂದೇಹವು [ಇವು) ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಯಾಃ = ಇದರ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸರ್ಗಯುಕ್ತ್ಣಂಋಸ್ಬಋ¯ = ಸಂಬಂಧದಿಂದ ಕೂಡಿರುವವನನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಧ್ರುವಂ ನ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ವಿಮುಂಚತ್ಣಿಋಸ್ಬಋ¯ = ಖಂಡಿತವಾಗಿ ಬಿಡುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ವಿಕ್ಷೇಪಶಕ್ತಿಃ =ವಿಕ್ಷೇಪಶಕ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಅಜಸ್ರ್ಣಂಋಸ್ಬಋ¯ = ಯಾವಾ ಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಷಪಯತಿ = ಬಾಧಿಸುತ್ತದೆ.

೧೧೫. ಅಭಾವನೆ೧ ವಿಪರೀತಭಾವನೆ೨ ಅಸಂಭಾವನೆ೩ ಸಂದೇಹ೪–(ಇವು) ಆವರಣಶಕ್ತಿಯ ಸಂಬಂಧದಿಂದ ಕೂಡಿರುವ ಮನುಷ್ಯನನ್ನು ಖಂಡಿತವಾಗಿ ಬಿಡುವು ದಿಲ್ಲ ; (ಅನಂತರ) ಅವನನ್ನು ವಿಕ್ಷೇಪಶಕ್ತಿಯು ಯಾವಾಗಲೂ ಬಾಧಿಸುವುದು.

[ ೧ ವಸ್ತುವಿನ ಸ್ವರೂಪವನ್ನು ತಿಳಿಯದಿರುವುದು.

೪. ವಸ್ತುವು ಆ ರೀತಿ ಇರುವುದೊ ಇಲ್ಲವೊ ಎಂಬುದು.]

ಅಜ್ಞಾನಂ =ಅಜ್ಞಾನವು ಆಲಸೃಯ-ಜಡತ್ವ-ನಿದ್ರಾ-ಪ್ರಮಾದ-ಮೂಢತ್ವ - ಮುಖಾಃ = ಆಲಸ್ಯ ಜಡತ್ವ ನಿದ್ರೆ ಪ್ರಮಾದ ಮೂಢತ್ವ ಇವೇ ಮೊದಲಾದುವು ತಮೋಗುಣಾಃ = ತಮಸ್ಸಿನಗುಣಗಳು; ಏತೈಃ = ಇವುಗಳಿಂದ ಪ್ರಯುಕ್ತಃ = ಕೂಡಿರುವನು ನಿದ್ರಾಲುವತ್ = ನಿದ್ರಿಸುವಂತೆ ಕಿಂಚಿತ್ = ಏನನ್ನೂ ನ ಹಿ ವೇತ್ತಿ = ಅರಿಯುವುದೇ ಇಲ್ಲ, ಸ್ತಂಭವತ್ ಏವ = ಕಂಬಂದತೆಯೇ ತಿಷ್ಠತಿ = ಇರುತ್ತಾನೆ.

೧೧೬. ಅಜ್ಞಾನ ಆಲಸ್ಯ ಜಡತ್ವ ನಿದ್ರೆ ಪ್ರಮಾದ ಮೂಢತ್ವ ಮೊದಲಾದವು ತಮಸ್ಸಿನ ಗುಣಗಳು; ಇವುಗಳಿಂದ ಕೂಡಿರುವವನು ನಿದ್ರಿಸುವವನಂತೆ ಏನನ್ನೂ ಅರಿಯುವುದಿಲ್ಲ, ಕಂಬಂದತೆಯೇ ಇರುತ್ತಾನೆ.೧.

೩೧. ಸತ್ತ್ವಗುಣ

ಪ್ರಕಾಶಯತ್ಯರ್ಕಇವಾಖಿಲಂ ಜಡಮ್ ॥ ೧೧೭ ॥

ಸತ್ತ್ವಂ = ಸತ್ತ್ವವು ಜಲವತ್ = ನೀರಿನ ಹಾಗೆ ವಿಶುದ್ಧಂ = ನಿರ್ಮಲವಾದುದು; ತಥಾ ಅಪಿ = ಆದರೂ ತಾಭ್ಯಾಂ = ಆ ರಜಸ್ತಮೋಗುಣಗಳೊಡನೆ ಮಿಲಿತ್ವಾ = ಬೆರೆತು ಸರಣಾಯ= ಸಂಸಾರಕ್ಕೆ ಕಲ್ಪತೇ = ಕಾರಣವಾಗಿರುತ್ತದೆ; ಯತ್ರ = ಯಾವ ಸತ್ತ್ವಗುಣದಲ್ಲಿ ಆತ್ಮಬಿಂಬಃ = ಆತ್ಮಸ್ವರೂಪವು ಪ್ರತಿಬಿಂಬಿತಃ = ಪ್ರತಿಬಿಂಬಿಸಿ ಆರ್ಕಃ ಇವ = ಸೂರ್ಯನಂತೆ ಅಖಿಲಂ = ಅಖಿಲಂ ಅಖಿಲವಾದ ಜಡಂ = ಜಡಜಗತ್ತನ್ನು ಪ್ರಕಾಶಯತಿ = ಪ್ರಕಾಶಿಸುತ್ತದೆಯೊ.

೧೧೭. ಸತ್ತ್ವಗುಣವು ನೀರಿನಂತೆ ನಿರ್ಮಲವಾದುದು. ಆದರೂ ರಜದಸ್ತಮೋಗುಣಗಳೊಂದಿಗೆ ಬೆರೆತು ಸಂಸಾರಕ್ಕೆ ಕಾರಣವಾಗಿರುತ್ತದಿ. ಸತ್ತ್ವಗುಣದಲ್ಲಿ ಆತ್ಮಸ್ವರೂಪವು ಪ್ರತಿಬಿಂಬಿಸಿ ಸೂರ್ಯನಂತೆ ಜಡಜಗತ್ತೆಲ್ಲವನ್ನೂ ಪ್ರಕಾಶಿಸುತ್ತದೆ.೧

[೧ ‘ ಸತ್ತ್ವಗುಣದಿಂದ ಜ್ಞಾನವುಂಟಾಗುತ್ತದೆ.’ ಸತ್ತ್ವಾತ್ ಸಂಜಾಯತೇ ಜ್ಞಾನಮ್ (ಗೀತಾ ೧೫, ೧೭) ಸತ್ತ್ವಗುಣವಿಲ್ಲದಿದ್ದರೆ ಆತ್ಮನು ಪ್ರತಿಬಿಂಬಿಸುವುದೂ ಅಲ್ಲ, ಮೋಕ್ಷಕ್ಕೆ ಕಾರಣವಾದ ಜ್ಞಾನವೂ ಉಂಟಾಗುವುದಿಲ್ಲ. ನೀರನಂತೆ ಸ್ವಚ್ಛವಾಗಿದ್ದರೂ ಸತ್ತ್ವವೂ ಇತರ ಗುಣಗಳೊಡನೆ ಕೂಡಿಕೊಂಡರೆ ಸಂಸಾರವನ್ನುಂಟುಮಾಡುವುದು.]
