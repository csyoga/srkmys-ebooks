್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಇದಂ ನ = ಇದಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಅದಃ ನ = ಅದಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಆದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಉಭಯ್ಣೋಃಋಸ್ಬಋ¯ = ಇವೆರಡರ ್ಣಋಸ್ಶ್ಯ್¯ಅವಭಾಸಕ್ಣಂಋಸ್ಬಋ¯ = ಪ್ರಕಾಶಕವಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಪರ್ಣಂಋಸ್ಬಋ¯ = ಶ್ರೇಷ್ಠವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಶುದ್ಧ್ಣಂಋಸ್ಬಋ¯ = ಶುದ್ಧವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಬಾಹ್ಯಾಭ್ಯಂತರ-ಶೂನ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಒಳಗು ಹೊರಗು ಎಂಬ ಭೇದವಿಲ್ಲದ ್ಣಋಸ್ಶ್ಯ್¯ಪೂರ್ಣ್ಣಂಋಸ್ಬಋ¯ =ಪೂರ್ಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಿತೀಯಂ = ಅದ್ವಿತೀಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ ಏವ್ಣಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮವೇ.

491. ನಾನು ಇದೂ1 ಅಲ್ಲ, ಅದೂ2 ಅಲ್ಲ ; ಆದರೆ ಇವರೆಡನ್ನೂ ಪ್ರಕಾಶಿ ಸುವ, ಶ್ರೇಷ್ಠವೂ ಶುದ್ಧವೂ ಆದ, ಒಳಗು ಹೊರಗು ಎಂಬ ಭೇದವಿಲ್ಲದ, ಪೂರ್ಣವಾದ ಅದ್ವಿತೀಯವಾದ ಬ್ರಹ್ಮವೇ ನಾನು.

[1 ಅಪರೋಕ್ಷಜ್ಞಾನಕ್ಕೆ ವಿಷಯವಾದುದು.
2ಪರೋಕ್ಷಜ್ಞಾನಕ್ಕೆ ವಿಷಯವಾದುದು.]
ನಿರುಪಮಮನಾದಿತತ್ತ್ವಂ ತ್ವಮಹಮಿದಮದ ಇತಿ ಕಲ್ಪನಾದೂರಮ್ ।
ನಿತ್ಯಾನಂದೈಕರಸಂ ಸತ್ಯಂ ಬ್ರಹ್ಮಾದ್ವಿತೀಯಮೇವಾಹಮ್ ॥ 492 ॥
್ಣಋಸ್ಶ್ಯ್¯ನಿರುಪಮ್ಣಂಋಸ್ಬಋ¯ = ಉಪಮಾನವಿಲ್ಲದ ್ಣಋಸ್ಶ್ಯ್¯ಅನಾದಿತತ್ತ್ವ್ಣಂಋಸ್ಬಋ¯ = ಅನಾದಿತತ್ತ್ವವಾದ ್ಣಋಸ್ಶ್ಯ್¯ತ್ವಮ್ ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನೀನು ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಇದಮ್ ಅದ್ಣಃಋಸ್ಬಋ¯ = ಇದು ಅದು ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಎಂಬ್ಣಋಸ್ಶ್ಯ್¯ ಕಲ್ಪನಾದೂರ್ಣಂಋಸ್ಬಋ¯ = ಕಲ್ಪನೆಗಳಿಗೆ ದೂರವಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯಾನಂದ-ಏಕರಸ್ಣಂಋಸ್ಬಋ¯ = ನಿತ್ಯಾನಂದೈಕರಸವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸತ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಸತ್ಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಿತೀಯ್ಣಂಋಸ್ಬಋ¯ = ಅದ್ವಿತೀಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ ಏವ = ಬ್ರಹ್ಮವೇ ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು.

492. ಅಸದೃಶವಾದ, ‘ನೀನು-ನಾನು, ಇದು-ಅದು’ ಎಂಬ ಕಲ್ಪನೆಗಳಿಗೆ ದೂರವಾಗಿರುವ, ನಿತ್ಯಾನಂದೈಕರಸವಾದ ಸತ್ಯವಾದ ಅದ್ವಿತೀಯವಾದ ಬ್ರಹ್ಮವೇ ನಾನು.

ನಾರಾಯಣೋsಹಂ ನರಕಾಂತಕೋsಹಂ
ಪುರಾಂತಕೋsಹಂ ಪುರಷೋsಹಮೀಶಃ ।
ಅಖಂಡಬೋಧೋsಹಮಶೇಷಸಾಕ್ಷೀ
ನಿರೀಶ್ವರೋsಹಂ ನಿರಹಂ ಚ ನಿರ್ಮಮಃ ॥ 493 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ನಾರಾಯಣ್ಣಋಸ್ಬಋ¯ಃ = ನಾರಾಯಣನು ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ ನರಕಾಂತಕಃ = ್ಣಋಸ್ಬಋ¯ನರ ಕಾಂತಕನ್ಣಋಸ್ಶ್ಯ್¯ು ಅಹಂ ಪುರಾಂತಕಃ = ್ಣಋಸ್ಬಋ¯ತ್ರಿಪುರಾಸುರಸಂಹಾರಿಯ್ಣುಋಸ್ಶ್ಯ್¯ ಅಹಂ ಪುರುಷ್ಣಋಸ್ಬಋ¯ಅಃ = ಹೃದಯ ದಲ್ಲಿರುವವನು ಈಶಃ = ಈಶ್ವರನು ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಅಖಂಡಬೋಧ್ಣಃಋಸ್ಬಋ¯ = ಅಖಂಡಜ್ಞಾನ ಸ್ವರೂಪನು ್ಣಋಸ್ಶ್ಯ್¯ಅಶೇಷಸಾಕ್ಷ್ಣೀಋಸ್ಬಋ¯ =ಸರ್ವಸಾಕ್ಷಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ ನಿರೀಶ್ವರ್ಣಃಋಸ್ಬಋ¯ = ತನಗೆ ಬೇರೊಬ್ಬ ಈಶ್ವರನಿಲ್ಲದವನು ್ಣಋಸ್ಶ್ಯ್¯ನಿರಹ್ಣಂಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಚ್ಣಋಸ್ಬಋ¯ = ಅಹಂಕಾರಶೂನ್ಯನು ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ಮಮ್ಣಋಸ್ಬಋ¯ಃ = ಮಮಕಾರ ಶೂನ್ಯನು.

493. ನಾನೇ ನಾರಾಯಣನು, ನರಕಾಂತಕನು,1 ತ್ರಿಪುರಾಸುರಸಂಹಾರಿಯಾದ ಶಿವನು ; ನಾನೇ ಹೃದಯಗುಹೆಯಲ್ಲಿರುವ ಅಂತರ್ಯಾಮಿಯು ಮತ್ತು ಈಶ್ವರನು ; ನಾನೇ ಅಖಂಡಜ್ಞಾನಸ್ವರೂಪನು ಮತ್ತು ಸರ್ವಸಾಕ್ಷಿಯು ; ನನಗೆ ಮತ್ತೊಬ್ಬ ಈಶ್ವರನಿಲ್ಲ ; ನನಗೆ ಅಹಂಕಾರ-ಮಮಕಾರಗಳಿಲ್ಲ.

[1 ಶ್ರೀಕೃಷ್ಣನು ; ಅಥವಾ ಭಕ್ತರನ್ನು ನರಕದುಃಖದಿಂದ ನಿವಾರಿಸುವವನು.]
ಸರ್ವೇಷು ಭೂತೇಷ್ವಹಮೇವ ಸಂಸ್ಥಿತೋ
ಜ್ಞಾನಾತ್ಮನಾsಂತರ್ಬಹಿರಾಶ್ರಯಃ ಸನ್ ।
ಭೋಕ್ತಾ ಚ ಭೋಗ್ಯಂ ಸ್ವಯಮೇವ ಸರ್ವಂ
ಯದ್ಯತ್ ಪೃಥಗ್ದೃಷ್ಟಮಿದಂತಯಾ ಪುರಾ ॥ 494 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಏವ್ಣಋಸ್ಬಋ¯ಅ = ನಾನೇ ್ಣಋಸ್ಶ್ಯ್¯ಜ್ಞಾನಾತ್ಮನಾ = ್ಣಋಸ್ಬಋ¯ಜ್ಞಾನಸ್ವರೂಪದಿಂದ್ಣಋಸ್ಶ್ಯ್¯ಅ ಅಂತಃ = ್ಣಋಸ್ಬಋ¯ಒಳಗ್ಣೂಋಸ್ಶ್ಯ್¯ ಬಹಿಃ = ಹೊರಗೂ ್ಣಋಸ್ಶ್ಯ್¯ಆಶ್ರಯಃ ಸನ್ಣ್ಋಸ್ಬಋ¯ = ಆಶ್ರಯನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವೇಷು ಭೂತೇಷ್ಣುಋಸ್ಬಋ¯ = ಸಮಸ್ತ ಭೂತಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸ್ಥಿತ್ಣಋಸ್ಬಋ¯ಃ = ಇರುತ್ತೇನೆ ; [ನಾನೇ] ್ಣಋಸ್ಶ್ಯ್¯ಭೋಕ್ತಾ ಚ = ಭೋಕ್ತೃವೂ ್ಣಋಸ್ಶ್ಯ್¯ಭೋಗ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಭೋಗ್ಯವೂ [ಆಗಿದ್ದೇನೆ] ; ್ಣಋಸ್ಶ್ಯ್¯ಪುರ್ಣಾಋಸ್ಬಋ¯ = ಮೊದಲ್ಣುಋಸ್ಶ್ಯ್¯ ಯತ್ ಯತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವಯಾವುದು ್ಣಋಸ್ಶ್ಯ್¯ಇದಂತಯ್ಣಾಋಸ್ಬಋ¯ = ‘ಇದು’ಎಂಬ ಭಾವನೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪೃಥಕ್ಣ್ಋಸ್ಬಋ¯ = ಬೇರೆಯಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ದೃಷ್ಟಂ = ನೋಡಲ್ಪಟ್ಟಿತ್ತೊ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವ್ಣಂಋಸ್ಬಋ¯ = ಅದೆಲ್ಲವೂ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಮ್ ಏವ = ನಾನೇ.

494. ನಾನೇ ಜ್ಞಾನಸ್ವರೂಪದಿಂದ ಒಳಗೂ ಹೊರಗೂ ಆಶ್ರಯನಾಗಿ ಸರ್ವ ಭೂತಗಳಲ್ಲಿಯೂ ಇದ್ದೇನೆ. ಭೋಕ್ತೃವೂ ಭೋಗ್ಯವೂ ನಾನೇ. ಮೊದಲು ಯಾವಯಾವುದು ‘ಇದು’ ಎಂಬ ಭಾವನೆಯಿಂದ ಬೇರೆಬೇರೆಯಾಗಿ ಕಾಣು ತ್ತಿತ್ತೊ ಅದೆಲ್ಲವೂ ನಾನೇ ಆಗಿದ್ದೇನೆ.

ಮಯ್ಯಖಂಡ-ಸುಖಾಂಭೋಧೌ ಬಹುಧಾ ವಿಶ್ವವೀಚಯಃ ।
ಉತ್ಪದ್ಯಂತೇ ವಿಲೀಯಂತೇ ಮಾಯಾಮಾರುತ-ವಿಭ್ರಮಾತ್ ॥ 495 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಖಂಡ-ಸುಖ-ಅಂಭೋಧ್ಣೌಋಸ್ಬಋ¯ = ಅಖಂಡಾನಂದಸಮುದ್ರಸ್ವರೂಪನಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಮಯಿ = ನನ್ನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ವಿಶ್ವವೀಚಯ್ಣಃಋಸ್ಬಋ¯ = ವಿಶ್ವಗಳೆಂಬ ತರಂಗಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಮಾಯಾ- ಮಾರುತ- ವಿಭ್ರಮಾತ್ಣಋಸ್ಬಋ¯್ = ಮಾಯೆಯೆಂಬ ಸುಂಟರಗಾಳಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಬಹುಧ್ಣಾಋಸ್ಬಋ¯ = ಅನೇಕವಿಧವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಉತ್ಪದ್ಯಂತ್ಣೇಋಸ್ಬಋ¯ = ಹುಟ್ಟುತ್ತವೆ, ್ಣಋಸ್ಶ್ಯ್¯ವಿಲೀಯಂತ್ಣೇಋಸ್ಬಋ¯ = ನಾಶವಾಗುತ್ತವೆ.

495. ಅಖಂಡಾನಂದ-ಸಮುದ್ರ-ಸ್ವರೂಪನಾಗಿರುವ ನನ್ನಲ್ಲಿ ವಿಶ್ವಗಳೆಂಬ ತರಂಗಗಳು ಮಾಯೆಯೆಂಬ ಸುಂಟರಗಾಳಿಯಿಂದ ಅನೇಕ ವಿಧವಾಗಿ ಹುಟ್ಟು ತ್ತವೆ, ನಾಶವಾಗುತ್ತವೆ.

ಸ್ಥೂಲಾದಿಭಾವಾ ಮಯಿ ಕಲ್ಪಿತಾ ಭ್ರಮಾ-
ದಾರೋಪಿತಾನುಸ್ಫುರಣೇನ ಲೋಕೈಃ ।
ಕಾಲೇ ಯಥಾ ಕಲ್ಪಕ-ವತ್ಸರಾಯ-
ನರ್ತ್ವಾದಯೋ ನಿಷ್ಕಲ-ನಿರ್ವಿಕಲ್ಪೇ ॥ 496 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯಥಾ = ಹೇಗೆ ್ಣಋಸ್ಶ್ಯ್¯ನಿಷ್ಕಲ-ನಿರ್ವಿಕಲ್ಪ್ಣೇಋಸ್ಬಋ¯ = ನಿರವಯವೂ ನಿರ್ವಿಶೇಷವೂ ಆದ್ಣಋಸ್ಶ್ಯ್¯ ಕಾಲ್ಣೇಋಸ್ಬಋ¯ = ಕಾಲದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಕಲ್ಪಕ-ವತ್ಸರ-ಅಯನ-ಪುತು-ಆದಯ್ಣಃಋಸ್ಬಋ¯ = ಕಲ್ಪ ಸಂವತ್ಸರ ಅಯನ ಪುತು ಮೊದಲಾದುವು [ಕಲ್ಪಿತ್ಣಾಋಸ್ಬಋ¯ಃ = ಕಲ್ಪಿತವಾಗಿಯೊ ್ಣಋಸ್ಶ್ಯ್¯ತಥ್ಣಾಋಸ್ಬಋ¯ = ಹಾಗೆ] ್ಣಋಸ್ಶ್ಯ್¯ಲೋಕ್ಣೈಃಋಸ್ಬಋ¯ = ಜನರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಥೂಲಾದಿಭಾವ್ಣಾಃಋಸ್ಬಋ¯ = ಸ್ಥೂಲವೇ ಮೊದಲಾದ ಭಾವಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಆರೋಪಿತ-ಅನುಸ್ಫುರಣೇನ್ಣಋಸ್ಬಋ¯ಅ = ಆರೋಪಿತವಾಗಿ ತೋರುತ್ತಿರುವುದರ ಮೂಲಕ ್ಣಋಸ್ಶ್ಯ್¯ಮಯ್ಣಿಋಸ್ಬಋ¯ = ನನ್ನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಭ್ರಮಾತ್ಣ್ಋಸ್ಬಋ¯ = ಭ್ರಮೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಕಲ್ಪಿತ್ಣಾಋಸ್ಬಋ¯ಃ = ಕಲ್ಪಿಸಲ್ಪಟ್ಟಿವೆ.

496. ನಿರವಯವವೂ ನಿರ್ವಿಶೇಷವೂ ಆದ ಕಾಲದಲ್ಲಿ ಕಲ್ಪ ಸಂವತ್ಸರ ಅಯನ ಪುತು ಮೊದಲಾದುವು ಹೇಗೆ ಕಲ್ಪಿತವಾಗಿವೆಯೊ ಹಾಗೆಯೇ ನನ್ನಲ್ಲಿ ಸ್ಥೂಲ-ಸೂಕ್ಷ್ಮಾದಿ-ಭಾವಗಳನ್ನು–ಆರೋಪಿತವಾಗಿ ತೋರುತ್ತಿರುವ ವಸ್ತುಗಳ ಮೂಲಕ –ಜನರು ಭ್ರಾಂತಿಯಿಂದ ಕಲ್ಪಿಸುತ್ತಾರೆ.

ಆರೋಪಿತಂ ನಾಶ್ರಯದೂಷಕಂ ಭವೇತ್
ಕದಾsಪಿ ಮೂಢೈರ್ಮತಿದೋಷದೂಷಿತೈಃ ।
ನಾರ್ದ್ರೀಕರೋತ್ಯೂಷರ-ಭೂಮಿಭಾಗಂ
ಮರೀಚಿಕಾ-ವಾರಿ-ಮಹಾಪ್ರವಾಹಃ ॥ 497 ॥
್ಣಋಸ್ಶ್ಯ್¯ಮತಿ-ದೋಷ-ದೂಷಿತ್ಣೈಃಋಸ್ಬಋ¯ = ಬುದ್ಧಿಯ ದೋಷದಿಂದ ಕಲುಷಿತರಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಮೂಢ್ಣೈಋಸ್ಬಋ¯ಃ = ಮೂಢರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಆರೋಪಿತ್ಣಂಋಸ್ಬಋ¯ = ಆರೋಪಿತವಾದದ್ದು ್ಣಋಸ್ಶ್ಯ್¯ಕದಾ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಎಂದಿಗ್ಣೂಋಸ್ಶ್ಯ್¯ ಆಶ್ರಯ-ದೂಷಕಂ ನ ಭವೇತ್ಣ್ಋಸ್ಬಋ¯ = ಅಧಿಷ್ಠಾನವಾದುದಕ್ಕೆ ದೋಷವನ್ನುಂಟುಮಾಡು ವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಮರೀಚಿಕಾ-ವಾರಿ-ಮಹಾಪ್ರವಾಹ್ಣಋಸ್ಬಋ¯ಃ = ಬಿಸಿಲ್ಗುದುರೆಯ ನೀರಿನ ಪ್ರವಾಹವು ್ಣಋಸ್ಶ್ಯ್¯ಊಷರ--ಭೂಮಿ-ಭಾಗ್ಣಂಋಸ್ಬಋ¯ = ಮರುಭೂಮಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನ ಆರ್ದ್ರೀಕರೋತ್ಣಿಋಸ್ಬಋ¯ = ನೆನೆಯಿಸ ಲಾರದು.

497. ಬುದ್ಧಿಯ ದೋಷದಿಂದ ಕಲುಷಿತರಾಗಿರುವ ಮೂಢರಿಂದ ಆರೋ ಪಿತವಾದದ್ದು ಅಧಿಷ್ಠಾನವಾದ ವಸ್ತುವಿಗೆ ಎಂದಿಗೂ ದೋಷವನ್ನುಂಟುಮಾಡು ವುದಿಲ್ಲ ; ಬಿಸಿಲ್ಗುದುರೆಯ ನೀರಿನ ಪ್ರವಾಹವು ಮರುಭೂಮಿಯನ್ನು ಎಂದಿಗೂ ನೆನೆಯಿಸಲಾರದು.

[‘ಹೀಗಿರುವಾಗ ಯಾವುದರಲ್ಲಿ ಯಾವುದನ್ನು ಅಧ್ಯಾಸಮಾಡಿದೆಯೊ ಅದರಿಂದ ಉಂಟಾದ ಗುಣವಾಗಲಿ ದೋಷವಾಗಲಿ ಅದಕ್ಕೆ (ಎಂದರೆ ಆಶ್ರಯಕ್ಕೆ) ಸ್ವಲ್ಪವೂ ಸಂಬಂಧಿಸಿರುವುದಿಲ್ಲ’ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ರ ಏವಂ ಸತಿ ಯತ್ರ ಯದಧ್ಯಾಸಃ ತತ್ಕೃತೇನ ಗುಣೇನ ದೋಷೇಣ ವಾ ಅಣುಮಾತ್ರೇಣಾಪಿ ಸ ನ ಸಂಬಧ್ಯತ್ಣೇಋಸ್ಬಋ¯ । (-ಬ್ರಹ್ಮಸೂತ್ರ-ಅಧ್ಯಾಸ mಭಾಷ್ಯ)]
ಆಕಾಶವಲ್ಲೇಪ-ವಿದೂರಗೋsಹಮ್
ಆದಿತ್ಯವದ್ಭಾಸ್ಯ-ವಿಲಕ್ಷಣೋsಹಮ್ ।
ಅಹಾರ್ಯವನ್ನಿತ್ಯವಿನಿಶ್ಚಲೋsಹಮ್
ಅಂಭೋಧಿವತ್ಪಾರ-ವಿವರ್ಜಿತೋsಹಮ್ ॥ 498 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಆಕಾಶವತ್ಣ್ಋಸ್ಬಋ¯ = ಆಕಾಶದಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಲೇಪ-ವಿದೂರಗಃ = ನಿರ್ಲಿಪ್ತನು; ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಆದಿತ್ಯವತ್ = ್ಣಋಸ್ಬಋ¯ಸೂರ್ಯನಂತ್ಣಋಸ್ಶ್ಯ್¯ೆ ಭಾಸ್ಯ-ವಿಲಕ್ಷಣಃ = ್ಣಋಸ್ಬಋ¯ಬೆಳಗಲ್ಪಡುವ್ಣಋಸ್ಶ್ಯ್¯ಅ ್ಣಋಸ್ಬಋ¯ವಸ್ತು ಗಳಿಗಿಂತ್ಣಋಸ್ಶ್ಯ್¯ಅ್ಣಋಸ್ಬಋ¯ ಭಿನ್ನನು ; ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಅಹಾರ್ಯವತ್ಣಋಸ್ಬಋ¯್ = ಪರ್ವತದಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ-ವಿನಿಶ್ಚಲಃ = ಯಾವಾಗಲೂ ಅಚಲನಾದವನು ; ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಅಂಭೋಧಿವತ್ಣ್ಋಸ್ಬಋ¯ = ಸಮುದ್ರದಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಪಾರ-ವಿವರ್ಜಿತ್ಣಃಋಸ್ಬಋ¯ = ಪಾರವಿಲ್ಲದವನು.

498. ನಾನು ಆಕಾಶದಂತೆ ನಿರ್ಲಿಪ್ತನು. ನಾನು ಸೂರ್ಯನಂತೆ ಬೆಳಗಲ್ಪಡುವ ವಸ್ತುಗಳಿಗಿಂತ ಅತ್ಯಂತ ಭಿನ್ನನು. ನಾನು ಪರ್ವತದಂತೆ ಯಾವಾಗಲೂ ಅಚಲ ನಾದವನು. ನಾನು ಸಮುದ್ರದಂತೆ ಪಾರವಿಲ್ಲದವನು.

ನ ಮೇ ದೇಹೇನ-ಸಂಬಂಧೋ ಮೇಘೇನೇವ ವಿಹಾಯಸಃ ।
ಅತಃ ಕುತೋ ಮೇ ತದ್ಧರ್ಮಾ ಜಾಗ್ರತ್ಸ್ವಪ್ನಸುಷುಪ್ತಯಃ ॥ 499 ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿಹಾಯಸ್ಣಋಸ್ಬಋ¯ಅಃ = ಆಕಾಶಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಮೇಘೇನ ಇವ್ಣಋಸ್ಬಋ¯ಅ = ಮೇಘದೊಂದಿಗೆ ಹೇಗೋ ಹಾಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಮ್ಣೆಋಸ್ಬಋ¯| = ನನಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹೇನ್ಣಋಸ್ಬಋ¯ಅ = ದೇಹದೊಂದಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಬಂಧ್ಣಋಸ್ಬಋ¯ಃ = ಸಂಬಂಧವು ್ಣಋಸ್ಶ್ಯ್¯ನ = ಇಲ್ಲವು ; ಅತ್ಣಃಋಸ್ಬಋ¯ = ಆದುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತದ್ಧರ್ಮಾ್ಣಋಸ್ಬಋ¯ಃ = ಅದರ ಧರ್ಮಗಳಾದ ್ಣಋಸ್ಶ್ಯ್¯ಜಾಗ್ರತ್-ಸ್ವಪ್ನ-ಸುಷುಪ್ತಯ್ಣಃಋಸ್ಬಋ¯ = ಜಾಗ್ರತ್ಸ್ವಪ್ನಸುಷುಪ್ತಿಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಮ್ಣೇಋಸ್ಬಋ¯ = ನನಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಕುತ್ಣಃಋಸ್ಬಋ¯ = ಎಲ್ಲಿಯವು ?

499. ಆಕಾಶಕ್ಕೆ ಮೇಘದ ಸಂಬಂಧವಿಲ್ಲದಿರುವಂತೆ ನನಗೆ ದೇಹದ ಸಂಬಂಧವಿರುವುದಿಲ್ಲ. ಆದುದರಿಂದ ಅದರ ಧರ್ಮಗಳಾದ ಜಾಗ್ರತ್-ಸ್ವಪ್ನ- ಸುಷುಪ್ತಿಗಳು ನನಗೆ ಎಲ್ಲಿಂದ ಬರುವುವು ?

ಉಪಾಧಿರಾಯಾತಿ ಸ ಏವ ಗಚ್ಛತಿ
ಸ ಏವ ಕರ್ಮಾಣಿ ಕರೋತಿ ಭುಂಕ್ತೇ ।
ಸ ಏವ ಜೀರ್ಯನ್ಮ್ರಿಯತೇ ಸದಾsಹಂ
ಕುಲಾದ್ರಿವನ್ನಿಶ್ಚಲ ಏವ ಸಂಸ್ಥಿತಃ ॥ 50ಂ ॥
್ಣಋಸ್ಶ್ಯ್¯ಉಪಾದ್ಣಿಋಸ್ಬಋ¯kಃ = ಉಪಾಧಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಆಯಾತ್ಣಿಋಸ್ಬಋ¯ = ಬರುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ಸಃ ಏವ = ಅದೇ ್ಣಋಸ್ಶ್ಯ್¯ಗಚ್ಛತ್ಣಿಋಸ್ಬಋ¯ = ಹೋಗುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ಸಃ ಏವ ಕರ್ಮಾಣ್ಣಿಋಸ್ಬಋ¯ = ಕರ್ಮಗಳನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಕರೋತ್ಣಿಋಸ್ಬಋ¯ = ಮಾಡುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ಭುಂಕ್ತ್ಣೇಋಸ್ಬಋ¯ = ಫಲವನ್ನು ಭೋಗಿಸುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಸಃ ಏವ ಜೀರ್ಯನ್ಣ್ಋಸ್ಬಋ¯ = ಮುಪ್ಪನ್ನು ಹೊಂದಿ ್ಣಋಸ್ಶ್ಯ್¯ಮ್ರಿಯತ್ಣೇಋಸ್ಬಋ¯ = ನಾಶವಾಗುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನ್ಣುಋಸ್ಶ್ಯ್¯ ಸದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗಲ್ಣೂಋಸ್ಶ್ಯ್¯ ಕುಲಾದ್ರಿವತ್ಣಋಸ್ಬಋ¯್ = ಕುಲಾಚಲದಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ನಿಶ್ಚಲಃ ಏವ್ಣಋಸ್ಬಋ¯ಅ = ನಿಶ್ಚಲನಾಗಿಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸ್ಥಿತ್ಣಃಋಸ್ಬಋ¯ = ಇರುತ್ತೇನೆ.

50ಂ. ಉಪಾಧಿಯು ಬರುತ್ತದೆ, ಅದೇ ಹೋಗುತ್ತದೆ ; ಅದೇ ಕರ್ಮಗಳನ್ನು ಮಾಡುತ್ತದೆ, ಫಲವನ್ನು ಭೋಗಿಸುತ್ತದೆ ; ಅದೇ ಮುಪ್ಪನ್ನು ಹೊಂದಿ ನಾಶ ವಾಗುತ್ತದೆ. (ಆದರೆ) ನಾನು ಯಾವಾಗಲೂ ಕುಲಾಚಲದಂತೆ ನಿಶ್ಚಲನಾಗಿಯೇ ಇರುತ್ತೇನೆ.

ನ ಮೇ ಪ್ರವೃತ್ತಿರ್ನ ಚ ಮೇ ನಿವೃತ್ತಿಃ
ಸದೈಕರೂಪಸ್ಯ ನಿರಂಶಕಸ್ಯ ।
ಏಕಾತ್ಮಕೋ ಯೋ ನಿಬಿಡೋ ನಿರಂತರೋ
ವ್ಯೋಮೇವ ಪೂರ್ಣಃ ಸ ಕಥಂ ನು ಚೇಷ್ಟತೇ ।। 501 ॥
್ಣಋಸ್ಶ್ಯ್¯ಸದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಏಕರೂಪಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಏಕರೂಪನಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರಂಶಕಸ್ಣಋಸ್ಬಋ¯್ಯ = ನಿರ ವಯವನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮ್ಣೇಋಸ್ಬಋ¯ = ನನಗೆ ್ಣಋಸ್ಶ್ಯ್¯ನ ಪ್ರವೃತ್ತಿಃ = ್ಣಋಸ್ಬಋ¯ಪ್ರವೃತ್ತಿಯಿಲ್ಲ,್ಣಋಸ್ಶ್ಯ್¯ ಮೇ ನ ಚ ನಿವೃತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ನಿವೃತ್ತಿಯಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣಃಋಸ್ಬಋ¯ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಏಕಾತ್ಮಕ್ಣಃಋಸ್ಬಋ¯ = ಏಕರೂಪನೊ ್ಣಋಸ್ಶ್ಯ್¯ನಿಬಿಡ್ಣಋಸ್ಬಋ¯ಃ = ಸಾಂದ್ರನೊ ್ಣಋಸ್ಶ್ಯ್¯ನಿರಂತರ್ಣಃಋಸ್ಬಋ¯ = ನಿರವಕಾಶನ್ಣೊಋಸ್ಶ್ಯ್¯ ವ್ಯೋಮ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಇವ = ಆಕಾಶದಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಪೂರ್ಣ್ಣಃಋಸ್ಬಋ¯ = ಪೂರ್ಣನೊ ್ಣಋಸ್ಶ್ಯ್¯ಸಃ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ಕಥಂ ನು ಚೇಷ್ಟತ್ಣೇಋಸ್ಬಋ¯ = ಹೇಗೆ ತಾನೇ ಕೆಲಸವನ್ನು ಮಾಡಿಯಾನು ?

501. ಯಾವಾಗಲೂ ಏಕರೂಪನಾದ ನಿರವಯವನಾದ ನನಗೆ ಪ್ರವೃತ್ತಿಯೂ ಇಲ್ಲ, ನಿವೃತ್ತಿಯೂ ಇಲ್ಲ. ಯಾವನು ಏಕರೂಪದವನೊ ಸಾಂದ್ರನೊ1 ನಿರವಕಾಶನೊ ಆಕಾಶದಂತೆ ಪೂರ್ಣನೊ ಅವನು ಹೇಗೆ ತಾನೇ ಕೆಲಸವನ್ನು ಮಾಡಿಯಾನು ?

[1 ಉಪ್ಪಿನ ಹರಳಿನಂತೆ ಘನವಾಗಿರುವನೊ.]
ಪುಣ್ಯಾನಿ ಪಾಪಾನಿ ನಿರಿಂದ್ರಿಯಸ್ಯ
ನಿಶ್ಚೇತಸೋ ನಿರ್ವಿಕೃತೇರ್ನಿರಾಕೃತೇಃ ।
ಕುತೋ ಮಮಾಖಂಡ-ಸುಖಾನುಭೂತೇ-
ಬ್ರೂRತೇ ಹ್ಯನನ್ವಾಗತಮಿತ್ಯಪಿ ಶ್ರುತಿಃ ॥ 502 ॥
್ಣಋಸ್ಶ್ಯ್¯ನಿರಿಂದ್ರಿಯಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಇಂದ್ರಿಯರಹಿತನಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿಶ್ಚೇತಸಃ = ಚಿತ್ತರಹಿತನಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕೃತ್ಣೇಋಸ್ಬಋ¯ಃ = ನಿರ್ವಿಕಾರನಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರಾಕೃತ್ಣೇಋಸ್ಬಋ¯ಃ = ಆಕಾರ-ರಹಿತನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅಖಂಡ-ಸುಖಾನುಭೂತೇಃ = ಅಖಂಡಾನಂದಾನುಭವನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮಮ್ಣಋಸ್ಬಋ¯ = ನನಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಪುಣ್ಯಾನ್ಣಿಋಸ್ಬಋ¯ = ಪುಣ್ಯಗಳಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಪಾಪಾನಿ =
ಪಾಪಗಳಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಕುತ್ಣಃಋಸ್ಬಋ¯ = ಎಲ್ಲಿಂದ ಬಂದಾವು ? ್ಣಋಸ್ಶ್ಯ್¯ಶ್ರುತಿಃ ಹ್ಣಿಋಸ್ಬಋ¯ = ಶ್ರುತಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಅನನ್ವಾಗತ್ಣಂಋಸ್ಬಋ¯ = ಸ್ಪೃಷ್ಟವಾಗಿಲ್ಲ ್ಣಋಸ್ಶ್ಯ್¯ಇತಿ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಎಂದೇ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರೂತ್ಣೇಋಸ್ಬಋ¯ = ಹೇಳುತ್ತದೆ.

502. ಇಂದ್ರಿಯರಹಿತನೂ ಚಿತ್ತವಿಲ್ಲದವನೂ ನಿರ್ವಿಕಾರನೂ ನಿರಾಕಾರನೂ ಅಖಂಡಾನಂದಾನುಭವ-ಸ್ವರೂಪನೂ ಆದ ನನಗೆ ಪುಣ್ಯಗಳಾಗಲಿ ಪಾಪಗಳಾಗಲಿ ಎಲ್ಲಿಂದ ಬಂದಾವು ? ‘ಸ್ಪೃಷ್ಟವಾಗಿಲ್ಲ’ ಎಂದೇ ಶ್ರುತಿಯು ಹೇಳುತ್ತದೆ.1

[1‘(ಈ ರೂಪವು) ಪುಣ್ಯದಿಂದ ಸ್ಪೃಷ್ಟವಾಗಿಲ್ಲ, ಪಾಪದಿಂದ ಸ್ಪೃಷ್ಟವಾಗಿಲ್ಲ ; ಏಕೆಂದರೆ ಅವನು ಬುದ್ಧಿಯ ಶೋಕಗಳನ್ನೆಲ್ಲ ದಾಟಿರುತ್ತಾನೆ’ ್ಣಋಸ್ಶ್ಯ್¯ಅನನ್ವಾಗತಂ ಪುಣ್ಯೇನ್ಣಋಸ್ಬಋ¯ಅ, ್ಣಋಸ್ಶ್ಯ್¯ಅನನ್ವಾಗತಂ ಪಾಪೇನ, ತೀರ್ಣೋ ಹಿ ತದಾ ಸರ್ವಾಂಛೋಕಾನ್ ಹೃದಯಸ್ಯ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಭವತ್ಣಿಋಸ್ಬಋ¯ (ಬೃಹದಾರಣ್ಯಕ ಉ. 4.3. 22) ಎಂದು ಶ್ರುತಿಯು ಸುಷುಪ್ತಾವಸ್ಥೆಯ ಅನುಭವವನ್ನು ತಿಳಿಸುತ್ತದೆ.]
ಛಾಯಯಾ ಸ್ಪೃಷ್ಟಮುಷ್ಣಂ ವಾ ಶೀತಂ ವಾ ಸುಷ್ಠು ದುಷ್ಠು ವಾ ।
ನ ಸ್ಪೃಶತ್ಯೇವ ಯತ್ಕಿಂಚಿತ್ ಪುರುಷಂ ತದ್ವಿಲಕ್ಷಣಮ್ ॥ 503 ॥
್ಣಋಸ್ಶ್ಯ್¯ಛಾಯಯ್ಣಾಋಸ್ಬಋ¯ = ನೆರಳಿನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಪೃಷ್ಟ್ಣಂಋಸ್ಬಋ¯ = ಮುಟ್ಟಲ್ಪಟ್ಟ್ಣಋಸ್ಶ್ಯ್¯ ಉಷ್ಣಂ ವ್ಣಾಋಸ್ಬಋ¯ = ಉಷ್ಣವಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಶೀತಂ ವ್ಣಾಋಸ್ಬಋ¯ = ಶೀತವಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸುಷ್ಣುಋಸ್ಬಋ¯್ಠ = ಒಳ್ಳೆಯದಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ದುಷ್ಠು ವ್ಣಾಋಸ್ಬಋ¯ = ಕೆಟ್ಟದಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಕಿಂಚಿತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವುದೊಂದೂ ್ಣಋಸ್ಶ್ಯ್¯ತದ್ವಿಲಕ್ಷಣ್ಣಂಋಸ್ಬಋ¯ = ಅದಕ್ಕಿಂತ ಭಿನ್ನನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪುರುಷ್ಣಂಋಸ್ಬಋ¯ = ಪುರುಷನನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನ ಸ್ಪೃಶತಿ ಏವ್ಣಋಸ್ಬಋ¯ = ಸೋಂಕುವುದೇ ಇಲ್ಲ.

503. ನೆರಳಿನ ಮೂಲಕ ಮುಟ್ಟಿದ ಉಷ್ಣವಾಗಲಿ ಶೀತವಾಗಲಿ, ಒಳ್ಳೆಯ ದಾಗಲಿ ಕೆಟ್ಟದಾಗಲಿ, ನೆರಳಿಗಿಂತ ಭಿನ್ನನಾದ ಪುರುಷನನ್ನು ಸೋಂಕುವುದೇ ಇಲ್ಲ.

ನ ಸಾಕ್ಷಿಣಂ ಸಾಕ್ಷ್ಯಧರ್ಮಾಃ ಸಂಸ್ಪೃಶಂತಿ ವಿಲಕ್ಷಣಮ್ । ಅವಿಕಾರಮುದಾಸೀನಂ ಗೃಹಧರ್ಮಾಃ ಪ್ರದೀಪವತ್ ॥ 504 ॥
್ಣಋಸ್ಶ್ಯ್¯ಗೃಹಧರ್ಮಾ್ಣಃಋಸ್ಬಋ¯ = ಮನೆಯ ಧರ್ಮಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರದೀಪವತ್ಣ್ಋಸ್ಬಋ¯ = ದೀಪವನ್ನು ಹೇಗೋ ಹಾಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಾಕ್ಷ್ಯಧರ್ಮಾ್ಣಋಸ್ಬಋ¯ಃ = ದೃಶ್ಯವಸ್ತುಗಳ ಧರ್ಮಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಅವಿಕಾರ್ಣಂಋಸ್ಬಋ¯ = ನಿರ್ವಿಕಾರನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಉದಾ ಸೀನ್ಣಂಋಸ್ಬಋ¯ = ಉದಾಸೀನನಾದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಲಕ್ಷಣ್ಣಂಋಸ್ಬಋ¯ = ವಿಲಕ್ಷಣನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸಾಕ್ಷಿಣಂ = ್ಣಋಸ್ಬಋ¯ಸಾಕ್ಷಿಯನ್ಣುಋಸ್ಶ್ಯ್¯್ನ ನ ಸಂಸ್ಪೃಶಂತಿ = ್ಣಋಸ್ಬಋ¯ಮುಟ್ಟುವುದಿಲ್ಲ.

504. ಮನೆಯ ಧರ್ಮಗಳು (ಅವುಗಳನ್ನು ಪ್ರಕಾಶಿಸುವ ) ಪ್ರದೀಪವನ್ನು ಮುಟ್ಟದಿರುವಂತೆ ನಿರ್ವಿಕಾರನೂ ಉದಾಸೀನನೂ ವಿಲಕ್ಷಣನೂ ಆದ ಸಾಕ್ಷಿ ಯನ್ನು ದೃಶ್ಯವಸ್ತುಗಳ ಧರ್ಮಗಳು ಮುಟ್ಟುವುದಿಲ್ಲ.

ರವೇರ್ಯಥಾ ಕರ್ಮಣಿ ಸಾಕ್ಷಿಭಾವೋ
ವಹ್ನೇರ್ಯಥಾ ದಾಹ- --ನಿಯಾಮಕತ್ವಮ್ ।
ರಜ್ಜೋರ್ಯಥಾssರೋಪಿತವಸ್ತು--ಸಂಗ-
ಸ್ತಥೈವ ಕೂಟಸ್ಥ-ಚಿದಾತ್ಮನೋ ಮೇ ॥ 505 ॥
್ಣಋಸ್ಶ್ಯ್¯ಕರ್ಮಣ್ಣಿಋಸ್ಬಋ¯ = ಕರ್ಮದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ರ ವ್ಣೇಋಸ್ಬಋ¯ಃ = ಸೂರ್ಯನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಾಕ್ಷಿಭಾವ್ಣಃಋಸ್ಬಋ¯ = ಸಾಕ್ಷಿತ್ವವು ್ಣಋಸ್ಶ್ಯ್¯ಯಥ್ಣಾಋಸ್ಬಋ¯ = ಹೇಗೋ, ್ಣಋಸ್ಶ್ಯ್¯ವಹ್ನ್ಣೇಃಋಸ್ಬಋ¯ = ಅಗ್ನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದಾಹ ನಿಯಾಮಕತ್ವ್ಣಂಋಸ್ಬಋ¯ = ದಹಿಸುವಿಕೆಯು ್ಣಋಸ್ಶ್ಯ್¯ಯಥ್ಣಾಋಸ್ಬಋ¯, ್ಣಋಸ್ಶ್ಯ್¯ರಜ್ಜ್ಣೋಋಸ್ಬಋ¯ಃ = ಹಗ್ಗಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಆರೋಪಿತ-ವಸ್ತು-ಸಂಗ್ಣಃಋಸ್ಬಋ¯ = ಆರೋಪಿಸಲ್ಪಟ್ಟ ವಸ್ತುವಿನ ಸಂಬಂಧವು ್ಣಋಸ್ಶ್ಯ್¯ಯಥಾ, ತಥಾ ಏವ್ಣಋಸ್ಬಋ¯ಅ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಕೂಟಸ್ಥ-ಚಿದಾತ್ಮನ್ಣಃಋಸ್ಬಋ¯ = ಕೂಟಸ್ಥ- ಚಿತ್ಸ್ವರೂಪನಾದ ನನಗೆ [ಸಾಕ್ಷಿಭಾವವು].

505. (ಮನುಷ್ಯರು ಮಾಡುವ ಒಳ್ಳೆಯ ಮತ್ತು ಕೆಟ್ಟ) ಕರ್ಮಗಳಿಗೆ ಸೂರ್ಯನು ಹೇಗೆ ಸಾಕ್ಷಿಯೊ,1 ಅಗ್ನಿಯು ಎಲ್ಲವನ್ನೂ ಹೇಗೆ ದಹಿಸುವುದೊ,2 ಹಗ್ಗವು ತನ್ನಲ್ಲಿ ಆರೋಪಿತವಾದ ವಸ್ತುವಿನೊಂದಿಗೆ ಹೇಗೆ ಸಂಬಂಧವನ್ನು ಪಡೆದಿದೆಯೊ3 ಹಾಗೆಯೇ ಕೂಟಸ್ಥ-ಚಿತ್ಸ್ವರೂಪನಾದ ನನಗೆ (ಸಾಕ್ಷಿಭಾವವಿರು ತ್ತದೆ).

[1 ಸೂರ್ಯನ ಬೆಳಕಿನಲ್ಲಿ ಮನುಷ್ಯರು ಎಂಥ ಕೆಲಸವನ್ನು ಮಾಡಿದರೂ ಅದರಿಂದ ಅವನು ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ.
2 ದಹಿಸಲ್ಪಟ್ಟ ವಸ್ತುಗಳ ಧರ್ಮಗಳಿಂದ ಅಗ್ನಿಯು ಲಿಪ್ತವಾಗುವುದಿಲ್ಲ.
3ಹಗ್ಗಕ್ಕೂ ಹಾವಿಗೂ ಇರುವ ಸಂಬಂಧವು ಕೇವಲ ಕಲ್ಪನೆ.]
ಕರ್ತಾsಪಿ ವಾ ಕಾರಯಿತಾsಪಿ ನಾಹಂ
ಭೋಕ್ತಾsಪಿ ವಾ ಭೋಜಯಿತಾsಪಿ ನಾಹಂ
ದ್ರಷ್ಟಾsಪಿ ವಾ ದರ್ಶಯಿತಾsಪಿ ನಾಹಂ
ಸೋsಹಂ ಸ್ವಯಂಜ್ಯೋತಿರನೀದೃಗಾತ್ಮಾ ॥ 506 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ತಾ ಅಪಿ ವ್ಣಾಋಸ್ಬಋ¯ = ಕರ್lತೃವೂ ್ಣಋಸ್ಶ್ಯ್¯ಕಾರಯಿತಾ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಮಾಡಿಸುವವನೂ ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ = ಅಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ ಭೋಕ್ತಾ ಅಪಿ ವ್ಣಾಋಸ್ಬಋ¯ = ಭೋಕ್ತೃವೂ ್ಣಋಸ್ಶ್ಯ್¯ಭೋಜಯಿತಾ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಅನುಭವಿಸುವಂತೆ ಮಾಡುವವನೂ ್ಣಋಸ್ಶ್ಯ್¯ನ, ಅಹ್ಣಂಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ದ್ರಷ್ಟಾ ಅಪಿ ವ್ಣಾಋಸ್ಬಋ¯ = ದ್ರಷ್ಟೃವೂ ್ಣಋಸ್ಶ್ಯ್¯ದರ್ಶ ಯಿತಾ ವ್ಣಋಸ್ಬಋ¯ಾ = ತೋರಿಸುವವನೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ; ಸಃ ಅಹಂ = ಆ ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಂಜ್ಯೋತಿಃ = ಸ್ವಯಂಜ್ಯೋತಿಯು, ್ಣಋಸ್ಶ್ಯ್¯ಅನೀದೃಕ್-ಆತ್ಮಾ= ಇದಕ್ಕಿಂತ ಬೇರೆಯ ಸ್ವರೂಪದವನೇ.

506. ನಾನು ಮಾಡುವವನಾಗಲಿ ಮಾಡಿಸುವವನಾಗಲಿ ಅಲ್ಲ, ಅನುಭವಿಸುವ ವಅನಾಗಲಿ (ಇತರರನ್ನು) ಅನುಭವಿಸುವಂತೆ ಮಾಡುವವನಾಗಲಿ ಅಲ್ಲ, ನೋಡುವ ವನಾಗಲಿ ತೋರಿಸುವವನಾಗಲಿ ಅಲ್ಲ ; ಆ ನಾನು ಸ್ವಯಂಜ್ಯೋತಿಯು, ದೃಶ್ಯ ಜಗತ್ತಿಗಿಂತ ಭಿನ್ನಸ್ವರೂಪದವನು.

ಸ್ವಬಿಂಬಭೂತಂ ರವಿವದ್ವಿನಿಷಿ­ಯಂ
ಕರ್ತಾsಸ್ಮಿ ಭೋಕ್ತಾsಸ್ಮಿ ಹತೋsಸ್ಮಿ ಹೇತಿ ॥ 507 ॥
್ಣಋಸ್ಶ್ಯ್¯ಮೂಢಧಿಯ್ಣಋಸ್ಬಋ¯ಃ = ಮೂಢಬುದ್ಧಿಯುಳ್ಳವರು ್ಣಋಸ್ಶ್ಯ್¯ಉಪಾಧೌ ಚಲತಿ [ಸತಿ] = ಉಪಾಧಿಯು ಚಲಿಸುತ್ತಿರಲಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಔಪಾಧಿಕ್ಣಂಋಸ್ಬಋ¯ = ಉಪಾಧಿಸಂಬಂಧವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತಿಬಿಂಬ-nಲೌಲ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಪ್ರತಿ ಬಿಂಬದ ಚಾಂಚಲ್ಯವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ರವಿವತ್ಣ್ಋಸ್ಬಋ¯ = ಸೂರ್ಯನಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ವಿನಿಷಿ­ಯ್ಣಂಋಸ್ಬಋ¯ = ನಿಷಿ­ಯನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಬಿಂಬಭೂತ್ಣಂಋಸ್ಬಋ¯ = ತಮ್ಮ ಬಿಂಬವಾದ [ಆತ್ಮನಿಗೆ] ್ಣಋಸ್ಶ್ಯ್¯ನಯಂತ್ಣಿಋಸ್ಬಋ¯ = ಸೇರಿಸುತ್ತಾರೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ತಾ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ಕರ್ತೃವಾಗಿದ್ದೇನೆ , ್ಣಋಸ್ಶ್ಯ್¯ಭೋಕ್ತಾ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ಭೋಕ್ತೃವಾಗಿದ್ದೇನೆ, ್ಣಋಸ್ಶ್ಯ್¯ಹಾ ಹತಃ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ಅಯ್ಯೊ ಸತ್ತೆನು ! ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಎಂದು (ತಿಳಿಯುತ್ತಾರೆ).

507. ಉಪಾಧಿಯು ಅಲುಗಾಡಿದಾಗ ಅದರಲ್ಲಿ ಪ್ರತಿಬಿಂಬಿಸಿರುವುದೂ ಚಲಿಸುತ್ತದೆ.1 ಮೂಢಬುದ್ಧಿಯುಳ್ಳವರು ಈ ಚಾಂಚಲ್ಯವನ್ನು ಸೂರ್ಯನಂತೆ ನಿಷಿ­ಯನಾದ ತಮ್ಮ ಬಿಂಬವಾದ ಆತ್ಮನಿಗೆ ಸೇರಿಸುತ್ತಾರೆ. ‘ನಾನು ಕರ್ತೃವು. ನಾನು ಭೋಕ್ತೃವು, ಅಯ್ಯೋ ನಾನು ಸತ್ತೆನು !’2 ಎಂದು ಭಾವಿಸುತ್ತಾರೆ.

[1 ಸೂರ್ಯನು ಪ್ರತಿಬಿಂಬಿಸಿರುವ ನೀರೇ ಉಪಾಧಿಯು. ನೀರು ಅಲುಗಾಡಿದಾಗ ಸೂರ್ಯನ ಪ್ರತಿಬಿಂಬವೂ ನೀರಿನೊಂದಿಗೆ ಅಲುಗಾಡುತ್ತದೆ, ಆದರೆ ಸೂರ್ಯನಿಗೆ ಈ ಅಲುಗಾಟವಿಲ್ಲ. ಇದನ್ನು ಕಂಡ ಯಾವನೋ ಮೂಢಬುದ್ಧಿಯು ಸೂರ್ಯನೇ ಚಲಿಸು ತ್ತಾನೆಂದು ಭಾವಿಸಿಕೊಳ್ಳಬಹುದು. ಹೀಗೆಯೇ ಬುದ್ಧಿಯ ವೃತ್ತಿಗಳಲ್ಲಿ ಪ್ರತಿಬಿಂಬಿ ಸಿರುವ ಆತ್ಮನಿಗೆ ವೃತ್ತಿಗಳ ಚಾಂಚಲ್ಯವನ್ನು ಮೂಢಬುದ್ಧಿಗಳು ಆರೋಪಿಸುತ್ತಾರೆ.
2 ಇದು ಅಜ್ಞಾನಿಗಳ ಗೋಳಾಟ.]
ಜಲೇ ವಾsಪಿ ಸ್ಥಲೇ ವಾsಪಿ ಲುಠತ್ವೇಷ ಜಡಾತ್ಮಕಃ ।
ನಾಹಂ ವಿಲಿಪ್ಯೇ ತದ್ಧರ್ಮೈಘRಟಧರ್ಮೈನRಭೋ ಯಥಾ ॥ 508 ॥
್ಣಋಸ್ಶ್ಯ್¯ಜಡಾತ್ಮಕ್ಣಋಸ್ಬಋ¯ಃ = ಜಡಸ್ವರೂಪವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಏಷ್ಣಋಸ್ಬಋ¯ಅಃ = ಈ ದೇಹವು ್ಣಋಸ್ಶ್ಯ್¯ಜಲೇ ವಾ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ನೀರಿನಲ್ಲಿಯಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಥಲೇ ವಾ ಅಪ್ಣಿಋಸ್ಬಋ¯ =ನೆಲದ ಮೇಲಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಲುಠತ್ಣುಋಸ್ಬಋ¯ =ಬೀಳಲಿ ; ್ಣಋಸ್ಶ್ಯ್¯ಘಟ-ಧರ್ಮೈ್ಣಃಋಸ್ಬಋ¯ = ಗಡಿಗೆಯ ಧರ್ಮಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನಭಃ ಯಥ್ಣಾಋಸ್ಬಋ¯ = ಆಕಾಶವು ಹೇಗೋ ಹಾಗೆ, ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ತದ್ಧರ್ಮೈ್ಣಃಋಸ್ಬಋ¯ = ಆ ಧರ್ಮಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನ ವಿಲಿಪ್ಯ್ಣೇಋಸ್ಬಋ¯ = ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ.

508. ಈ ಜಡಸ್ವರೂಪವಾದ ದೇಹವು ನೀರಿನಲ್ಲಿಯಾದರೂ ನೆಲದ ಮೇಲಾ ದರೂ ಬೀಳಲಿ ! ಗಡಿಗೆಯ ಧರ್ಮಗಳು ಆಕಾಶಕ್ಕೆ ಅಂಟದಿರುವಂತೆ ಈ ಶರೀರ ಧರ್ಮಗಳು ನನಗೆ ಅಂಟುವುದಿಲ್ಲ.

[ಗಡಿಗೆಯಲ್ಲಿ ಅವಚ್ಛಿನ್ನವಾಗಿರುವಂತೆ ಕಾಣುವ ಆಕಾಶವು ಬಾಹ್ಯಾಕಾಶದೊಂದಿಗೆ ಏಕವಾಗಿದೆ ; ಗಡಿಗೆಯು ಒಡೆಯಲಿ ಒಡೆಯದಿರಲಿ ಆಕಾಶಕ್ಕೆ ಯಾವ ಭೇದವೂ ಉಂಟಾ ಗುವುದಿಲ್ಲ. ಆತ್ಮನಿಗೆ ಶರೀರದೊಂದಿಗೆ ಸಂಬಂಧವುಂಟಾದಂತೆ ಕಂಡುಬಂದರೂ ಅವನ ನಿರಂಜನತ್ವಕ್ಕೆ ಯಾವ ಹಾನಿಯೂ ಆಗುವುದಿಲ್ಲ.]
ಕರ್ತೃತ್ವ-ಭೋಕ್ತೃತ್ವ-ಖಲತ್ವ-ಮತ್ತತಾ-
- ಜಡತ್ವ-ಬದ್ಧತ್ವ-ವಿಮುಕ್ತತಾದಯಃ ।
ಬುದ್ಧೇರ್ವಿಕಲ್ಪಾ ನ ತು ಸಂತಿ ವಸ್ತುತಃ
ಸ್ವಸ್ಮಿನ್ ಪರೇ ಬ್ರಹ್ಮಣಿ ಕೇವಲೇsದ್ವಯೇ ॥ 509 ॥
್ಣಋಸ್ಶ್ಯ್¯ಕರ್ತೃತ್ವ-ಭೋಕ್ಚತ್ವ-ಖಲತ್ವ-ಮತ್ತತಾ-ಜಡತ್ವ-ಬದ್ಧತ್ವ-ವಿಮುಕ್ತತಾ-ಆದಯಃ = ಕರ್ತೃತ್ವ ಭೋಕ್ತೃತ್ವ ನೀಚತ್ವ ಮತ್ತತೆ ಜಡತ್ವ ಬದ್ಧತ್ವ ಮುಕ್ತತ್ವ ಮೊದಲಾದುವು ್ಣಋಸ್ಶ್ಯ್¯ಬುದ್ಧ್ಣೇಃಋಸ್ಬಋ¯ = ಬುದ್ಧಿಯ ್ಣಋಸ್ಶ್ಯ್¯ವಿಕಲ್ಪ್ಣಾಃಋಸ್ಬಋ¯ = ಧರ್ಮಗಳು, ್ಣಋಸ್ಶ್ಯ್¯ಕೇವಲ್ಣೇಋಸ್ಬಋ¯ = ಕೇವಲವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಯ್ಣೇಋಸ್ಬಋ¯ = ಅದ್ವಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಸ್ಮಿನ್ಣ್ಋಸ್ಬಋ¯ = ತನ್ನ ಆತ್ಮನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪರೇ ಬ್ರಹ್ಮಣಿ = ಪರಬ್ರಹ್ಮದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ವಸ್ತುತ್ಣಃಋಸ್ಬಋ¯ = ನಿಜವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ನ ತು ಸಂತ್ಣಿಋಸ್ಬಋ¯ = ಇಲ್ಲವು.

509. ಕರ್ತೃತ್ವ ಭೋಕ್ತೃತ್ವ ನೀಚತ್ವ ಮತ್ತತೆ ಜಡತ್ವ ಬದ್ಧತ್ವ ಮುಕ್ತತ್ವ– ಇವೇ ಮೊದಲಾದುವು ಬುದ್ಧಿಯ ಧರ್ಮಗಳೇ ಹೊರತು ಕೇವಲವೂ ಅದ್ವಯವೂ ಆತ್ಮವೂ ಆದ ಪರಬ್ರಹ್ಮದಲ್ಲಿ ನಿಜವಾಗಿ ಇಲ್ಲವು.

ಸಂತು ವಿಕಾರಾಃ ಪ್ರಕೃತೇರ್ದಶಧಾ ಶತಧಾ ಸಹಸ್ರಧಾ ವಾsಪಿ ।
ಕಿಂ ಮೇsಸಂಗಚಿತೇಸ್ತೈರ್ನ ಘನಃ ಕ್ವಚಿದಂಬರಂ ಸ್ಪೃಶತಿ ॥ 510 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಕೃತ್ಣೇಋಸ್ಬಋ¯ಃ = ಪ್ರಕೃತಿಯ ್ಣಋಸ್ಶ್ಯ್¯ವಿಕಾರ್ಣಾಃಋಸ್ಬಋ¯ = ವಿಕಾರಗಳು ್ಣಋಸ್ಶ್ಯ್¯ದಶಧ್ಣಾಋಸ್ಬಋ¯ = ಹತ್ತು ಬಗೆಯಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಶತಧ್ಣಾಋಸ್ಬಋ¯ = ನೂರು ಬಗೆಯಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸಹಸ್ರಧಾ ವಾ ಅಪಿನ್ಶ್ನಿì = ಅಥವಾ ಸಾವಿರ ಬಗೆಯಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸಂತ್ಣುಋಸ್ಬಋ¯ = ಇರಲಿ ; ಅಸಂಗಚಿತ್ಣೇಃಋಸ್ಬಋ¯ = ಅಸಂಗಚಿದ್ರೂಪನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮ್ಣೇಋಸ್ಬಋ¯ = ನನಗೆ ್ಣಋಸ್ಶ್ಯ್¯ತ್ಣೈಃಋಸ್ಬಋ¯ = ಅವುಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಣಿಂಋಸ್ಬಋ¯ = ಏನಾಗಬೇಕು ? ್ಣಋಸ್ಶ್ಯ್¯ಘನಃ = ಮೇಘವು ್ಣಋಸ್ಶ್ಯ್¯ಕ್ವಚಿತ್ಣ್ಋಸ್ಬಋ¯ = ಎಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಅಂಬರ್ಣಂಋಸ್ಬಋ¯ = ಆಕಾಶವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನ ಸ್ಪೃಶತ್ಣಿಋಸ್ಬಋ¯ = ಸೋಂಕುವುದಿಲ್ಲ.

510. ಪ್ರಕೃತಿಯ ವಿಕಾರಗಳು ಹತ್ತು ಬಗೆಯಾಗಿರಲಿ ನೂರು ಬಗೆಯಾಗಿರಲಿ ಅಥವಾ ಸಾವಿರ ಬಗೆಯಾಗಿರಲಿ ; ಅಸಂಗಚಿದ್ರೂಪನಾದ ನನಗೆ ಅವುಗಳಿಂದ ಏನಾಗಬೇಕು ? ಮೇಘವು ಎಲ್ಲಿಯೂ ಆಕಾಶವನ್ನು ಸೋಂಕುವುದಿಲ್ಲವಷ್ಟೆ ?

ಅವ್ಯಕ್ತಾದಿ-ಸ್ಥೂಲಪರ್ಯಂತಮೇತ-
ದ್ವಿಶ್ವಂ ಯತ್ರಾಭಾಸಮಾತ್ರಂ ಪ್ರತೀತಮ್ ।
ವ್ಯೋಮಪ್ರಖ್ಯಂ ಸೂಕ್ಷ್ಮಮಾದ್ಯಂತಹೀನಂ
ಬ್ರಹ್ಮಾದ್ವೈತಂ ಯತ್ತದೇವಾಹಮಸ್ಮಿ ॥ 511 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯತ್ರ = ಯಾವುದರಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅವ್ಯಕ್ತಾದಿ-ಸ್ಥೂಲಪರ್ಯಂತ್ಣಂಋಸ್ಬಋ¯ =ಅವ್ಯಕ್ತವೇ ಮೊದಲಾಗಿ ಸ್ಥೂಲಶರೀರದವರೆಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಏತತ್ ವಿಶ್ವ್ಣಂಋಸ್ಬಋ¯ = ಈ ವಿಶ್ವವು ್ಣಋಸ್ಶ್ಯ್¯ಆಭಾಸಮಾತ್ರ್ಣಂಋಸ್ಬಋ¯ = ಕೇವಲ ತೋರ್ಕೆಯಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತೀತ್ಣಂಋಸ್ಬಋ¯ = ಕಾಣುತ್ತಿರುವುದೊ, ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವುದು ್ಣಋಸ್ಶ್ಯ್¯ವ್ಯೋಮ- ಪ್ರಖ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಆಕಾಶದಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಸೂಕ್ಷ್ಮ್ಣಂಋಸ್ಬಋ¯ = ಸೂಕ್ಷ್ಮವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆದ್ಯಂತ-ಹೀನ್ಣಂಋಸ್ಬಋ¯ = ಆದ್ಯಂತ ರಹಿತವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವೈತಂ ಬ್ರಹ್ಮ್ಣಋಸ್ಬಋ¯ = ಅದ್ವಿತೀಯ-ಬ್ರಹ್ಮವೊ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ ಏವ್ಣಋಸ್ಬಋ¯ಅ = ಅದೇ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ನಾನಾಗಿರುವೆನು.

511. ಯಾವುದರಲ್ಲಿ ಅವ್ಯಕ್ತವೇ ಮೊದಲಾಗಿ ಸ್ಥೂಲಶರೀರದವರೆಗಿರುವ ಈ ವಿಶ್ವವು ಕೇವಲ ತೋರ್ಕೆಯಾಗಿ ಕಾಣುತ್ತಿರುವುದೊ ಯಾವುದು ಆಕಾಶದಂತೆ ಸೂಕ್ಷ್ಮವಾದ ಕೊನೆಮೊದಲಿಲ್ಲದ ಅದ್ವಿತೀಯ-ಬ್ರಹ್ಮವೊ ಅದೇ ನಾನಾಗಿರುವೆನು.

ಸರ್ವಾಧಾರಂ ಸರ್ವವಸ್ತುಪ್ರಕಾಶಂ
ಸರ್ವಾಕಾರಂ ಸರ್ವಗಂ ಸರ್ವಶೂನ್ಯಮ್ ।
ನಿತ್ಯಂ ಶುದ್ಧಂ ನಿಶ್ಚಲಂ ನಿರ್ವಿಕಲ್ಪಂ
ಬ್ರಹ್ಮಾದ್ವೈತಂ ಯತ್ತದೇವಾಹಮಸ್ಮಿ ॥ 512 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವುದು ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾಧಾರ್ಣಂಋಸ್ಬಋ¯ = ಸರ್ವಾಶ್ರಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವವಸ್ತು-ಪ್ರಕಾಶ್ಣಋಸ್ಬಋ¯ಂ =ವಸ್ತುಗಳನ್ನೆಲ್ಲ ಬೆಳಗಿಸುವ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾಕಾರ್ಣಂಋಸ್ಬಋ¯ = ಅಪರಿಚ್ಛಿನ್ನಾಕಾರವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಗ್ಣಂಋಸ್ಬಋ¯ = ಸರ್ವವ್ಯಾಪಕವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಶೂನ್ಯ್ಣಂಋಸ್ಬಋ¯ = ದ್ವೈತಶೂನ್ಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ್ಣಂಋಸ್ಬಋ¯ = ನಿತ್ಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಶುದ್ಧ್ಣಂಋಸ್ಬಋ¯ =ಶುದ್ಧವಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿಶ್ಚಲ್ಣಂಋಸ್ಬಋ¯ = ನಿಶ್ಚಲವಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕಲ್ಪ್ಣಂಋಸ್ಬಋ¯ = ನಿರ್ವಿಕಲ್ಪವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವೈತಂ ಬ್ರಹ್ಣಋಸ್ಬಋ¯ಅ್ಮ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ ಏವ ಅಹಮ್ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯.

512. ಯಾವುದು ಸರ್ವಾಶ್ರಯವೂ ವಸ್ತುಗಳನ್ನೆಲ್ಲ ಪ್ರಕಾಶಿಸುವುದೂ ಅಪರಿ ಚ್ಛಿನ್ನಾಕಾರವೂ ಸರ್ವವ್ಯಾಪಿಯೂ ದ್ವೈತಶೂನ್ಯವೂ ನಿತ್ಯವೂ ಶುದ್ಧವೂ ನಿಶ್ಚಲವೂ ನಿರ್ವಿಕಲ್ಪವೂ ಆದ ಅದ್ವಿತೀಯ-ಬ್ರಹ್ಮವೊ ಅದೇ ನಾನಾಗಿರುವೆನು.

ಯತ್ಪ್ರತ್ಯಸ್ತಾಶೇಷ-ಮಾಯಾವಿಶೇಷಂ
ಪ್ರತ್ಯಗ್ರೂಪಂ ಪ್ರತ್ಯಯಾಗಮ್ಯಮಾನಮ್ ।
ಸತ್ಯಜ್ಞಾನಾನಂತಮಾನಂದರೂಪಂ
ಬ್ರಹ್ಮಾದ್ವೈತಂ ಯತ್ತದೇವಾಹಮಸ್ಮಿ ॥ 513 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವುದು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತ್ಯಸ್ತ-ಅಶೇಷ-ಮಾಯಾವಿಶೇಷ್ಣಂಋಸ್ಬಋ¯ = ಮಾಯೆಯ ವಿಶೇಷ ಗಳನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಹೋಗಲಾಡಿಸಿಕೊಂಡಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತ್ಯಕ್-ರೂಪ್ಣಂಋಸ್ಬಋ¯ = ಸರ್ವಾಂತರ ವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತ್ಯಯ-ಅಗಮ್ಯಮಾನ್ಣಂಋಸ್ಬಋ¯ = ಮನಸ್ಸಿಗೆ ಅಗೋಚರವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸತ್ಯ-ಜ್ಞಾನ-ಅನಂತಂ = ಸತ್ಯಜ್ಞಾನಾನಂತವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆನಂದರೂಪ್ಣಂಋಸ್ಬಋ¯ = ಆನಂದಸ್ವರೂಪವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ ಅದ್ವೈತಂ ಬ್ರಹ್ಮ ತತ್ ಏವ ಅಹಮ್ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯.

513. ಯಾವುದು ಮಾಯೆಯ ವಿಶೇಷಗಳನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಹೋಗಲಾ ಡಿಸಿಕೊಂಡಿರುವುದೊ ಸರ್ವಾಂತರವೊ ಮನಸ್ಸಿಗೆ ಅಗೋಚರವೊ ಸತ್ಯಜ್ಞಾನಾ ನಂತವೊ ಆನಂದಸ್ವರೂಪವೊ ಅದ್ವಿತೀಯ-ಬ್ರಹ್ಮವೊ ಅದೇ ನಾನಾಗಿರುವೆನು.

ನಿಷಿ­ಯೋsಸ್ಮ್ಯವಿಕಾರೋsಸ್ಮಿ ನಿಷ್ಕಲೋsಸ್ಮಿ ನಿರಾಕೃತಿಃ ।
ನಿರ್ವಿಕಲ್ಪೋsಸ್ಮಿ ನಿತ್ಯೋsಸ್ಮಿ ನಿರಾಲಂಬೋsಸ್ಮಿ ನಿರ್ದ್ವಯಃ ॥ 514 ॥
್ಣಋಸ್ಶ್ಯ್¯ನಿಷಿ­ಯಃ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = [ನಾನು] ನಿಷಿ­ಯನಾಗಿರುವೆನು ್ಣಋಸ್ಶ್ಯ್¯ಅವಿಕಾರಃ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ನಿರ್ವಿಕಾರ ನಾಗಿರುವೆನು ್ಣಋಸ್ಶ್ಯ್¯ನಿಷ್ಕಲಃ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ನಿಷ್ಕಲನಾಗಿರುವೆನು ್ಣಋಸ್ಶ್ಯ್¯ನಿರಾಕೃತಿಃ [ಅಸ್ಮಿ] = ನಿರಾಕಾರ ನಾಗಿರುವೆನು ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕಲ್ಪಃ ಅಸ್ಮಿ = ನಿರ್ವಿಕಲ್ಪನಾಗಿರುವೆನು ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯಃ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ನಿತ್ಯನಾಗಿರುವೆನು ್ಣಋಸ್ಶ್ಯ್¯ನಿರಾಲಂಬಃ ಅಸ್ಮ್ಣಿಋಸ್ಬಋ¯ = ನಿರಾಶ್ರಯನಾಗಿರುವೆನು ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ದ್ವಯಃ [ಅಸ್ಮಿ] = ಅದ್ವಯನಾಗಿರು ವೆನು.

514. ನನಗೆ ಕ್ರಿಯೆಯಿಲ್ಲ ವಿಕಾರವಿಲ್ಲ ಕಲೆಯಿಲ್ಲ ಆಕಾರವಿಲ್ಲ ವಿಕಲ್ಪವಿಲ್ಲ; ನಾನು ನಿತ್ಯನು ನಿರಾಶ್ರಯನು ಮತ್ತು ಅದ್ವಿತೀಯನು.

ಸರ್ವಾತ್ಮಕೋsಹಂ ಸರ್ವೋsಹಂ ಸರ್ವಾತೀತೋsಹಮದ್ವಯಃ । ಕೇವಲಾಖಂಡಬೋಧೋsಹಮಾನಂದೋsಹಂ ನಿರಂತರಃ ॥ 515 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾತ್ಮಕ್ಣಃಋಸ್ಬಋ¯ = ಸರ್ವಸ್ವರೂಪನು ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ ಸರ್ವ್ಣಃಋಸ್ಬಋ¯ = ನಾನು ಸರ್ವವು ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ ಸರ್ವಾತೀತ್ಣಃಋಸ್ಬಋ¯ = ನಾನು ಎಲ್ಲವನ್ನೂ ಮೀರಿದವನು ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಯ್ಣಃಋಸ್ಬಋ¯ = ಅದ್ವಯನು ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಕೇವಲ-ಅಖಂಡಬೋಧಃ = ಕೇವಲಾಖಂಡಜ್ಞಾನ-ಸ್ವರೂಪನು ್ಣಋಸ್ಶ್ಯ್¯ಅಹಮ್ ಆನಂದಃ = ಆನಂದಸ್ವರೂಪನು ್ಣಋಸ್ಶ್ಯ್¯ನಿರಂತರ್ಣಃಋಸ್ಬಋ¯ = ನಿರವಕಾಶನು.

515. ನಾನು ಸರ್ವಾತ್ಮನೂ ಸರ್ವವೂ ಎಲ್ಲವನ್ನೂ ಮೀರಿದವನೂ ಅದ್ವಯನೂ ಆಗಿರುತ್ತೇನೆ ; ನಾನು ಕೇವಲ ಅಖಂಡ-ಜ್ಞಾನಾನಂದ-ಸ್ವರೂಪನು, ನಾನು ನಿರವಕಾಶನು.

ಸ್ವಾರಾಜ್ಯ-ಸಾಮ್ರಾಜ್ಯ-ವಿಭೂತಿರೇಷಾ
ಭವತ್ಕೃಪಾಶ್ರೀ-ಮಹಿಮಪ್ರಸಾದಾತ್ ।
ಪ್ರಾಪ್ತಾ ಮಯಾ ಶ್ರೀಗುರವೇ ಮಹಾತ್ಮನೇ
ನಮೋ ನಮಸ್ತೇsಸ್ತು ಪುನರ್ನಮೋsಸ್ತು ॥ 516 ॥
್ಣಋಸ್ಶ್ಯ್¯ಭವತ್-ಕೃಪಾಶ್ರೀ-ಮಹಿಮ-ಪ್ರಸಾದಾತ್ಣ್ಋಸ್ಬಋ¯ = ನಿಮ್ಮ ಕೃಪೆಯೆಂಬ ಲಕ್ಷ್ಮಿಯ ಮಹಿಮೆಯ ಪ್ರಸಾದದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಏಷಾ = ಈ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಾರಾಜ್ಯ-ಸಾಮ್ರಾಜ್ಯ-ವಿಭೂತ್ಣಿಃಋಸ್ಬಋ¯ = ಸ್ವಾರಾಜ್ಯ ಸಾಮ್ರಾಜ್ಯ ವೈಭವವು ್ಣಋಸ್ಶ್ಯ್¯ಮಯ್ಣಾಋಸ್ಬಋ¯ = ನನ್ನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಪ್ಣಾಋಸ್ಬಋ¯್ತ = ಹೊಂದಲ್ಪಟ್ಟಿತು; ್ಣಋಸ್ಶ್ಯ್¯ಮಹಾತ್ಮನ್ಣೇಋಸ್ಬಋ¯ = ಮಹಾತ್ಮನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಶ್ರೀಗುರವ್ಣೇಋಸ್ಬಋ¯ = ಶ್ರೀಗುರುವಾದ ್ಣಋಸ್ಶ್ಯ್¯ತ್ಣೇಋಸ್ಬಋ¯ = ನಿನಗೆ ್ಣಋಸ್ಶ್ಯ್¯ನಮಃ ನಮಃ ಅಸ್ತ್ಣುಋಸ್ಬಋ¯ = ಪುನಃ ಪುನಃ ನಮಸ್ಕಾರವಿರಲಿ, ್ಣಋಸ್ಶ್ಯ್¯ಪುನಃ ನಮಃ ಅಸ್ತು.್ಣಋಸ್ಬಋ¯

516. ನಿಮ್ಮ ಕೃಪೆಯಿಂದ ಲಕ್ಷ್ಮಿಯ ಮಹಿಮೆಯ ಪ್ರಸಾದದಿಂದ ಈ ಸ್ವಾರಾಜ್ಯ-ಸಾಮ್ರಾಜ್ಯದ ವೈಭವವು ನನಗೆ ಪ್ರಾಪ್ತವಾಯಿತು. ಮಹಾತ್ಮನಾದ ಶ್ರೀಗುರುವೇ, ನಿನಗೆ ಪುನಃ ಪುನಃ ನಮಸ್ಕಾರಗಳು.

ಮಹಾಸ್ವಪ್ನೇ ಮಾಯಾಕೃತ-ಜನಿ-ಜರಾ-ಮೃತ್ಯು-ಗಹನೇ
ಭ್ರಮಂತಂ ಕ್ಲಿಶ್ಯಂತಂ ಬಹುಲತರ-ತಾಪೈರನುದನಮ್ ।
ಅಹಂಕಾರವ್ಯಾಘ್ರ-ವ್ಯಥಿತಮಿಮಮತ್ಯಂತ-ಕೃಪಯಾ
ಪ್ರಬೋಧ್ಯ ಪ್ರಸ್ವಾಪಾತ್ ಪರಮವಿತವಾನ್ ಮಾಮಸಿ
ಗುರೋ ॥ 517 ॥
್ಣಋಸ್ಶ್ಯ್¯ಗುರ್ಣೋಋಸ್ಬಋ¯ = ಹೇ ಗುರುವೆ,್ಣಋಸ್ಶ್ಯ್¯ ಮಹಾಸ್ವಪ್ನ್ಣೇಋಸ್ಬಋ¯ = ಮಹಾಸ್ವಪ್ನದಂತಿರುವ್ಣಋಸ್ಶ್ಯ್¯ ಮಾಯಾ- ಕೃತ-ಜನಿ-ಜರಾ-ಮೃತ್ಯು-ಗಹನೇ = ಮಾಯೆಯಿಂದ ಮಾಡಲ್ಪಟ್ಟ ಹುಟ್ಟು-ಮುಪ್ಪು-ಸಾವು ಗಳೆಂಬ ಕಾಡಿನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಭ್ರಮ್ಣಂಋಸ್ಬಋ¯ತ್ಣಋಸ್ಶ್ಯ್¯ಅ್ಣಂಋಸ್ಬಋ¯ = ಅಲೆಯುತ್ತಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಬಹುಲತರ-ತಾಪೈಃ = ಬಗೆಬಗೆಯ ತಾಪಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅನುದಿನ್ಣಂಋಸ್ಬಋ¯ = ಪ್ರತಿದಿನವೂ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಲಿಶ್ಯಂತ್ಣಂಋಸ್ಬಋ¯ = ಕ್ಲೇಶಪಡುತ್ತಿರುವ, ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂಕಾರ- ವ್ಯಾಘ್ರ-ವ್ಯಥಿತ್ಣಂಋಸ್ಬಋ¯ = ಅಹಂಕಾರವೆಂಬ ಹುಲಿಯಿಂದ ವ್ಯಥೆಪಡುತ್ತಿರುವ, ್ಣಋಸ್ಶ್ಯ್¯ಇಮಂ ಮ್ಣಾಂಋಸ್ಬಋ¯ = ಈ ನನ್ನನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅತ್ಯಂತಕೃಪಯ್ಣಾಋಸ್ಬಋ¯ = ಅಪಾರಕೃಪೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಸ್ವಾಪಾತ್ಣ್ಋಸ್ಬಋ¯ = ನಿದ್ರೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಬೋಧ್ಣಋಸ್ಬಋ¯್ಯ = ಎಚ್ಚರಗೊಳಿಸಿ ್ಣಋಸ್ಶ್ಯ್¯ಪರ್ಣಂಋಸ್ಬಋ¯ = ಸಂಪೂರ್ಣವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಅವಿತವಾನ್ ಅಸ್ಣಿಋಸ್ಬಋ¯ = ಕಾಪಾಡಿರುವೆ.

517. ಹೇ ಗುರುವೆ, ಮಹಾಸ್ವಪ್ನದಂತಿರುವ ಮಾಯಾಕೃತವಾದ ಹುಟ್ಟು- ಮುಪ್ಪು-ಸಾವುಗಳೆಂಬ ಕಾಡಿನಲ್ಲಿ ಅಲೆಯುತ್ತಿರುವ, ಬಗೆಬಗೆಯ ತಾಪಗಳಿಂದ ಪ್ರತಿದಿನವೂ ಕ್ಲೇಶಪಡುತ್ತಿರುವ ಅಹಂಕಾರವೆಂಬ ಹುಲಿಯಿಂದ ವ್ಯಥೆಪಡು ತ್ತಿರುವ ಈ ನನ್ನನ್ನು ಅಪಾರಕೃಪೆಯಿಂದ (ಅವಿದ್ಯೆಯೆಂಬ) ನಿದ್ರೆಯಿಂದ ಎಚ್ಚರ ಗೊಳಿಸಿ ಸಂಪೂರ್ಣವಾಗಿ ಕಾಪಾಡಿರುವೆ !

ನಮಸ್ತಸ್ಮೈ ಸದೈಕಸ್ಮೈ ಕಸ್ಮೈಚಿನ್ಮಹಸೇ ನಮಃ ।
ಯದೇತದ್ವಿಶ್ವರೂಪೇಣ ರಾಜತೇ ಗುರುರಾಜ ತೇ ॥ 518 ॥
್ಣಋಸ್ಶ್ಯ್¯ಗುರುರಾಜ್ಣಋಸ್ಬಋ¯ = ಹೇ ಗುರುರಾಜನೆ, ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ ಏತತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವ ಈ [ಬ್ರಹ್ಮವು] ್ಣಋಸ್ಶ್ಯ್¯ವಿಶ್ವರೂಪೇಣ್ಣಋಸ್ಬಋ¯ = ವಿಶ್ವರೂಪದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ರಾಜತ್ಣೇಋಸ್ಬಋ¯ = ಪ್ರಕಾಶಿಸುತ್ತಿರುವುದೊ ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಮೈಮಹಸ್ಣೇಋಸ್ಬಋ¯ = ಆ ತೇಜಃಸ್ವರೂಪನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸದೈಕಸ್ಮ್ಣೆಋಸ್ಬಋ¯ೈ = ಸದಾ ಏಕರೂಪನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಕಸ್ಮೈಚಿತ್ಣ್ಋಸ್ಬಋ¯ = ಮನೋ ವಾಕ್ಕುಗಳಿಗೆ ಅಗೋಚರನಾದ ್ಣಋಸ್ಶ್ಯ್¯ತೇ = ನಿನಗೆ ್ಣಋಸ್ಶ್ಯ್¯ನಮಃ = ನಮಸ್ಕಾರ, ್ಣಋಸ್ಶ್ಯ್¯ನಮಃ.್ಣಋಸ್ಬಋ¯

518. ಹೇ ಗುರುರಾಜನೇ, ವಿಶ್ವರೂಪದಿಂದ ಪ್ರಕಾಶಿಸುತ್ತಿರುವ ತೇಜಸ್ವರೂಪ ನಾದ ಸದೈಕರೂಪನಾದ ಮನೋವಾಕ್ಕುಗಳಿಗೆ ಅಗೋಚರನಾದ ನಿನಗೆ ಪುನಃ ಪುನಃ ನಮಸ್ಕಾರ.

73. ಉಪದೇಶದ ಉಪಸಂಹಾರ
ಇತಿ ನತಮವಲೋಕ್ಯ ಶಿಷ್ಯವರ್ಯಂ
ಸಮಧಿಗತಾತ್ಮಸುಖಂ ಪ್ರಬುದ್ಧತತ್ತ್ವಮ್ ।
ಪ್ರಮುದಿತ-ಹೃದಯಃ ಸ ದೇಶಿಕೇಂದ್ರಃ
ಪುನರಿದಮಾಹ ವಚಃ ಪರಂ ಮಹಾತ್ಮಾ ॥ 519 ॥
್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಹೀಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಮಧಿಗತ-ಆತ್ಮಸುಖ್ಣಂಋಸ್ಬಋ¯ = ಆತ್ಮಾನಂದವನ್ನು ಸಾಕ್ಷಾತ್ಕರಿಸಿದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಬುದ್ಧ-ತತ್ತ್ವಂ = ತತ್ತ್ವವನ್ನುರಿತುಕೊಂಡ ್ಣಋಸ್ಶ್ಯ್¯ನತ್ಣಂಋಸ್ಬಋ¯ = ನಮಸ್ಕರಿಸಿದ ್ಣಋಸ್ಶ್ಯ್¯ಶಿಷ್ಯವರ್ಯಂ = ಶಿಷ್ಯವರ್ಯನನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅವಲೋಕ್ಯ = ನೋಡಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಮುದಿತ-ಹೃದಯಃ = ಸಂತೋಷ ಹೃದಯ ನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪರಂ ಮಹಾತ್ಮಾ= ಶ್ರೇಷ್ಠಮಹಾತ್ಮನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸಃ ದೇಶಿಕೇಂದ್ರ್ಣಃಋಸ್ಬಋ¯ = ಆ ಆಚಾರ್ಯ ವರ್ಯನು ್ಣಋಸ್ಶ್ಯ್¯ಪುನ್ಣಋಸ್ಬಋ¯ಅಃ = ಮರಳಿ ್ಣಋಸ್ಶ್ಯ್¯ಇದಂ ವಚ್ಣಃಋಸ್ಬಋ¯ = ಈ ಮಾತನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಆಹ್ಣಋಸ್ಬಋ¯ = ಹೇಳುತ್ತಾನೆ.

519. ಹೀಗೆ ಆತ್ಮಾನಂದವನ್ನು ಸಾಕ್ಷಾತ್ಕರಿಸಿಕೊಂಡು ತತ್ತ್ವವನ್ನರಿತುಕೊಂಡು ನಮಸ್ಕರಿಸಿದ ಶಿಷ್ಯವರ್ಯನನ್ನು ನೋಡಿ ಸಂತೋಷಹೃದಯನೂ ಶ್ರೇಷ್ಠ ಮಹಾತ್ಮನೂ ಆದ ಆ ಆಚಾರ್ಯವರ್ಯನು ಪುನಃ ಈ ಮಾತನ್ನು ಹೇಳುತ್ತಾನೆ.

ಬ್ರಹ್ಮಪ್ರತ್ಯಯ-ಸಂತತಿರ್ಜಗದತೋ ಬ್ರಹ್ಮೈವ ತತ್ಸರ್ವತಃ
ಪಶ್ಯಾಧ್ಯಾತ್ಮದೃಶಾ ಪ್ರಶಾಂತ-ಮನಸಾ ಸರ್ವಾಸ್ವವಸ್ಥಾಸ್ವಪಿ ।
ರೂಪಾದನ್ಯದವೇಕ್ಷಿತುಂ ಕಿಮಭಿತಶ್ಚಕ್ಷುಷ್ಮತಾಂ ವಿದ್ಯತೇ
ತದ್ವದ್ಬ್ರಹ್ಮವಿದಃ ಸತಃ ಕಿಮಪರಂ ಬುದ್ಧೇರ್ವಿಹಾರಾಸ್ಪದಮ್
॥ 520 ॥
್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಪ್ರತ್ಯಯ-ಸಂತತ್ಣಿಃಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಪ್ರತ್ಯಯದ ಪ್ರವಾಹವೇ ್ಣಋಸ್ಶ್ಯ್¯ಜಗತ್ = ಜಗತ್ತು ; ್ಣಋಸ್ಶ್ಯ್¯ಅತ್ಣಃಋಸ್ಬಋ¯ = ಆದುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಣ್ಋಸ್ಬಋ¯ = ಅದು ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವತ್ಣಃಋಸ್ಬಋ¯ = ಎಲ್ಲಾ ಕಡೆಯಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ ಏವ್ಣಋಸ್ಬಋ¯ಅ = ಬ್ರಹ್ಮವೇ ; ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಶಾಂತಮನಸಾ = ಪ್ರಶಾಂತವಾದ ಮನಸ್ಸಿನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅಧ್ಯಾತ್ಮದೃಶ್ಯ್ಣಾಋಸ್ಬಋ¯ = ಅಧ್ಯಾತ್ಮದೃಷ್ಟಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾಸು ಅವಸ್ಥಾಸು ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಎಲ್ಲ ಅವಸ್ಥೆಗಳಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಪಶ್ಯ್ಣಋಸ್ಬಋ¯ =[ಬ್ರಹ್ಮವನ್ನೇ] ನೋಡು ; ್ಣಋಸ್ಶ್ಯ್¯ಚಕ್ಷುಷ್ಮತ್ಣಾಂಋಸ್ಬಋ¯ = ಕಣ್ಣಿರುವವರಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಭಿತಃ = ಸುತ್ತಲೂ ್ಣಋಸ್ಶ್ಯ್¯ರೂಪಾತ್ಣ್ಋಸ್ಬಋ¯ =ರೂಪಕ್ಕಿಂತ ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ಯತ್ಣ್ಋಸ್ಬಋ¯ = ಬೇರೊಂದು ್ಣಋಸ್ಶ್ಯ್¯ಅವೇಕ್ಷಿತುಂ = ನೋಡಲು ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಣಿಂಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ವಿದ್ಯತ್ಣೇಋಸ್ಬಋ¯ =ಇರುವುದೆ ? ್ಣಋಸ್ಶ್ಯ್¯ತದ್ವತ್ಣ್ಋಸ್ಬಋ¯ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮವಿದ್ಣಃಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಜ್ಞಾನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸತ್ಣಃಋಸ್ಬಋ¯ = ಸದ್ರೂಪಕ್ಕಿಂತ ್ಣಋಸ್ಶ್ಯ್¯ಅಪರಂ ಕ್ಣಿಂಋಸ್ಬಋ¯ = ಬೇರೆ ಯಾವುದು ್ಣಋಸ್ಶ್ಯ್¯ಬುದ್ಧ್ಣೇಃಋಸ್ಬಋ¯ = ಬುದ್ಧಿಯ ್ಣಋಸ್ಶ್ಯ್¯ವಿಹಾರಾಸ್ಪದ್ಣಂಋಸ್ಬಋ¯ = ಸಂಚಾರಕ್ಕೆ ಆಶ್ರಯವಾಗಿ [ಇರುತ್ತದೆ ?]

520. ಈ ಜಗತ್ತು ಬ್ರಹ್ಮಪ್ರಯತ್ನದ ಪ್ರವಾಹವಾಗಿರುತ್ತದೆ. ಆದುದರಿಂದ ಅದು ಎಲ್ಲಾ ಕಡೆಯಲ್ಲಿಯೂ ಬ್ರಹ್ಮವೇ. ಪ್ರಶಾಂತಮನಸ್ಸಿನಿಂದ ಮತ್ತು ಆಧ್ಯಾತ್ಮದೃಷ್ಟಿಯಿಂದ ಎಲ್ಲ ಅವಸ್ಥೆಗಳಲ್ಲಿಯೂ (ಬ್ರಹ್ಮವನ್ನೇ) ನೋಡು. kಕಣ್ಣಿರುವವರೆಗೆ ಸುತ್ತಲೂ ರೂಪಕ್ಕಿಂತ ಬೇರೆಯಾದುದು ನೋಡಲು ಇರುವುದೆ? ಹೀಗೆಯೇ ಬ್ರಹ್ಮಜ್ಞಾನಿಗೆ ಸದ್ರೂಪವಾದ (ಬ್ರಹ್ಮಕ್ಕಿಂತ ) ಮತ್ತೇನು ಬುದ್ಧಿಯ ಸಂಚಾರಕ್ಕೆ ಆಶ್ರಯವಾದೀತು ?

ಕಸ್ತಾಂ ಪರಾನಂದ-ರಸಾನುಭೂತಿ-
ಮುತ್ಸೃಜ್ಯ ಶೂನ್ಯೇಷು ರಮೇತ ವಿದ್ವಾನ್ ।
ಚಂದ್ರೇ ಮಹಾಹ್ಲಾದಿನಿ ದೀಪ್ಯಮಾನೇ
ಚಿತ್ರೇಂದುಮಾಲೋಕಯಿತಂ ಕ ಇಚ್ಛೇತ್ ॥ 521 ॥
್ಣಋಸ್ಶ್ಯ್¯ಕಃ ವಿದ್ವಾನ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವ ವಿದ್ವಾಂಸನು ್ಣಋಸ್ಶ್ಯ್¯ತಾಂ = ್ಣಋಸ್ಬಋ¯ಆ್ಣಋಸ್ಶ್ಯ್¯ ಪರಾನಂದ-ರಸಾನುಭೂತಿಂ = ್ಣಋಸ್ಬಋ¯ಪರಾನಂದರಸವಾದ ಅನುಭವವನ್ನ್ಣುಋಸ್ಶ್ಯ್¯ ಉತ್ಸೃಜ್ಣಋಸ್ಬಋ¯್ಯ = ಬಿಟ್ಟು ್ಣಋಸ್ಶ್ಯ್¯ಶೂನ್ಯೇಷ್ಣುಋಸ್ಬಋ¯ = ನೀರಸವಿಷಯ ಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ರಮೇಶ್ಣಋಸ್ಬಋ¯ = ರಮಿಸಿಯಾನು ? ್ಣಋಸ್ಶ್ಯ್¯ಮಹಾಹ್ಲಾದಿನಿ ಚಂದ್ರ್ಣೇಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ದೀಪ್ಯಮಾನೇ[ಸತಿ] = ಮಹಾಹ್ಲಾದವನ್ನುಂಟುಮಾಡುವ ಚಂದ್ರನು ಪ್ರಕಾಶಿಸುತ್ತಿರುವಾಗ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಣಃಋಸ್ಬಋ¯ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಚಿತ್ರ-ಇಂದ್ಣುಂಋಸ್ಬಋ¯ = ಚಿತ್ರದಲ್ಲಿರುವ ಚಂದ್ರನನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಆಲೋಕಯಿತ್ಣುಂಋಸ್ಬಋ¯ = ನೋಡುವುದಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಇಚ್ಛೇತ್ಣ್ಋಸ್ಬಋ¯ = ಬಯಸುವನು ?

521. ಆ ಪರಾನಂದದ ಅನುಭವವನ್ನು ಬಿಟ್ಟು, ನೀರಸವಾದ ವಿಷಯಗಳಲ್ಲಿ ಯಾವ ವಿದ್ವಾಂಸನು ತಾನೇ ರಮಿಸಿಯಾನು ? ಮಹಾಹ್ಲಾದವನ್ನುಂಟುಮಾಡುವ ಚಂದ್ರನು ಪ್ರಕಾಶಿಸುತ್ತಿರುವಾಗ ಯಾವನು ತಾನೇ ಚಿತ್ರದಲ್ಲಿರುವ ಚಂದ್ರನನ್ನು ನೋಡುವುದಕ್ಕೆ ಬಯಸುವನು ?

ಅಸತ್ಪದಾರ್ಥಾನುಭವೇನ ಕಿಂಚಿ-
ನ್ನ ಹ್ಯಸ್ತಿ ತೃಪ್ತಿರ್ನ ಚ ದುಃಖಹಾನಿಃ ।
ತದದ್ವಯಾನಂದ-ರಸಾನುಭೂತ್ಯಾ
ತೃಪ್ತಃ ಸುಖಂ ತಿಷ್ಠ ಸದಾತ್ಮನಿಷ್ಠಯಾ ॥ 522 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್ಪದಾರ್ಥ-ಅನುಭವೇನ್ಣಋಸ್ಬಋ¯ಅ = ಮಿಥ್ಯಾವಸ್ತುಗಳ ಅನುಭವದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂಚಿತ್ಣ್ಋಸ್ಬಋ¯ = [ಕಾಚಿತ್]ತೃಪ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಸ್ಪಲ್ಪ ತೃಪ್ತಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ಹಿ ಅಸ್ತ್ಣಿಋಸ್ಬಋ¯ = ಇಲ್ಲವು, ್ಣಋಸ್ಶ್ಯ್¯ದುಃಖಹಾನಿಃ ಚ ನ್ಣಋಸ್ಬಋ¯ಅ = ದುಃಖನಿವೃತ್ತಿಯೂ ಇಲ್ಲವು ; ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಣ್ಋಸ್ಬಋ¯ = ಆದುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಯಾನಂದರಸ್ಣಋಸ್ಬಋ¯ಅ್ಣಋಸ್ಶ್ಯ್¯ ಅನುಭೂತ್ಣಾಋಸ್ಬಋ¯್ಯ = ಅದ್ವಯಾನಂದರಸದ ಅನುಭವದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತೃಪ್ತ್ಣಃಋಸ್ಬಋ¯ = ತೃಪ್ತನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಸದಾತ್ಮನಿಷ್ಠಯ್ಣಾಋಸ್ಬಋ¯ = ಸದ್ರೂಪನಾದ ಆತ್ಮನ ನಿಷ್ಠೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸುಖಂ ತಿಷ್ಠ = ಸುಖವಾಗಿರು.

522. ಮಿಥ್ಯಾವಸ್ತುಗಳ ಅನುಭವದಿಂದ ಸ್ಪಲ್ಪ ತೃಪ್ತಿಯೂ ಉಂಟಾಗುವು ದಿಲ್ಲ. ದುಃಖನಿವೃತ್ತಿಯೂ ಆಗುವುದಿಲ್ಲ. ಆದುದರಿಂದ ಅದ್ವಯಾನಂದರಸದ ಅನುಭವದಿಂದ ತೃಪ್ತನಾಗಿ ಸದ್ರೂಪಾತ್ಮನ ನಿಷ್ಠೆಯಿಂದಲೇ ಸುಖವಾಗಿರು.

ಸ್ವಮೇವ ಸರ್ವಥಾ ಪಶ್ಯನ್ಮನ್ಯಮಾನಃ ಸ್ವಮದ್ವಯಮ್ ।
ಸ್ವಾನಂದಮನುಭುಂಜಾನಃ ಕಾಲಂ ನಯ ಮಹಾಮತೇ ॥ 523 ॥
್ಣಋಸ್ಶ್ಯ್¯ಮಹಾಮತ್ಣೇಋಸ್ಬಋ¯ = ಎಲೈ ಮಹಾವತಿಯೆ, ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಥ್ಣಾಋಸ್ಬಋ¯ = ಸರ್ವಪ್ರಕಾರದಿಂದಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಮ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಏವ = ಸ್ವಾತ್ಮನನ್ನೇ ್ಣಋಸ್ಶ್ಯ್¯ಪಶ್ಯನ್ಣ್ಋಸ್ಬಋ¯ = ನೋಡುತ್ತಾ, ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಯಂ ಸ್ವಮ್ = ಅದ್ವಯನಾದ ಆತ್ಮನನ್ನ್ಣೇಋಸ್ಶ್ಯ್¯ ಮನ್ಯಮಾನ್ಣಃಋಸ್ಬಋ¯ = ಮನನಮಾಡುತ್ತ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ-ಆನಂದ್ಣಂಋಸ್ಬಋ¯ = ಸ್ವಾತ್ಮಾನಂದವನ್ನೇ ್ಣಋಸ್ಶ್ಯ್¯ಅನುಭುಂಜಾನಃ = ಅನುಭವಿಸುತ್ತ ್ಣಋಸ್ಶ್ಯ್¯ಕಾಲ್ಣಂಋಸ್ಬಋ¯ = ಕಾಲವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನಯ್ಣಋಸ್ಬಋ¯ = ಕಳೆ.

523. ಎಲೈ ಮಹಾಮತಿಯೆ, ಎಲ್ಲ ಪ್ರಕಾರದಿಂದಲೂ ಸ್ವಾತ್ಮನನ್ನೇ ನೋಡುತ್ತ, ಅದ್ವಯನಾದ ಆತ್ಮನನ್ನೇ ಮನನಮಾಡುತ್ತ, ಸ್ವಾತ್ಮಾನಂದವನ್ನೇ ಅನುಭವಿಸುತ್ತ, ಕಾಲವನ್ನು ಕಳೆ.

ಅಖಂಡಬೋಧಾತ್ಮನಿ ನಿರ್ವಿಕಲ್ಪೇ
ವಿಕಲ್ಪನಂ ವ್ಯೋಮ್ನಿ ಪುರಪ್ರಕಲ್ಪನಮ್ ।
ತದದ್ವಯಾನಂದಮಯಾತ್ಮನಾ ಸದಾ
ಶಾಂತಿಂ ಪಾರಮೇತ್ಯ ಭಜಸ್ವ ಮೌನಮ್ ॥ 524 ॥
್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕಲ್ಪೇ = ವಿಕಲ್ಪಶೂನ್ಯನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅಖಂಡಬೋಧಾತ್ಮನ್ಣಿಋಸ್ಬಋ¯ = ಅಖಂಡಜ್ಞಾನಾತ್ಮನಲ್ಲಿ n್ಣಋಸ್ಶ್ಯ್¯ವಿಕಲ್ಪನ್ಣಂಋಸ್ಬಋ¯ = ವಿವಿಧಕಲ್ಪನೆಯು ್ಣಋಸ್ಶ್ಯ್¯ವ್ಯೋಮ್ನ್ಣಿಋಸ್ಬಋ¯ = ಆಕಾಶದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಪುರಪ್ರಕಲ್ಪನ್ಣಂಋಸ್ಬಋ¯ = ನಗರವನ್ನು ಕಲ್ಪಿಸಿದಂತೆ ; ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಣ್ಋಸ್ಬಋ¯ = ಆದುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಯಾನಂದ್ಣಋಸ್ಬಋ¯ಅ ್ಣಋಸ್ಶ್ಯ್¯ಮಯಾತ್ಮನ್ಣಾಋಸ್ಬಋ¯ = ಅದ್ವಯಾನಂದಸ್ವರೂಪದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪರಾಂ ಶಾಂತಿಂ = ಪರಮಶಾಂತಿಯನ್ನ್ಣುಋಸ್ಶ್ಯ್¯ ಏತ್ಯ್ಣಋಸ್ಬಋ¯ = ಹೊಂದಿ ್ಣಋಸ್ಶ್ಯ್¯ಮೌನ್ಣಂಋಸ್ಬಋ¯ = ಮೌನವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಭಜಸ್ವ್ಣಋಸ್ಬಋ¯ = ಸೇವಿಸು.

524. ವಿಕಲ್ಪಶೂನ್ಯನಾದ ಅಖಂಡಜ್ಞಾನಾತ್ಮನಲ್ಲಿ ವಿವಿಧ ಕಲ್ಪನೆಗಳನ್ನು ಮಾಡುವುದು ಆಕಾಶದಲ್ಲಿ ನಗರವನ್ನು ಕಲ್ಪಿಸಿದಂತೆ. ಆದುದರಿಂದ ಅದ್ವಯಾನಂದ ಸ್ವರೂಪದಿಂದ ಯಾವಾಗಲೂ ಪರಮಶಾಂತಿಯನ್ನು ಹೊಂದಿ ಮೌನವನ್ನು ಸೇವಿಸು.

ತೂಷ್ಣೀಮನಸ್ಥಾ ಪರಮೋಶಾಂತಿ
ಬುRದ್ದೇರಸತ್ಕಲ್ಪ-ವಿಕಲ್ಪ-ಹೇತೋಃ ।
ಬ್ರಹ್ಮಾತ್ಮನಾ ಬ್ರಹ್ಮವಿದೋ ಮಾಹತ್ಮನೋ
ಯತ್ರಾದ್ವಯಾನಂದಸುಖಂ ನಿರಂತರಮ್ ॥ 525 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್ಕಲ್ಪ- ವಿಕಲ್ಪ- ಹೇತ್ಣೋಃಋಸ್ಬಋ¯ = ಮಿಥ್ಯಾವಿಷಯಗಳ ಕಲ್ಪನೆಗಳಿಗೆ ಕಾರಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಬುದ್ಧ್ಣೇಋಸ್ಬಋ¯ಃ =ಬುದ್ಧಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಮಹಾತ್ಮನ್ಣಃಋಸ್ಬಋ¯ = ಮಹಾತ್ಮನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮವಿದ್ಣಃಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಜ್ಞಾನಿಯ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಾತ್ಮನ್ಣಾಋಸ್ಬಋ¯ =ಬ್ರಹ್ಮಭಾವದಿಂದ (ಆಗುವ) ್ಣಋಸ್ಶ್ಯ್¯ತೂಷ್ಮೀಮವಸ್ಥ್ಣಾಋಸ್ಬಋ¯ = ಮೌನವೇ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮೋರಶಾಂತಿಃ =ಪರಮಶಾಂತಿಯು ; ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ರ್ಣಋಸ್ಬಋ¯ = ಎಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ನಿರಂತರ್ಣಂಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಅದ್ವಯಾನಂದಸುಖ್ಣಂಋಸ್ಬಋ¯ = ಅದ್ವಯಾನಂದಸುಖವು ( ಆಗುತ್ತದೆಯೋ).

525. ಮಿಥ್ಯಾವಿಷಯಗಳ ಕಲ್ಪನೆಗಳಿಗೆ ಕಾರಣವಾದ ಬುದ್ಧಿಗೆ, ಮಹಾತ್ಮನಾದ ಬ್ರಹ್ಮಜ್ಞಾನಿಯ ಬ್ರಹ್ಮಭಾವದಿಂದ ಉಂಟಾಗುವ ಮೌನವೇ ಪರಮೋಪಶಾಂತಿ. ಅಲ್ಲಿ ನಿರಂತರವೂ ಅದ್ವಯಾನಂದಸುಖವಾಗುತ್ತದೆ.

ನಾಸ್ತಿ ನಿರ್ವಾಸನಾನ್ಮೌನಾತ್ ಪರಂ ಸುಖಕೃದುತ್ತಮಮ್।
ವಿಜ್ಞಾತಾತ್ಮಸ್ವರೂಪಸ್ಯ ಸ್ವಾನಂದರಸ-ಪಾಯಿನಃ ॥ 526 ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾllತ ----- ಆತ್ಮಸ್ವರೂಪಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಆತ್ಮಸ್ವರೂಪವನ್ನು ತಿಳಿದುಕೊಂಡಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ-ಆನಂದ-ರಸ-ಪಾಯಿನ್ಣಃಋಸ್ಬಋ¯ = ಆತ್ಮಾನಂದರಸವನ್ನು ಕುಡಿಯುತ್ತಿರುವವನಿಗೆ ಬಿ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಾಸನಾತ್ಣ್ಋಸ್ಬಋ¯ = ನಿರ್ವವಾಸನಾರಹಿತವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮೌನಾತ್ಣ್ಋಸ್ಬಋ¯ = ಮೌನಕ್ಕಿಂತ ್ಣಋಸ್ಶ್ಯ್¯ಉತ್ತಮ್ಣಂಋಸ್ಬಋ¯ = ಶ್ರೇಷ್ಠವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸುಖಕೃತ್ಣ್ಋಸ್ಬಋ¯ = ಸುಖಕರವಾದದ್ದು ್ಣಋಸ್ಶ್ಯ್¯ಪರಂ ನ ಅಸ್ಣಿಋಸ್ಬಋ¯್ತ = ಬೇರೊಂದಿಲ್ಲವು.

526. ಆತ್ಮಸ್ವರೂಪವನ್ನು ತಿಳಿದುಕೊಂಡು ಆತ್ಮಾನಂದರಸವನ್ನು ಕುಡಿಯುತ್ತಿರುವವನಿಗೆ ಸರ್ವವಾಸನಾರಹಿತವಾದ ಮೌನಕ್ಕಿಂತ ಶ್ರೇಷ್ಠ-ಸುಖಕರವಾದದ್ದು ಬೇರೊಂದಿಲ್ಲ.

ಗಚ್ಛಂಸ್ತಿಷ್ಠನ್ನುಪವಿಶಂಛಯಾನೋ ವಾsವಾನ್ಯಥಾsಪಿ ವಾ ।
ಯಥೇಚ್ಛಯಾ ವಸೇದ್ವಿದ್ವಾನಾತ್ಮಾರಾಮಃ ಸದಾ ಮುನಿಃ ॥ 527 ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿದ್ವಾನ್ಣ್ಋಸ್ಬಋ¯ = ವಿದ್ವಾಂಸನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮಾರಾಮ್ಣಃಋಸ್ಬಋ¯ = ಆತ್ಮಾರಾಮನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮುನ್ಣಿಃಋಸ್ಬಋ¯ = ಮುನಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಗಚ್ಛನ್ಣ್ಋಸ್ಬಋ¯ = ನಡೆಯುತ್ತ ್ಣಋಸ್ಶ್ಯ್¯ತಿಷ್ಠನ್ಣ್ಋಸ್ಬಋ¯ = ನಿಲ್ಲುತ್ತ ್ಣಋಸ್ಶ್ಯ್¯ಉಪವಿಶನ್ಣ್ಋಸ್ಬಋ¯ = ಕುಳಿತುಕೊಳ್ಳುತ್ತ ್ಣಋಸ್ಶ್ಯ್¯ಶಯಾನಃ ವ್ಣಾಋಸ್ಬಋ¯ = ಮಲಗುತ್ತ ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ಯಥಾ ಅಪಿ ವಾ = ಅಥವಾ ಬೇರೆ ಸ್ಥಿತಿಯಲ್ಲಿ ಇರುತ್ತ ್ಣಋಸ್ಶ್ಯ್¯ಯಥೇಚ್ಛಯ್ಣಾಋಸ್ಬಋ¯ =ಸ್ವೇಚ್ಛೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ವಸೇತ್ಣ್ಋಸ್ಬಋ¯ = ಇರಬೇಕು.

527. ವಿದ್ವಾಂಸನೂ ಆತ್ಮಾರಾಮನೂ ಆದ ಮುನಿಯು ನಡೆಯುವಾಗಲೂ ನಿಲ್ಲುವಾಗಲೂ ಕುಳಿತುಕೊಳ್ಳವಾಗಲೂ ಮಲಗುವಾಗಲೂ ಅಥವಾ ಬೇರೆ ಯಾವ ಸ್ಥಿlತಿಯಲ್ಲಿರುವಾಗಲೂ ಸ್ವೇಚ್ಛೆಯಿಂದ್ 1 ಯಾವಾಗಲೂ ಇರಬೇಕು.

[1 ಪರತಂತ್ರನಾಗದೆ.]

ನ ದೇಶಕಾಲಾಸನದಿಗ್ಯಮಾದಿ-
ಲಕ್ಷ್ಯಾದ್ಯಪೇಕ್ಷಾsಪ್ರತಿಬದ್ಧವೃತ್ತೇಃ ।
ಸಂಸಿದ್ಧತತ್ತ್ವಸ್ಯ ಮಹಾತ್ಮನೋsಸ್ತಿ
ಸ್ವವೇದನೇ ಕಾ ನಿಯಮಾದ್ಯಪೇಕ್ಷಾ ॥ 528 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಪ್ರತಿಬದ್ಧವೃತ್ತ್ಣೇಃಋಸ್ಬಋ¯ = ಪ್ರತಿಬಂಧವಿಲ್ಲದ ಬುದ್ಧಿವೃತ್ತಿಯುಳ್ಳ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸಿದ್ಧ = ತತ್ತ್ವಸ್ಯ್ಣಋಸ್ಬಋ¯ =ಆತ್ಮಸ್ವರೂಪವನ್ನು ನಿಶ್ಚಯಪಡಿಸಿಕೊಂಡಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಮಹಾತ್ಮನ್ಣಃಋಸ್ಬಋ¯ = ಮಹಾತ್ಮನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದೇಶಕಾಲ-ಆಸನ-ದಿಕ್-ಲಕ್ಷ್ಯ-ಆತಿ-ಅಪೇಕ್ಷಾ = ದೇಶ ಕಾಲ ಆಸನ ದಿಕ್ಕು ಯಮಾದಿಗಳು, ಲಕ್ಷ್ಯವೇ ಮೊದಲಾದುವು_ಇವುಗಳ ಅಪೇಕ್ಷೆಯು ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ = ಇರುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ- ವೇದನ್ಣಋಸ್ಬಋ¯ೇ =ಸ್ವಾತ್ಮಸಾಕ್ಷಾತ್ಕಾರಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ನಿಯಮಾದಿ-ಅಪೇಕ್ಷ್ಣಾಋಸ್ಬಋ¯ = ನಿಯಮಾದಿಗಳು ಅಪೇಕ್ಷೆಯು ್ಣಋಸ್ಶ್ಯ್¯ಕಾ ಅಸ್ತ್ಣಿಋಸ್ಬಋ¯ =ಎಲ್ಲಿಯದು?

528. ಯಾವ ಪ್ರತಿಬಂಧವೂ ಇಲ್ಲದ ಬುದ್ಧಿವೃತ್ತಿಯುಳ್ಳ, ಆತ್ಮಸ್ವರೂಪವನ್ನು ನಿಶ್ಚಯಪಡಿಸಿಕೊಂಡಿರುವ ಮಹಾತ್ಮನಿಗೆ ದೇಶ1 ಕಾಲ ಆಸನ ದಿಕ್ಕು ಯಮನಿಯಮಾದಿಗಳು ಲಕ್ಷ್ಯಾದಿಗಳು2 – ಇವುಗಳ ಅಪೇಕ್ಷೆಯಿರುವುದಿಲ್ಲ. ತನ್ನ ಆತ್ಮನನ್ನು ಸಾಕ್ಷಾತ್ಕರಿಸಿಕೊಳ್ಳುವುದಕ್ಕೆ ನಿಯಮಾದಿಗಳ ಅಪೇಕ್ಷೆ ಎಲ್ಲಿಯದು?

[1 ಪವಿತ್ರಸ್ಥಾನಗಳು ; ಹೀಗೆಯೇ ಪವಿತ್ರವಾದ kಕಾಲ.

2. ಧ್ಯಾನಕ್ಕೆ ಬೇಕಾದ ಲಕ್ಷ್ಯಗಳು.]

ಘಟೋsಯಮಿತಿ ವಿಜ್ಞಾತುಂ ನಿಯಮಃ ಕೋ ನ್ವಪೇಕ್ಷ್ಯತೇ ।
ವಿನಾ ಪ್ರಮಾಣಸುಷ್ಠುತ್ವಂ ಯಸ್ಮಿನ್ಸತಿ ಪದಾರ್ಥಧೀಃ ॥ 529 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಮಿನ್ ಸತ್ಣಿಋಸ್ಬಋ¯ = ಯಾವುದಿದ್ದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಪದಾರ್ಥಧೀಃ = ಪದಾರ್ಥಜ್ಞಾನವು (ಉಂಟಾಗುವುದೊ ಅಂಥ) ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಮಾಣ-ಸುಷ್ಠುತ್ವಂ ವಿನಾ = ್ಣಋಸ್ಬಋ¯ಪ್ರಮಾಣಸೌಷ್ಠವವನ್ನ್ಣುಋಸ್ಶ್ಯ್¯ ಬಿಟ್ಟು ಅಯಂ = ್ಣಋಸ್ಬಋ¯ಇದ್ಣುಋಸ್ಶ್ಯ್¯ ಘಟಃ = ್ಣಋಸ್ಬಋ¯ಗಡಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಎಂದು ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾತುಂ = ತಿಳಿದುಕೊಳ್ಳುವುದಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಕಃ ನಿಯಮ್ಣಃಋಸ್ಬಋ¯ = ಯಾವ ನಿಯಮವು ್ಣಋಸ್ಶ್ಯ್¯ಅಪೇಕ್ಷ್ಯತೇ ನ್ಣುಋಸ್ಬಋ¯ = ಅಪೇಕ್ಷಿಸುಲ್ಪಡುತ್ತದೆ ?

529. ‘ಇದು ಗಡಿಗೆ’ಎಂದು ತಿಳಿದುಕೊಳ್ಳುವುದಕ್ಕೆ ಪ್ರಮಾಣ ಸೌಷ್ಠವನ್ನು ಬಿಟ್ಚು ಯಾವ ನಿಯಮವು ಬೇಕಾಗಿರುತ್ತದೆ ? ಅದಿದ್ದರೆ ಮಾತ್ರ ಪದಾರ್ಥಜ್ಞಾನವು ಉಂಟಾಗುತ್ತದೆ.

ಅಯಮಾತ್ಮಾ ನಿತ್ಯಸಿದ್ಧಃ ಪ್ರಮಾಣೇ ಸತಿ ಭಾಸತೇ ।
ನ ದೇಶಂ ನಾಪಿ ವಾ ಕಾಲಂ ನ ಶುದ್ಧಿಂ ವಾಪ್ಯಪೇಕ್ಷತೇ ॥ 530 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಯಮ್ ಆತ್ಮ್ಣಾಋಸ್ಬಋ¯ = ಈ ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯಸಿದ್ಧ್ಣಃಋಸ್ಬಋ¯ = ನಿತ್ಯಸಿದ್ಧಸ್ವರೂಪನು ; ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಮಾಣ್ಣೇಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಸತಿ =ಪ್ರಮಾಣವಿರುವಾಗ ್ಣಋಸ್ಶ್ಯ್¯ಭಾಸತ್ಣೇಋಸ್ಬಋ¯ = ಪ್ರಕಾಶಿಸುತ್ತಾನೆ ; ್ಣಋಸ್ಶ್ಯ್¯ದೇಶ್ಣಂಋಸ್ಬಋ¯ = ದೇಶವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಅಪೇಕ್ಷತ್ಣೇಋಸ್ಬಋ¯ =ಅಪೇಕ್ಷಿಸುವುದಿಲ್ಲ. ್ಣಋಸ್ಶ್ಯ್¯ಕಾಲಮ್ ಅಪಿ ವ್ಣಾಋಸ್ಬಋ¯ = ಕಾಲವನ್ನೂ ್ಣಋಸ್ಶ್ಯ್¯ನ, ಶುದ್ಧಿಂ ವ್ಣಾಋಸ್ಬಋ¯ = ಶುದ್ಧಿಯನ್ನೂ ್ಣಋಸ್ಶ್ಯ್¯ನ.್ಣಋಸ್ಬಋ¯

530. ಈ ಆತ್ಮನು ನಿತ್ಯಸಿದ್ಧ-ಸ್ವರೂಪನು ; ಪ್ರಮಾಣವಿರುವಾಗ ಪ್ರಕಾಶಸಿಸುತ್ತಾನೆ ; ದೇಶವನ್ನಾಗಲಿ ಕಾಲವನ್ನಾಗಲಿ ಶುದ್ಧಿಯನ್ನಾಗಲಿ ಅಪೇಕ್ಷಿಸುವುದಿಲ್ಲ.

(ಯಜ್ಞಾದಿ-ಕರ್ಮಗಳು ಕ್ರಿಯಾರೂಪವಾದುರಿಂದ ಅವುಗಳಿಗೆ ದೇಶಕಾಲ ಅಪೇಕ್ಷೆಯಿದೆ. ಆತ್ಮನು ನಿತ್ಯಸಿದ್ಧನಾದುದರಿಂದ ದೇಶಕಾಲಗಳ ಅಪೇಕ್ಷೆಯಿಲ್ಲದಿದ್ದರೂ ಅವನ ಅಭಿವ್ಯಕ್ತಿಗಾಗಿ ಪ್ರಮಾಣ-ಪ್ರವೃತ್ತಿಯ ಅಪೇಕ್ಷೆಯಿದೆ ; ಇಲ್ಲದಿದ್ದರೆ ಎಲ್ಲರೂ ಆತ್ಮಜ್ಞರಾಗಬೇಕಾಗಿತ್ತು.]

ದೇವದತೋsಹಮಿತ್ಯೇತದ್ವಿಜ್ಞಾನಂ ನಿರಪೇಕ್ಷಕಮ್ ।
ತದ್ವದ್ಬ್ರಹ್ಮವಿದೋsಪ್ಯಸ್ಯ ಬ್ರಹ್ಮಾಹಮಿತ ವೇದನಮ್ ॥ 531 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಋಸ್ಬಋ¯ಂ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ದೇವದತ್ತ್ಣಃಋಸ್ಬಋ¯ = ದೇವೇಂದ್ರನು ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಎಂಬ್ಣಋಸ್ಶ್ಯ್¯ ಏತತ್ ವಿಜ್ಞಾನ್ಣಂಋಸ್ಬಋ¯ = ಈ ಜ್ಞಾನವು ್ಣಋಸ್ಶ್ಯ್¯ನಿರಪೇಕ್ಷಕ್ಣಂಋಸ್ಬಋ¯ = ನಿರಪೇಕ್ಷಕವಾದುದು. ್ಣಋಸ್ಶ್ಯ್¯ತದ್ವತ್ಣ್ಋಸ್ಬಋ¯ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮವಿದ್ಣಃಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಅಪ್ಣಿಋಸ್ಬಋ¯ =ಬ್ರಹ್ಮಜ್ಞಾನಿಗೂ ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ್ಣಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮವು ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಎಂಬ್ಣಋಸ್ಶ್ಯ್¯ ವೇದನ್ಣಂಋಸ್ಬಋ¯ = ಜ್ಞಾನವು (ನಿರಪೇಕ್ಷಕವು )

531. ‘ನಾನು ದೇವದತ್ತನು’ ಎಂಬ ಜ್ಞಾನವು ಯಾವುದನೂ1 ಅಪೇಕ್ಷಿಸದೆ ಆಗುವುದು ; ಹಾಗೆಯೇ ಬ್ರಹ್ಮಜ್ಞಾನಿಗೂ ‘ನಾನು ಬ್ರಹ್ಮವು’ ಎಂಬ ಜ್ಞಾನವು (ಯಾವುದನ್ನೂ ಅಪೇಕ್ಷಿಸದೆ ಆಗುವುದು.)

[1 ದೇಶ-ಕಾಲಾದಿಗಳನ್ನು]

ಭಾನುನೇವ ಜಗತ್ಸರ್ವಂ ಭಾಸತೇ ಯಸ್ಯ ತೇಜಸಾ ।
ಅನಾತ್ಮಕಮಸತುಚ್ಛಂ ಕಿಂ ನು ತಸ್ಯಾವಭಾಸಕಮ್ ॥ 532 ॥
್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಂ ಜಗತ್ಣಋಸ್ಬಋ¯್ = ಸರ್ವಜಗತ್ತೂ ್ಣಋಸ್ಶ್ಯ್¯ಭಾನುನಾ ಇವ್ಣಋಸ್ಬಋ¯ಅ = ಸೂರ್ಯನಿಂದ ಹೇಗೋ ಹಾಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಯ್ಣಋಸ್ಬಋ¯ =ಯಾವ ್ಣಋಸ್ಶ್ಯ್¯ತೇಜಸ್ಣಾಋಸ್ಬಋ¯ = ತೇಜಸ್ಸಿನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್ಣ್ಋಸ್ಬಋ¯ = ಮಿಥ್ಯಾರೂಪವಾದ ್ಣಋಸ್ಶ್ಯ್¯ತುಚ್ಛ್ಣಂಋಸ್ಬಋ¯ = ತುಚ್ಛವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆನಾತ್ಮಕ್ಣಂಋಸ್ಬಋ¯ = ಅನಾತ್ಮವಸ್ತುವು ್ಣಋಸ್ಶ್ಯ್¯ಭಾಸತ್ಣೇಋಸ್ಬಋ¯ = ತೋರುತ್ತದೆಯೊ ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಣಋಸ್ಬಋ¯ಅ್ಯ = ಅವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಅವಭಾಸಕಂ = ಪ್ರಕಾಶವಾದದ್ದು ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂ ನು [ ಸ್ಯಾತ್ ] = ್ಣಋಸ್ಬಋ¯ಯಾವುದಿದ್ದೀತು ?
532. ಸೂರ್ಯನಿಂದ ಸರ್ವಜಗತ್ತೂ ಬೆಳಗುವ ಹಾಗೆ ಯಾವನ ತೇಜಸ್ಸಿನಿಂದ್ಣಋಸ್ಶ್ಯ್¯ ್ಣಋಸ್ಬಋ¯ಮಿಥ್ಯಾರೂಪವೂ ತುಚ್ಛವೂ ಅದ ಅನಾತ್ಮವಸ್ತುವು ತೋರುತ್ತದೆಯೊ ಅವನನ್ನು ಪ್ರಕಾಶಿಸುವುದು ಯಾವುದಿದ್ಧೀತು ?
ವೇದಶಾಸ್ತ್ರಪುರಾಣಾನಿ ಭೂತಾನಿ ಸಕಲಾನ್ಯಪಿ ।
ಯೇನಾರ್ಥವಂತಿ ತಂ ಕಿನ್ನು ವಿಜ್ಞಾತಾರಂ ಪ್ರಕಾಶಯೇತ್ ॥ 533 ॥
್ಣಋಸ್ಶ್ಯ್¯ವೇದ-ಶಾಸ್ತ್ರ-ಪುರಾಣಾನ್ಣಿಋಸ್ಬಋ¯ = ವೇದಶಾಸ್ತ್ರಪುರಾಣಗಳೂ ್ಣಋಸ್ಶ್ಯ್¯ಸಕಲಾನಿ ಭೂತಾನಿ ಅಪ್ಣಿಋಸ್ಬಋ¯ =ಭೂತಗಳೆಲ್ಲವೂ ್ಣಋಸ್ಶ್ಯ್¯ಯೇನ್ಣಋಸ್ಬಋ¯ = ಯಾವನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅರ್ಥವಂತ್ಣಿಋಸ್ಬಋ¯ = ಅರ್ಥವತ್ತಾಗುವುವೊ ್ಣಋಸ್ಶ್ಯ್¯ತ್ಣಂಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾತಾರಂ = ್ಣಋಸ್ಬಋ¯ಆ ವಿಜ್ಞಾತೃವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂ ನು = ್ಣಋಸ್ಬಋ¯ಯಾವುದು ್ಣಋಸ್ಶ್ಯ್¯ತಾನೇ ಪ್ರಕಾಶಯೇತ್ =ಪ್ರಕಾಶಪಡಿಸೀತು ?

533. ವೇದ-ಶಾಸ್ತ್ರ-ಪುರಾಣಗಳೂ ಸಮಸ್ತ ಭೂತಗಳೂ ಯಾವುದರಿಂದ ಅರ್ಥವತ್ತಾಗುವುವೊ ಆ ವಿಜ್ಞಾತೃವನ್ನು ಯಾವುದು ತಾನೇ ಪ್ರಕಾಶಪಡಿಸೀತು ?1

[1 ಯಾವನಿಂದ ಇದೆಲ್ಲವನ್ನೂ ತಿಳಿದುಕೊಳ್ಳುತ್ತಾನೆಯೊ ಅವನನ್ನು ಯಾವುದರಿಂದ ತಿಳಿದುಕೊಂಡಾನು ? ಎಲೆ ಮೈತ್ರೇಯಿ, ವಿಜ್ಞಾತೃವನ್ನು ಯಾವುದರಿಂದ ತಿಳಿದುಕೊಂಡಾನು ? ್ಣಋಸ್ಶ್ಯ್¯ಯೇನೇದಂ ಸರ್ವಂ ವಿಜಾನಾತಿ ತಂ ಕೇನ ವಿಜಾನೀಯಾತ್ ? ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾತಾರಮರೇ ಕೇನ ವಿಜಾನೀಯಾದಿತ್ಣಿಋಸ್ಬಋ¯ (ಬೃಹದಾರಣ್ಯಕ ಉ.2 , 4, 14 )
ಏಷ ಸ್ವಯಂಜ್ಯೋತಿರನಂತಶಕ್ತಿ-
ಅ ರಾತ್ಮಾsಪ್ರಮೇಯಃ ಸಕಲಾನುಭೂತಿಃ ।
ಯಮೇವ ವಿಜ್ಞಾಯ ವಿಮುಕ್ತಬಂಧೋ
ಜಯತ್ಯಯಂ ಬ್ರಹ್ಮವಿದುತ್ತಮೋತ್ತಮಃ ॥ 534 ॥
್ಣಋಸ್ಶ್ಯ್¯ಏಷಃ ಆತ್ಮಾ = ಈ ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಂಜ್ಯೋತ್ಣಿಃಋಸ್ಬಋ¯ = ಸ್ವಯಂಪ್ರಕಾಶನು ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ಣಂಋಸ್ಬಋ¯ತ ್ಣಋಸ್ಶ್ಯ್¯ಶಕ್ತಿಃ =ಅನಂತಶಕ್ತಿಯುಳ್ಳವನು ್ಣಋಸ್ಶ್ಯ್¯ಅಪ್ರಮೇಯ್ಣಃಋಸ್ಬಋ¯ = ಅಪ್ರಮೇಯನು ್ಣಋಸ್ಶ್ಯ್¯ಸಕಲಾನುಭೂತ್ಣಿಃಋಸ್ಬಋ¯ =
ಸರ್ವಾನುಭವಿಸಿದ್ಧನು ; ್ಣಋಸ್ಶ್ಯ್¯ಯಮ್ ಏವ್ಣಋಸ್ಬಋ¯ಅ = ಯಾವನನ್ನೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾಯ್ಣಋಸ್ಬಋ¯ = ತಿಳಿದುಕೊಂಡು ್ಣಋಸ್ಶ್ಯ್¯ವಿಮುಕ್ತಬಂಧ್ಣಃಋಸ್ಬಋ¯ = ಬಂಧದಿಂದ ಮುಕ್ತನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಇವನು ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮವಿತ್-ಉತ್ತಮೋತ್ತಮ್ಣಃಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಜ್ಞಾನಿಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವೋತ್ತಮನಾಗಿ ಜಯತಿ =
ಸರ್ವೋತ್ಕೃಷ್ಟನಾಗಿರುತ್ತಾನೆ.

534. ಈ ಆತ್ಮನು ಸ್ವಯಂಪ್ರಕಾಶನು, ಅನಂತಶಕ್ತಿಯುಳ್ಳವನು, ಅಪ್ರಮೇಯನಾಗಿದ್ದರೂ ಎಲ್ಲರ ಅನುಭವಕ್ಕೂ ಗೋಚರನಾದವನು ; ಅವನನ್ನೇ ಅರಿತುಕೊಂಡು ಸಂಸಾರಬಂಧದಿಂದ ಮುಕ್ತನಾದ ಇವನು ಬ್ರಹ್ಮವಿದರಲ್ಲಿ ಸರ್ವೋತ್ತಮನಾಗಿ ಸರ್ವೋತ್ಕೃಷ್ಟನಾಗಿರುತ್ತಾನೆ.

ಸ್ವಸ್ಮಿನ್ಸದಾ ಕ್ರೀಡತಿ ನಂದತಿ ಸ್ವಯಂ
ನಿರಂತರಾನಂದರಸೇನ ತೃಪ್ತಃ ॥ 535 ॥
(ಇವನು) ನ ಖಿದ್ಯತೇ = ದುಃಖಿಸುವದಿಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ವಿಷಯ್ಣೈಃಋಸ್ಬಋ¯ = ವಿಷಯಗಳಿಂದ್ಣಋಸ್ಶ್ಯ್¯ ನ ಉ = ಸಂತೋಷಪಡುವುದಿಲ್ಲ ; (ಅವುಗಳಲ್ಲಿ) ನ ಸಜ್ಜತ್ಣೇಋಸ್ಬಋ¯ = ಆಸಕ್ತನಾಗುವುದಿಲ್ಲ. ್ಣಋಸ್ಶ್ಯ್¯ನ ಅಪ್ಣಿಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ವಿರಜ್ಯತೇ ಚ್ಣಋಸ್ಬಋ¯ = ವಿರಕ್ತನಾಗುವುದೂ ಇಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ನಿರಂತರ-ಆನಂದ-ರಸೇನ = ನಿತ್ಯಾನಂದರಸದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತೃಪ್ತ್ಣಃಋಸ್ಬಋ¯ = ತೃಪ್ತನಾಗ್ಣಿಋಸ್ಶ್ಯ್¯ ಸದಾ = ್ಣಋಸ್ಬಋ¯ಯಾವಾಗಲ್ಣೂಋಸ್ಶ್ಯ್¯ ಸ್ವಯಂ = ತಾನೇ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಸ್ಮಿನ್ ಪ್ ತನ್ನಲ್ಲಿಯೇ ಕ್ರೀಡತ್ಣಿಋಸ್ಬಋ¯ = ಕ್ರೀಡಿಸುತ್ತಾನೆ, ್ಣಋಸ್ಶ್ಯ್¯ನಂದತ್ಣಿಋಸ್ಬಋ¯ = ಆನಂದಿಸುತ್ತಾನೆ.

535. ಅವನು (ವಿಷಯ-ವಿಯೋಗದಿಂದ) ದುಃಖಿಸುವುದಿಲ್ಲ. ವಿಷಯಗಳಿಂದ ಸಂತೋಷಪಡುವುದಿಲ್ಲ ; (ಸುಖ-ಸಾಧನಗಳಲ್ಲಿ) ಆಸಕ್ತನಾಗುವುದೂ ಇಲ್ಲ. ವಿರಕ್ತನಾಗುವುದೂಇಲ್ಲ. ನಿತ್ಯಾನಂದರಸದಿಂದ ತೃಪ್ತನಾಗಿ ಯಾವಾಗಲೂ ತಾನೇ ತನ್ನಲ್ಲಿಯೇ ಕ್ರೀಡಿಸುತ್ತಾನೆ, ಆನಂದಪಡುತ್ತಾನೆ.

ಕ್ಷುಧಾಂ ದೇಹವ್ಯಥಾಂ ತ್ಯಕ್ತ್ವಾ ಬಾಲಃ ಕ್ರೀಡತಿ ವಸ್ತುನಿ ।
ತಥೈವ ವಿದ್ವಾನ್ ರಮತೇ ನಿರ್ಮಮೋ ನಿರಹಂ ಸುಖೀ ॥ 53್ಣಋಸ್ಶ್ಯ್¯6 ॥
್ಣಋಸ್ಶ್ಯ್¯ಬಾnಲಃ = ಬಾಲಕನು ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಷುಧ್ಣಾಂಋಸ್ಬಋ¯ = ಹಸಿವನ್ನೂ ್ಣಋಸ್ಶ್ಯ್¯ದೇವವ್ಯಥ್ಣಾಂಋಸ್ಬಋ¯ = ದೇಹಾಯಾಸದ ವ್ಯಥೆಯನ್ನ್ಣೂಋಸ್ಶ್ಯ್¯ ತ್ಯಕ್ತ್ವಾ್ಣಋಸ್ಬಋ¯ = ಬಿಟ್ಟು ್ಣಋಸ್ಶ್ಯ್¯ವಸ್ತುನಿ = (ಆಟದ) ವಸ್ತುವಿನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ರೀಡತ್ಣಿಋಸ್ಬಋ¯ = ಆಡುತ್ತಾನೆ. ್ಣಋಸ್ಶ್ಯ್¯ತಥಾ ಏವ್ಣಋಸ್ಬಋ¯ಅ =ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿದ್ವಾನ್ಣ್ಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಜ್ಞಾನಿಯು ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ಮಮ್ಣಃಋಸ್ಬಋ¯ = ಮಮಕಾರವಿಲ್ಲದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರಹ್ಣಂಋಸ್ಬಋ¯ =ಅಹಂಕಾರವಿಲ್ಲದೆ ್ಣಋಸ್ಶ್ಯ್¯ಸುಖ್ಣಿಋಸ್ಬಋ¯| = ಸುಖಿಯಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ರಮತ್ಣೇಋಸ್ಬಋ¯ = ರಮಿಸುತ್ತಾನೆ.

536. ಬಾಲಕನು ಹಸಿವನ್ನೂ ದೇಹಾಯಾಸದ ವ್ಯಥೆಯನ್ನೂ ಮರೆತು ತನ್ನ ಆಟದ ವಸ್ತುವಿನೊಂದಿಗೆ ಆಡುವಂತೆ ಬ್ರಹ್ಮಜ್ಞಾನಿಯು ಅಹಂಕಾರ ಮಮಕಾರಗಳಿಲ್ಲದೆ ಸುಖಿಯಾಗಿ (ತನ್ನ ಆತ್ಮನಲ್ಲಿ) ರಮಿಸುತ್ತಾನೆ.
