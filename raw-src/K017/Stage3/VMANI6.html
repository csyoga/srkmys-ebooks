142. ಸೂರ್ಯನ ಕಿರಣಗಳಿಂದ ಉಂಟಾದ ಮೇಘಪಂಕ್ತಿಯು ಸೂರ್ಯ ನನ್ನೇ ಮರೆಮಾಡಿ ಹೇಗೆ ಮೆರೆಯುವುದೊ ಹಾಗೆಯೇ ಆತ್ಮನಿಂದಲೇ ಉತ್ಪನ್ನ ವಾದ ಅಹಂಕಾರವೂ ಕೂಡ ಆತ್ಮಸ್ವರೂಪವನ್ನು ಮರೆಮಾಡಿ ತಾನೇ ಮೆರೆಯು ತ್ತದೆ.

ವ್ಯRಥಯತಿ ಹಿಮ-ಝಂರst-ವಾಯುರುಗ್ರೋ ಯಥೈತಾನ್ ।
ಅವಿರತ-ತಮಸಾತ್ಮನ್ಯಾವೃತೇ ಮೂಢಬುದ್ಧಿಂ

್ಣಋಸ್ಶ್ಯ್¯ಸಾಂದ್ರಮೇಘೈಃ = ನಿಬಿಡವಾದ ಮೇಘಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಕಬಲಿತ-ದಿನನಾಥ್ಣೇಋಸ್ಬಋ¯ = ನುಂಗಲ್ಪಟ್ಟ ಸೂರ್ಯನುಳ್ಳ ್ಣಋಸ್ಶ್ಯ್¯ದುರ್ದಿನೇ = ದುರ್ದಿನದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಉಗ್ರಃ = ಉಗ್ರವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಹಿಮ-ಝಂರst- ವಾಯ್ಣುಃಋಸ್ಬಋ¯ = ಶೀತಲವಾದ ಬಿರುಗಾಳಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಏತಾನ್ಣ್ಋಸ್ಬಋ¯ = ಈ ಜನರನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಯಥ್ಣಾಋಸ್ಬಋ¯ = ಹೇಗೆ ್ಣಋಸ್ಶ್ಯ್¯ವ್ಯಥಯತ್ಣಿಋಸ್ಬಋ¯ = ವ್ಯಥೆಗೊಳಿಸುವುದೊ [ಹಾಗೆಯೇ], ್ಣಋಸ್ಶ್ಯ್¯ಅವಿರತ-ತಮಸಾ ಯಥ್ಣಾಋಸ್ಬಋ¯ = ನಿಬಿಡ ವಾದ ಅಜ್ಞಾನಾಂಧಕಾರದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನಿ ಆವೃತೇ [ಸತಿ] = ಆತ್ಮನು ಆವರಿಸಲ್ಪಟ್ಟಾಗ ್ಣಋಸ್ಶ್ಯ್¯ತೀವ್ರವಿಕ್ಷೇಪಶಕ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ತೀವ್ರವಾದ ವಿಕ್ಷೇಪಶಕ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಬಹುದುಃಖ್ಣೈಃಋಸ್ಬಋ¯ = ಅನೇಕ ವ್ಯಸನ ಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಮೂಢಬುದ್ಧ್ಣಿಂಋಸ್ಬಋ¯ = ಮೂಡಬುದ್ಧಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಷಪಯತ್ಣಿಋಸ್ಬಋ¯ = ವ್ಯಥೆಗೊಳಿಸುತ್ತದೆ.

್ಣಋಸ್ಶ್ಯ್¯ಏತಾಭ್ಯಾಂ ಶಕ್ತಿಭ್ಯಾಮ್ ಏವ = ಈ ಎರಡು ಶಕ್ತಿಗಳಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಪುಂಸ್ಣಃಋಸ್ಬಋ¯ = ಮನುಷ್ಯ ನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಬಂಧ್ಣಃಋಸ್ಬಋ¯ = ಬಂಧವು ್ಣಋಸ್ಶ್ಯ್¯ಸಮಾಗತ್ಣಃಋಸ್ಬಋ¯ = ಬಂದಿರುವುದು ; ್ಣಋಸ್ಶ್ಯ್¯ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಇವನ್ಣುಋಸ್ಶ್ಯ್¯ ಯಾಭ್ಯ್ಣಾಂಋಸ್ಬಋ¯ = ಯಾವ ಶಕ್ತಿಗಳಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿಮೋಹಿತಃ [ಸನ್] = ವಿಮೋಹಿತನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹ್ಣಂಋಸ್ಬಋ¯ = ದೇಹ ವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮಾನಂ ಮತ್ವಾ = ಆತ್ಮನೆಂದು ತಿಳಿದು ್ಣಋಸ್ಶ್ಯ್¯ಭ್ರಮತ್ಣಿಋಸ್ಬಋ¯ = ಅಲೆಯುತ್ತಾನೆ.

37. ಬಂಧನಿರೂಪಣ
ಬೀಜಂ ಸಂಸೃತಿ-ಭೂಮಿಜಸ್ಯ ತು ತಮೋ ದೇಹಾತ್ಮಧೀರಂಕುರೋ

ಅಗ್ರಾಣೀಂದ್ರಿಯ-ಸಂಹತಿಶ್ಚ ವಿಷಯಾಃ ಪುಷ್ಪಾಣಿ ದುಃಖಂ ಫಲಂ

ವಿವೇಕವಿಜ್ಞಾನ-ಮಹಾsಸಿನಾ ವಿನಾ
ಧಾತುಃ ಪ್ರಸಾದೇನ ಶಿತೇನ ಮಂಜುನಾ ॥ 147 ॥
[ಈ ಬಂಧವು] ್ಣಋಸ್ಶ್ಯ್¯ಧಾತುಃ ಪ್ರಸಾದೇನ್ಣಋಸ್ಬಋ¯ = ಪರಮಾತ್ಮನ ಅನುಗ್ರಹದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಶಿತೇನ್ಣಋಸ್ಬಋ¯ = ತೀಕ್ಷ್ಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮಂಜುನ್ಣಾಋಸ್ಬಋ¯ = ಮನೋಹರವಾದ ್ಣಋಸ್ಶ್ಯ್¯ವಿವೇಕವಿಜ್ಞಾನ-ಮಹಾ-ಅಸಿನಾ ವಿನ್ಣಾಋಸ್ಬಋ¯ = ವಿವೇಕವಿಜ್ಞಾನವೆಂಬ ಮಹಾಖಡ್ಗದಿಂದಲ್ಲದೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ತ್ರೆ್ಣೈಃಋಸ್ಬಋ¯ = ಅಸ್ತ್ರಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಶಸ್ತ್ರೆ್ಣೈಃಋಸ್ಬಋ¯ = ಶಸ್ತ್ರ ಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅನಿಲೇನ್ಣಋಸ್ಬಋ¯ = ಗಾಳಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಹ್ನಿನ್ಣಾಋಸ್ಬಋ¯ = ಬೆಂಕಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಛೇತ್ತ್ಣುಂಋಸ್ಬಋ¯ = ಕತ್ತರಿಸಲು ್ಣಋಸ್ಶ್ಯ್¯ನ ಶಕ್ಯ್ಣಃಋಸ್ಬಋ¯ = ಸಾಧ್ಯವಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ಮಕೋಟಿಭಿಃ ಚ್ಣಋಸ್ಬಋ¯ = ಕರ್ಮಗಳ ಕೋಟಿಗಳಿಂದಾಗಲಿ [ಕತ್ತರಿಸಲಾಗು ವುದಿಲ್ಲ].

147. ಈ ಬಂಧವನ್ನು ಅಸ್ತ್ರಗಳಿಂದಾಗಲಿ ಶಸ್ತ್ರಗಳಿಂದಾಗಲಿ ಗಾಳಿಯಿಂದಾಗಲಿ ಬೆಂಕಿಯಿಂದಾಗಲಿ ಕತ್ತರಿಸಲು ಸಾಧ್ಯವಿಲ್ಲ ; ಕೋಟಿಕರ್ಮಗಳಿಂದಲೂ ಸಾಧ್ಯ ವಿಲ್ಲ ; ಪರಮಾತ್ಮನ ಅನುಗ್ರಹದಿಂದ1 ತೀಕ್ಷ್ಣವಾದ ಮನೋಹರವಾದ ವಿವೇಕ ವಿಜ್ಞಾನವೆಂಬ ಮಹಾಖಡ್ಗದಿಂದಲೇ ಸಾಧ್ಯ.

[1 ‘ಧಾತುವಿನ ಅನುಗ್ರಹದಿಂದ ಆತ್ಮನ ಮಹಿಮೆಯನ್ನು ನೋಡುತ್ತಾನೆ.’ ್ಣಋಸ್ಶ್ಯ್¯ಧಾತು ಪ್ರಸಾದಾನ್ಮಹಿಮಾನಮಾತ್ಮನ್ಣಃಋಸ್ಬಋ¯ ಎಂದು ಶ್ರುತಿಯು (ಕಠ ಉ. 1. 2. 20) ಹೇಳುತ್ತದೆ.]
ಶ್ರುತಿಪ್ರಮಾಣೈಕಮತೇಃ ಸ್ವಧರ್ಮ-
ನಿಷ್ಠಾ ತಯೈವಾತ್ಮಮವಿಶುದ್ಧಿರಸ್ಯ ।
ವಿಶುದ್ಧಬುದ್ಧೇಃ ಪರಮಾತ್ಮವೇದನಂ
ತೇನೈವ ಸಂಸಾರ-ಸಮೂಲ-ನಾಶಃ ॥ 148 ॥
್ಣಋಸ್ಶ್ಯ್¯ಶ್ರುತಿಪ್ರಮಾಣಃ-ಏಕಮತ್ಣೇಃಋಸ್ಬಋ¯ = ಶ್ರುತಿಪ್ರಮಾಣದಲ್ಲಿಯೇ ಬುದ್ಧಿಯುಳ್ಳವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ ಧರ್ಮನಿಷ್ಣಾಋಸ್ಬಋ¯್ಠ = ಸ್ವಧರ್ಮದಲ್ಲಿ ನಿಷ್ಠೆಯು [ಉಂಟಾಗುತ್ತದೆ] ; ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಇವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ತಯಾ ಏವ್ಣಋಸ್ಬಋ¯ = ಅದರಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮವಿಶುದ್ಧ್ಣಿಃಋಸ್ಬಋ¯ = ಚಿತ್ತಶುದ್ಧಿಯು ; ್ಣಋಸ್ಶ್ಯ್¯ವಿಶುದ್ಧಬುದ್ಧ್ಣೇಃಋಸ್ಬಋ¯ = ಚಿತ್ತಶುದ್ಧಿ ಯನ್ನು ಹೊಂದಿದವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮಾತ್ಮ-ವೇದನ್ಣಂಋಸ್ಬಋ¯ = ಪರಮಾತ್ಮಜ್ಞಾನವು ; ್ಣಋಸ್ಶ್ಯ್¯ತೇನ ಏವ = ಅದರಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸಾರ-ಸಮೂಲ-ನಾಶಃ = ಬೇರುಸಹಿತ ಸಂಸಾರನಾಶ.

148. ಶ್ರುತಿಪ್ರಮಾಣದಲ್ಲಿಯೇ1 ಬುದ್ಧಿಯನ್ನಿಟ್ಟುಕೊಂಡಿರುವವನಿಗೆ ಸ್ವಧರ್ಮದಲ್ಲಿ ನಿಷ್ಠೆಯುಂಟಾಗುವುದು. ಅದರಿಂದಲೇ ಅವನಿಗೆ ಚಿತ್ತಶುದ್ಧಿ ಯುಂಟಾಗುವುದು. ಚಿತ್ತಶುದ್ಧಿಯನ್ನು ಹೊಂದಿದವನಿಗೆ ಪರಮಾತ್ಮಜ್ಞಾನವು ಉಂಟಾಗುವುದು. ಅದರಿಂದಲೇ ಸಂಸಾರವು ನಿರ್ಮೂಲವಾಗಿ ನಾಶವಾಗುತ್ತದೆ.

[1 ‘ಕಾರ್ಯ, ಅಕಾರ್ಯ–ಇವುಗಳ ನಿರ್ಧಾರಣೆಯಲ್ಲಿ ಶಾಸ್ತ್ರವೇ ನಿನಗೆ ಪ್ರಮಾಣ’ ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಮಾಚ್ಛಾಸ್ತ್ರಂ ಪ್ರಮಾಣಂ ತೇ ಕಾರ್ಯಾಕಾರ್ಯವ್ಯವಸ್ಥಿತ್ಣೌಋಸ್ಬಋ¯ ಎಂದು ಭಗವಂತನು (ಗೀತಾ 16. 24) ಹೇಳಿದ್ದಾನೆ.]
ಕೋಶೈರನ್ನಮಯಾದ್ಯೈಃ ಪಂಚಭಿರಾತ್ಮಾ ನ ಸಂವೃತೋ ಭಾತಿ ।
ನಿಜಶಕ್ತಿ-ಸಮುತ್ಪನ್ನೈಃ ಶೈವಾಲ-ಪಟಲೈರಿವಾಂಬು ವಾಪಿಸ್ಥಮ್ ॥ 149 ॥
್ಣಋಸ್ಶ್ಯ್¯ಶೈವಾಲ-ಪಟಲ್ಣೈಃಋಸ್ಬಋ¯ = ಪಾಚಿಯ ಪದರಗಳಿಂದ [ಮುಚ್ಚಲ್ಪಟ್ಟ] ್ಣಋಸ್ಶ್ಯ್¯ವಾಪಿಸ್ಥ್ಣಂಋಸ್ಬಋ¯ = ಕೊಳ ದಲ್ಲಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಅಂಬು ಇವ್ಣಋಸ್ಬಋ¯ = ನೀರಿನಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ನಿಜಶಕ್ತಿ-ಸಮುತ್ಪನ್ನ್ಣೈಃಋಸ್ಬಋ¯ = ತನ್ನ ಶಕ್ತಿಯಿಂದ ಉಂಟಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ನಮಯಾದ್ಯ್ಣೈಃಋಸ್ಬಋ¯ = ಅನ್ನಮಯವೇ ಮೊದಲಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪಂಚಭಿಃ ಕೋಶ್ಣೈಃಋಸ್ಬಋ¯ = ಪಂಚಕೋಶಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸಂವೃತ್ಣಃಋಸ್ಬಋ¯ = ಮುಚ್ಚಲ್ಪಟ್ಟ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮ್ಣಾಋಸ್ಬಋ¯ = ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ನ ಭಾತ್ಣಿಋಸ್ಬಋ¯ = ಕಾಣಿಸಿ ಕೊಳ್ಳುವುದಿಲ್ಲ.

149. ಪಾಚಿಯ ಪದರಗಳಿಂದ ಮುಚ್ಚಲ್ಪಟ್ಟ ಕೊಳದಲ್ಲಿ ನೀರು ಕಾಣದೆ ಇರುವಂತೆ ತನ್ನ ಶಕ್ತಿಯಿಂದಲೇ ಉಂಟಾಗಿರುವ ಅನ್ನಮಯಾದಿ ಪಂಚಕೋಶ ಗಳಿಂದ ಮುಚ್ಚಲ್ಪಟ್ಟ ಆತ್ಮನು ಕಾಣಿಸಿಕೊಳ್ಳದೆ ಇರುತ್ತಾನೆ.

ತಚ್ಛೈವಾಲಾಪನಯೇ ಸಮ್ಯಕ್ ಸಲಿಲಂ ಪ್ರತೀಯತೇ ಶುದ್ಧಮ್ ।
ತೃಷ್ಣಾ-ಸಂತಾಪಹರಂ ಸದ್ಯಃ ಸೌಖ್ಯಪ್ರದಂ ಪರಂ ಪುಂಸಃ ॥ 150 ॥
್ಣಋಸ್ಶ್ಯ್¯ತತ್-ಶೈವಾಲ-ಅಪನಯ್ಣೇಋಸ್ಬಋ¯ = [ನೀರನ್ನು ಮುಚ್ಚಿಕೊಂಡಿರುವ] ಆ ಪಾಚಿಯನ್ನು ತೆಗೆದಾಗ ್ಣಋಸ್ಶ್ಯ್¯ಪುಂಸ್ಣಃಋಸ್ಬಋ¯ = ಮನುಷ್ಯನ ್ಣಋಸ್ಶ್ಯ್¯ತೃಷ್ಣಾ-ಸಂತಾಪಹರ್ಣಂಋಸ್ಬಋ¯ = ತೃಷ್ಣೆಯ ಸಂಕಟವನ್ನು ಹೋಗಲಾಡಿಸುವ, ್ಣಋಸ್ಶ್ಯ್¯ಸದ್ಯ್ಣಃಋಸ್ಬಋ¯ = ಕೂಡಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಪರಂ ಸೌಖ್ಯಪ್ರದ್ಣಂಋಸ್ಬಋ¯ = ಅತ್ಯಂತ ಸುಖಪ್ರದವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಶುದ್ಧ್ಣಂಋಸ್ಬಋ¯ = ನಿರ್ಮಲವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸಲಿಲ್ಣಂಋಸ್ಬಋ¯ = ನೀರು ್ಣಋಸ್ಶ್ಯ್¯ಸಮ್ಯಕ್ಣ್ಋಸ್ಬಋ¯ = ಚೆನ್ನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತೀಯತ್ಣೇಋಸ್ಬಋ¯ = ಕಾಣಬರುವುದು.

150. (ನೀರನ್ನು ಮುಚ್ಚಿಕೊಂಡಿರುವ) ಆ ಪಾಚಿಯನ್ನು ತೆಗೆದರೆ ಮನುಷ್ಯನ ತೃಷ್ಣೆಯ ಸಂಕಟವನ್ನು ಹೋಗಲಾಡಿಸುವ, ಕೂಡಲೇ ಅತ್ಯಂತ ಸುಖವನ್ನು ಉಂಟುಮಾಡುವ, ನಿರ್ಮಲವಾದ ನೀರು ಚೆನ್ನಾಗಿ ಕಾಣಬರುವುದು.

[ನೀರನ್ನು ಎಲ್ಲಿಂದಲೂ ತರಬೇಕಾದ ಆವಶ್ಯಕತೆಯಿಲ್ಲ. ನೀರು ಅಲ್ಲೇ ಇದೆ. ಅದರ ಮೇಲಿರುವ ಆವರಣವನ್ನು ಮಾತ್ರ ತೆಗೆಯಬೇಕು. ಹೀಗೆಯೇ ಆತ್ಮನ ವಿಷಯದಲ್ಲಿಯೂ ಅರಿಯಬೇಕು.]
ಪಂಚಾನಾಮಪಿ ಕೋಶಾನಾಮಪವಾದೇ ವಿಭಾತ್ಯಯಂ ಶುದ್ಧಃ ।
ನಿತ್ಯಾನಂದೈಕರಸಃ ಪ್ರತ್ಯಗ್ರೂಪಃ ಪರಂ ಸ್ವಯಂಜ್ಯೋತಿಃ ॥ 151 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪಂಚಾನಾಮ್ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಪಂಚವಿಧವಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಕೋಶಾನ್ಣಾಂಋಸ್ಬಋ¯ = ಕೋಶಗಳ ್ಣಋಸ್ಶ್ಯ್¯ಅಪವಾದ್ಣೇಋಸ್ಬಋ¯ = ತೆಗೆಯಲ್ಪಟ್ಟಾಗ ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯಾನಂದೈಕರಸ್ಣಃಋಸ್ಬಋ¯ = ನಿತ್ಯಾನಂದೈಕರಸನೂ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತ್ಯಕ್ರೂಪ್ಣಃಋಸ್ಬಋ¯ = ಸರ್ವಾಂತರನೂ ್ಣಋಸ್ಶ್ಯ್¯ಪರ್ಣಂಋಸ್ಬಋ¯ = ಸರ್ವೋತ್ಕೃಷ್ಟನೂ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯಂಜ್ಯೋತ್ಣಿಃಋಸ್ಬಋ¯ = ಸ್ವಯಂಪ್ರಕಾಶನೂ ್ಣಋಸ್ಶ್ಯ್¯ಶುದ್ಧ್ಣಃಋಸ್ಬಋ¯ = ಶುದ್ಧನೂ [ಆದ] ್ಣಋಸ್ಶ್ಯ್¯ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಈ ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ವಿಭಾತ್ಣಿಋಸ್ಬಋ¯ = ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತಾನೆ.

151. ಅನ್ನಮಯವೇ ಮೊದಲಾದ ಪಂಚಕೋಶಗಳನ್ನು ತೆಗೆದು ಹಾಕಿದರೆ ನಿತ್ಯಾನಂದೈಕರಸನೂ ಸರ್ವಾಂತರನೂ ಸರ್ವೋತ್ಕೃಷ್ಟನೂ ಸ್ವಯಂಪ್ರಕಾಶನೂ ಶುದ್ಧನೂ ಆದ ಈ ಆತ್ಮನು ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತಾನೆ.

ಆತ್ಮಾsನಾತ್ಮವಿವೇಕಃ ಕರ್ತವ್ಯೋ ಬಂಧಮುಕ್ತಯೇ ವಿದುಷಾ ।
ತೇನೈವಾನಂದೀ ಭವತಿ ಸ್ವಂ ವಿಜ್ಞಾಯ ಸಚ್ಚಿದಾನಂದಮ್ ॥ 152 ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿದುಷ್ಣಾಋಸ್ಬಋ¯ = ಪಂಡಿತನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಬಂಧಮುಕ್ತಯ್ಣೇಋಸ್ಬಋ¯ = ಬಂಧದ ಬಿಡುಗಡೆಗಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮ- ಅನಾತ್ಮ-ವಿವೇಕಃ = ಆತ್ಮ-ಅನಾತ್ಮಗಳ ವಿವೇಚನವು ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ತವ್ಯ್ಣಃಋಸ್ಬಋ¯ = ಮಾಡತಕ್ಕದ್ದು ; ್ಣಋಸ್ಶ್ಯ್¯ತೇನ ಏವ್ಣಋಸ್ಬಋ¯ = ಅದರಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಂ ಸಚ್ಚಿದಾನಂದ್ಣಂಋಸ್ಬಋ¯ = ತನ್ನ ಸಚ್ಚಿದಾನಂದ ಸ್ವರೂಪವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾಯ್ಣಋಸ್ಬಋ¯ = ಅರಿತುಕೊಂಡು ್ಣಋಸ್ಶ್ಯ್¯ಆನಂದೀ ಭವತ್ಣಿಋಸ್ಬಋ¯ = ಆನಂದಿಯಾಗುತ್ತಾನೆ.

152. ಪಂಡಿತನಾದವನು ಬಂಧದಿಂದ ಬಿಡುಗಡೆಯನ್ನು ಹೊಂದಲು ಆತ್ಮ, ಅನಾತ್ಮ ಇವುಗಳ ವಿವೇಕವನ್ನು ಮಾಡಿಕೊಳ್ಳಬೇಕು. ಆ ವಿವೇಕದಿಂದಲೇ ತನ್ನ ಸಚ್ಚಿದಾನಂದಸ್ವರೂಪವನ್ನು ತಿಳಿದುಕೊಂಡು ಆನಂದಿಯಾಗುತ್ತಾನೆ.

ಮುಂಜಾದಿಷೀಕಾಮಿವ ದೃಶ್ಯವರ್ಗಾತ್
ಪ್ರತ್ಯಂಚಮಾತ್ಮಾನಮಸಂಗಮಕ್ರಿಯಮ್ ।
ವಿವಿಚ್ಯ ತತ್ರ ಪ್ರವಿಲಾಪ್ಯ ಸರ್ವಂ
ತದಾತ್ಮನಾ ತಿಷ್ಠತಿ ಯಃ ಸ ಮುಕ್ತಃ ॥ 153 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣಃಋಸ್ಬಋ¯ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಮುಂಜಾತ್ಣ್ಋಸ್ಬಋ¯ = ಮುಂಜವೆಂಬ ಹುಲ್ಲಿನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಇಷೀಕಾಮ್ ಇವ್ಣಋಸ್ಬಋ¯ = ದಂಟನ್ನು ಬೇರ್ಪಡಿಸುವಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ದೃಶ್ಯವರ್ಗಾತ್ಣ್ಋಸ್ಬಋ¯ = ದೃಶ್ಯಸಮೂಹದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅಸಂಗ್ಣಂಋಸ್ಬಋ¯ = ಸಂಗರಹಿತನೂ ್ಣಋಸ್ಶ್ಯ್¯ಅಕ್ರಿಯ್ಣಂಋಸ್ಬಋ¯ = ಕ್ರಿಯಾಶೂನ್ಯನೂ ಆದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತ್ಯಂಚಮ್ ಆತ್ಮಾನ್ಣಂಋಸ್ಬಋ¯ = ಪ್ರತ್ಯಗಾತ್ಮನನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ವಿವಿಚ್ಯ್ಣಋಸ್ಬಋ¯ = ಬೇರ್ಪಡಿಸಿ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ರ್ಣಋಸ್ಬಋ¯ = ಆ ಆತ್ಮನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವ್ಣಂಋಸ್ಬಋ¯ = ಅನಾತ್ಮ ವರ್ಗವೆಲ್ಲವನ್ನೂ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರವಿಲಾಪ್ಯ್ಣಋಸ್ಬಋ¯ = ಲಯಹೊಂದಿಸಿ ್ಣಋಸ್ಶ್ಯ್¯ತತ್-ಆತ್ಮನ್ಣಾಋಸ್ಬಋ¯ = ಆ ಆತ್ಮಸ್ವರೂಪ ದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ತಿಷ್ಠತ್ಣಿಋಸ್ಬಋ¯ = ನಿಲ್ಲುತ್ತಾನೆಯೊ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನೇ ್ಣಋಸ್ಶ್ಯ್¯ಮುಕ್ತ್ಣಃಋಸ್ಬಋ¯ = ಮುಕ್ತನು.

153. ಮುಂಜವೆಂಬ ಹುಲ್ಲಿನಿಂದ ಅದರ ದಂಟನ್ನು ಬೇರ್ಪಡಿಸುವಂತೆ1 ಯಾವನು ದೃಶ್ಯಸಮೂಹದಿಂದ2 ಸಂಗರಹಿತನೂ ಕ್ರಿಯಾಶೂನ್ಯನೂ ಆದ ಪ್ರತ್ಯ ಗಾತ್ಮನನ್ನು ಬೇರ್ಪಡಿಸಿ, ಆ ಆತ್ಮನಲ್ಲಿ ಅನಾತ್ಮವರ್ಗವೆಲ್ಲವನ್ನೂ ಲಯ ಹೊಂದಿಸಿ ಆ ಆತ್ಮಸ್ವರೂಪದಿಂದಲೇ ನಿಲ್ಲುವನೊ ಅವನೇ ಮುಕ್ತನು.

[ಇಲ್ಲಿ ವಿವೇಚನಪ್ರಕಾರವನ್ನು ದೃಷ್ಟಾಂತದ ಮೂಲಕ ಹೇಳಿದೆ.
1 ್ಣಋಸ್ಶ್ಯ್¯ಮುಂಜಾದಿವ ಇಷೀಕಾಮ್ಣಋಸ್ಬಋ¯ (ಕಠ. ಉ. 2. 6. 17)
2 ದೇಹೇಂದ್ರಿಯಗಳ ಗುಂಪಿನಿಂದ.]
39. ಅನ್ನಮಯ ಕೋಶ
ದೇಹೋsಯಮನ್ನಭವನೋsನ್ನಮಯಸ್ತು ಕೋಶ-
ಶ್ಚಾನ್ನೇನ ಜೀವತಿ ವಿನಶ್ಯತಿ ತದ್ವಿಹೀನಃ ।
ತ್ವಕ್ಚರ್ಮ-ಮಾಂಸ-ರುಧಿರಾಸ್ಥಿ-ಪುರೀಷರಾಶಿ-
ನಾRಯಂ ಸ್ವಯಂ ಭವಿತುಮರ್ಹತಿ ನಿತ್ಯಶುದ್ಧಃ
॥ 154 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಯಂ ದೇಹಃ ತ್ಣುಋಸ್ಬಋ¯ = ಈ ಶರೀರವು ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ನಭವನ್ಣಃಋಸ್ಬಋ¯ = ಅನ್ನದಿಂದ ಉತ್ಪನ್ನವಾದದ್ದು, ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ನಮಯಃ ಕೋಶ್ಣಃಋಸ್ಬಋ¯ = ಅನ್ನಮಯ ಕೋಶವು, ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ನೇನ್ಣಋಸ್ಬಋ¯ = ಅನ್ನದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಜೀವತ್ಣಿಋಸ್ಬಋ¯ = ಬದುಕಿರುವುದು, ್ಣಋಸ್ಶ್ಯ್¯ತದ್ವಿಹೀನ್ಣಃಋಸ್ಬಋ¯ = ಅದಿಲ್ಲದಿದ್ದರೆ ್ಣಋಸ್ಶ್ಯ್¯ವಿನಶ್ಯತ್ಣಿಋಸ್ಬಋ¯ = ನಾಶವಾಗುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ತ್ವಕ್- ಚರ್ಮ-ಮಾಂಸ-ರುಧಿರ-ಅಸ್ಥಿ-ಪುರೀಷ-ರಾಶ್ಣಿಃಋಸ್ಬಋ¯ = ತ್ವಕ್ಕು, ಚರ್ಮ, ಮಾಂಸ, ರಕ್ತ, ಮೂಳೆ, ಮಲ ಇವುಗಳ ರಾಶಿಯಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಇದು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯ್ಣಂಋಸ್ಬಋ¯ = ತಾನೇ ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯಶುದ್ಧ್ಣಃಋಸ್ಬಋ¯ = ನಿತ್ಯ ಶುದ್ಧವು ್ಣಋಸ್ಶ್ಯ್¯ಭವಿತ್ಣುಂಋಸ್ಬಋ¯ = ಆಗಲು ್ಣಋಸ್ಶ್ಯ್¯ನ ಅರ್ಹತ್ಣಿಋಸ್ಬಋ¯ = ಅರ್ಹವಲ್ಲ.

154. ಈ ಶರೀರವು ಅನ್ನದಿಂದ ಹುಟ್ಟಿದೆ, ಅನ್ನಮಯಕೋಶವಾಗಿದೆ. ಇದು ಅನ್ನದಿಂದ ಬದುಕಿರುವುದು. ಅದಿಲ್ಲದಿದ್ದರೆ ನಾಶವಾಗುತ್ತದೆ. ತ್ವಕ್ಕು ಚರ್ಮ ಮಾಂಸ ರಕ್ತ ಮೂಳೆ ಮಲ ಇವುಗಳ ರಾಶಿಯಾಗಿರುವ ಈ ಶರೀರವು ನಿತ್ಯಶುದ್ಧ ವಾದ ಆತ್ಮವಸ್ತುವಾಗಲಾರದು.

[ಇಲ್ಲಿಂದ ಅನ್ನಮಯಕೋಶದ ಲಕ್ಷಣವನ್ನು ಹೇಳಿದೆ. ‘ಆ ಈ ಪುರುಷನು ಅನ್ನರಸ ಮಯನು’ ್ಣಋಸ್ಶ್ಯ್¯ಸ ವಾ ಏಷ ಪುರುಷೋsನ್ನರಸಮಯ್ಣಃಋಸ್ಬಋ¯ (ತೈತ್ತಿರೀಯ ಉ. 2. 1). ಅನ್ನರಸ ಮಯನು ಎಂದರೆ ಅನ್ನರಸದ ವಿಕಾರ ಅಥವಾ ಕಾರ್ಯ.]
ಪೂರ್ವಂ ಜನೇರಪಿ ಮೃತೇರಥ ನಾಯಮಸ್ತಿ
ಜಾತಕ್ಷಣಃ ಕ್ಷಣಗುಣೋsನಿಯತಸ್ವಭಾವಃ ।
ನೈಕೋ ಜಡಶ್ಚ ಘಟವತ್ಪರಿದೃಶ್ಯಮಾನಃ
ಸ್ವಾತ್ಮಾ ಕಥಂ ಭವತಿ ಭಾವವಿಕಾರವೇತ್ತಾ ॥ 155 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಈ ಶರೀರವು ್ಣಋಸ್ಶ್ಯ್¯ಜನೇಃ ಪೂರ್ವಮ್ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಜನಿಸುವುದಕ್ಕೆ ಮೊದಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಮೃತೇಃ ಅಥ್ಣಋಸ್ಬಋ¯ = ಸತ್ತಮೇಲೆಯೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ಅಸ್ತ್ಣಿಋಸ್ಬಋ¯ = ಇರುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಜಾತಕ್ಷಣಃ = ಕ್ಷಣಕಾಲ ಬದುಕಿರುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಷಣಗುಣಃ = ಕ್ಷಣಿಕವಾದ ಗುಣವುಳ್ಳದ್ದು ; ್ಣಋಸ್ಶ್ಯ್¯ಅನಿಯತಸ್ವಭಾವ್ಣಃಋಸ್ಬಋ¯ = ನಿಯತವಾದ ಸ್ವಭಾವವಿಲ್ಲದ್ದು ; ್ಣಋಸ್ಶ್ಯ್¯ನ ಏಕ್ಣಃಋಸ್ಬಋ¯ = ಅನೇಕ ರೂಪವುಳ್ಳದ್ದು ; ್ಣಋಸ್ಶ್ಯ್¯ಜಡ್ಣಃಋಸ್ಬಋ¯ = ಜಡವಾದದ್ದು ; ್ಣಋಸ್ಶ್ಯ್¯ಘಟವತ್ಣ್ಋಸ್ಬಋ¯ = ಗಡಿಗೆಯಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಪರಿದೃಶ್ಯಮಾನ್ಣಃಋಸ್ಬಋ¯ = ದೃಶ್ಯವಾಗಿದೆ ; [ಇದು] ್ಣಋಸ್ಶ್ಯ್¯ಭಾವ-ವಿಕಾರ-ವೇತ್ತ್ಣಾಋಸ್ಬಋ¯ = ದೇಹಾದಿಸಂಘಾತವಾದ ವಿಕಾರವನ್ನರಿಯುವ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಾತ್ಮ್ಣಾಋಸ್ಬಋ¯ = ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ಕಥಂ ಭವತ್ಣಿಋಸ್ಬಋ¯ = ಹೇಗಾದೀತು ?

155. ಈ ಶರೀರವು ಹುಟ್ಟುವುದಕ್ಕಿಂತ ಮೊದಲೂ ಸತ್ತಮೇಲೆಯೂ ಇರುವು ದಿಲ್ಲ. ನಡುವೆ ಕೆಲವು ಕಾಲವಿರುತ್ತದೆ. ಇದು ಕ್ಷಣಿಕವಾದ ಗುಣಗಳುಳ್ಳದ್ದು, ನಿಯತವಾದ ಸ್ವಭಾವವಿಲ್ಲದ್ದು, ಅನೇಕ ರೂಪಗಳುಳ್ಳದ್ದು ;1 ಇದು ಜಡವಾಗಿ ಗಡಿಗೆಯಂತೆ ದೃಶ್ಯವಾಗಿರುತ್ತದೆ. ದೇಹವೇ ಮೊದಲಾದ ಸಂಘಾತದ ವಿಕಾರ ಗಳನ್ನು ಅರಿಯುವ ಆತ್ಮನು ಹೇಗಾದೀತು ?

[1 ಅವಯವಗಳ ಉಪಚಯ-ಅಪಚಯಗಳಿಂದ ಭಿನ್ನಭಿನ್ನವಾದ ರೂಪವನ್ನು ಹೊಂದುವುದು.]
ಪಾಣಿ-ಪಾದಾದಿಮಾನ್ ದೇಹೋ ನಾತ್ಮಾ ವ್ಯಂಗೇsಪಿ ಜೀವನಾತ್ ।
ತತ್ತಚ್ಛಕ್ತೇರನಾಶಾಚ್ಚ ನ ನಿಯಮ್ಯೋ ನಿಯಾಮಕಃ ॥ 156 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪಾಣಿ-ಪಾದ-ಆದಿಮಾನ್ = ಕೈ ಕಾಲು ಮೊದಲಾದುವುಗಳನ್ನುಳ್ಳ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹ್ಣಃಋಸ್ಬಋ¯ = ಶರೀರವು ್ಣಋಸ್ಶ್ಯ್¯ನ ಆತ್ಮಾ= ಆತ್ಮನಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ವ್ಯಂಗೇ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಯಾವುದಾದರೂ ಒಂದು ಅಂಗವು ನಾಶ ವಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಜೀವನಾತ್= ಬದುಕಿರುವುದರಿಂದಲೂ, ್ಣಋಸ್ಶ್ಯ್¯ತತ್-ತತ್-ಶಕ್ತ್ಣೇಃಋಸ್ಬಋ¯ = ಆಯಾ ಶಕ್ತಿಯ ್ಣಋಸ್ಶ್ಯ್¯ಅನಾಶಾತ್ ಚ = ನಾಶವಾಗದೆ ಇರುವುದರಿಂದಲೂ [ಆತ್ಮನಲ್ಲ] ; ್ಣಋಸ್ಶ್ಯ್¯ನ ನಿಯಮ್ಯ್ಣಃಋಸ್ಬಋ¯ = ನಿಯಮಿಸಲ್ಪಡತಕ್ಕವನಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ನಿಯಾಮಕ್ಣಃಋಸ್ಬಋ¯ = ನಿಯಾಮಕನು.

156. ಕೈ ಕಾಲು ಮೊದಲಾದ ಅವಯವಗಳುಳ್ಳ ದೇಹವು ಆತ್ಮನಲ್ಲ ; ಏಕೆಂದರೆ ಅವನು ಯಾವುದಾದರೂ ಒಂದು ಅಂಗವು ನಾಶವಾದರೂ ಬದುಕಿರು ತ್ತಾನೆ1 ಮತ್ತು ಆಯಾ ಶಕ್ತಿಯೂ ನಾಶವಾಗುವುದಿಲ್ಲ.2 ಈ ಆತ್ಮನು ನಿಯಮಿಸ ಲ್ಪಡತಕ್ಕವನಲ್ಲ, ಆದರೆ ನಿಯಾಮಕನು.

[1 ‘ಜೀವನು ಈ ಮರದ ಒಂದು ಕೊಂಬೆಯನ್ನು ಬಿಟ್ಟರೆ ಆಗ ಅದು ಒಣಗುತ್ತದೆ . . . ಜೀವನು ಇಡೀ ಮರವನ್ನು ಬಿಟ್ಟರೆ ಮರವೆಲ್ಲ ಒಣಗುತ್ತದೆ.’ (ಛಾಂದೋಗ್ಯ ಉ. 6. 11. 2)
2 ಯಾವುದಾದರೂ ಒಂದು ಅಂಗವು ನಾಶವಾದರೂ ಕೃತಕವಾದ ಅಂಗಗಳ ಮೂಲಕ ಆ ಕಾರ್ಯವನ್ನು ನಿರ್ವಹಿಸಬಹುದು.]
ದೇಹ-ತದ್ಧರ್ಮ-ತತ್ಕರ್ಮ-ತದವಸ್ಥಾದಿ-ಸಾಕ್ಷಿಣಃ ।
ಸ್ವತ ಏವ ಸ್ವತಃಸಿದ್ಧಂ ತದ್ವೈಲಕ್ಷಣ್ಯಮಾತ್ಮನಃ ॥ 157 ॥
್ಣಋಸ್ಶ್ಯ್¯ದೇಹ-ತದ್ಧರ್ಮ-ತತ್ಕರ್ಮ-ತದವಸ್ಥಾದಿ-ಸಾಕ್ಷಿಣ್ಣಃಋಸ್ಬಋ¯ = ದೇಹ, ಅದರ ಧರ್ಮಗಳು, ಅದರ ಕರ್ಮಗಳು, ಅದರ ಅವಸ್ಥೆಗಳು ಇವೇ ಮೊದಲಾದುವುಗಳಿಗೆ ಸಾಕ್ಷಿಯಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನ್ಣಃಋಸ್ಬಋ¯ = ಆತ್ಮನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವತಃ ಏವ್ಣಋಸ್ಬಋ¯ = ಸ್ವಭಾವದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ತದ್ವೈಲಕ್ಷಣ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಅದಕ್ಕಿಂತ ವಿಲಕ್ಷಣತ್ವವು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವತಃ ಸಿದ್ಧ್ಣಂಋಸ್ಬಋ¯ = ಸ್ವತಃಸಿದ್ಧವಾಗಿದೆ.

157. ದೇಹ, ಅದರ ಧರ್ಮಗಳು,1 ಅದರ ಕರ್ಮಗಳು,2 ಅದರ ಅವಸ್ಥೆಗಳು3 –ಇವೇ ಮೊದಲಾದವುಗಳಿಗೆ ಸಾಕ್ಷಿಯಾಗಿರುವ ಆತ್ಮನು ಅವುಗಳಿಗಿಂತ ಸ್ವಭಾವ ದಿಂದಲೇ ವಿಲಕ್ಷಣನೆಂಬುದು ಸ್ವತಃಸಿದ್ಧವಾಗಿದೆ.

[1 ಸ್ಥೂಲತ್ವ ಕೃಶತ್ವ ಮೊದಲಾದವು.
2 ಗಮನಾಗಮನಗಳು.
3 ಬಾಲ್ಯ ಯೌವನ ಮೊದಲಾದುವು.]
ಶಲ್ಯರಾಶಿರ್ಮಾಂಸಲಿಪ್ತೋ ಮಲಪೂರ್ಣೋsತಿಕಶ್ಮಲಃ ।
ಕಥಂ ಭವೇದಯಂ ವೇತ್ತಾ ಸ್ವಯಮೇತದ್ವಿಲಕ್ಷಣಃ ॥ 158 ॥
್ಣಋಸ್ಶ್ಯ್¯ಶಲ್ಯರಾಶ್ಣಿಃಋಸ್ಬಋ¯ = ಮೂಳೆಗಳ ರಾಶಿಯಾದ ್ಣಋಸ್ಶ್ಯ್¯ಮಾಂಸಲಿಪ್ತ್ಣಃಋಸ್ಬಋ¯ = ಮಾಂಸದಿಂದ ಮುಚ್ಚ ಲ್ಪಟ್ಟಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಮಲಪೂರ್ಣ್ಣಃಋಸ್ಬಋ¯ = ಮಲದಿಂದ ತುಂಬಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಅತಿಕಶ್ಮಲ್ಣಃಋಸ್ಬಋ¯ = ಅತ್ಯಂತ ಕಶ್ಮಲ ವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಈ ಶರೀರವು ್ಣಋಸ್ಶ್ಯ್¯ಏತತ್ವಿಲಕ್ಷಣ್ಣಃಋಸ್ಬಋ¯ = ಇದಕ್ಕಿಂತ ವಿಲಕ್ಷಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ವೇತ್ತ್ಣಾಋಸ್ಬಋ¯ = ಅರಿಯುವವನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯ್ಣಂಋಸ್ಬಋ¯ = ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ಕಥಂ ಭವೇತ್ಣ್ಋಸ್ಬಋ¯ = ಹೇಗಾದೀತು ?

158. ಮೂಳೆಗಳ ರಾಶಿಯಾದ ಮಾಂಸದಿಂದ ಮುಚ್ಚಲ್ಪಟ್ಟಿರುವ ಮಲ ಪೂರ್ಣವಾಗಿ ಅತ್ಯಂತ ಕಶ್ಮಲವಾದ ಶರೀರವು ತನಗಿಂತ ವಿಲಕ್ಷಣವಾದ, ಅರಿಯು ವವನಾದ ಆತ್ಮನು ಹೇಗಾದೀತು ?

ಣಿ4ಣಬ13ಶ್ಲಸ್ನ್ಇವ್ಶ್ಶ್ ಪ್ ತ್ವಙ್ಮಾಂಸ-ಮೇದೋsಸ್ಥಿ-ಪುರೀಷ-ರಾಶಾ-
 ವಹಂಮತಿಂ ಮೂಢಜನಃ ಕರೋತಿ ।
ವಿಲಕ್ಷಣಂ ವೇತ್ತಿ ವಿಚಾರಶೀಲೋ
ನಿಜಸ್ವರೂಪಂ ಪರಮಾರ್ಥಭೂತಮ್ ॥ 159 ॥

್ಣಋಸ್ಶ್ಯ್¯ಮೂಢಜನ್ಣಃಋಸ್ಬಋ¯ = ಮೂಢಜನವು ್ಣಋಸ್ಶ್ಯ್¯ತ್ವಕ್-ಮಾಂಸ-ಮೇದಃ-ಅಸ್ಥಿ-ಪುರೀಷ-ರಾಶ್ಣೌಋಸ್ಬಋ¯ = ಚರ್ಮ ಮಾಂಸ ಮೇದಸ್ಸು ಮೂಳೆ ಮಲ–ಇವುಗಳ ರಾಶಿಯಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ-ಮತ್ಣಿಂಋಸ್ಬಋ¯ = ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಕರೋತ್ಣಿಋಸ್ಬಋ¯ = ಮಾಡುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ವಿಚಾರಶೀಲ್ಣಃಋಸ್ಬಋ¯ = ವಿವೇಕಿಯು ್ಣಋಸ್ಶ್ಯ್¯ನಿಜಸ್ವರೂಪ್ಣಂಋಸ್ಬಋ¯ = ತನ್ನ ಸ್ವರೂಪವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ವಿಲಕ್ಷಣ್ಣಂಋಸ್ಬಋ¯ = ವಿಲಕ್ಷಣವೆಂದೂ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮಾರ್ಥ ಭೂತ್ಣಂಋಸ್ಬಋ¯ = ಸತ್ಯವೆಂದೂ ್ಣಋಸ್ಶ್ಯ್¯ವೇತ್ತ್ಣಿಋಸ್ಬಋ¯ = ಅರಿತುಕೊಳ್ಳುತ್ತಾನೆ.

159. ಚರ್ಮ ಮಾಂಸ ಮೇದಸ್ಸು ಮೂಳೆ ಮಲ–ಇವುಗಳ ರಾಶಿಯಾಗಿರುವ ಈ ಶರೀರದಲ್ಲಿ ಮೂಢಜನವು ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು1 ಮಾಡುತ್ತದೆ. ವಿವೇಕಿಯಾದವನು (ಶರೀರಾದಿಗಳಿಗಿಂತ) ಭಿನ್ನವಾದ ಮತ್ತು ಪರಮಾರ್ಥವಾದ ತನ್ನ ಸ್ವರೂಪವನ್ನು ಅರಿತುಕೊಳ್ಳುತ್ತಾನೆ.

[1 ನಾನು ಸ್ಥೂಲನು, ನಾನು ಕೃಶನು, ಇತ್ಯಾದಿ ಬುದ್ಧಿಯನ್ನು.]
ದೇಹೋsಹಮಿತ್ಯೇವ ಜಡಸ್ಯ ಬುದ್ಧಿ-
ದೆR|ಹೇ ಚ ಜೀವೇ ವಿದುಷಸ್ತ್ವಹಂಧೀಃ ।
ವಿವೇಕ-ವಿಜ್ಞಾನವತೋ ಮಹಾತ್ಮನೋ
ಬ್ರಹ್ಮಾಹಮಿತ್ಯೇವ ಮತಿಃ ಸದಾತ್ಮನಿ ॥ 160 ॥
್ಣಋಸ್ಶ್ಯ್¯ಜಡಸ್ಣಋಸ್ಬಋ¯್ಯ = ಜಡನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹಃ = ಶರೀರವು ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಇತಿ ಏವ್ಣಋಸ್ಬಋ¯ = ಎಂದೇ ್ಣಋಸ್ಶ್ಯ್¯ಬುದ್ಧ್ಣಿಃಋಸ್ಬಋ¯ = ಬುದ್ಧಿಯುಂಟಾಗುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ವಿದುಷಃ ತ್ಣುಋಸ್ಬಋ¯ = ಲೌಕಿಕ ಪಂಡಿತನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹೇ ಜೀವೇ ಚ್ಣಋಸ್ಬಋ¯ = ಶರೀರದಲ್ಲಿ ಮತ್ತು ಜೀವನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂಧ್ಣೀಃಋಸ್ಬಋ¯ = ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯುಂಟಾಗುತ್ತದೆ; ್ಣಋಸ್ಶ್ಯ್¯ವಿವೇಕವಿಜ್ಞಾನವತ್ಣಃಋಸ್ಬಋ¯ = ಆತ್ಮಾನಾತ್ಮ-ವಿವೇಚನವನ್ನು ಮಾಡಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಮಹಾತ್ಮನ್ಣಃಋಸ್ಬಋ¯ = ಮಹಾತ್ಮನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸತ್-ಆತ್ಮನ್ಣಿಋಸ್ಬಋ¯ = ಅವಿನಾಶಿಯಾದ ಆತ್ಮಸ್ವರೂಪದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ ಬ್ರಹ್ಮ್ಣಋಸ್ಬಋ¯ = ನಾನು ಬ್ರಹ್ಮವು ್ಣಋಸ್ಶ್ಯ್¯ಇತಿ ಏವ್ಣಋಸ್ಬಋ¯ = ಎಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಮತ್ಣಿಃಋಸ್ಬಋ¯ = ಬುದ್ದಿಯು [ಉಂಟಾಗುತ್ತದೆ].

160. ಜಡನಾದ ಮನುಷ್ಯನು ತಾನೇ ಶರೀರವೆಂದು ಭಾವಿಸಿಕೊಳ್ಳುತ್ತಾನೆ. ಲೌಕಿಕಪಂಡಿತನಾದವನು ಸ್ಥೂಲಶರೀರದಲ್ಲಿಯೂ ಮತ್ತು ಜೀವನಲ್ಲಿಯೂ1, ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು ಮಾಡುತ್ತಾನೆ. ಆತ್ಮಾನಾತ್ಮ ವಿವೇಚನದಲ್ಲಿ ನಿಪುಣ ನಾದ ಮಹಾತ್ಮನಿಗೆ ಅವಿನಾಶಿಯಾದ ಆತ್ಮಸ್ವರೂಪದಲ್ಲಿಯೇ ‘ನಾನು ಬ್ರಹ್ಮ’ ಎಂಬ ಬುದ್ಧಿಯುಂಟಾಗುತ್ತದೆ.

[ಇಲ್ಲಿ ಮೂಢ, ಕೇವಲ ಶಾಸ್ತ್ರಗಳನ್ನು ಅಧ್ಯಯನ ಮಾಡಿರುವ ಪಂಡಿತ ಮತ್ತು ನಿಜವಾದ ಆತ್ಮಜ್ಞಾನಿ–ಇವರು ಶರೀರಾದಿಗಳಲ್ಲಿ ಅಹಂಬುದ್ಧಿಯನ್ನು ಹೇಗೆ ಮಾಡು ತ್ತಾರೆ ಎಂಬುದನ್ನು ಹೇಳಿದೆ.
1 ಪ್ರಾಣವೇ ಮೊದಲಾದುವನ್ನು ಉಪಾಧಿಯಾಗಿ ಹೊಂದಿರುವ ಚೈತನ್ಯದಲ್ಲಿ.]
ಅತ್ರಾತ್ಮಬುದ್ಧಿಂ ತ್ಯಜ ಮೂಢಬುದ್ಧೇ
ತ್ವಙ್ಮಾಂಸ-ಮೇದೋsಸ್ಥಿ-ಪುರೀಷರಾಶೌ ।
ಸರ್ವಾತ್ಮನಿ ಬ್ರಹ್ಮಣಿ ನಿರ್ವಿಕಲ್ಪೇ
ಕುರುಷ್ವ ಶಾಂತಿಂ ಪರಮಾಂ ಭಜಸ್ವ ॥ 161 ॥
್ಣಋಸ್ಶ್ಯ್¯ಮೂಢಬುದ್ಧ್ಣೇಋಸ್ಬಋ¯ = ಹೇ ಮೂಢಬುದ್ಧಿ, ್ಣಋಸ್ಶ್ಯ್¯ತ್ವಕ್-ಮಾಂಸ-ಮೇದಃ-ಅಸ್ಥಿ-ಪುರೀಷ-ರಾಶ್ಣೌಋಸ್ಬಋ¯ = ತ್ವಕ್ಕು ಮಾಂಸ ಮೇದಸ್ಸು ಮೂಳೆ ಮಲ ಇವುಗಳ ರಾಶಿಯಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅತ್ರ= ಈ ಶರೀರದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮಬುದ್ಧ್ಣಿಂಋಸ್ಬಋ¯ = ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ತ್ಯಜ್ಣಋಸ್ಬಋ¯ = ತೊರೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವಾತ್ಮನ್ಣಿಋಸ್ಬಋ¯ = ಸರ್ವಾತ್ಮವಾದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕಲ್ಪ್ಣೇಋಸ್ಬಋ¯ = ನಿರ್ವಿಕಲ್ಪವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಣ್ಣಿಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮದಲ್ಲಿ [ಆತ್ಮಬುದ್ಧಿ ಯನ್ನು] ್ಣಋಸ್ಶ್ಯ್¯ಕುರುಷ್ವ್ಣಋಸ್ಬಋ¯ = ಮಾಡು ್ಣಋಸ್ಶ್ಯ್¯ಪರಮಾಂ ಶಾಂತ್ಣಿಂಋಸ್ಬಋ¯ = ಪರಮ ಶಾಂತಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಭಜಸ್ಣಋಸ್ಬಋ¯್ವ = ಪಡೆ.

161. ಹೇ ಮೂಢಬುದ್ಧಿ ! ತ್ವಕ್ಕು ಮಾಂಸ ಮೇದಸ್ಸು ಮೂಳೆ ಮಲ– ಇವುಗಳ ರಾಶಿಯಾದ ಈ ಶರೀರದಲ್ಲಿ ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು ಬಿಡು. ಸರ್ವಾತ್ಮವೂ ನಿರ್ವಿಕಲ್ಪವೂ ಆದ ಬ್ರಹ್ಮದಲ್ಲಿ ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು ಮಾಡು. ಪರಮಶಾಂತಿಯನ್ನು ಪಡೆ.

ತಾವನ್ನ ತಸ್ಯಾಸ್ತಿ ವಿಮುಕ್ತಿವಾರ್ತಾ-
ಪ್ಯಸ್ತ್ವೇಷ ವೇದಾಂತ-ನಯಾಂತ-ದರ್ಶೀ ॥ 162 ॥
್ಣಋಸ್ಶ್ಯ್¯ವಿದ್ವಾನ್ಣ್ಋಸ್ಬಋ¯ = ವಿವೇಚನಕುಶಲನು ್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್ಣಿಋಸ್ಬಋ¯ = ಅಸತ್ಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹೇಂದ್ರಿಯಾದ್ಣೌಋಸ್ಬಋ¯ = ದೇಹೇಂದ್ರಿಯಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಭ್ರಮ-ಉದಿತ್ಣಾಂಋಸ್ಬಋ¯ = ಭ್ರಾಂತಿಯಿಂದುಂಟಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂತ್ಣಾಂಋಸ್ಬಋ¯ = ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಯಾವತ್ಣ್ಋಸ್ಬಋ¯ = ಎಲ್ಲಿಯವರೆಗೆ ್ಣಋಸ್ಶ್ಯ್¯ನ ಜಹಾತ್ಣಿಋಸ್ಬಋ¯ = ಬಿಡುವು ದಿಲ್ಲವೊ ್ಣಋಸ್ಶ್ಯ್¯ತಾವತ್ಣ್ಋಸ್ಬಋ¯ = ಅಲ್ಲಿಯವರೆಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಏಷ್ಣಃಋಸ್ಬಋ¯ = ಇವನು ್ಣಋಸ್ಶ್ಯ್¯ವೇದಾಂತ-ನಯ-ಅಂತದರ್ಶೀ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ತ್ಣುಋಸ್ಬಋ¯ = ವೇದಾಂತಶಾಸ್ತ್ರದ ದಡವನ್ನು ಬಲ್ಲವನೇ ಆಗಿರಲಿ, ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಅವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ವಿಮುಕ್ತಿ ವಾರ್ತಾ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಮುಕ್ತಿಯ ಸುದ್ದಿಯೂ ಕೂಡ ್ಣಋಸ್ಶ್ಯ್¯ನ ಅಸ್ತ್ಣಿಋಸ್ಬಋ¯ = ಇರುವುದಿಲ್ಲ.

162. ವಿವೇಚನದಲ್ಲಿ ಕುಶಲನಾದವನು ದೇಹೇಂದ್ರಿಯವೇ ಮೊದಲಾದವು ಗಳಲ್ಲಿ ಭ್ರಾಂತಿಯಿಂದ ಉಂಟಾಗಿರುವ ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯನ್ನು ಎಲ್ಲಿಯ ವರೆಗೆ ಬಿಡುವುದಿಲ್ಲವೋ ಅಲ್ಲಿಯವರೆಗೆ–ಅವನು ವೇದಾಂತಶಾಸ್ತ್ರದ ದಡವನ್ನು ಬಲ್ಲವನಾಗಿದ್ದರೂ1–ಅವನಿಗೆ ಮುಕ್ತಿಯ ಸುದ್ದಿಯೂ ಕೂಡ ಇರುವುದಿಲ್ಲ.

[ದೇಹಾದಿಗಳಲ್ಲಿ ಅಭಿಮಾನವನ್ನು ಬಿಡುವ ವರೆಗೆ ಕೇವಲ ಶಾಸ್ತ್ರಜನ್ಯ-ಜ್ಞಾನದಿಂದ ಮುಕ್ತಿಯುಂಟಾಗುವುದಿಲ್ಲ ಎಂಬುದು ಅಭಿಪ್ರಾಯ.
1 ವೇದಾಂತದರ್ಶನದ ಅಧ್ಯಯನ-ಅಧ್ಯಾಪನಗಳಲ್ಲಿ ನಿಪುಣನಾಗಿದ್ದರೂ.]
ಛಾಯಾಶರೀರೇ ಪ್ರತಿಬಿಂಬಗಾತ್ರೇ
ಯತ್ಸ್ವಪ್ನದೇಹೇ ಹೃದಿ ಕಲ್ಪಿತಾಂಗೇ ।
ಯಥಾssತ್ಮಬುದ್ಧಿಸ್ತವ ನಾಸ್ತಿ ಕಾಚಿ-
ಜ್ಜೀವಚ್ಛರೀರೇ ಚ ತಥೈವ ಮಾsಸ್ತು ॥ 163 ॥
್ಣಋಸ್ಶ್ಯ್¯ಛಾಯಾಶರೀರ್ಣೇಋಸ್ಬಋ¯ = ನೆರಳಿನ ಶರೀರದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತಿಬಿಂಬಗಾತ್ರ್ಣೇಋಸ್ಬಋ¯ = ಪ್ರತಿಬಿಂಬಿತ ಶರೀರ ದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್-ಸ್ವಪ್ನದೇಹ್ಣೇಋಸ್ಬಋ¯ = ಯಾವ ಕನಸಿನ ಶರೀರದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಹೃದಿ ಕಲ್ಪಿತಾಂಗ್ಣೇಋಸ್ಬಋ¯ = ಮನಸ್ಸಿನಲ್ಲಿ ಕಲ್ಪಿತವಾದ ಶರೀರದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ತವ್ಣಋಸ್ಬಋ¯ = ನಿನಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮಬುದ್ಧ್ಣಿಃಋಸ್ಬಋ¯ = ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಯಥ್ಣಾಋಸ್ಬಋ¯ = ಹೇಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಕಾಚಿತ್ ನ ಅಸ್ಣಿಋಸ್ಬಋ¯್ತ =ಸ್ವಲ್ಪವೂ ಇರುವುದಿಲ್ಲವೋ ್ಣಋಸ್ಶ್ಯ್¯ತಥಾ ಏವ್ಣಋಸ್ಬಋ¯ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಜೀವತ್-ಶರೀರೇ ಚ್ಣಋಸ್ಬಋ¯ = ಜೀವಿಸಿರುವ ಈ ಶರೀರದಲ್ಲಿಯೂ [ಆತ್ಮ ಬುದ್ಧಿಯು] ್ಣಋಸ್ಶ್ಯ್¯ಮಾ ಅಸ್ತ್ಣುಋಸ್ಬಋ¯ = ಉಂಟಾಗದಿರಲಿ.

163. ಶರೀರದ ನೆರಳಿನಲ್ಲಿ, (ಕನ್ನಡಿಯೇ ಮೊದಲಾದುವುಗಳಲ್ಲಿ) ಪ್ರತಿ ಬಿಂಬಿತವಾದ ಶರೀರದಲ್ಲಿ, ಕನಸಿನಲ್ಲಿ ಕಾಣುವ ಶರೀರದಲ್ಲಿ ಮತ್ತು ಮನಸ್ಸಿನಲ್ಲಿ ಕಲ್ಪಿತವಾದ ಶರೀರದಲ್ಲಿ ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯು ಹೇಗೆ ಸ್ವಲ್ಪವೂ ಇರುವು ದಿಲ್ಲವೋ ಹಾಗೆಯೇ ಜೀವಿಸಿರುವ ಈ ಶರೀರದಲ್ಲಿಯೂ (‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯು) ನಿನಗೆ ಉಂಟಾಗದಿರಲಿ.

[ಶರೀರಾದಿಗಳಲ್ಲಿ ಅಭಿಮಾನವನ್ನು ಬಿಡಲು ಸುಲಭೋಪಾಯವನ್ನು ಇಲ್ಲಿ ಹೇಳಿದೆ.]
ದೇಹಾತ್ಮಧೀರೇವ ನೃಣಾಮಸದ್ಧಿಯಾಂ
ಜನ್ಮಾದಿ-ದುಃಖ-ಪ್ರಭವಸ್ಯ ಬೀಜಮ್ ।
ಯತಸ್ತತಸ್ತ್ವಂ ಜಹಿ ತಾಂ ಪ್ರಯತ್ನಾತ್
ತ್ಯಕ್ತೇ ತು ಚಿತ್ತೇ ನ ಪುನರ್ಭವಾಶಾ ॥ 164 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್-ಧಿಯ್ಣಾಂಋಸ್ಬಋ¯ = ಅವಿವೇಕಬುದ್ಧಿಯುಳ್ಳ ್ಣಋಸ್ಶ್ಯ್¯ನೃಣ್ಣಾಂಋಸ್ಬಋ¯ = ಜನರಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹ-ಆತ್ಮ-ಧೀಃ ಏವ್ಣಋಸ್ಬಋ¯ = ದೇಹದಲ್ಲಿ ಆತ್ಮಬುದ್ಧಿಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಜನ್ಮಾದಿ-ದುಃಖ-ಪ್ರಭವಸ್ಯ್ಣಋಸ್ಬಋ¯ =ಜನ್ಮವೇ ಮೊದಲಾದ ದುಃಖದ ಉತ್ಪತ್ತಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣಃಋಸ್ಬಋ¯ = ಯಾವ ಕಾರಣದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಬೀಜ್ಣಂಋಸ್ಬಋ¯ = ಕಾರಣವೊ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಣಃಋಸ್ಬಋ¯ = ಆ ಕಾರಣದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ತ್ವ್ಣಂಋಸ್ಬಋ¯ = ನೀನು ್ಣಋಸ್ಶ್ಯ್¯ತ್ಣಾಂಋಸ್ಬಋ¯ = ಅಂಥ ಬುದ್ಧಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಯತ್ನಾತ್ಣ್ಋಸ್ಬಋ¯ = ಪ್ರಯತ್ನದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಜಹ್ಣಿಋಸ್ಬಋ¯ = ಬಿಡು ; ್ಣಋಸ್ಶ್ಯ್¯ಚಿತ್ತೇ ತು ತ್ಯಕ್ತ್ಣೇಋಸ್ಬಋ¯ = ಅಂಥ ಬುದ್ಧಿಯನ್ನು ಬಿಟ್ಟರೆ ್ಣಋಸ್ಶ್ಯ್¯ನ ಪುನರ್ಭವ ಆಶ್ಣಾಋಸ್ಬಋ¯ = ಪುನರ್ಜನ್ಮದ ಆಶೆಯಿಲ್ಲ.

164. ಅವಿವೇಕಬುದ್ಧಿಯುಳ್ಳ ಜನರಿಗೆ ದೇಹಾದಿಗಳಲ್ಲಿ ‘ನಾನು’ ಎಂಬ ಬುದ್ಧಿಯೇ ಜನ್ಮವೇ ಮೊದಲಾದ ದುಃಖಗಳಿಗೆ ಕಾರಣವಾಗಿರುತ್ತದೆ. ಆದುದ ರಿಂದ ನೀನು ಪ್ರಯತ್ನಪೂರ್ವಕವಾಗಿ ಅಂಥ ದೇಹಾತ್ಮಬುದ್ಧಿಯನ್ನು ತೊರೆ. ದೇಹಾತ್ಮಬುದ್ಧಿಯು ಹೋದರೆ ಪುನರ್ಜನ್ಮದ ಸಂಭವವೇ ಇಲ್ಲ.

[ಛಾಂದೋಗ್ಯೋಪನಿಷತ್ತಿನಲ್ಲಿ (8. 12. 1) ಶರೀರವುಳ್ಳವನು ಪ್ರಿಯಾಪ್ರಿಯ ಗಳಿಂದ ಗ್ರಸ್ತನಾಗಿರುವನೆಂದು ಹೇಳಿದೆ.]
40. ಪ್ರಾಣಮಯಕೋಶ
ಕರ್ಮೇಂದ್ರಿಯೈಃ ಪಂಚಭಿರಂಚಿತೋsಯಂ
ಪ್ರಾಣೋ ಭವೇತ್ ಪ್ರಾಣಮಯಸ್ತು ಕೋಶಃ ।
ಯೇನಾತ್ಮವಾನನ್ನಮಯೋsನುಪೂರ್ಣಃ
ಪ್ರವರ್ತತೇsಸೌ ಸಕಲಕ್ರಿಯಾಸು ॥ 165 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪಂಚಭಿಃ ಕರ್ಮೇಂದ್ರಿಯ್ಣೈಃಋಸ್ಬಋ¯ = ಪಂಚಕರ್ಮೇಂದ್ರಿಯಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅಂಚಿತ್ಣಃಋಸ್ಬಋ¯ = ಕೂಡಿ ರುವ ್ಣಋಸ್ಶ್ಯ್¯ಅಯಂ ಪ್ರಾಣಃ ತ್ಣುಋಸ್ಬಋ¯ = ಈ ಪ್ರಾಣವಾದರೊ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಣಮಯಃ ಕೋಶ್ಣಃಋಸ್ಬಋ¯ = ಪ್ರಾಣಮಯ ಕೋಶವು ್ಣಋಸ್ಶ್ಯ್¯ಭವೇತ್ಣ್ಋಸ್ಬಋ¯ = ಆಗುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ನಮಯ್ಣಃಋಸ್ಬಋ¯ = ಅನ್ನಮಯಕೋಶವು ್ಣಋಸ್ಶ್ಯ್¯ಯೇನ = ಯಾವ ಈ ಪ್ರಾಣಮಯಕೋಶದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅನುಪೂರ್ಣ್ಣಃಋಸ್ಬಋ¯ = ತುಂಬಲ್ಪಟ್ಟು ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮವಾನ್ಣ್ಋಸ್ಬಋ¯ = ಆತ್ಮ ವಂತವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಣೌಋಸ್ಬಋ¯ = ಈ ಕೋಶವು ್ಣಋಸ್ಶ್ಯ್¯ಸಕಲಕ್ರಿಯಾಸ್ಣುಋಸ್ಬಋ¯ = ಎಲ್ಲಾ ಕ್ರಿಯೆಗಳಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರವರ್ತತ್ಣೇಋಸ್ಬಋ¯ = ಪ್ರರ್ವತಿಸುತ್ತದೆಯೊ.

165. ಪಂಚಕರ್ಮೇಂದ್ರಿಯಗಳೊಂದಿಗೆ ಕೂಡಿರುವ ಈ ಪ್ರಾಣವು ಪ್ರಾಣ ಮಯಕೋಶವಾಗುತ್ತದೆ. ಅನ್ನಮಯಕೋಶವು ಈ ಪ್ರಾಣಮಯಕೋಶದಿಂದ ತುಂಬಲ್ಪಟ್ಟು1 ಆತ್ಮವಂತವಾಗಿ ಎಲ್ಲಾ ಕ್ರಿಯೆಗಳಲ್ಲಿಯೂ ಪ್ರವರ್ತಿಸುತ್ತದೆ.

[ಇಲ್ಲಿ ಮತ್ತು ಮುಂದಿನ ಶ್ಲೋಕದಲ್ಲಿ ಪ್ರಾಣಮಯಕೋಶವನ್ನು ವರ್ಣಿಸಿದೆ.
1 ‘ಅಂಥ ಈ ಅನ್ನರಸಮಯನಾದ ಆತ್ಮನಿಗಿಂತ ಬೇರೆಯವನೂ ಒಳಗಿರುವವನೂ ಪ್ರಾಣಮಯನೂ ಆದ ಆತ್ಮನಿರುವನು. ಅವನಿಂದ ಈ ಅನ್ನರಸಮಯನಾದ ಆತ್ಮನು ತುಂಬಿರುವನು’ ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಮಾದ್ವಾ ಏತಸ್ಮಾದನ್ನರಸಮಯಾತ್ ಅನ್ಯೋsಂತರ ಆತ್ಮಾ ಪ್ರಾಣ ಮಯಃ, ತೇನೈಷ ಪೂರ್ಣ್ಣಃಋಸ್ಬಋ¯ (ತೈತ್ತಿರೀಯ ಉ. 2. 2 )
ನೈವಾತ್ಮಾsಪಿ ಪ್ರಾಣಮಯೋ ವಾಯುವಿಕಾರೋ
ಗಂತಾssಗಂತಾ ವಾಯುವದಂತರ್ಬಹಿರೇಷಃ ।
ಯಸ್ಮಾತ್ ಕಿಂಚಿತ್ ಕ್ವಾಪಿ ನ ವೇತ್ತೀಷ್ಟಮನಿಷ್ಟಂ
ಸ್ವಂ ವಾsನ್ಯಂ ವಾ ಕಿಂಚನ ನಿತ್ಯಂ ಪರತಂತ್ರಃ ॥ 166 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಣಮಯಃ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಪ್ರಾಣಮಯವೂ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮಾ ನ ಏವ್ಣಋಸ್ಬಋ¯ = ಆತ್ಮವಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಮಾತ್ಣ್ಋಸ್ಬಋ¯ = ಏಕೆಂದರೆ ್ಣಋಸ್ಶ್ಯ್¯ವಾಯುವಿಕಾರ್ಣಃಋಸ್ಬಋ¯ = ವಾಯುವಿನ ರೂಪಾಂತರವು ; ್ಣಋಸ್ಶ್ಯ್¯ಏಷ್ಣಃಋಸ್ಬಋ¯ = ಇದು ್ಣಋಸ್ಶ್ಯ್¯ವಾಯು ವತ್ಣ್ಋಸ್ಬಋ¯ = ವಾಯುವಿನಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಂತ್ಣಃಋಸ್ಬಋ¯ = ಒಳಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಗಂತ್ಣಾಋಸ್ಬಋ¯ = ಹೋಗುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ಬಹ್ಣಿಃಋಸ್ಬಋ¯ = ಹೊರಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಆಗಂತ್ಣಾಋಸ್ಬಋ¯ = ಬರುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ನಿತ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಪರತಂತ್ರ್ಣಃಋಸ್ಬಋ¯ = ಪರಾಧೀನವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ವ ಅಪ್ಣಿಋಸ್ಬಋ¯ =ಎಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಇಷ್ಟ್ಣಂಋಸ್ಬಋ¯ = ಇಷ್ಟವನ್ನಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅನಿಷ್ಟ್ಣಂಋಸ್ಬಋ¯ = ಅನಿಷ್ಟವನ್ನಾಗಲಿ, ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂಚಿತ್ ನ ವೇತ್ತ್ಣಿಋಸ್ಬಋ¯ = ಸ್ವಲ್ಪವೂ ಅರಿಯುವುದಿಲ್ಲ. ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಂ ವ್ಣಾಋಸ್ಬಋ¯ = ತನ್ನನ್ನಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ಯಂ ವ್ಣಾಋಸ್ಬಋ¯ = ಮತ್ತೊಂದ ನ್ನಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂಚನ [ನ ವೇತ್ತಿ] =ಸ್ವಲ್ಪವೂ ಅರಿಯುವುದಿಲ್ಲ.

166. ಈ ಪ್ರಾಣಮಯಕೋಶವು ವಾಯುವಿನ ರೂಪಾಂತರವಾಗಿರುವುದ ರಿಂದ ಆತ್ಮವಲ್ಲ ; ಏಕೆಂದರೆ ವಾಯುವಿನಂತೆ ಒಳಕ್ಕೆ ಹೋಗುತ್ತಲೂ ಹೊರಕ್ಕೆ ಬರುತ್ತಲೂ ಇರುತ್ತದೆ. ಇದು ಎಲ್ಲಿಯೂ ಇಷ್ಟವನ್ನಾಗಲಿ ಅನಿಷ್ಟವನ್ನಾಗಲಿ ಸ್ವಲ್ಪವೂ ಅರಿತುಕೊಳ್ಳಲಾರದು. ತನ್ನನ್ನಾಗಲಿ ಮತ್ತೊಂದನ್ನಾಗಲಿ ಸ್ವಲ್ಪವೂ ಅರಿಯಲಾರದು. ಯಾವಾಗಲೂ ಪರಾಧೀನವಾಗಿರುತ್ತದೆ.

41. ಮನೋಮಯಕೋಶ
ಜ್ಞಾನೇಂದ್ರಿಯಾಣಿ ಚ ಮನಶ್ಚ ಮನೋಮಯಃ ಸ್ಯಾತ್
ಕೋಶೋ ಮಮಾಹಮಿತಿ ವಸ್ತು-ವಿಕಲ್ಪ-ಹೇತುಃ ।
ಸಂಜ್ಞಾದಿಭೇದ-ಕಲನಾ-ಕಲಿತೋ ಬಲೀಯಾಂ-
ಸ್ತತ್ಪೂರ್ವಕೋಶಮಭಿಪೂರ್ಯ ವಿಜೃಂಭತೇ ಯಃ ॥ 167 ॥
್ಣಋಸ್ಶ್ಯ್¯ಜ್ಞಾನೇಂದ್ರಿಯಾಣಿ ಚ್ಣಋಸ್ಬಋ¯ = ಜ್ಞಾನೇಂದ್ರಿಯಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಮನಃ ಚ್ಣಋಸ್ಬಋ¯ಅ = ಮತ್ತು ಮನಸ್ಸು [ಸೇರಿ] ್ಣಋಸ್ಶ್ಯ್¯ಮನೋಮಯಃ ಕೋಶಃ ಸ್ಯಾತ್ಣ್ಋಸ್ಬಋ¯ = ಮನೋಮಯಕೋಶವಾಗುತ್ತದೆ ; [ಇದು] ಮಮ = ನನ್ನದು, ್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಎಂಬ ್ಣಋಸ್ಶ್ಯ್¯ವಸ್ತು-ವಿಕಲ್ಪ- ಹೇತ್ಣುಃಋಸ್ಬಋ¯ = ವಸ್ತುಗಳ ಭೇದಕ್ಕೆ ಕಾರಣವಾಗಿರುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣಃಋಸ್ಬಋ¯ = ಯಾವುದು ್ಣಋಸ್ಶ್ಯ್¯ಸಂಜ್ಞಾದಿ-ಭೇದ-ಕಲನಾ-ಕಲಿತ್ಣಃಋಸ್ಬಋ¯ = ನಾಮಾದಿ ಭೇದಗಳ ಕಲ್ಪನೆಯಿಂದ ಕೂಡಿರುತ್ತದೆಯೊ, ್ಣಋಸ್ಶ್ಯ್¯ಬಲೀಯಾನ್ಣ್ಋಸ್ಬಋ¯ = ಬಲಿಷ್ಠವಾಗಿರುತ್ತದೆಯೊ, ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಪೂರ್ವಕೋಶ್ಣಂಋಸ್ಬಋ¯ = ಹಿಂದಿನ ಕೋಶವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅಭಿಪೂರ್ಯ್ಣಋಸ್ಬಋ¯ = ವ್ಯಾಪಿಸಿಕೊಂಡು ್ಣಋಸ್ಶ್ಯ್¯ವಿಜೃಂ ಭತ್ಣೇಋಸ್ಬಋ¯ = ವಿಜೃಂಬಿಸುತ್ತದೆಯೊ.

167. ಜ್ಞಾನೇಂದ್ರಿಯಗಳೂ ಮನಸ್ಸೂ ಸೇರಿಕೊಂಡು ಮನೋಮಯಕೋಶ ವಾಗಿದೆ. ಇದು ‘ನಾನು, ನನ್ನದು’ ಎಂಬ ವಸ್ತುಗಳ ಭೇದಕ್ಕೆ ಕಾರಣವಾಗಿದೆ.1 ನಾಮವೇ ಮೊದಲಾದ ಭೇದಗಳ ಕಲ್ಪನೆಯಿಂದ ಕೂಡಿರುತ್ತದೆ, ಬಲಿಷ್ಠವಾಗಿರು ತ್ತದೆ. ಹಿಂದಿನ ಪ್ರಾಣಮಯಕೋಶವನ್ನು ವ್ಯಾಪಿಸಿಕೊಂಡು ಹರಡುತ್ತದೆ.

[ಇಲ್ಲಿಂದ ಮನೋಮಯಕೋಶವನ್ನು ವಿವರಿಸಿದೆ.
1 ದೇಹೇಂದ್ರಿಯ-ಗೃಹಕ್ಷೇತ್ರಗಳಲ್ಲಿ ‘ನಾನು’ ‘ನನ್ನದು’ ಎಂಬ ಬುದ್ಧಿ.]
ಪಂಚೇಂದ್ರಿಯೈಃ ಪಂಚಭಿರೇವ ಹೋತೃಭಿಃ
ಪ್ರಚೀಯಮಾನೋ ವಿಷಯಾಜ್ಯ-ಧಾರಯಾ ।
ಜಾಜ್ವಲ್ಯಮಾನೋ ಬಹುವಾಸನೇಂಧನೈ-
ಮRನೋಮಯಾಗ್ನಿರ್ದಹತಿ ಪ್ರಪಂಚಮ್ ॥ 168 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪಂಚೇಂದ್ರಿಯ್ಣೈಃಋಸ್ಬಋ¯ = ಪಂಚೇಂದ್ರಿಯಗಳೆಂಬ ್ಣಋಸ್ಶ್ಯ್¯ಪಂಚಭಿಃ ಹೋತೃಭಿಃ ಏವ್ಣಋಸ್ಬಋ¯ = ಐವರು ಹೋತೃಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಷಯ-ಆಜ್ಯ-ಧಾರಯ್ಣಾಋಸ್ಬಋ¯ = ವಿಷಯಗಳೆಂಬ ತುಪ್ಪದ ಧಾರೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಚೀಯಮಾನ್ಣಃಋಸ್ಬಋ¯ = ವರ್ಧಿಸುತ್ತಿರುವ, ್ಣಋಸ್ಶ್ಯ್¯ಬಹು-ವಾಸನಾ-ಇಂಧನ್ಣೈಃಋಸ್ಬಋ¯ = ನಾನಾ ವಾಸನೆ ಗಳೆಂಬ ಕಟ್ಟಿಗೆಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಜಾಜ್ವಲ್ಯಮಾನ್ಣಃಋಸ್ಬಋ¯ =ಪ್ರಜ್ವಲಿಸುತ್ತಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಮನೋಮಯಾಗ್ನ್ಣಿಃಋಸ್ಬಋ¯ = ಮನೋಮಯವೆಂಬ ಅಗ್ನಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಪಂಚ್ಣಂಋಸ್ಬಋ¯ = ಪ್ರಪಂಚವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ದಹತ್ಣಿಋಸ್ಬಋ¯ = ಸುಡುತ್ತಿರು ವುದು.

168. ಪಂಚೇಂದ್ರಿಯಗಳೆಂಬ ಪಂಚಹೋತೃಗಳು ಸುರಿಯುತ್ತಿರುವ ವಿಷಯ ಗಳೆಂಬ ತುಪ್ಪದ ಧಾರೆಯಿಂದ ವರ್ಧಿಸುತ್ತಿರುವ, ಬಹುಬಗೆಯ ವಾಸನೆಗಳೆಂಬ ಕಟ್ಟಿಗೆಗಳಿಂದ ಪ್ರಜ್ವಲಿಸುತ್ತಿರುವ ಮನೋಮಯಕೋಶವೆಂಬ ಅಗ್ನಿಯು ಇಡೀ ಪ್ರಪಂಚವನ್ನೇ ಸುಡುತ್ತಿರುವುದು.

[ಮನೋಮಯಕೋಶವು ಜಗತ್ತೆಲ್ಲವನ್ನೂ ಹೇಗೆ ಬಂಧಿಸಿದೆಯೆಂದು ಇಲ್ಲಿ ಹೇಳಿದೆ.]
ನ ಹ್ಯಸ್ತ್ಯವಿದ್ಯಾ ಮನಸೋsತಿರಿಕ್ತಾ
ಮನೋ ಹ್ಯವಿದ್ಯಾ ಭವಬಂಧಹೇತುಃ ।
ತಸ್ಮಿನ್ ವಿನಷ್ಟೇ ಸಕಲಂ ವಿನಷ್ಟಂ
ವಿಜೃಂಭಿತೇsಸ್ಮಿನ್ ಸಕಲಂ ವಿಜೃಂಭತೇ ॥ 169 ॥
್ಣಋಸ್ಶ್ಯ್¯ಮನಸ್ಣಃಋಸ್ಬಋ¯ = ಮನಸ್ಸಿಗಿಂತಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಅತಿರಿಕ್ತ್ಣಂಋಸ್ಬಋ¯ = ಬೇರೆಯಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಅವಿದ್ಯ್ಣಾಋಸ್ಬಋ¯ = ಅವಿದ್ಯೆಯು ್ಣಋಸ್ಶ್ಯ್¯ನ ಹಿ ಅಸ್ತ್ಣಿಋಸ್ಬಋ¯ = ಇಲ್ಲವೇ ಇಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಮನಃ ಹ್ಣಿಋಸ್ಬಋ¯ = ಮನಸ್ಸೇ ್ಣಋಸ್ಶ್ಯ್¯ಭವಬಂಧುಹೇತುಃ = ಸಂಸಾರ ಬಂಧಕ್ಕೆ ಕಾರಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಅವಿದ್ಯ್ಣಾಋಸ್ಬಋ¯ = ಅವಿದ್ಯೆಯು ; ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಮಿನ್ ವಿನಷ್ಟ್ಣೇಋಸ್ಬಋ¯ [ಸತಿ] = ಅದು ನಷ್ಟವಾದಾಗ ್ಣಋಸ್ಶ್ಯ್¯ಸಕಲಂ ವಿನಷ್ಟ್ಣಂಋಸ್ಬಋ¯ = ಎಲ್ಲವೂ ನಷ್ಟವಾಗುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಮಿನ್ ವಿಜೃಂಭಿತೇ [ಸತಿ] ಅದು ವಿಜೃಂಭಿಸುತ್ತಿರುವಾಗ ್ಣಋಸ್ಶ್ಯ್¯ಸಕಲ್ಣಂಋಸ್ಬಋ¯ = ಎಲ್ಲವೂ ್ಣಋಸ್ಶ್ಯ್¯ವಿಜೃಂಭತ್ಣೇಋಸ್ಬಋ¯ = ವಿಜೃಂಭಿಸುತ್ತದೆ.

169. ಮನಸ್ಸಿಗಿಂತಲೂ ಬೇರೆಯಾಗಿರುವ ಅವಿದ್ಯೆಯು ಇಲ್ಲವೇ ಇಲ್ಲ ; ಮನಸ್ಸೇ ಸಂಸಾರಬಂಧಕ್ಕೆ ಕಾರಣವಾದ ಅವಿದ್ಯೆಯು. ಅದು ನಷ್ಟವಾದಾಗ ಎಲ್ಲವೂ ನಷ್ಟವಾಗುತ್ತದೆ. ಅದು ವಿಜೃಂಭಿಸುತ್ತಿರುವಾಗ ಎಲ್ಲವೂ ವಿಜೃಂಭಿಸು ತ್ತದೆ.

[ಸುಷುಪ್ತಿಯಲ್ಲಿ ಮನಸ್ಸು ಲಯವಾದಾಗ ಇಡೀ ಜಗತ್ತು ನಷ್ಟವಾದಂತೆಯೇ. ಆದು ದರಿಂದ ಮನಸ್ಸೇ ಭವಬಂಧಕ್ಕೆ ಕಾರಣವೆಂಬುದು ಸ್ಪಷ್ಟವಾಗಿದೆ.]
ಸ್ವಪ್ನೇsಥRಶೂನ್ಯೇ ಸೃಜತಿ ಸ್ವಶಕ್ತ್ಯಾ
ಭೋಕ್ತ್ರಾದಿ-ವಿಶ್ವಂ ಮನ ಏವ ಸರ್ವಮ್ ।
ತಥೈವ ಜಾಗ್ರತ್ಯಪಿ ನೋ ವಿಶೇಷ-
ಸ್ತತ್ಸರ್ವಮೇತನ್ಮನಸೋ ವಿಜೃಂಭಣಮ್ ॥ 170 ॥
್ಣಋಸ್ಶ್ಯ್¯ಮನಃ ಏವ್ಣಋಸ್ಬಋ¯ = ಮನಸ್ಸೇ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಶಕ್ತ್ಯಾ್ಣಋಸ್ಬಋ¯ = ತನ್ನ ಶಕ್ತಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಅರ್ಥಶೂನ್ಯ್ಣೇಋಸ್ಬಋ¯ = ಯಾವ ಬಾಹ್ಯವಿಷಯವೂ ಇಲ್ಲದಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಪ್ನ್ಣೇಋಸ್ಬಋ¯ = ಸ್ವಪ್ನಾವಸ್ಥೆಯಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವ್ಣಂಋಸ್ಬಋ¯ = ಎಲ್ಲ ್ಣಋಸ್ಶ್ಯ್¯ಭೋಕ್ತ್ರಾ ದಿವಿಶ್ವ್ಣಂಋಸ್ಬಋ¯ = ಭೋಕ್ತ ೃವೇ ಮೊದಲಾದ ಪ್ರಪಂಚವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಸೃಜತ್ಣಿಋಸ್ಬಋ¯ = ಸೃಷ್ಟಿಸುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ತಥಾ ಏವ್ಣಋಸ್ಬಋ¯ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಜಾಗ್ರತಿ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಜಾಗ್ರದವಸ್ಥೆಯಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ವಿಶೇಷಃ ನ ಉ್ಣಋಸ್ಬಋ¯ = ವಿಶೇಷವಿಲ್ಲ
