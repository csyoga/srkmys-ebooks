[1 ಬಾಹ್ಯವಿಷಯಗಳಿಂದ ಚಿತ್ತವೃತ್ತಿಗಳನ್ನು ಹಿಂತಿರುಗಿಸಿಕೊಳ್ಳುವಿಕೆ.]
ಯದ್ಯುತ್ತರೋತ್ತರಾಭಾವಃ ಪೂರ್ವಪೂರ್ವಂ ತು ನಿಷ್ಫಲಮ್ ।
ನಿವೃತ್ತಿಃ ಪರಮಾ ತೃಪ್ತಿರಾನಂದೋsನುಪಮಃ ಸ್ವತಃ ॥ 419 ॥
್ಣಋಸ್ಶ್ಯ್¯ಉತ್ತರೋತ್ತರ-ಅಭಾವ್ಣಃಋಸ್ಬಋ¯ = ಮುಂದುಮುಂದಿನದರ ಅಭಾವವು ್ಣಋಸ್ಶ್ಯ್¯ಯದಿ [ಸ್ಯಾತ್] = ಉಂಟಾದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಪೂರ್ವಪೂರ್ವಂ ತ್ಣುಋಸ್ಬಋ¯ = ಹಿಂದುಹಿಂದಿನದು ್ಣಋಸ್ಶ್ಯ್¯ನಿಷ್ಫಲ್ಣಂಋಸ್ಬಋ¯ = ನಿಷ್ಫಲವು ; ್ಣಋಸ್ಶ್ಯ್¯ನಿವೃತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ನಿವೃತ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಪರಮಾ ತೃಪ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಪರಮತೃಪ್ತಿ ್ಣಋಸ್ಶ್ಯ್¯ಅನುಪಮ್ಣಃಋಸ್ಬಋ¯ = ಅಸದೃಶವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಆನಂದ್ಣಃಋಸ್ಬಋ¯ = ಆನಂದವು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವತ್ಣಃಋಸ್ಬಋ¯ = ಸ್ವಾಭಾವಿಕವಾಗಿಯೇ [ಉಂಟಾಗುತ್ತವೆ ].

419. ಇವುಗಳಲ್ಲಿ ಮುಂದುಮುಂದಿನದು ಸಿದ್ಧಿಸದೆ ಹೋದರೆ ಹಿಂದು ಹಿಂದಿನದು ನಿಷ್ಫಲವಾಗುವುದು.1 ಸಮಸ್ತಕಾರ್ಯಗಳಿಂದ ಹಿಂತಿರುಗುವುದು, ಪರಮತೃಪ್ತಿ, ಅಸದೃಶವಾದ ಆನಂದ–ಇವು ಸ್ವಾಭಾವಿಕವಾಗಿಯೇ ಉಂಟಾಗು ತ್ತವೆ.

[1 ಜ್ಞಾನವು ಸಿದ್ಧಿಸದಿದ್ದರೆ ವೈರಾಗ್ಯವಿದ್ದರೂ ನಿಷ್ಫಲವಾಗುವುದು. ಉಪರತಿಯು ಸಿದ್ಧಿಸದಿದ್ದರೆ ಜ್ಞಾನವಿದ್ದರೂ ನಿಷ್ಫಲವಾಗುವುದು. ಶಾಂತಿಯು ಸಿದ್ಧಿಸದಿದ್ದರೆ ಉಪರತಿ ಯಿದ್ದರೂ ನಿಷ್ಫಲವಾಗುವುದು.]
ದೃಷ್ಟದುಃಖೇಷ್ವನುದ್ವೇಗೋ ವಿದ್ಯಾಯಾಃ ಪ್ರಸ್ತುತಂ ಫಲಮ್ ।
ಯತ್ಕೃತಂ ಭ್ರಾಂತಿವೇಲಾಯಾಂ ನಾನಾ ಕರ್ಮ ಜುಗುಪ್ಸಿತಮ್ ।
ಪಶ್ಚಾನ್ನರೋ ವಿವೇಕೇನ ತತ್ ಕಥಂ ಕರ್ತುಮರ್ಹತಿ ॥ 420 ॥
್ಣಋಸ್ಶ್ಯ್¯ದೃಷ್ಟ-ದುಃಖೇಷ್ಣುಋಸ್ಬಋ¯ = ಪ್ರಾಪ್ತವಾದ ದುಃಖಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅನುದ್ವೇಗ್ಣಃಋಸ್ಬಋ¯ = ಉದ್ವೇಗವಿಲ್ಲದಿರು ವುದು ್ಣಋಸ್ಶ್ಯ್¯ವಿದ್ಯಾಯಾಃ = ಜ್ಞಾನದ್ಣಋಸ್ಶ್ಯ್¯ಅ ಪ್ರಸ್ತುತಂ = ಮುಖ್ಯವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಫಲಂ = ್ಣಋಸ್ಬಋ¯ಫಲವ್ಣುಋಸ್ಶ್ಯ್¯ ಭ್ರಾಂತಿ ವೇಲಾಯ್ಣಾಂಋಸ್ಬಋ¯ = ಭ್ರಾಂತಿಯ ಕಾಲದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಜುಗುಪ್ಸಿತಂ = ್ಣಋಸ್ಬಋ¯ಅಸಹ್ಯಕರವಾದ್ಣಋಸ್ಶ್ಯ್¯ ಯತ್ = ್ಣಋಸ್ಬಋ¯ಯಾವ ್ಣಋಸ್ಶ್ಯ್¯ನಾನಾ ಕರ್ಮ = ಅನೇಕವಿಧವಾದ ಕರ್ಮವ್ಣುಋಸ್ಶ್ಯ್¯ ಕೃತ್ಣಂಋಸ್ಬಋ¯ = ಮಾಡಲ್ಪಟ್ಟಿತೊ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಣ್ಋಸ್ಬಋ¯ = ಅದನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಪಶ್ಚಾತ್ಣ್ಋಸ್ಬಋ¯ = ಅನಂತರ ್ಣಋಸ್ಶ್ಯ್¯ವಿವೇಕೇನ = ವಿವೇಕದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನರ್ಣಃಋಸ್ಬಋ¯ = ಮನುಷ್ಯನು ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ತು್ಣಂಋಸ್ಬಋ¯ = ಮಾಡಲು ್ಣಋಸ್ಶ್ಯ್¯ಕಥಮ್ ಅರ್ಹತ್ಣಿಋಸ್ಬಋ¯ = ಹೇಗೆ ಯೋಗ್ಯನಾಗುತ್ತಾನೆ ?

420. ದುಃಖಗಳು ಪ್ರಾಪ್ತವಾದಾಗ ಉದ್ವೇಗಪಡದಿರುವುದೇ ಜ್ಞಾನದ ಮುಖ್ಯ ವಾದ ಫಲವು. ಮನುಷ್ಯನು ಭ್ರಾಂತಿಯ ಕಾಲದಲ್ಲಿ ಯಾವ ಅಸಹ್ಯಕರವಾದ ನಾನಾ ಕರ್ಮವನ್ನು ಮಾಡಿದ್ದನೊ ಅದನ್ನು ಅನಂತರ ವಿವೇಕವಿರುವಾಗಲೂ ಹೇಗೆ ತಾನೇ ಮಾಡಿಯಾನು ?

ವಿದ್ಯಾಫಲಂ ಸ್ಯಾದಸತೋ ನಿವೃತ್ತಿಃ
ಪ್ರವೃತ್ತಿರಜ್ಞಾನಫಲಂ ತದೀಕ್ಷಿತಮ್ ।
ತಜ್ಞಾಜ್ಞಯೋರ್ಯನ್ಮೃಗತೃಷ್ಣಿಕಾದೌ
ನೋಚೇದ್ವಿದಾಂ ದೃಷ್ಟಫಲಂ ಕಿಮಸ್ಮಾತ್ ॥ 421 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಸತ್ಣಃಋಸ್ಬಋ¯ = ಅಸದ್ವಸ್ತುವಿನಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನಿವೃತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಹಿಂತಿರುಗುವುದೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿದ್ಯಾಫಲಂ ಸ್ಯಾತ್ಣ್ಋಸ್ಬಋ¯ = ಜ್ಞಾನದ ಫಲವಾಗುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರವೃತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = [ಅಸದ್ವಸ್ತುವಿನಲ್ಲಿ] ಪ್ರವೃತ್ತನಾಗುವುದೇ ್ಣಋಸ್ಶ್ಯ್¯ಅಜ್ಞಾನಫಲ್ಣಂಋಸ್ಬಋ¯ = ಅಜ್ಞಾನದ ಫಲವು ; ್ಣಋಸ್ಶ್ಯ್¯ಮೃಗತೃಷ್ಣಿಕಾದ್ಣೌಋಸ್ಬಋ¯ = ಬಿಸಿಲ್ಗುದುರೆ ಮೊದಲಾದುವು ಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ತಜ್ಞ-ಅಜ್ಞಯ್ಣೋಃಋಸ್ಬಋ¯ = ಅದನ್ನು ಅರಿತವನು ಮತ್ತು ಅರಿಯದವನು–ಇವರ ವಿಷಯದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ತತ್= ಅದು ್ಣಋಸ್ಶ್ಯ್¯ಈಕ್ಷಿತ್ಣಂಋಸ್ಬಋ¯ = ಕಂಡುಬರುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ನ ಉ ಚೇತ್ಣ್ಋಸ್ಬಋ¯ = ಹಾಗಿಲ್ಲದೆ ಹೋದರೆ ್ಣಋಸ್ಶ್ಯ್¯ವಿದ್ಣಾಂಋಸ್ಬಋ¯ = ಜ್ಞಾನಿಗಳಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಮಾತ್ಣ್ಋಸ್ಬಋ¯ = ಇದಕ್ಕಿಂತಲೂ ್ಣಋಸ್ಶ್ಯ್¯ದೃಷ್ಟಫಲ್ಣಂಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಕಿಮ್ಣಋಸ್ಬಋ¯ = ದೃಷ್ಟಫಲವು ಯಾವುದಿದ್ದೀತು ?

421. ಅಸದ್ವಸ್ತುವಿನಿಂದ ಹಿಂತಿರುಗುವುದೇ ಜ್ಞಾನದ ಫಲವು. ಅಸದ್ವಸ್ತುವಿನ ಕಡೆಗೆ ನುಗ್ಗುವುದೇ ಅಜ್ಞಾನದ ಫಲವು. ಬಿಸಿಲ್ಗುದುರೆ ಮೊದಲಾದುವುಗಳನ್ನು ಅರಿತವನಲ್ಲಿ ಅಥವಾ ಅರಿಯದವನಲ್ಲಿ1 ಇದು ಕಂಡುಬರುತ್ತದೆ. ಹಾಗಿಲ್ಲದಿದ್ದರೆ ಜ್ಞಾನಿಗಳಿಗೆ ಇದಕ್ಕಿಂತಲೂ ದೃಷ್ಟಫಲವು ಮತ್ತಾವುದಿರುವುದು ?

[[1 ಬಿಸಿnಲ್ಗುದುರೆಯನ್ನು ಅರಿತವನಿಗೆ ಯಾವ ಭ್ರಮೆಯೂ ಉಂಟಾಗುವುದಿಲ್ಲ ; ಅದನ್ನು ಅರಿಯದವನು ಮಾತ್ರ ನೀರಿದೆಯೆಂದು ಭಾವಿಸಿಕೊಂಡು ನುಗ್ಗುತ್ತಾನೆ. ಹಾಗೆಯೇ ಜಗತ್ತು ಜ್ಞಾನಿಗೆ ಕಾಣಿಸಿಕೊಂಡರೂ ಅವನು ಅದರ ವಿಲಾಸಗಳಿಂದ ಮೋಹಿತ ನಾಗುವುದಿಲ್ಲ. ಆದರೆ ಲೌಕಿಕನಿಗೆ ಹಾಗಾಗುವುದಿಲ್ಲ.]
ಅಜ್ಞಾನ-ಹೃದಯಗ್ರಂಥೇರ್ವಿನಾಶೋ ಯದ್ಯಶೇಷತಃ ।
ಅನಿಚ್ಛೋರ್ವಿಷಯಃ ಕಿಂ ನು ಪ್ರವೃತ್ತೇಃ ಕಾರಣಂ ಸ್ವತಃ ॥ 422 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಜ್ಞಾನ-ಹೃದಯ-ಗ್ರಂಥ್ಣೇಃಋಸ್ಬಋ¯ = ಅಜ್ಞಾನವೆಂಬ ಹೃದಯದ ಗ್ರಂಥಿಯ ್ಣಋಸ್ಶ್ಯ್¯ವಿನಾಶ್ಣಃಋಸ್ಬಋ¯ = ವಿನಾಶವು ್ಣಋಸ್ಶ್ಯ್¯ಅಶೇಷತಃ ಯದ್ಣಿಋಸ್ಬಋ¯ = ಸಂಪೂರ್ಣವಾಗಿ ಉಂಟಾದರೆ [ಅಗ] ್ಣಋಸ್ಶ್ಯ್¯ಅನಿಚ್ಛ್ಣೋಃಋಸ್ಬಋ¯ = ಇಚ್ಛಿಸದವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ವಿಷಯ್ಣಃಋಸ್ಬಋ¯ = ವಿಷಯವು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವತ್ಣಃಋಸ್ಬಋ¯ = ತಾನೇ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರವೃತ್ತ್ಣೇಃಋಸ್ಬಋ¯ =ಪ್ರವೃತ್ತಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಕಾರಣ್ಣಂಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂ ನ್ಣುಋಸ್ಬಋ¯ = ಕಾರಣವಾಗುತ್ತದೆಯೇ ?

422. ಅಜ್ಞಾನವೆಂಬ ಹೃದಯಗ್ರಂಥಿಯು ಸಂಪೂರ್ಣವಾಗಿ ನಾಶವಾಗಿದ್ದರೆ ಯಾವುದನ್ನೂ ಇಚ್ಛಿಸದವನಿಗೆ ವಿಷಯವು ತಾನಾಗಿಯೇ ಪ್ರವೃತ್ತಿಗೆ ಕಾರಣ ವಾಗುತ್ತದೆಯೇ ?

ವಾಸನಾsನುದಯೋ ಭೋಗ್ಯೇ ವೈರಾಗ್ಯಸ್ಯ ತದಾವಧಿಃ । ಅಹಂಭಾವೋದಯಾಭಾವೋ ಬೋಧಸ್ಯ ಪರಮಾವಧಿಃ ।
ಲೀನವೃತ್ತೇರನುತ್ಪತ್ತಿರ್ಮಯಾRದೋಪರತೇಸ್ತು ಸಾ ॥ 423 ॥
[ಯದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗ] ಭೋಗ್ಯ್ಣೇಋಸ್ಬಋ¯ = ವಿಷಯಭೋಗದಲ್ಲ್ಣಿಋಸ್ಶ್ಯ್¯ ವಾಸನಾ -ಅನುದಯ್ಣಃಋಸ್ಬಋ¯ = ವಾಸನೆಯು ಉಂಟಾಗುವುದಿಲ್ಲವೊ ್ಣಋಸ್ಶ್ಯ್¯ತದ್ಣಾಋಸ್ಬಋ¯ = ಆಗ ್ಣಋಸ್ಶ್ಯ್¯ವೈರಾಗ್ಯಸ್ಯ್ಣಋಸ್ಬಋ¯ = ವೈರಾಗ್ಯದ ್ಣಋಸ್ಶ್ಯ್¯ಅವಧ್ಣಿಃಋಸ್ಬಋ¯ =ಎಲ್ಲೆಯು ; ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂಭಾವ-ಉದಯ-ಅಭಾವ್ಣಃಋಸ್ಬಋ¯ = ಅಹಂಕಾರೋದಯದ ಶೂನ್ಯತೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಬೋಧಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಜ್ಞಾನದ ್ಣಋಸ್ಶ್ಯ್¯ಪರಮಾವಧ್ಣಿಃಋಸ್ಬಋ¯ = ಕೊನೆಯ ಎಲ್ಲೆಯು ; ್ಣಋಸ್ಶ್ಯ್¯ಲೀನವೃತ್ತ್ಣೇಃಋಸ್ಬಋ¯ = ಲಯವಾದ ಅಂತಃಕರಣವೃತ್ತಿಯ ್ಣಋಸ್ಶ್ಯ್¯ಸಾ ಅನುತ್ಪತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಆ ಅನುತ್ಪತ್ತಿಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಉಪರತ್ಣೇಃಋಸ್ಬಋ¯ = ಉಪರತಿಯ ್ಣಋಸ್ಶ್ಯ್¯ಮರ್ಯಾದ್ಣಾಋಸ್ಬಋ¯ = ಎಲ್ಲೆಯು.

423. ಯಾವಾಗ ವಿಷಯಭೋಗದಲ್ಲಿ ವಾಸನೆಯು ಉಂಟಾಗುವುದಿಲ್ಲವೊ ಆಗ ವೈರಾಗ್ಯದ ಎಲ್ಲೆಯನ್ನು ಸೇರಿದಂತಾಗುವುದು ; ಅಹಂಭಾವವು ಉಂಟಾಗ ದಿರುವುದೇ ಜ್ಞಾನದ ಕೊನೆಯ ಎಲ್ಲೆ ; ಮತ್ತು ಲಯವಾದ ಅಂತಃಕರಣವೃತ್ತಿಯು ಪುನಃ ಉತ್ಪನ್ನವಾಗದಿರುವುದೇ ಉಪರತಿಯ ಎಲ್ಲೆಯು.

[‘ಬ್ರಹ್ಮಲೋಕವನ್ನು ತೃಣವೆಂದು ಭಾವಿಸುವುದು ವೈರಾಗ್ಯದ ಎಲ್ಲೆಯು ; ದೇಹಾ ತ್ಮದ ಐಕ್ಯದಂತೆ ಜೀವಅ-ಪರಮಾತ್ಮರ ಐಕ್ಯವಾದಾಗ ಜ್ಞಾನವು ಸಮಾಪ್ತಿಯನ್ನು ಹೊಂದು ತ್ತದೆ ; ಸುಷುಪ್ತಿಯಲ್ಲಿರುವ ಹಾಗೆ ದ್ವೈತಪ್ರಪಂಚವು ಲಯವಾಗುವುದು ಉಪರತಿಯ ಎಲ್ಲೆಯು’ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಲೋಕ-ತೃಣೀಕಾರೋ ವೈರಾಗ್ಯಸ್ಯಾವಧಿರ್ಮತಃ । ದೇಹಾತ್ಮವತ್ ಪರಾ ತ್ಮತ್ವ-ದಾರ್ಢ್ಯೇ ಬೋಧಃ ಸಮಾಪ್ಯತೇ ॥ ಸುಪ್ತಿವದ್ ವಿಸ್ಮೃತ್ಣಿಃಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಸೀಮಾ ಭವೇದುಪರಮಸ್ಯ ಹಿ । (ಪಂಚದಶಿಯ ಚಿತ್ರದೀಪಪ್ರಕರಣ–285-286.]
68. ಜೀವನ್ಮುಕ್ತನ ಲಕ್ಷಣ
ಬ್ರಹ್ಮಾಕಾರತಯಾ ಸದಾ ಸ್ಥಿತತಯಾ ನಿರ್ಮುಕ್ತ-ಬಾಹ್ಯಾರ್ಥಧೀ-
ರನ್ಯಾವೇದಿತ-ಭೋಗ್ಯಭೋಗ-ಕಲನೋ ನಿದ್ರಾಲುವದ್ಬಾಲವತ್ ।
ಸ್ವಪ್ನಾಲೋಕಿತ-ಲೋಕವಜ್ಜಗದಿದಂ ಪಶ್ಯನ್ ಕ್ವಚಿಲ್ಲಬ್ಧಧೀ-
ರಾಸ್ತೇ ಕಶ್ಚಿದನಂತ-ಪುಣ್ಯಫಲಭುಗ್ ಧನ್ಯಃ ಸ ಮಾನ್ಯೋ ಭುವಿ ॥ 424 ॥
್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಾಕಾರತಯ್ಣಾಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮರೂಪದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಸದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಥಿತ ತಯ್ಣಾಋಸ್ಬಋ¯ = ಇರುವುದರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ಮುಕ್ತ-ಬಾಹ್ಯ-ಅರ್ಥ-ಧ್ಣೀಃಋಸ್ಬಋ¯ = ಬಾಹ್ಯವಿಷಯಗಳ ಜ್ಞಾನವೇ ಇಲ್ಲ ದವವಾಗಿ, ್ಣಋಸ್ಶ್ಯ್¯ನಿದ್ರಾಲುವತ್ಣಋಸ್ಬಋ¯್ = ನಿದ್ರೆಯುಳ್ಳವನಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಬಾಲವತ್ಣ್ಋಸ್ಬಋ¯ = ಮಗುವಿನಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಅನ್ಯ-ಆವೇದಿತ-ಭೋಗ್ಯ-ಭೋಗ-ಕಲನ್ಣಃಋಸ್ಬಋ¯ = ಇತರರಿಂದ ತಿಳಿಸಲ್ಪಟ್ಟಭೋಗ್ಯವಸ್ತುಗಳ ಅನುಭವವುಳ್ಳವನಾಗಿ, ್ಣಋಸ್ಶ್ಯ್¯ಇದಂ ಜಗತ್ಣ್ಋಸ್ಬಋ¯ = ಈ ಜಗತ್ತನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಪ್ನ-ಆಲೋಕಿತ-ಲೋಕವತ್ಣ್ಋಸ್ಬಋ¯ = ಕನಸಿನಲ್ಲಿ ನೋಡಲ್ಪಟ್ಟ ಲೋಕದಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಪಶ್ಯನ್ಣ್ಋಸ್ಬಋ¯ =ನೋಡುವವನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ವಚಿತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವಾಗಲೋ ್ಣಋಸ್ಶ್ಯ್¯ಲಬ್ಧ-ಧ್ಣೀಃಋಸ್ಬಋ¯ = ಎಚ್ಚರಗೊಂಡವನಾಗಿ, ್ಣಋಸ್ಶ್ಯ್¯ಅನಂತ-ಪುಣ್ಯಫಲ-ಭುಕ್ = ಅನಂತಪುಣ್ಯಫಲವನ್ನು ಅನುಭವಿಸುವ ್ಣಋಸ್ಶ್ಯ್¯ಕಶ್ಚಿತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವನೊ ಒಬ್ಬನು ್ಣಋಸ್ಶ್ಯ್¯ಆಸ್ತ್ಣೇಋಸ್ಬಋ¯ = ಇರುತ್ತಾನೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ಭುವಿ = ಭೂಲೋಕದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಧನ್ಯಃ = ಧನ್ಯನು ್ಣಋಸ್ಶ್ಯ್¯ಮಾನ್ಯ್ಣಃಋಸ್ಬಋ¯ = ಮಾನ್ಯನು.

424. ಬ್ರಹ್ಮರೂಪದಿಂದಲೇ ಯಾವಾಗಲೂ ಇರುವುದರಿಂದ ಬಾಹ್ಯ ವಿಷಯ ಗಳ ಜ್ಞಾನವೇ ಇಲ್ಲದವನಾಗಿ, ನಿದ್ರಿಸುತ್ತಿರುವವನಂತೆ, ಮಗುವಿನಂತೆ ಇತರರಿಂದ ಕೊಡಲ್ಪಟ್ಟ ಭೋಗ್ಯವಸ್ತುಗಳನ್ನು ಅನುಭವಿಸುತ್ತ, ಈ ಜಗತ್ತನ್ನು ಕನಸಿನಲ್ಲಿ ಕಾಣುವ ಲೋಕದಂತೆ ನೋಡುತ್ತ, ಯಾವಾಗಲೋ ಒಮ್ಮೆ ಬಹಿರ್ಮುಖವಾದ ಜ್ಞಾನವುಳ್ಳವನಾಗಿ ಅನಂತ-ಪುಣ್ಯಫಲವನ್ನು ಅನುಭವಿಸುವ ಯಾವನೊ ಒಬ್ಬನು ಇರುತ್ತಾನೆ ; ಭೂಲೋಕದಲ್ಲಿ ಅವನೇ ಧನ್ಯನು ಮತ್ತು ಮಾನ್ಯನು.

ಸ್ಥಿತಪ್ರಜ್ಞೋ ಯತಿರಯಂ ಯಃ ಸದಾನಂದಮಶ್ನುತೇ ।
ಬ್ರಹ್ಮಣ್ಯೇವ ವಿಲೀನಾತ್ಮಾ ನಿರ್ವಿಕಾರೋ ವಿನಿಷಿ­ಯಃ ॥ 425 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣಃಋಸ್ಬಋ¯ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಣಿ ಏವ್ಣಋಸ್ಬಋ¯ಅ = ಬ್ರಹ್ಮದಲ್ಲಿಯೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿಲೀನಾತ್ಮ್ಣಾಋಸ್ಬಋ¯ = ಲಯವಾದ ಅಂತಃಕರಣವುಳ್ಳವನೊ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕಾರ್ಣಃಋಸ್ಬಋ¯ = ನಿರ್ವಿಕಾರನೊ ್ಣಋಸ್ಶ್ಯ್¯ವಿನಿಷಿ­ಯ್ಣಃಋಸ್ಬಋ¯ = ನಿಷ್ಖ್ರಿಯನೊ ್ಣಋಸ್ಶ್ಯ್¯ಸದಾ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಆನಂದಮ್ ಅಶ್ನುತ್ಣೇಋಸ್ಬಋ¯ = ಆನಂದವನ್ನು ಹೊಂದಿರುವನೊ ್ಣಋಸ್ಶ್ಯ್¯ಅಯ್ಣಂಋಸ್ಬಋ¯ = ಈ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣಿಃಋಸ್ಬಋ¯ = ಯತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಥಿತಪ್ರಜ್ಞ್ಣಃಋಸ್ಬಋ¯ = ಸ್ಥಿತಪ್ರಜ್ಞನು.

425. ಯಾವನು ಬ್ರಹ್ಮದಲ್ಲಿಯೇ ಅಂತಃಕರಣವನ್ನು ಲಯಮಾಡಿಕೊಂಡು ಸರ್ವವಿಕಾರ-ರಹಿತನಾಗಿ ಸರ್ವಕರ್ಮರಹಿತನಾಗಿ ಯಾವಾಗಲೂ ಆನಂದವನ್ನು ಹೊಂದಿರುವನೊ ಆ ಯತಿಯು ಸ್ಥಿತಪ್ರಜ್ಞನು.

ಬ್ರಹ್ಮಾತ್ಮನೋಃ ಶೋಧಿತಯೋರೇಕಭಾವಾವಗಾಹಿನೀ ।
ನಿರ್ವಿಕಲ್ಪಾ ಚ ಚಿನ್ಮಾತ್ರಾ ವೃತ್ತಿಃ ಪ್ರಜ್ಞೇತಿ ಕಥ್ಯತೇ ।
ಸುಸ್ಥಿತಾsಸೌ ಭವೇದ್ಯಸ್ಯ ಸ್ಥಿತಪ್ರಜ್ಞಃ ಸ ಉಚ್ಯತೇ ॥ 426 ॥
್ಣಋಸ್ಶ್ಯ್¯ಶೋಧಿತಯ್ಣೋಃಋಸ್ಬಋ¯ = ಶೋಧಿಸಲ್ಪಟ್ಟ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಾತ್ಮನ್ಣೋಃಋಸ್ಬಋ¯ = ಪರಮಾತ್ಮ-ಜೀವಾತ್ಮರ ್ಣಋಸ್ಶ್ಯ್¯ಏಕ ಭಾವ-ಅವಗಾಹಿನ್ಣೀಋಸ್ಬಋ¯ = ಐಕ್ಯಭಾವವನ್ನು ಗ್ರಹಿಸುವ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಿಕಲ್ಣಾಋಸ್ಬಋ¯್ಪ = ನಿರ್ವಿಕಲ್ಪವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಚಿನ್ಮಾತ್ರಾ ಚ = ಚಿನ್ಮಾತ್ರವಾದ ್ಣಋಸ್ಶ್ಯ್¯ವೃತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ವೃತ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಜ್ಞಾ ಇತ್ಣಿಋಸ್ಬಋ¯ = ಪ್ರಜ್ಞೆಯೆಂದು ್ಣಋಸ್ಶ್ಯ್¯ಕಥ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಹೇಳಲ್ಪಡುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಯಾವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಣೌಋಸ್ಬಋ¯ = ಈ ಪ್ರಜ್ಞೆಯು ್ಣಋಸ್ಶ್ಯ್¯ಸುಸ್ಥಿತ್ಣಾಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಭವೇತ್ಣಋಸ್ಬಋ¯್ = ಸ್ಥಿರವಾಗಿರುವುದೊ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಥಿತಪ್ರಜ್ಞ್ಣಃಋಸ್ಬಋ¯ = ಸ್ಥಿತಪ್ರಜ್ಞನೆಂದು ್ಣಋಸ್ಶ್ಯ್¯ಉಚ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಹೇಳಲ್ಪ ಡುತ್ತಾನೆ.

426. (ತತ್, ತ್ವಮ್ ಎಂಬ ಪದಗಳಿಗೆ ವಾಚ್ಯರಾಗಿರುವ) ಪರಮಾತ್ಮ, ಜೀವಾತ್ಮ–ಇವರನ್ನು ಶೋಧಿಸಿ1 ಐಕ್ಯಭಾವವನ್ನು ಗ್ರಹಿಸುವ, ನಿರ್ವಿಕಲ್ಪವೂ ಚಿನ್ಮಾತ್ರವೂ ಆದ ವೃತ್ತಿಯು ಪ್ರಜ್ಞೆ ಎಂದು ಹೇಳಲ್ಪಡುತ್ತದೆ. ಯಾವನಿಗೆ ಈ ಪ್ರಜ್ಞೆಯು ಸ್ಥಿರವಾಗಿರುವುದೊ ಅವನು ಸ್ಥಿತಪ್ರಜ್ಞನು ಎನಿಸುತ್ತಾನೆ.

[1 241ನೆಯ ಶ್ಲೋಕವನ್ನು ನೋಡಿ.]
ಯಸ್ಯ ಸ್ಥಿತಾ ಭವೇತ್ ಪ್ರಜ್ಞಾ ಯಸ್ಯಾನಂದೋ ನಿರಂತರಃ ।
ಪ್ರಪಂಚೋ ವಿಸ್ಮೃತಪ್ರಾಯಃ ಸ ಜೀವನ್ಮುಕ್ತ ಇಷ್ಯತೇ ॥ 427 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಯಾವನ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಜ್ಞ್ಣಾಋಸ್ಬಋ¯ = ಪ್ರಜ್ಞೆಯ್ಣುಋಸ್ಶ್ಯ್¯ ಸ್ಥಿತಾ ಭವೇತ್ಣ್ಋಸ್ಬಋ¯ = ಸ್ಥಿತವಾಗಿರುವುದೊ, ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಣಋಸ್ಬಋ¯್ಯ = ಯಾವನ ್ಣಋಸ್ಶ್ಯ್¯ಆನಂದ್ಣಃಋಸ್ಬಋ¯ = ಆನಂದವು ್ಣಋಸ್ಶ್ಯ್¯ನಿರಂತರ್ಣಃಋಸ್ಬಋ¯ = ನಿರಂತರವಾಗಿರುವುದೊ, [ಯಾವನಿಗೆ] ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಪಂಚ್ಣಃಋಸ್ಬಋ¯ = ಪ್ರಪಂಚವು ್ಣಋಸ್ಶ್ಯ್¯ವಿಸ್ಮೃತಪ್ರಾಯ್ಣಃಋಸ್ಬಋ¯ = ಮರೆತುಹೋದಂತೆ ಆಗಿರುವುದೊ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ಜೀವನ್ಮುಕ್ತ್ಣಃಋಸ್ಬಋ¯ = ಜೀವನ್ಮುಕ್ತನೆಂದು ್ಣಋಸ್ಶ್ಯ್¯ಇಷ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಹೇಳಲ್ಪ ಡುತ್ತಾನೆ.

427. ಯಾವನ ಪ್ರಜ್ಞೆಯು ಸ್ಥಿತವಾಗಿರುವುದೊ ಯಾವನ ಆನಂದವು ನಿರಂತರವಾಗಿರುವುದೊ ಯಾವನಿಗೆ ಪ್ರಪಂಚವು ಮರೆತುಹೋದಂತೆ ಆಗಿರು ವುದೊ ಅವನು ಜೀವನ್ಮುಕ್ತನು ಎನಿಸುತ್ತಾನೆ.

ಲೀನಧೀರಪಿ ಜಾಗರ್ತಿ ಯೋ ಜಾಗ್ರದ್ಧರ್ಮವರ್ಜಿತಃ ।
ಬೋಧೋ ನಿರ್ವಾಸನೋ ಯಸ್ಯ ಸ ಜೀವನ್ಮುಕ್ತ ಇಷ್ಯತೇ ॥ 428 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣಃಋಸ್ಬಋ¯ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಲೀನಧೀಃ ಅಪ್ಣಿಋಸ್ಬಋ¯ = [ಪರಮಾತ್ಮನಲ್ಲಿ] ಲಯವಾಗಿರುವ ಬುದ್ಧಿ ‡ಯುಳ್ಳವನಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಜಾಗ್ರದ್-ಧರ್ಮವರ್ಜಿತಃ = ಜಾಗ್ರದವಸ್ಥೆಯ ಧರ್ಮಗಳಿಂದ ಬಿಡಲ್ಪಟ್ಟವನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಜಾಗತ್ಣಿಋಸ್ಬಋ¯R = ಎಚ್ಚತ್ತಿರುವನೊ, ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಣಋಸ್ಬಋ¯್ಯ = ಯಾವನ ್ಣಋಸ್ಶ್ಯ್¯ಬೋಧ್ಣಋಸ್ಬಋ¯ಃ = ಜ್ಞಾನವು ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ವಾಸನ್ಣಃಋಸ್ಬಋ¯ = ವಾಸನಾರಹಿತವಾಗಿರುವುದೊ ್ಣಋಸ್ಶ್ಯ್¯ಸಃ ಜೀವನ್ಮುಕ್ತ್ಣಃಋಸ್ಬಋ¯ ಇಷ್ಯತೇ.

428. ಯಾವನು ತನ್ಮ ಬುದ್ಧಿವೃತ್ತಿಯನ್ನು (ಪರಮಾತ್ಮನಲ್ಲಿ) ಲಯಗೊಳಿಸಿ ಕೊಂಡಿದ್ದರೂ ಜಾಗ್ರದವಸ್ಥೆಯ ಧರ್ಮಗಳಿಂದ ಬಿಡಲ್ಪಟ್ಟು1 ಎಚ್ಚತ್ತಿರುವನೊ, ಯಾವನ ಜ್ಞಾನವು ವಿಷಯ-ವಾಸನಾರಹಿತವಾಗಿರುವುದೊ ಅವನು ಜೀವನ್ಮುಕ್ತನು ಎನಿಸುತ್ತಾನೆ.

[1 ‘ಇಂದ್ರಿಯಗಳ ಮೂಲಕ ವಿಷಯಗಳನ್ನು ಗ್ರಹಿಸುವುದು ಜಾಗ್ರದವಸ್ಥೆಯ ಲಕ್ಷಣವು’ ್ಣಋಸ್ಶ್ಯ್¯ಇಂದ್ರಿಯೈರರ್ಥೋಪಲಬ್ಧಿರ್ಜಾಗರಿತಮಿತಿ ಲಕ್ಷಣಮ್ಣಋಸ್ಬಋ¯.]
ಶಾಂತಸಂಸಾರಕಲನಃ ಕಲಾವಾನಪಿ ನಿಷ್ಕಲಃ ।
ಯಃ ಸಚಿತ್ತೋsಪಿ ನಿಶ್ಚಿತ್ತಃ ಸ ಜೀವನ್ಮುಕ್ತ ಇಷ್ಯತೇ ॥ 429 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣಃಋಸ್ಬಋ¯ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಶಾಂತ-ಸಂಸಾರ-ಕಲನ್ಣಋಸ್ಬಋ¯ಅಃ = ಶಾಂತವಾದ ಸಂಸಾರ ಚಿಂತೆಯುಳ್ಳವ ನಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಕಲಾವಾನ್ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಅವಯವಗಳುಳ್ಳವನಾಗಿದ್ದರೂ ್ಣಋಸ್ಶ್ಯ್¯ನಿಷ್ಕಲ್ಣಃಋಸ್ಬಋ¯ = ನಿರವಯವ ನಾಗಿರುವನೊ ್ಣಋಸ್ಶ್ಯ್¯ಸಚಿತ್ತಃ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಚಿತ್ತವುಳ್ಳವನಾಗಿದ್ದರೂ ್ಣಋಸ್ಶ್ಯ್¯ನಿಶ್ಚಿತ್ತ್ಣಃಋಸ್ಬಋ¯ = ಚಿತ್ತವಿಲ್ಲದವನಂ ತಿರುವನೊ ್ಣಋಸ್ಶ್ಯ್¯ಸಃ ಜೀವನ್ಮುಕ್ತಃ ಇಷ್ಯತೇ.್ಣಋಸ್ಬಋ¯

429. ಯಾವನು ಸಂಸಾರದ ಚಿಂತೆಯನ್ನೆಲ್ಲ ಕಳೆದುಕೊಂಡು, ಅವಯವ ಗಳುಳ್ಳವನಾಗಿದ್ದರೂ ಅವಯವರಹಿತನಂತಿರುವನೊ, ಚಿತ್ತವುಳ್ಳವನಾಗಿದ್ದರೂ ಚಿತ್ತವಿಲ್ಲದವನಂತಿರುವನೊ1 ಅವನು ಜೀವನ್ಮುಕ್ತನು ಎನಿಸುವನು.

[1 ಹೊರಗಿನ ವೃತ್ತಿಗಳೆಲ್ಲವೂ ಲಯವಾಗಿರುವುದರಿಂದ ಮುಗ್ಧನಂತೆ ಕಂಡುಬರು ವನು ಅಥವಾ ರಾಗಾದಿಶೂನ್ಯನಾಗಿರುವನು.]
ವರ್ತಮಾನೇsಪಿ ದೇಹೇsಸ್ಮಿಂಛಾಯಾವದನುವರ್ತಿನಿ ।
ಅಹಂತಾ-ಮಮತಾsಭಾವೋ ಜೀವನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ ॥ 430 ॥
್ಣಋಸ್ಶ್ಯ್¯ಛಾಯಾವತ್ಣ್ಋಸ್ಬಋ¯ = ನೆರಳಿನಂತೆ ್ಣಋಸ್ಶ್ಯ್¯ಅನುವರ್ತಿನಿ ಅಸ್ಮಿನ್ ದೇಹೇ ವರ್ತಮಾನೇ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಅನುಸರಿಸಿಕೊಂಡಿರುವ ಈ ದೇಹವು ಇರುತ್ತಿದ್ದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂತಾ-ಮಮತಾ-ಅಭಾವ್ಣಋಸ್ಬಋ¯ಃ = ಅಹಂಕಾರ-ಮಮಕಾರಗಳ ಶೂನ್ಯತೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಜೀವನ್ಮುಕ್ತಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಜೀವನ್ಮುಕ್ತನ ್ಣಋಸ್ಶ್ಯ್¯ಲಕ್ಷಣ್ಣಂಋಸ್ಬಋ¯ = ಲಕ್ಷಣವು.

430. ನೆರಳಿನಂತೆ ಈ ದೇಹವು ತನ್ನನ್ನು ಅನುಸರಿಸಿಕೊಂಡೇ ಇದ್ದರೂ ‘ನಾನು, ನನ್ನದು’ ಎಂಬ ಭಾವವಿಲ್ಲದಿರುವುದೇ ಜೀವನ್ಮುಕ್ತನ ಲಕ್ಷಣವು.

[ರಾಗ-ದ್ವೇಷಗಳು ಜರಾಮರಣಾದಿಗಳು ಇವೇ ಮೊದಲಾದ ವಿಕಾರಗಳು ಸ್ಥೂಲ ದೇಹದ ಅಭಿಮಾನದಿಂದ ಉಂಟಾಗಿರುವುದರಿಂದ ಅಂಥ ಅಭಿಮಾನವಿಲ್ಲದಿದ್ದರೆ ನಿರ್ವಿಕಾರತ್ವವು ಸಿದ್ಧಿಸುವುದೆಂಬುದು ಸ್ಪಷ್ಟವಾಗಿದೆ.]
ಅತೀತಾನನುಸಂಧಾನಂ ಭವಿಷ್ಯದವಿಚಾರಣಮ್ ।
ಔದಾಸೀನ್ಯಮಪಿ ಪ್ರಾಪ್ತೇ ಜೀವನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ ॥ 431 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅತೀತ-ಅನನುಸಂಧಾನ್ಣಂಋಸ್ಬಋ¯ = ಕಳೆದುಹೋದದ್ದನ್ನು ಸ್ಮರಿಸಿಕೊಳ್ಳದಿರುವುದು, ್ಣಋಸ್ಶ್ಯ್¯ಭವಿ ಷ್ಯತ್-ಅವಿಚಾರಣ್ಣಂಋಸ್ಬಋ¯ = ಮುಂದೆ ಒದಗುವುದನ್ನು ಯೋಚಿಸದಿರುವುದು, ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಪ್ತ್ಣೇಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಅಪಿ = ಈಗ ಒದಗಿರುವುದರಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಔದಾಸೀನ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಉದಾಸೀನತೆ–[ಇವುಗಳೇ] ್ಣಋಸ್ಶ್ಯ್¯ಜೀವ ನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ಣಋಸ್ಬಋ¯.

431. ಕಳೆದುಹೋದದ್ದನ್ನು ಸ್ಮರಿಸಿಕೊಳ್ಳದಿರುವುದು, ಮುಂದೆ ಒದಗುವು ದನ್ನು ಯೋಚಿಸದಿರುವುದು. ಈಗ ಒದಗಿರುವುದರಲ್ಲಿಯೂ ಉದಾಸೀನನಾಗಿರು ವುದು–(ಇವುಗಳೇ) ಜೀವನ್ಮುಕ್ತನ ಲಕ್ಷಣ.

ಗುಣ-ದೋಷ-ವಿಶಿಷ್ಟೇsಸ್ಮಿನ್ ಸ್ವಭಾವೇನ ವಿಲಕ್ಷಣೇ ।
ಸರ್ವತ್ರ ಸಮದರ್ಶಿತ್ವಂ ಜೀವನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ ॥ 432 ॥
್ಣಋಸ್ಶ್ಯ್¯ಗುಣ-ದೋಷ-ವಿಶಿಷ್ಟ್ಣೇಋಸ್ಬಋ¯ = ಗುಣದೋಷಗಳಿಂದ ಕೂಡಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಭಾವೇನ್ಣಋಸ್ಬಋ¯ಅ = ಸ್ವಭಾವ ದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿಲಕ್ಷಣ್ಣೇಋಸ್ಬಋ¯ = ವಿಚಿತ್ರವಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಮಿನ್ಣಋಸ್ಬಋ¯್ [ಜಗತಿ] = ಜಗತ್ತಿನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವತ್ರ = ಎಲ್ಲೆಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಸಮದರ್ಶಿತ್ವ್ಣಂಋಸ್ಬಋ¯ = ಸಮದರ್ಶಿಯಾಗಿರುವುದು ್ಣಋಸ್ಶ್ಯ್¯ಜೀವನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ಣಋಸ್ಬಋ¯

432. ಗುಣದೋಷಗಳಿಂದ ಕೂಡಿ ಸ್ವಭಾವದಿಂದಲೇ ವಿಚಿತ್ರವಾಗಿರುವ ಈ ಜಗತ್ತಿನಲ್ಲಿ ಎಲ್ಲೆಲ್ಲಿಯೂ ಸಮಾನದೃಷ್ಟಿಯಿಂದ1 ನೋಡುವುದು ಜೀವನ್ಮುಕ್ತನ ಲಕ್ಷಣವು.

[ಎಲ್ಲವೂ ಅವಿದ್ಯಾತ್ಮಕವಾಗಿರುವುದರಿಂದ ಜೀವನ್ಮುಕ್ತನು ಯಾವುದನ್ನೂ ದೋಷ ದೃಷ್ಟಿಯಿಂದ ಖಂಡಿಸುವುದಿಲ್ಲ. ಗುಣದೃಷ್ಟಿಯಿಂದ ಸ್ವೀಕರಿಸುವುದಿಲ್ಲ.
1 ‘ಜ್ಞಾನಿಗಳು ಸಮದರ್ಶಿಗಳಾಗಿರುವರು’ ್ಣಋಸ್ಶ್ಯ್¯ಪಂಡಿತಾಃ ಸಮದರ್ಶಿನ್ಣಃಋಸ್ಬಋ¯ (ಗೀತಾ 5.18)]
ಇಷ್ಟಾನಿಷ್ಟಾರ್ಥ-ಸಂಪ್ರಾಪ್ತೌ ಸಮದರ್ಶಿತಯಾssತ್ಮನಿ ।
ಉಭಯತ್ರಾವಿಕಾರಿತ್ವಂ ಜೀವನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ ॥ 433 ॥
್ಣಋಸ್ಶ್ಯ್¯ಇಷ್ಟ-ಅನಿಷ್ಟ-ಅರ್ಥ-ಸಂಪ್ರಾಪ್ಣೌಋಸ್ಬಋ¯್ತ = ಇಷ್ಟವಾದ ಅಥವಾ ಅನಿಷ್ಟವಾದ ವಿಷಯವು ಪ್ರಾಪ್ತವಾದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಮದರ್ಶಿತಯ್ಣಾಋಸ್ಬಋ¯ = ಸಮದೃಷ್ಟಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಉಭಯತ್ಣಋಸ್ಬಋ¯ಅ್ರ = ಎರಡರ ವಿಷಯ ದಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಮನ್ಣಿಋಸ್ಬಋ¯ = ತನ್ನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅವಿಕಾರಿತ್ವ್ಣಂಋಸ್ಬಋ¯ = ವಿಕಾರವನ್ನು ಪಡೆಯದಿರುವುದು ್ಣಋಸ್ಶ್ಯ್¯ಜೀವನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ಣಋಸ್ಬಋ¯.

433. ಇಷ್ಟವಾದ ಅಥವಾ ಅನಿಷ್ಟವಾದ ವಿಷಯವು ಪ್ರಾಪ್ತವಾದಾಗ ಸಮ ದೃಷ್ಯಿಯಿಂದ ಈ ಎರಡರ ವಿಷಯವಾಗಿಯೂ ತನ್ನಲ್ಲಿ ವಿಕಾರವನ್ನು ಪಡೆಯ ದಿರುವುದು ಜೀವನ್ಮುಕ್ತನ ಲಕ್ಷಣವು.

[‘ಯಾವನು ಸರ್ವವಸ್ತುಗಳಲ್ಲಿಯೂ ಆಸಕ್ತಿರಹಿತನಾಗಿ, ಆಯಾ ಪ್ರಿಯಾಪ್ರಿಯ ವಿಷಯಗಳನ್ನು ಪಡೆದಾಗ ಸಂತೋಷಿಸುವುದಿಲ್ಲವೊ ದ್ವೇಷಿಸುವುದಿಲ್ಲವೊ ಅವನ ಪ್ರಜ್ಞೆಯು ಸ್ಥಿರವಾಗಿರುವುದು.’ (ಗೀತಾ 2. 57)]
ಬ್ರಹ್ಮಾನಂದ-ರಸಾಸ್ವಾದಾಸಕ್ತ-ಚಿತ್ತತಯಾ ಯತೇಃ ।
ಅಂತರ್ಬಹಿರವಿಜ್ಞಾನಂ ಜೀವನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ ॥ 434 ॥
್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಾನಂದರಸ-ಆಸ್ವಾದ -ಆಸಕ್ತ-ಚಿತ್ತತಯ್ಣಾಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಾನಂದರಸದ ಸವಿಯಲ್ಲಿ ಆಸಕ್ತವಾಗಿರುವ ಚಿತ್ತವಿರುವುದರಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣೇಋಸ್ಬಋ¯ಃ = ಯತಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಅಂತ್ಣಋಸ್ಬಋ¯ಃ = ಒಳಗೂ ್ಣಋಸ್ಶ್ಯ್¯ಬಹ್ಣಿಃಋಸ್ಬಋ¯ = ಹೊರಗೂ ್ಣಋಸ್ಶ್ಯ್¯ಅವಿಜ್ಞಾನ್ಣಂಋಸ್ಬಋ¯ = ಅರಿವಿಲ್ಲದಿರುವುದು ್ಣಋಸ್ಶ್ಯ್¯ಜೀವನ್ಮುಕ್ತಸ್ಯ ಲಕ್ಷಣಮ್ಣಋಸ್ಬಋ¯್.

434. ಬ್ರಹ್ಮಾನಂದರಸವನ್ನು ಸವಿಯುವುದರಲ್ಲಿ ಚಿತ್ತವು ಆಸಕ್ತವಾಗಿರು ವುದರಿಂದ ಯತಿಗೆ ಒಳಗಿನ ಮತ್ತು ಹೊರಗಿನ ಅರಿವು ಇರುವುದಿಲ್ಲ ; ಇದು ಜೀವನ್ಮುಕ್ತನ ಲಕ್ಷಣವು.

ದೇಹೇಂದ್ರಿಯಾದೌ ಕರ್ತವ್ಯೇ ಮಮಾಹಂಭಾವ-ವರ್ಜಿತಃ ।
ಔದಾಸೀನ್ಯೇನ ಯಸ್ತಿಷ್ಠೇತ್ ಸ ಜೀವನ್ಮುಕ್ತಲಕ್ಷಣಃ ॥ 435 ॥
್ಣಋಸ್ಶ್ಯ್¯ದೇಹ-ಇಂದ್ರಿಯ-ಆದ್ಣೌಋಸ್ಬಋ¯ = ದೇಹ ಇಂದ್ರಿಯ ಇವೇ ಮೊದಲಾದುದರಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ತವ್ಯ್ಣೇಋಸ್ಬಋ¯ = ಮಾಡತಕ್ಕ ಕೆಲಸದಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ಮಮ-ಅಹಂ-ಭಾವ-ವರ್ಜಿತ್ಣಃಋಸ್ಬಋ¯ = ‘ನನ್ನದು’ ‘ನಾನು’ ಎಂಬ ಭಾವವಿಲ್ಲದೆ ್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣಃಋಸ್ಬಋ¯ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಔದಾಸೀನ್ಯೇನ್ಣಋಸ್ಬಋ¯ = ಉದಾಸೀನ ಭಾವದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ತಿಷ್ಠೇತ್ಣ್ಋಸ್ಬಋ¯ = ಇರುವನೊ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ಜೀವನ್ಮುಕ್ತಲಕ್ಷಣ್ಣಃಋಸ್ಬಋ¯

435. ದೇಹ ಇಂದ್ರಿಯಗಳು ಇವೇ ಮೊದಲಾದುವುಗಳಲ್ಲಿಯೂ ಮಾಡತಕ್ಕ ಕೆಲದಲ್ಲಿಯೂ ‘ನಾನು’ ‘ನನ್ನದು’ ಎಂಬ ಭಾವವಿಲ್ಲದೆ ಯಾವನು ಉದಾ ಸೀನನಾಗಿರುವನೊ ಅವನು ಜೀವನ್ಮುಕ್ತನ ಲಕ್ಷಣವುಳ್ಳವನು.

ವಿಜ್ಞಾತ ಆತ್ಮನೋ ಯಸ್ಯ ಬ್ರಹ್ಮಭಾವಃ ಶ್ರುತೇರ್ಬಲಾತ್ ।
ಭವಬಂಧವಿನಿರ್ಮುಕ್ತಃ ಸ ಜೀವನ್ಮುಕ್ತಲಕ್ಷಣಃ ॥ 436 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಯಾವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಶ್ರುತ್ಣೇಃಋಸ್ಬಋ¯ = ಶ್ರುತಿಯ ್ಣಋಸ್ಶ್ಯ್¯ಬಲಾತ್ಣ್ಋಸ್ಬಋ¯ = ಬಲದಿಂದ್ಣಋಸ್ಶ್ಯ್¯ ಆತ್ಮನ್ಣಋಸ್ಬಋ¯ಃ = ತನ್ನ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮಭಾವಃ = ಬ್ರಹ್ಮಭಾವವು ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾತ್ಣಃಋಸ್ಬಋ¯ = ತಿಳಿದಿರುವುದೊ, ್ಣಋಸ್ಶ್ಯ್¯ಭವಬಂಧವಿನಿರ್ಮುಕ್ತ್ಣಃಋಸ್ಬಋ¯ = ಸಂಸಾರಬಂಧದಿಂದ ಮುಕ್ತನಾದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ಜೀವನ್ಮುಕ್ತಲಕ್ಷಣಃ.್ಣಋಸ್ಬಋ¯

436. ಯಾವನಿಗೆ ಶ್ರುತಿಯ ಬಲದಿಂದ ತನ್ನ ಬ್ರಹ್ಮಭಾವವು ತಿಳಿದು ಬಂದಿರುವುದೊ ಸಂಸಾರಬಂಧದಿಂದ ಮುಕ್ತನಾದ ಅವನು ಜೀವನ್ಮುಕ್ತನ ಲಕ್ಷಣ ವುಳ್ಳವನು.

ದೇಹೇಂದ್ರಿಯೇಷ್ವಹಂಭಾವ ಇದಂ ಭಾವಸ್ತದನ್ಯಕೇ ।
ಯಸ್ಯ ನೋ ಭವತಃ ಕ್ವಾಪಿ ಸ ಜೀವನ್ಮುಕ್ತ ಇಷ್ಯತೇ ॥ 437 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಣಋಸ್ಬಋ¯್ಯ = ಯಾವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ದೇಹ-ಇಂದ್ರಿಯೇಷ್ಣುಋಸ್ಬಋ¯ = ದೇಹ ಇಂದ್ರಿಯ–ಇವುಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂ-ಭಾವ್ಣಃಋಸ್ಬಋ¯ = ‘ನಾನು’ ಎಂಬ ಭಾವವೂ ್ಣಋಸ್ಶ್ಯ್¯ತತ್-ಅನ್ಯಕ್ಣೇಋಸ್ಬಋ¯ = ಅವುಗಳಿಗಿಂತ ವ್ಯತಿ ರಿಕ್ತವಾದ ವಸ್ತುಗಳಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಇದಂ-ಭಾವ್ಣಃಋಸ್ಬಋ¯ = ‘ಇದು’ ಎಂಬ ಭಾವವೂ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ವ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಎಂದಿಗೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ಉ ಭವತ್ಣಃಋಸ್ಬಋ¯ = ಉಂಟಾಗುವುದಿಲ್ಲವೊ ್ಣಋಸ್ಶ್ಯ್¯ಸ ಜೀವನ್ಮುಕ್ತಃ ಇಷ್ಯತ್ಣೇಋಸ್ಬಋ¯.

437. ಯಾವನಿಗೆ ದೇಹೇಂದ್ರಿಯಗಳಲ್ಲಿ ‘ನಾನು’ ಎಂಬ ಭಾವವೂ, ಅವುಗಳಿ ಗಿಂತ ವ್ಯತಿರಿಕ್ತವಾದ ವಸ್ತುಗಳಲ್ಲಿ ‘ಇದು’ ಎಂಬ ಭಾವವೂ ಎಂದಿಗೂ ಉಂಟಾ ಗುವುದಿಲ್ಲವೊ ಅವನು ಜೀವನ್ಮುಕ್ತನು ಎನಿಸುವನು.

ನ ಪ್ರತ್ಯಗ್ಬ್ರಹ್ಮಣೋರ್ಭೇದಂ ಕದಾsಪಿ ಬ್ರಹ್ಮಸರ್ಗಯೋಃ ।
ಪ್ರಜ್ಞಯಾ ಯೋ ವಿಜಾನಾತಿ ಸ ಜೀವನ್ಮುಕ್ತ ಇಷ್ಯತೇ ॥ 438 ॥
್ಣಋಸ್ಶ್ಯ್¯ಯ್ಣಃಋಸ್ಬಋ¯ = ಯಾವನು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತ್ಯಕ್-ಬ್ರಹ್ಮಣ್ಣೋಃಋಸ್ಬಋ¯ = ಜೀವ, ಬ್ರಹ್ಮ ಇವರಲ್ಲಾಗಲಿ, ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ--ಸರ್ಗಯೋಃ = ್ಣಋಸ್ಬಋ¯ಬ್ರಹ್ಮ, ಜಗತ್ತು ಇವುಗಳಲ್ಲಾಗಲ್ಣಿಋಸ್ಶ್ಯ್¯ ಪ್ರಜ್ಞಯಾ = ್ಣಋಸ್ಬಋ¯ಯಥಾರ್ಥ್ಣಋಸ್ಶ್ಯ್¯ ್ಣಋಸ್ಬಋ¯ದೃಷ್ಟಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಭೇದ್ಣಂಋಸ್ಬಋ¯ = ಭೇದವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಕದಾ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಕಾಲತ್ರಯದಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ವಿಜಾನಾತ್ಣಿಋಸ್ಬಋ¯ = ಅರಿಯುವುದಿಲ್ಲವೊ ್ಣಋಸ್ಶ್ಯ್¯ಸಃ ಜೀವನ್ಮುಕ್ತಃ ಇಷ್ಯತೇ.್ಣಋಸ್ಬಋ¯

438. ಯಾವನು ಜೀವ ಮತ್ತು ಬ್ರಹ್ಮ–ಇವರಲ್ಲಾಗಲಿ, ಬ್ರಹ್ಮ ಮತ್ತು ಜಗತ್ತು–ಇವುಗಳಲ್ಲಾಗಲಿ ಯಥಾರ್ಥದೃಷ್ಟಿಯಿಂದ ಭೇದವನ್ನು ಕಾಲತ್ರಯ ದಲ್ಲಿಯೂ ಅರಿಯುವುದಿಲ್ಲವೊ ಅವನು ಜೀವನ್ಮುಕ್ತನು ಎನಿಸುವನು.

ಸಾಧುಭಿಃ ಪೂಜ್ಯಮಾನೇsಸ್ಮಿನ್ ಪೀಡ್ಯಮಾನೇsಪಿ ದುರ್ಜನೈಃ ।
ಸಮಭಾವೋ ಭವೇದ್ಯಸ್ಯ ಸ ಜೀವನ್ಮುಕ್ತ ಇಷ್ಯತೇ ॥ 439 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಮಿನ್ [ಶರೀರೇ] = ಈ ಶರೀರವು ್ಣಋಸ್ಶ್ಯ್¯ಸಾಧುಭ್ಣಿಋಸ್ಬಋ¯ಃ = ಸಾಧುಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪೂಜ್ಯಮಾನ್ಣೇಋಸ್ಬಋ¯ = ಗೌರವಿಸಲ್ಪಟ್ಟರೂ, ್ಣಋಸ್ಶ್ಯ್¯ದುರ್ಜನ್ಣೈಃಋಸ್ಬಋ¯ = ದುಷ್ಟರಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಪೀಡ್ಯಮಾನೇ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಪೀಡಿಸಲ್ಪಟ್ಟರೂ ್ಣಋಸ್ಶ್ಯ್¯ಯಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಯಾವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸಮಭಾವಃ ಭವೇತ್ಣ್ಋಸ್ಬಋ¯ = ಸಮಭಾವವಿರುವುದೊ ್ಣಋಸ್ಶ್ಯ್¯ಸಃ ಜೀವನ್ಮುಕ್ತ್ಣಃಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಇಷ್ಯತೇ.್ಣಋಸ್ಬಋ¯

439. ಸಾಧುಗಳು ಈ ಶರೀರವನ್ನು ಗೌರವಿಸಲಿ ಅಥವಾ ದುಷ್ಟರು ಪೀಡಿಸಲಿ, ಯಾವನಿಗೆ ಸಮಭಾವಿರುವುದೊ ಅವನು ಜೀವನ್ಮುಕ್ತನು ಎನಿಸುವನು.

ಯತ್ರ ಪ್ರವಿಷ್ಟಾ ವಿಷಯಾಃ ಪರೇರಿತಾ
ನದೀಪ್ರವಾಹಾ ಇವ ವಾರಿರಾಶೌ ।
ಲೀಯಂತಿ ಸನ್ಮಾತ್ರತಯಾ ನ ವಿಕ್ರಿಯಾ---
ಮುತ್ಪಾದಯಂತ್ಯೇಷ ಯತಿರ್ವಿಮುಕ್ತಃ ॥ 440 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪರ----ಈರಿತ್ಣಾಃಋಸ್ಬಋ¯ = ಇತರರಿಂದ ಪ್ರೇರಿತವಾದ ್ಣಋಸ್ಶ್ಯ್¯ವಿಷಯ್ಣಾಃಋಸ್ಬಋ¯ = ವಿಷಯಗಳು ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ರ್ಣಋಸ್ಬಋ¯ = ಯಾವನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರವಿಷ್ಟ್ಣಾಋಸ್ಬಋ¯ಃ = ಪ್ರವೇಶಿಸಿದವುಗಳಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ವಾರಿರಾಶ್ಣೌಋಸ್ಬಋ¯ = ಸಮುದ್ರದಲ್ಲಿ [ಪ್ರವೇಶಿ ಸಿದ] ನದೀ--ಪ್ರವಾಹ್ಣಾಃಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಇವ್ಣಋಸ್ಬಋ¯ = ನದೀಪ್ರವಾಹಗಳ ಹಾಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸನ್ಮಾತ್ರತಯ್ಣಾಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮಾ ತ್ಮೈಕ್ಯ ಭಾವನೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಲೀಯಂತ್ಣಿಋಸ್ಬಋ¯ = ಲಯವಾಗುವುವೊ, ್ಣಋಸ್ಶ್ಯ್¯ವಿಕ್ರಿಯ್ಣಾಂಋಸ್ಬಋ¯ = ವಿಕಾರವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನ ಉತ್ಪಾದಯಂತ್ಣಿಋಸ್ಬಋ¯ = ಉಂಟುಮಾಡುವುದಿಲ್ಲವೊ ್ಣಋಸ್ಶ್ಯ್¯ಏಷಃ ಯತ್ಣಿಃಋಸ್ಬಋ¯ = ಈ ಯತಿಯೇ ್ಣಋಸ್ಶ್ಯ್¯ವಿಮುಕ್ತ್ಣಃಋಸ್ಬಋ¯ = ಮುಕ್ತನು.

440. ಇತರರಿಂದ ಪ್ರೇರಿತವಾಗಿ ಬರುವ ವಿಷಯಗಳು ಪ್ರವೇಶಿಸಿ–ಸಮುದ್ರ ವನ್ನು ಪ್ರವೇಶಿಸಿದ ನದೀಪ್ರವಾಹಗಳಂತೆ–ಬ್ರಹಾತ್ಮೈಕ್ಯಭಾವನೆಯಿಂದ ಯಾವ ಯತಿಯಲ್ಲಿ ಲಯವಾಗುವುವೋ ಮತ್ತು ಯಾವ ವಿಕಾರವನ್ನೂ ಉಂಟುಮಾಡು ವುದಿಲ್ಲವೊ ಅವನೇ ಮುಕ್ತನು.

[[ಜೀವನ್ಮುಕ್ತನು ಸ್ವತಃ ಬಹಿರ್ಮುಖನಾಗದಿದ್ದರೂ, ಇತರರು ಮಾಡುವ ಸ್ತುತಿ- ನಿಂದೆಗಳು–ಮತ್ತೊಬ್ಬರ ದೃಷ್ಟಿಯಿಂದ ಒಳ್ಳೆಯದು ಕೆಟ್ಟದು ಎನಿಸಿದ್ದರೂ–ಇವ ನಲ್ಲಿ ಯಾವ ವಿಕಾರವನ್ನೂ ಉಂಟುಮಾಡದೆ ಅಡಗಿ ಹೋಗುವುವು (ಗೀತಾ 2. 70 ನೋಡಿ).]

ಣಿ2ಣಬ13ಶ್ಲಸ್ನ್ಇವ್ಶ್ಶ್ ಪ್ ವಿಜ್ಞಾತಬ್ರಹ್ಮತತ್ತ್ವಸ್ಯ ಯಥಾಪೂರ್ವಂ ನ ಸಂಸೃತಿಃ ।
ಅಸ್ತಿ ಚೇನ್ನ ಸ ವಿಜ್ಞಾತಬ್ರಹ್ಮಭಾವೋ ಬಹಿರ್ಮುಖಃ ॥ 441 ॥

್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾತ-ಬ್ರಹ್ಮತತ್ತ್ವಸ್ಯ = ್ಣಋಸ್ಬಋ¯ಬ್ರಹ್ಮತತ್ತ್ವವನ್ನು ಅರಿತುಕೊಂಡವನಿಗ್ಣೆಋಸ್ಶ್ಯ್¯ ಯಥ್ಣಾಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಪೂರ್ವ್ಣಂಋಸ್ಬಋ¯ = ಮೊದಲಿನಂತೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸೃತ್ಣಿಋಸ್ಬಋ¯ಃ = ಸಂಸಾರವು ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ಅ = ಇರುವುದಿಲ್ಲ. ್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ತಿಚೇತ್ಣ್ಋಸ್ಬಋ¯ = ಇದ್ದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾತ- -ಬ್ರಹ್ಮಭಾವಃ ನ್ಣಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮತತ್ತ್ವವನ್ನು ಅರಿತು ಕೊಂಡವನು ಆಗುವುದಿಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ಬಹಿರ್ಮುಖ್ಣಋಸ್ಬಋ¯ಃ = ಬಹಿರ್ಮುಖನಾದವನೇ.

442. ಬ್ರಹ್ಮತತ್ತ್ವವನ್ನು ಅರಿತುಕೊಂಡವನಿಗೆ ಮೊದಲಿನಂತೆಯೇ ಸಂಸಾರ ವಿರುವುದಿಲ್ಲ ; ಮುನ್ನಿನಂತೆ ಸಂಸಾರವಾಸನೆಯಿದ್ದರೆ ಅವನು ಬ್ರಹ್ಮತತ್ತ್ವವನ್ನು ಅರಿತುಕೊಂಡವನಾಗುವುದಿಲ್ಲ. ಬಹಿರ್ಮುಖನೇ1 (ಆಗಿರುತ್ತಾನೆ).

[[[1 ಇಂದ್ರಿಯಗಳು ವಿಷಯಗಳ ಕಡೆಗೆ ಹೋಗುವ ಪ್ರವೃತ್ತಿಯುಳ್ಳವನು.]
ಪ್ರಾಚೀನವಾಸನಾವೇಗಾದಸೌ ಸಂಸರತೀತಿ ಚೇತ್ ।
ನ ಸದೇಕತ್ವವಿಜ್ಞಾನಾನ್ಮಂದೀಭವತಿ ವಾಸನಾ ॥ 442 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಸ್ಣೌಋಸ್ಬಋ¯ = ಇವನು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಚೀನ---ವಾಸನಾ-ವೇಗಾತ್ಣ್ಋಸ್ಬಋ¯ = ಹಿಂದಿನ ವಾmಸನಾವೇಗದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸರತ್ಣಿಋಸ್ಬಋ¯ = ಸಂಸಾರಹೊಂದುತ್ತಾನೆ–-್ಣಋಸ್ಶ್ಯ್¯ಇತಿ ಚೇತ್ಣ್ಋಸ್ಬಋ¯ = ಎಂದು ಹೇಳಿದರೆ, ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ಅ =ಇಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಸತ್-ಏಕತ್ವ--ವಿಜ್ಞಾನಾತ್ಣ್ಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮ ಅಭೇದಜ್ಞಾನದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಾಸನ್ಣಾಋಸ್ಬಋ¯ = ವಾಸನೆಯು ್ಣಋಸ್ಶ್ಯ್¯ಮಂದೀಭವತಿ = ಕೃಶವಾಗುತ್ತದೆ.

442. ‘ಇವನು ಹಿಂದಿನ ವಾಸನಾವೇಗದಿಂದ ಸಂಸರಿಸುತ್ತಾನೆ’ ಎಂದರೆ ಅದಕ್ಕೆ ಉತ್ತರವು ‘ಇಲ್ಲ’ ; (ಏಕೆಂದರೆ) ಬ್ರಹ್ಮದ ಅಭೇದಜ್ಞಾನದಿಂದ ಸಂಸಾರ ವಾಸನೆಯು ಕೃಶಿಸಿಹೋಗುತ್ತದೆ.

ಅತ್ಯಂತಕಾಮುಕಸ್ಯಾಪಿ ವೃತ್ತಿಃ ಕುಂಠತಿ ಮಾತರಿ ।
ತಥೈವ ಬ್ರಹ್ಮಣಿ ಜ್ಞಾತೇ ಪೂರ್ಣಾನಂದೇ ಮನೀಷಿಣಃ ॥ 443 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅತ್ಯಂತಕಾಮುಕಸ್ಯ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಅತ್ಯಂತ ಕಾಮುಕನಿಗೂ ್ಣಋಸ್ಶ್ಯ್¯ಮಾತರ್ಣಿಋಸ್ಬಋ¯ = ತಾಯಿಯ ವಿಷಯ ದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ವೃತ್ತ್ಣಿಃಋಸ್ಬಋ¯ = ಕಾಮವೃತ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಕುಂಠತ್ಣಿಋಸ್ಬಋ¯ = ಹಿಂಜರಿಯುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ತಥಾ ಏವ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಪೂರ್ಣಾನಂದೇ ಬ್ರಹ್ಮಣಿ ಜ್ಞಾತ್ಣೇಋಸ್ಬಋ¯ = ಪೂರ್ಣಾನಂದಬ್ರಹ್ಮವನ್ನು ಅರಿತುಕೊಂಡಾಗ ್ಣಋಸ್ಶ್ಯ್¯ಮನೀಷಿಣ್ಣಃಋಸ್ಬಋ¯ = ಜ್ಞಾನಿಗೆ [ಆಗುತ್ತದೆ].

443. ಅತ್ಯಂತ ಕಾಮುಕನಿಗೂ ತನ್ನ ತಾಯಿಯ ವಿಷಯದಲ್ಲಿ ಕಾಮರೂಪ ವಾದ ಚಿತ್ತವೃತ್ತಿಯು ಹಿಂಜರಿಯುತ್ತದೆ ; ಪೂರ್ಣಾನಂದವಾದ ಬ್ರಹ್ಮವನ್ನು ಅರಿತುಕೊಂಡಾಗ ಜ್ಞಾನಿಗೂ ಹಾಗೆಯೇ (ಸಂಸಾರವೃತ್ತಿಯು ಹಿಂಜರಿಯುತ್ತದೆ).

69. ಪ್ರಾರಬ್ಧದ ವಿಚಾರ
ನಿದಿಧ್ಯಾಸನಶೀಲಸ್ಯ ಬಾಹ್ಯಪ್ರತ್ಯಯ ಈಕ್ಷ್ಯತೇ ।
ಬ್ರವೀತಿ ಶ್ರುತಿರೇತಸ್ಯ ಪ್ರಾರಬ್ಧಂ ಫಲದರ್ಶನಾತ್ ॥ 444 ॥
್ಣಋಸ್ಶ್ಯ್¯ನಿದಿಧ್ಯಾಸನ-ಶೀಲಸ್ಣಋಸ್ಬಋ¯್ಯ = ನಿದಿಧ್ಯಾಸನವನ್ನು ಮಾಡುತ್ತಿರುವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಬಾಹ್ಯಪ್ರತ್ಯಯ್ಣಃಋಸ್ಬಋ¯ = ಬಾಹ್ಯವಿಷಯಗಳ ಜ್ಞಾನವು ್ಣಋಸ್ಶ್ಯ್¯ಈಕ್ಷ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಕಂಡುಬರುತ್ತದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಶ್ರುತ್ಣಿಃಋಸ್ಬಋ¯ = ಶ್ರುತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಏತಸ್ಣಋಸ್ಬಋ¯್ಯ ಇವನಿಗೆ–ಫಲದರ್ಶನಾತ್ಣ್ಋಸ್ಬಋ¯ = ಕರ್ಮಫಲವು ಕಂಡುಬರುತ್ತಿರುವುದರಿಂದ– ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾರಬ್ಧ್ಣಂಋಸ್ಬಋ¯ = ಪ್ರಾರಬ್ಧಕರ್ಮವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರವೀತ್ಣಿಋಸ್ಬಋ¯ = ಹೇಳುತ್ತದೆ.

444. ಸಮಾಧಿಶೀಲನಾದ ಜ್ಞಾನಿಗೆ ಬಾಹ್ಯವಿಷಯಗಳ ಜ್ಞಾನವು1 ಕಂಡುಬರು ತ್ತದೆ. ಹೀಗೆ ಅವನಿಗೆ ಕರ್ಮಫಲದ ಅನುಭವವು ಕಂಡುಬರುತ್ತಿರುವುದರಿಂದ ಪ್ರಾರಬ್ಧಕರ್ಮವಿದೆಯೆಂದು2 ಶ್ರುತಿಯು ಹೇಳುತ್ತದೆ.3

[[[[ 1 ಉಪದೇಶ ಭೋಜನ ಇವೇ ಮೊದಲಾದ ವ್ಯವಹಾರ.
2 450ನೆಯ ಶ್ಲೋಕವನ್ನು ನೋಡಿ.
3 ‘ಅವನಿಗೆ ಎಲ್ಲಿಯವರೆಗೆ ಮೋಕ್ಷವಾಗುವುದಿಲ್ಲವೊ ಅಲ್ಲಿಯವರೆಗೇ ವಿಳಂಬ’ ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಯ ತಾವದೇವ ಚಿರಂ ಯಾವನ್ನ ವಿಮೋಕ್ಷ್ಯೇ (ಛಾಂದೋಗ್ಯ ಉ 6. 14. 2).]
ಸುಖಾದ್ಯನುಭವೋ ಯಾವತ್ ತಾವತ್ ಪ್ರಾರಬ್ಧಮಿಷ್ಯತೇ ।
ಫಲೋದಯಃ ಕ್ರಿಯಾಪೂರ್ವೋ ನಿಷಿ­ಯೋ ನ ಹಿ ಕುತ್ರಚಿತ್ ॥ 445 ॥
್ಣಋಸ್ಶ್ಯ್¯ಸುಖಾದಿ-ಅನುಭವ್ಣಋಸ್ಬಋ¯ಃ = ಸುಖಾದಿಗಳ ಅನುಭವವು ್ಣಋಸ್ಶ್ಯ್¯ಯಾವತ್ಣ್ಋಸ್ಬಋ¯ = ಎಲ್ಲಿಯವರೆಗೆ ಇರುವುದೊ ್ಣಋಸ್ಶ್ಯ್¯ತಾವತ್= ಅಲ್ಲಿಯವರೆಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾರಬ್ಧ್ಣಂಋಸ್ಬಋ¯ = ಪ್ರಾರಬ್ಧವು ್ಣಋಸ್ಶ್ಯ್¯ಇಷ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಇರುವು ದೆಂದು ಸ್ವೀಕರಿಸಲ್ಪಟ್ಟಿದೆ ; ್ಣಋಸ್ಶ್ಯ್¯ಹ್ಣಿಋಸ್ಬಋ¯ = ಏಕೆಂದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಫಲೋದಯಃ = ಫಲೋತ್ಪತ್ತಿಯು ್ಣಋಸ್ಶ್ಯ್¯ಕ್ರಿಯಾ ಪೂರ್ವ್ಣಃಋಸ್ಬಋ¯ = ಕರ್ಮವನ್ನು ಮೊದಲೇ ಹೊಂದಿರಬೇಕು. ್ಣಋಸ್ಶ್ಯ್¯ಕುತ್ರಚಿತ್ಣಋಸ್ಬಋ¯್ = ಎಲ್ಲಿಯೂ ್ಣಋಸ್ಶ್ಯ್¯ನಿಷಿ­ಯ್ಣಃಋಸ್ಬಋ¯ = ಕರ್ಮವಿಲ್ಲದೆ ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ = ಆಗುವುದಿಲ್ಲ.

445. ಸುಖದುಃಖಾದಿಗಳ ಅನುಭವವು ಎಲ್ಲಿಯವರೆಗೆ ಇರುವುದೊ ಅಲ್ಲಿಯ ವರೆಗೆ ಪ್ರಾರಬ್ಧಕರ್ಮವನ್ನು ಒಪ್ಪಿಕೊಳ್ಳಬೇಕು ; ಏಕೆಂದರೆ ಫಲವುಂಟಾಗುವುದಕ್ಕೆ ಮೊದಲು ಕರ್ಮವು ಕಾರಣವಾಗಿರಬೇಕು. ಎಲ್ಲಿಯೂ ಕರ್ಮವಿಲ್ಲದೆ ಫಲ ವುಂಟಾಗಲಾರದು.

ಅಹಂ ಬ್ರಹ್ಮೇತಿ ವಿಜ್ಞಾನಾತ್ ಕಲ್ಪಕೋಟಿ-ಶತಾರ್ಜಿತಮ್ ।
ಸಂಚಿತಂ ವಿಲಯಂ ಯಾತಿ ಪ್ರಬೋಧಾತ್ ಸ್ವಪ್ನ-ಕರ್ಮವತ್ ॥ 446 ॥
್ಣಋಸ್ಶ್ಯ್¯ಅಹ್ಣಂಋಸ್ಬಋ¯ = ನಾನು ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ್ಣಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮವು ್ಣಋಸ್ಶ್ಯ್¯ಇತ್ಣಿಋಸ್ಬಋ¯ = ಎಂಬ ್ಣಋಸ್ಶ್ಯ್¯ವಿಜ್ಞಾನಾತ್ಣ್ಋಸ್ಬಋ¯ = ವಿಜ್ಞಾನದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಕಲ್ಪಕೋಟಿ-ಶತಾರ್ಜಿತ್ಣಂಋಸ್ಬಋ¯ = ನೂರು ಕೋಟಿ ಕಲ್ಪಗಳಿಂದ ಸಂಪಾದಿಸಿದ್ದ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಚಿತ್ಣಂಋಸ್ಬಋ¯ = ಸಂಚಿತ ಕರ್ಮವು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಬೋಧಾತ್ಣ್ಋಸ್ಬಋ¯ = ಎಚ್ಚರದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಪ್ನ-ಕರ್ಮವತ್ಣ್ಋಸ್ಬಋ¯ = ಸ್ವಪ್ನಕರ್ಮದ ಹಾಗೆ ್ಣಋಸ್ಶ್ಯ್¯ವಿಲಯ್ಣಋಸ್ಬಋ¯ಂ = ನಾಶವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಯಾತ್ಣಿಋಸ್ಬಋ¯ = ಹೊಂದುತ್ತದೆ.

446. ‘ನಾನು ಬ್ರಹ್ಮವು’ ಎಂಬ ಐಕ್ಯಜ್ಞಾನದಿಂದ ನೂರಾರು ಕೋಟಿ ಕಲ್ಪಗಳಿಂದ ಸಂಪಾದಿಸಿರುವ ಸಂಚಿತಕರ್ಮವೂ1 ಕೂಡ–ಎಚ್ಚರವಾದ ಮೇಲೆ ಸ್ವಪ್ನದಲ್ಲಿ ಮಾಡಿದ ಕರ್ಮವು ನಾಶವಾಗುವಂತೆ–ನಾಶವಾಗುತ್ತದೆ.

[1ಫಲವನ್ನು ಕೊಡಲು ಪ್ರಾರಂಭವಾಗದಿರುವ ಪೂರ್ವಕರ್ಮ.]
ಯತ್ಕೃತಂ ಸ್ವಪ್ನವೇಲಾಯಾಂ ಪುಣ್ಯಂ ವಾ ಪಾಪಮುಲ್ಬಣಮ್ । ಸುಪ್ತೋತ್ಥಿತಸ್ಯ ಕಿಂ ತತ್ ಸ್ಯಾತ್ ಸ್ವರ್ಗಾಯ ನರಕಾಯ ವಾ ॥ 447 ॥
್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಪ್ನವೇಲಾಯ್ಣಾಂಋಸ್ಬಋ¯ = ಸ್ವಪ್ನದ ಕಾಲದಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಪುಣ್ಯಂ ವ್ಣಾಋಸ್ಬಋ¯ = ಪುಣ್ಯವಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಉಲ್ಬಣ್ಣಂಋಸ್ಬಋ¯ = ಭೀಕರವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪಾಪಂ [ವಾ][[[ = ್ಣಋಸ್ಬಋ¯ಪಾಪವಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಣ್ಋಸ್ಬಋ¯ = ಯಾವುದು ್ಣಋಸ್ಶ್ಯ್¯ಕೃತ್ಣಂಋಸ್ಬಋ¯ = ಮಾಡ ಲ್ಪಟ್ಟಿತೊ ್ಣಋಸ್ಶ್ಯ್¯ಸುಪ್ತ-ಉತ್ಥಿತಸ್ಯ್ಣಋಸ್ಬಋ¯ = ನಿದ್ರೆಯಿಂದ ಎಚ್ಚತ್ತವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ಣಋಸ್ಬಋ¯್ = ಅದು ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವರ್ಗಾಯ್ಣಋಸ್ಬಋ¯ = ಸ್ವರ್ಗಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ನರಕಾಯ ವ್ಣಾಋಸ್ಬಋ¯ =ನರಕಕ್ಕೆ ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂ ಸ್ಯಾತ್ಣ್ಋಸ್ಬಋ¯ = ಕಾರಣವಾಗುವುದೆ ?

447. ಸ್ವಪ್ನದಲ್ಲಿ ಮಾಡಿದ್ದ ಪುಣ್ಯಕರ್ಮವಾಗಲಿ ಅಥವಾ ಭೀಕರವಾದ ಪಾಪವಾಗಲಿ ನಿದ್ರೆಯಿಂದ ಎಚ್ಚತ್ತವನಿಗೆ ಸ್ವರ್ಗಕ್ಕೆ ಅಥವಾ ನರಕಕ್ಕೆ ಕಾರಣ ವಾಗುವುದೆ ?

ಸ್ವಮಸಂಗಮುದಾಸೀನಂ ಪರಿಜ್ಞಾಯ ನಭೋ ಯಥಾ ।
ನ ಶ್ಲಿಷ್ಯತೇ ಚ ಯತ್ಕಿಂಚಿತ್ ಕದಾಚಿದ್ಭಾವಿಕರ್ಮಭಿಃ ॥ 448 ॥
್ಣಋಸ್ಶ್ಯ್¯ನಭ್ಣಃಋಸ್ಬಋ¯ = ಆಕಾಶವು ್ಣಋಸ್ಶ್ಯ್¯ಯಥಾ = ಹೇಗೋ [ಹಾಗೆಯೇ] ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವ್ಣಂಋಸ್ಬಋ¯ = ತನ್ನ ಆತ್ಮನನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅಸಂಗ್ಣಂಋಸ್ಬಋ¯ = ಅಸಂಗನೆಂದೂ ್ಣಋಸ್ಶ್ಯ್¯ಉದಾಸೀನ್ಣಂಋಸ್ಬಋ¯ =ಉದಾಸೀನನೆಂದೂ ್ಣಋಸ್ಶ್ಯ್¯ಪರಿಜ್ಞಾಯ್ಣಋಸ್ಬಋ¯ = ತಿಳಿದು ಕೊಂಡು [ಯತಿಯು] ್ಣಋಸ್ಶ್ಯ್¯ಭಾವಿಕರ್ಮಭ್ಣಿಃಋಸ್ಬಋ¯ = ಆಗಾಮಿಕರ್ಮಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ಯತ್ಕಿಂಚಿತ್ಣ್ಋಸ್ಬಋ¯ = ಸ್ವಲ್ಪವಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ಕದಾಚಿತ್ಣ್ಋಸ್ಬಋ¯ = ಎಂದಿಗೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ಶ್ಲಿಷ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ.

448. ಆಕಾಶದಂತೆ ತನ್ನ ಆತ್ಮನು ಅಸಂಗನೆಂದೂ ಉದಾಸೀನನೆಂದೂ ತಿಳಿದುಕೊಂಡ ಯತಿಯು ಆಗಾಮಿಕರ್ಮಗಳಿಂದ1 ಸ್ವಲ್ಪವೂ ಎಂದಿಗೂ ಲಿಪ್ತ ನಾಗುವುದಿಲ್ಲ.

[ 1 ಆತ್ಮಜ್ಞಾನವಾದಮೇಲೆ ಮುಂದೆ ಮಾಡತಕ್ಕ ಕರ್ಮಗಳಿಂದ.]
್ಣಋಸ್ಶ್ಯ್¯ನ ನಭೋ ಘಟಯೋಗೇನ ಸುರಾಗಂಧೇನ ಲಿಪ್ಯತೇ । ತಥಾತ್ಮೋಪಾಧಿಯೋಗೇನ ತದ್ಧರ್ಮೈನೈRವ ಲಿಪ್ಯತೇ <+>॥ 449 ॥
್ಣಋಸ್ಶ್ಯ್¯ನಭ್ಣಃಋಸ್ಬಋ¯ = ಆಕಾಶವು ್ಣಋಸ್ಶ್ಯ್¯ಘಟ-ಯೋಗೇನ್ಣಋಸ್ಬಋ¯ಅ = ಗಡಿಗೆಯ ಸಂಬಂಧದ ಮೂಲಕ ್ಣಋಸ್ಶ್ಯ್¯ಸುರಾಗಂಧೇ = [ಅದರಲ್ಲಿರುವ] ಮದ್ಯದ ವಾಸನೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನ ಲಿಪ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಲಿಪ್ತವಾಗುವುದಿಲ್ಲ, ್ಣಋಸ್ಶ್ಯ್¯ತಥ್ಣಾಋಸ್ಬಋ¯ = ಹಾಗೆಯೇ ್ಣಋಸ್ಶ್ಯ್¯ಆತ್ಣಾಋಸ್ಬಋ¯್ಮ = ಆತ್ಮನು ್ಣಋಸ್ಶ್ಯ್¯ಉಪಾಧಿಯೋಗೇನ್ಣಋಸ್ಬಋ¯ಅ = ಉಪಾಧಿ ಸಂಬಂಧದ ಮೂಲಕ ್ಣಋಸ್ಶ್ಯ್¯ತತ್-ಧರ್ಮೈ್ಣಃಋಸ್ಬಋ¯ = ಅದರ ಧರ್ಮಗಳಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನ ಏವ ಲಿಪ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ಲಿಪ್ತನಾಗುವುದೇ ಇಲ್ಲ.

449. ಹೇಗೆ ಆಕಾಶವು ಗಡಿಗೆಯ ಸಂಬಂಧದ ಮೂಲಕ (ಅದರಲ್ಲಿರುವ) ಮದ್ಯದ ವಾಸನೆಯಿಂದ ಲಿಪ್ತವಾಗುವುದಿಲ್ಲವೊ ಹಾಗೆಯೇ ಆತ್ಮನು ಉಪಾಧಿಯ ಸಂಬಂಧದ ಮೂಲಕ ಅದರ ಧರ್ಮಗಳಿಂದ ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ.

[ಸ್ಥೂಲ-ಸೂಕ್ಷ್ಮ-ಶರೀರಗಳ ಸಂಬಂಧವಿದ್ದರೂ ಅವುಗಳ ಧರ್ಮಗಳಿಂದ ಆತ್ಮನು ಲಿಪ್ತನಾಗುವುದಿಲ್ಲ.]
ಜ್ಞಾನೋದಯಾತ್ ಪುರಾರಬ್ಧಂ ಕರ್ಮ ಜ್ಞಾನಾನ್ನ ನಶ್ಯತಿ ।
ಅದತ್ವಾ ಸ್ವಫಲಂ ಲಕ್ಷ್ಯಮುದ್ದಿಶ್ಯೋತ್ಸೃಷ್ಟಬಾಣವತ್ ॥ 450 ॥
್ಣಋಸ್ಶ್ಯ್¯ಜ್ಞಾನೋದಯಾತ್ಣ್ಋಸ್ಬಋ¯ = ಜ್ಞಾನವು ಉದಯವಾಗುವುದಕ್ಕಿಂತ ್ಣಋಸ್ಶ್ಯ್¯ಪುರ್ಣಋಸ್ಬಋ¯ಾ = ಮೊದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಆರಬ್ಧ್ಣಋಸ್ಬಋ¯ಂ = [ಫಲವನ್ನು ಕೊಡುವುದಕ್ಕೆ] ಪ್ರಾರಂಭವಾಗಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಕರ್ಮ್ಣಋಸ್ಬಋ¯ = ಕರ್ಮವು ್ಣಋಸ್ಶ್ಯ್¯ಜ್ಞಾನಾತ್ಣ್ಋಸ್ಬಋ¯ = ಜ್ಞಾನದಿಂದ, ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಫಲ್ಣಂಋಸ್ಬಋ¯ = ತನ್ನ ಫಲವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅದತ್ವಾ = ಕೊಡದೆ–ಲಕ್ಷ್ಯ್ಣಂಋಸ್ಬಋ¯ ಗುರಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಉದ್ದಿಶ್ಣಋಸ್ಬಋ¯್ಯ = ಉದ್ದೇಶಿಸಿ ್ಣಋಸ್ಶ್ಯ್¯ಉತ್ಸೃಷ್ಟ-ಬಾಣವತ್ = ್ಣಋಸ್ಬಋ¯ಬಿಡಲ್ಪಟ್ಟ ಬಾಣ ದಂತೆ–್ಣಋಸ್ಶ್ಯ್¯ನ ನಶ್ಯತ್ಣಿಋಸ್ಬಋ¯ = ್ಣಋಸ್ಬಋ¯ನಾಶವಾಗುವುದಿಲ್ಲ.

450. ಜ್ಞಾನೋದಯಕ್ಕೆ ಮೊದಲೇ ಫಲವನ್ನು ಕೊಡುವುದಕ್ಕೆ ಆರಂಭವಾಗಿ ರುವ ಕರ್ಮವು–ಗುರಿಯನ್ನು ಉದ್ದೇಶಿಸಿ ಬಿಟ್ಟ ಬಾಣವು (ಗುರಿಯನ್ನು ಹೊಡೆ ಯುವಂತೆ)–ತನ್ನ ಫಲವನ್ನು ಕೊಡದೆ ಜ್ಞಾನದಿಂದ ನಾಶವಾಗುವುದಿಲ್ಲ.

ವ್ಯಾಘ್ರಬುದ್ಧ್ಯಾ ವಿನಿರ್ಮುಕ್ತೋ ಬಾಣಃ ಪಶ್ಚಾತ್ತು ಗೋಮತೌ ।
ನ ತಿಷ್ಠತಿ ಛಿನತ್ತ್ಯೇವ ಲಕ್ಷ್ಯಂ ವೇಗೇನ ನಿರ್ಭರಮ್ ॥ 451 ॥
್ಣಋಸ್ಶ್ಯ್¯ವ್ಯಾಘ್ರಬುದ್ಧ್ಣಾಋಸ್ಬಋ¯್ಯ = ಹುಲಿ ಎಂಬ ಬುದ್ಧಿಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ವಿನಿರ್ಮುಕ್ತ್ಣಃಋಸ್ಬಋ¯ = ಬಿಡಲ್ಪಟ್ಟ ್ಣಋಸ್ಶ್ಯ್¯ಬಾಣ್ಣಃಋಸ್ಬಋ¯ = ಬಾಣವು ್ಣಋಸ್ಶ್ಯ್¯ಪಶ್ಚಾತ್ಣ್ಋಸ್ಬಋ¯ = ಅನಂತರ ್ಣಋಸ್ಶ್ಯ್¯ಗೋಮತೌ ತ್ಣುಋಸ್ಬಋ¯ = ಹಸು ಎಂಬ ಬುದ್ಧಿಯುಂಟಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ನ ತಿಷ್ಠತ್ಣಿಋಸ್ಬಋ¯ = ನಿಲ್ಲುವುದಿಲ್ಲ. ್ಣಋಸ್ಶ್ಯ್¯ವೇಗೇನ್ಣಋಸ್ಬಋ¯ = ವೇಗದಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ಭರ್ಣಂಋಸ್ಬಋ¯ = ಖಂಡಿತವಾಗಿ ್ಣಋಸ್ಶ್ಯ್¯ಲಕ್ಷ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಗುರಿಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಛಿನತ್ತಿ ಏವ್ಣಋಸ್ಬಋ¯ = ಹೊಡೆದೇ ಹೊಡೆಯುತ್ತದೆ.

451. ಹುಲಿಯೆಂಬ ಬುದ್ಧಿಯಿಂದ ಬಿಡಲ್ಪಟ್ಟ ಬಾಣವು ಅನಂತರ ‘ ಅದು ಹಸು’ ಎಂಬ ಬುದ್ಧಿಯುಂಟಾದರೂ ನಿಲ್ಲುವುದಿಲ್ಲ ; ವೇಗದಿಂದ ತನ್ನ ಗುರಿ ಯನ್ನು ಖಂಡಿತವಾಗಿ ಹೊಡೆದೇಬಿಡುತ್ತದೆ.

ಪ್ರಾರಬ್ಧಂ ಬಲವತ್ತರಂ ಖಲು ವಿದಾಂ ಭೋಗೇನ ತಸ್ಯ ಕ್ಷಯಃ
ಸಮ್ಯಗ್ಜ್ಞಾನಹುತಾಶನೇನ ವಿಲಯಃ ಪ್ರಾಕ್ಸಂಚಿತಾಗಾಮಿನಾಮ್ । ಬ್ರಹ್ಮಾತ್ಮೈಕ್ಯಮವೇಕ್ಷ್ಯ ತನ್ಮಯತಯಾ ಯೇ ಸರ್ವದಾ ಸಂಸ್ಥಿತಾ-
ಸ್ತೇಷಾಂ ತತ್ತ್ರಿತಯಂ ನ ಹಿ ಕ್ವಚಿದಪಿ ಬ್ರಹ್ಮೈವ ತೇ ನಿರ್ಗುಣಮ್ ॥ 452 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾರಬ್ಧ್ಣಂಋಸ್ಬಋ¯ = ಪ್ರಾರಬ್ಧಕರ್ಮವು ್ಣಋಸ್ಶ್ಯ್¯ಬಲವತ್ತರಂ ಖಲ್ಣುಋಸ್ಬಋ¯ = ನಿಜವಾಗಿ ಅತ್ಯಂತ ಬಲವಾದದ್ದು ; ್ಣಋಸ್ಶ್ಯ್¯ವಿದ್ಣಾಂಋಸ್ಬಋ¯ = ಜ್ಞಾನಿಗಳಿಗೂ ್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಣಋಸ್ಬಋ¯್ಯ = ಅದರ ್ಣಋಸ್ಶ್ಯ್¯ಭೋಗೇನ = ಅನುಭವದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ಷಯ್ಣಋಸ್ಬಋ¯ಃ = ನಾಶವು [ಆಗಬೇಕು]; ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಾಕ್-ಸಂಚಿತ-ಆಗಾಮಿನಾಂ = ಹಿಂದೆ ಸಂಪಾದಿಸಿದ ಮlತ್ತು ಮುಂದೆ ಬರುವ [ಕರ್ಮಗಳ] ್ಣಋಸ್ಶ್ಯ್¯ವಿಲಯ್ಣಃಋಸ್ಬಋ¯ = ನಾಶವು ್ಣಋಸ್ಶ್ಯ್¯ಸಮ್ಯಗ್ಜ್ಞಾನ-ಹುತಾಶನೇನ = ್ಣಋಸ್ಬಋ¯ಸಮ್ಯಗ್ಜ್ಞಾನವೆಂಬ ಅಗ್ನಿಯಿಂದ [[ಆಗುತ್ತದೆ]; ಯೇ = ್ಣಋಸ್ಬಋ¯ಯಾರು ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ-ಆತ್ಮ-ಐಕ್ಯ್ಣಂಋಸ್ಬಋ¯ = ಜೀವಾತ್ಮ-ಪರಮಾತ್ಮರ ಐಕ್ಯವನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಅವೇಕ್ಷ್ಣಋಸ್ಬಋ¯್ಯ = ತಿಳಿದುಕೊಂಡು ್ಣಋಸ್ಶ್ಯ್¯ತನ್ಮಯತಯ್ಣಾಋಸ್ಬಋ¯ = ತನ್ಮಯತೆಯಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ಸರ್ವದ್ಣಾಋಸ್ಬಋ¯ = ಯಾವಾಗಲೂ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಸ್ಥಿತ್ಣಾಋಸ್ಬಋ¯ಃ = ಇರುವರ್ಣೊಋಸ್ಶ್ಯ್¯ ತೇಷ್ಣಾಂಋಸ್ಬಋ¯ = ಅವರಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ತತ್ ತ್ರಿತಯ್ಣಂಋಸ್ಬಋ¯ = ಆ ಮೂರೂ ್ಣಋಸ್ಶ್ಯ್¯ಕ್ವಚಿತ್ ಅಪ್ಣಿಋಸ್ಬಋ¯ = ಸ್ವಲ್ಪವಾದರೂ ್ಣಋಸ್ಶ್ಯ್¯ನ್ಣಋಸ್ಬಋ¯ = ಇರುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಹಿ = ಏಕೆಂದರೆ ್ಣಋಸ್ಶ್ಯ್¯ತೇ = ಅವರು ್ಣಋಸ್ಶ್ಯ್¯ನಿರ್ಗುಣ್ಣಂಋಸ್ಬಋ¯ = ನಿರ್ಗುಣವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಬ್ರಹ್ಮ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ಏವ್ಣಋಸ್ಬಋ¯ = ಬ್ರಹ್ಮವೇ ಆಗಿರುತ್ತಾರೆ.

452. ಪ್ರಾರಬ್ಧಕರ್ಮವು ನಿಜವಾಗಿ ಅತ್ಯಂತ ಬಲವಾದದ್ದು. ಜ್ಞಾನಿಗಳು ಅದನ್ನು ಅನುಭವಿಸಿಯೇ ನಾಶಗೊಳಿಸಬೇಕು. ಸಮ್ಯಗ್ಜ್ಞಾನವೆಂಬ ಅಗ್ನಿಯಿಂದ ಹಿಂದೆ ಸಂಪಾದಿಸಿದ ಮತ್ತು ಮುಂದೆ ಬರುವ ಕರ್ಮಗಳೆಲ್ಲ ನಾಶವಾಗುತ್ತವೆ. ಆದರೆ ಯಾರು ಜೀವಾತ್ಮ-ಪರಮಾತ್ಮರ ಐಕ್ಯವನ್ನು ತಿಳಿದುಕೊಂಡು ತನ್ಮಯತೆ ಯಿಂದಲೇ1 ಯಾವಾಗಲೂ ಇರುವರೊ ಅವರಿಗೆ ಈ ಮೂರು ಬಗೆಯ ಕರ್ಮವೂ ಸ್ವಲ್ಪವಾದರೂ ಇರುವುದಿಲ್ಲ ; ಏಕೆಂದರೆ ಅವರು ನಿರ್ಗುಣವಾದ ಬ್ರಹ್ಮವೇ ಆಗಿರುತ್ತಾರೆ.

[1 ಬ್ರಹ್ಮಾತ್ಮಭಾವದಿಂದಲೇ.]
ಉಪಾಧಿ-ತಾದಾತ್ಮ್ಯ-ವಿಹೀನ-ಕೇವಲ-
ಬ್ರಹ್ಮಾತ್ಮನೈವಾತ್ಮನಿ ತಿಷ್ಠತೋ ಮುನೇಃ ।
ಪ್ರಾರಬ್ಧ-ಸದ್ಭಾವ-ಕಥಾ ನ ಯುಕ್ತಾ
ಸ್ವಪ್ನಾರ್ಥ-ಸಂಬಂಧ-ಕಥೇವ ಜಾಗ್ರತಃ ॥ 453 ॥
್ಣಋಸ್ಶ್ಯ್¯ಉಪಾಧಿ-ತಾದಾತ್ಮ್ಯ-ವಿಹೀನ-ಕೇವಲ-ಬ್ರಹ್ಮ-ಆತ್ಮನಾ ಏವ = ಉಪಾಧಿಗಳೊಂದಿಗೆ ತಾದಾತ್ಮ್ಯವಿಲ್ಲದೆ ಕೇವಲ ಬ್ರಹ್ಮಭಾವದಿಂದಲೇ ್ಣಋಸ್ಶ್ಯ್¯ತಿಷ್ಠತ್ಣಃಋಸ್ಬಋ¯ = ಇರುತ್ತಿರುವ ್ಣಋಸ್ಶ್ಯ್¯ಮುನ್ಣೇಃಋಸ್ಬಋ¯ = ಮುನಿಗೆ–ಜಾಗ್ರತಃ = ಎಚ್ಚತ್ತಿರುವವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಪ್ನ-ಅರ್ಥ-ಸಂಬಂಧ-ಕಥಾ ಇವ = ಸ್ವಪ್ನದ ವಿಷಯಗಳ ಸಂಬಂಧದ ಮಾತು ಹೇಗೋ ಹಾಗೆ–ಪ್ರಾರಬ್ಧ-ಸದ್ಭಾವ --ಕಥ್ಣಾಋಸ್ಬಋ¯ = ಪ್ರಾರಬ್ಧವಿದೆಯೆಂಬ ಮಾತು ್ಣಋಸ್ಶ್ಯ್¯ನ ಯುಕ್ತ್ಣಾಋಸ್ಬಋ¯ = ಯುಕ್ತವಲ್ಲ.

453. ಉಪಾಧಿಗಳೊಂದಿಗೆ ತಾದಾತ್ಮ್ಯವಿಲ್ಲದೆ ಕೇವಲ ಬ್ರಹ್ಮಭಾವದಿಂದಲೇ ಇರುತ್ತಿರುವ ಮುನಿಗೆ–ಎಚ್ಚತ್ತಿರುವವನಿಗೆ ಸ್ವಪ್ನದಲ್ಲಿ ಕಂಡ ವಿಷಯಗಳ ಸಂಬಂಧವಿದೆಯೆಂಬ ಮಾತು ಹೇಗೆ ಯುಕ್ತವಲ್ಲವೊ ಹಾಗೆ–ಪ್ರಾರಬ್ಧವಿದೆ ಯೆಂಬ ಮಾತು ಯುಕ್ತವಲ್ಲ.

ನ ಹಿ ಪ್ರಬುದ್ಧಃ ಪ್ರತಿಭಾಸ-ದೇಹೇ
ದೇಹೋಪಯೋಗಿನ್ಯಪಿ ಚ ಪ್ರಪಂಚೇ ।
ಕರೋತ್ಯಹಂತಾಂ ಮಮತಾಮಿದಂತಾಂ
ಕಿಂತು ಸ್ವಯಂ ತಿಷ್ಠತಿ ಜಾಗರೇಣ ॥ 454 ॥
್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಬುದ್ಧ್ಣಃಋಸ್ಬಋ¯ = ಎಚ್ಚತ್ತಿರುವವನು ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರತಿಭಾಸದೇಹ್ಣೇಋಸ್ಬಋ¯ = [ಸ್ವಪ್ನದಲ್ಲಿ] ತೋರಿದ ದೇಹ ದಲ್ಲಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಪಿ ಚ್ಣಋಸ್ಬಋ¯ = ಮತ್ತು ್ಣಋಸ್ಶ್ಯ್¯ದೇಹ-ಉಪಯೋಗಿನ್ಣಿಋಸ್ಬಋ¯ = ಆ ದೇಹಕ್ಕೆ ಉಪಯೋಗವಾದ ್ಣಋಸ್ಶ್ಯ್¯ಪ್ರಪಂಚ್ಣೇಋಸ್ಬಋ¯ = ವಿಷಯವಸ್ತುವಿನಲ್ಲಾಗಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅಹಂತ್ಣಾಂಋಸ್ಬಋ¯ = ‘ನಾನು’ ಎಂಬ ಭಾವನೆಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಮಮತ್ಣಾಂಋಸ್ಬಋ¯ = ‘ನನ್ನದು’ ಎಂಬ ಭಾವನೆಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ಇದಂತ್ಣಾಂಋಸ್ಬಋ¯ = ‘ಇದು’ ಎಂಬ ಭಾವನೆಯನ್ನು ್ಣಋಸ್ಶ್ಯ್¯ನ ಹಿ ಕರೋತ್ಣಿಋಸ್ಬಋ¯ = ಮಾಡುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ಕಿಂ ತ್ಣುಋಸ್ಬಋ¯ = ಮತ್ತೇನೆಂದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ವಯ್ಣಂಋಸ್ಬಋ¯ = ತಾನು ್ಣಋಸ್ಶ್ಯ್¯ಜಾಗರೇಣ್ಣಋಸ್ಬಋ¯ ್ಣಋಸ್ಶ್ಯ್¯ತಿಷ್ಠತ್ಣಿಋಸ್ಬಋ¯ = ಎಚ್ಚತ್ತಿರುತ್ತಾನೆ.

454. (ಸ್ವಪ್ನದಿಂದ) ಎಚ್ಚತ್ತಿರುವವನು (ಸ್ವಪ್ನದಲ್ಲಿ) ತೋರಿದ ದೇಹದಲ್ಲಾ ಗಲಿ ಅದಕ್ಕೆ ಉಪಯೋಗಕರವಾದ ವಿಷಯವಸ್ತುವಿನಲ್ಲಾಗಲಿ ‘ನಾನು’ ‘ನನ್ನದು’ ‘ಇದು’ ಎಂಬ ಭಾವನೆಯನ್ನು ಮಾಡುವುದಿಲ್ಲ ; ಮತ್ತೇನೆಂದರೆ ತನ್ನ ಸ್ವರೂಪ ದಲ್ಲಿಯೇ ಎಚ್ಚತ್ತಿರುತ್ತಾನೆ.

ನ ತಸ್ಯ ಮಿಥ್ಯಾರ್ಥಸಮರ್ಥನೇಚ್ಛಾ
ನ ಸಂಗ್ರಹಸ್ತಜ್ಜಗತೋsಪಿ ದೃಷ್ಟಃ ।
ತತ್ರಾನುವೃತ್ತಿರ್ಯದಿ ಚೇನ್ಮೃಷಾರ್ಥೇ
ನ ನಿದ್ರಯಾ ಮುಕ್ತ ಇತೀಷ್ಯತೇ ಧ್ರುವಮ್ ॥ 455 ॥
್ಣಋಸ್ಶ್ಯ್¯ತಸ್ಯ್ಣಋಸ್ಬಋ¯ = ಅವನಿಗೆ ್ಣಋಸ್ಶ್ಯ್¯ಮಥ್ಯಾರ್ಥ-ಸಮರ್ಥನ-ಇಚ್ಛ್ಣಾಋಸ್ಬಋ¯ = ಮಿಥ್ಯಾವಸ್ತುಗಳ ಸಮರ್ಥನದಲ್ಲಿ ಬಯಕೆಯು ್ಣಋಸ್ಶ್ಯ್¯ನ = ಇರುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ತಜ್ಜಗತ್ಣಃಋಸ್ಬಋ¯ = ಆ ಸ್ವಪ್ನಜಗತ್ತಿನ ್ಣಋಸ್ಶ್ಯ್¯ಸಂಗ್ರಹಃ ಅಪಿ = ಸಂಗ್ರಹವೂ ಕೂಡ ್ಣಋಸ್ಶ್ಯ್¯ನ ದೃಷ್ಟ್ಣಃಋಸ್ಬಋ¯ = ಕಂಡುಬರುವುದಿಲ್ಲ ; ್ಣಋಸ್ಶ್ಯ್¯ತತ್ರ ಮೃಷಾರ್ಥೇ = ಆ ಅನೃತ ವಸ್ತುವಿನಲ್ಲಿ ್ಣಋಸ್ಶ್ಯ್¯ಅನುವೃತ್ತಿಃ ಚೇತ್ ಯದ್ಣಿಋಸ್ಬಋ¯ = ಅನುವರ್ತನವಿದ್ದರೆ ್ಣಋಸ್ಶ್ಯ್¯ಸ್ಣಃಋಸ್ಬಋ¯ = ಅವನು ್ಣಋಸ್ಶ್ಯ್¯ನಿದ್ರಯ್ಣಾಋಸ್ಬಋ¯ = ನಿದ್ರೆಯಿಂದ ್ಣಋಸ್ಶ್ಯ್¯ನ ಮುಕ್ತಃ ಇತ್ಣಿಋಸ್ಬಋ¯ = ಬಿಡಲ್ಲಟ್ಟವನಲ್ಲವೆಂದು ್ಣಋಸ್ಶ್ಯ್¯ಇಷ್ಯತ್ಣೇಋಸ್ಬಋ¯ = ತಿಳಿಯಲ್ಪಡುತ್ತದೆ, ್ಣಋಸ್ಶ್ಯ್¯ಧ್ರುವ್ಣಂಋಸ್ಬಋ¯ = ಇದು ನಿಶ್ಚಯವು.

455. ಅವನಿಗೆ ಮಿಥ್ಯಾವಸ್ತುಗಳ ಸಮರ್ಥದಲ್ಲಿ ಬಯಕೆಯು ಇರುವುದಿಲ್ಲ; ಆ ಸ್ಪಪ್ನಜಗತ್ತಿನ (ವಸ್ತುಗಳ) ಸಂಗ್ರಹವೂ ಕೂಡ ಕಂಡುಬರುವುದಿಲ್ಲ. ಆ ಅನೃತ ವಸ್ತುವಿನಲ್ಲಿ (ಮನಸ್ಸಿನ)ಅನುವರ್ತನವಿದ್ದರೆ ಅವನು ನಿದ್ರೆಯಿಂದ ಎಚ್ಚತ್ತಿಲ್ಲವೆಂದು ತಿಳಿಯಲ್ಪಡುತ್ತದೆ. ಇದು ನಿಶ್ಚಯವು.

ತದ್ವತ್ = ಹಾಗೆಯೇ ಪರೇ ಬ್ರಹ್ಮಣಿ = ಪರಬ್ರಹ್ಮದಲ್ಲಿ ವರ್ತಮಾನಃ = ಇರುವವನು ಸದಾತ್ಮನಾ = ಬ್ರಹ್ಮಭಾವದಿಂದಲೇ ತಿಷ್ಠತಿ = ಇರುತ್ತಾನೆ, ಅನ್ಯತ್ = ಬೇರೆ ಯಾವುದನ್ನೂ ನ ಈಕ್ಷತೇ = ನೋಡುವುದಿಲ್ಲ; ಸ್ವಪ್ನ-ವಿಲೋಕಿನ-ಅರ್ಥೇ = ಸ್ವಪ್ನದಲ್ಲಿ ಕಂಡ ವಿಷಯದಲ್ಲಿ ಸ್ಮೃತಿಃ = ನೆನಪು ಯಥಾ = ಹೇಗೋ ತಥಾ = ಹಾಗೆಯೇ ವಿದಃ =ಬ್ರಹ್ಮವಿದವನಿಗೆ ಪ್ರಾಶನ-ಮೋಚನ-ಅದೌ = ಊಟಮಾಡುವುದು ಬಿಡುವುದು ಮೊದಲಾದುವುಗಳಲ್ಲಿ (ನೆನಪು ಮಾತ್ರ ಇರುತ್ತದೆ).
