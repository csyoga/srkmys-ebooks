೧. ಮಂಗಲಾಚರಣ

ಸರ್ವ-ವೇದಾಂತ-ಸಿದ್ಧಾಂತ-ಗೋಚರಂ ತಮಗೋಚರಮ್ ।
ಗೋವಿಂದಂ ಪರಮಾನಂದಂ ಸದ್ಗುರುಂ ಪ್ರಣತೋಽಸ್ಮ್ಯಹಮ್ ॥ ೧ ॥

ಸರ್ವ-ವೇದಾಂತ-ಸಿದ್ಧಾಂತ-ಗೋಚರ್ಣಂ = ಸಮಸ್ತ ಉಪನಿಷತ್ತುಗಳ ತಾತ್ಪರ್ಯಕ್ಕೆ ವಿಷಯನಾಗಿರುವ ಅಗೋಚರ್ಣಂ = [ವಾಕ್ಕು, ಮನಸ್ಸು ಇವುಗಳಿಗೆ] ವಿಷಯನಲ್ಲದ ಪರಮಾನಂದ್ಣಂ = ಪರಮಾನಂದಸ್ವರೂಪನಾದ ಸದ್ಗುರ್ಣುಂ = ಸದ್ಗುರುವಾದ ತಂ ಗೋವಿಂದ್ಣಂ = ಆ ಗೋವಿಂದನನ್ನು ಅಹ್ಣಂ =ನಾನು ಪ್ರಣತ್ಣಃ ಅಸ್ಮಿ = ಪ್ರಣಾಮ ಮಾಡುವವನಾಗಿದ್ದೇನೆ.

೧. ಸಮಸ್ತ ಉಪನಿಷತ್ತುಗಳ ತಾತ್ಪರ್ಯಕ್ಕೆ ಮಾತ್ರ ವಿಷಯನಾಗಿರುವವನೂ ವಾಙ್ಮನಸ್ಸುಗಳಿಗೆ೧ ವಿಷಯನಲ್ಲದವನೂ ಪರಮಾನಂದಸ್ವರೂಪನೂ ಸದ್ಗುರುವೂ ಆದ ಗೋವಿಂದನಿಗೆ ಮಣಿಯುತ್ತೇನೆ.

[ಗ್ರಂಥವು ನಿರ್ವಿಘ್ನವಾಗಿ ಸಮಾಪ್ತವಾಗಲು ಶ್ರೀಶಂಕರಭಗವತ್ಪಾದರು ಗುರುಗಳಿಗೆ ಗುರುವೂ ತಮ್ಮ ಇಷ್ಟದೇವತೆಯೂ ಆದ ವಿಷ್ಣುವಿಗೆ ನಮಸ್ಕರಿಸುತ್ತಾರೆ ; ಅಥವಾ ಇಲ್ಲಿ ಗೋವಿಂದನೆಂದರೆ ಆಚಾರ್ಯರ ಗುರುವಾದ ಶ್ರೀಗೋವಿಂದಭಗವತ್ಪಾದರೂ ಆಗಬಹುದು.

೨. ಮುಕ್ತಿಯ ಮಹತ್ತ್ವ

ಜಂತೂನಾಂ ನರಜನ್ಮ ದುರ್ಲಭಮತಃ ಪುಂಸ್ತ್ವಂ ತತೋ ವಿಪ್ರತಾ
ತಸ್ಮಾದ್ವೈದಿಕ-ಧರ್ಮಮಾರ್ಗ-ಪರತಾ ವಿದ್ವತ್ತ್ವಮಸ್ಮಾತ್​ಪರಮ್ ।
ಆತ್ಮಾನಾತ್ಮ-ವಿವೇಚನಂ ಸ್ವನುಭವೋ ಬ್ರಹ್ಮಾತ್ಮನಾ ಸಂಸ್ಥಿತಿ-
ರ್ಮುಕ್ತಿರ್ನೋ ಶತಕೋಟಿ-ಜನ್ಮ-ಸುಕೃತೈಃ ಪುಣ್ಯೈರ್ವಿನಾ ಲಭ್ಯತೇ ೨

ಜಂತೂನ್ಣಾಂ = ಪ್ರಾಣಿಗಳಿಗೆ ನರಜನ್ಮ್ಣ = ಮನುಷ್ಯಜನ್ಮವು ದುರ್ಲಭ್ಣಂ = ಕಷ್ಟ ಸಾಧ್ಯವಾದುದು ಅತ್ಣಃ = ಇದಕ್ಕಿಂತಲೂ ಪುಂಸ್ತ್ವ್ಣ = ಪುರುಷತ್ವವು [ದುರ್ಲಭವು]. ತತ್ಣಃ = ಅದಕ್ಕಿಂತಲೂ ವಿಪ್ರತ್ಣಾ = ಬ್ರಾಹ್ಮಣತ್ವವು [ದುರ್ಲಭವು]. ತಸ್ಮಾತ್ಣ್​ = ಅದಕ್ಕಿಂ ತಲೂ ವೈದಿಕ-ಧರ್ಮಮಾರ್ಗ-ಪರತ್ಣಾ = ವೈದಿಕ ಧರ್ಮಮಾರ್ಗದಲ್ಲಿ ವಿಶ್ವಾಸವು [ದುರ್ಲಭವು]. ಅಸ್ಮಾತ್ಣ್​ = ಇದಕ್ಕಿಂತಲೂ ವಿದ್ವತ್ತ್ವ್ಣಂ = ಶಾಸ್ತ್ರಪಾಂಡಿತ್ಯವು ಪರ್ಣಂ = ಅಧಿಕವಾದುದು ; ಆತ್ಮಾನಾತ್ಮ-ವಿವೇಚನ್ಣಂ = ಆತ್ಮ ಅನಾತ್ಮ ಇವುಗಳ ವಿಚಾರ ಸ್ವನುಭವ್ಣಃ = ಸ್ವಾತ್ಮಾನುಭವ ಬ್ರಹ್ಮಾತ್ಮನಾ ಸಂಸ್ಥಿತ್ಣಿಃ = ಬ್ರಹ್ಮಾತ್ಮ-ಭಾವದಲ್ಲಿರು ವುದು [ಇವು ಕ್ರಮವಾಗಿ ಒಂದಕ್ಕಿಂತ ಒಂದು ಅಧಿಕವಾದುವು] ; ಮುಕ್ತ್ಣಿಃ = ಮೋಕ್ಷವು ಶತಕೋಟಿ-ಜನ್ಮ-ಸುಕೃತೈಃ ಪುಣ್ಯೈಃ ವಿನ್ಣಾ = ನೂರುಕೋಟಿ ಜನ್ಮಗಳಲ್ಲಿ ಸಂಪಾದಿಸಿದ ಪುಣ್ಯವಿಲ್ಲದೆ ನೋ ಲಭ್ಯತ್ಣೇ = ಲಭಿಸುವುದಿಲ್ಲ.

೨. ಪ್ರಾಣಿಗಳಿಗೆ ಮನುಷ್ಯಜನ್ಮವು ಕಷ್ಟಸಾಧ್ಯವಾದುದು, ಮನುಷ್ಯಜನ್ಮ ಕ್ಕಿಂತಲೂ ಪುರುಷತ್ವವು ದುರ್ಲಭವು, ಪುರುಷತ್ವಕ್ಕಿಂತಲೂ ಬ್ರಾಹ್ಮಣತ್ವವು ದುರ್ಲಭವು, ಬ್ರಾಹ್ಮಣತ್ವಕ್ಕಿಂತಲೂ ವೈದಿಕ-ಧರ್ಮಮಾರ್ಗದಲ್ಲಿ ವಿಶ್ವಾಸವು ದುರ್ಲಭವು. ಇದಕ್ಕಿಂತಲೂ ಶಾಸ್ತ್ರಪಾಂಡಿತ್ಯವು ಅಧಿಕವಾದುದು ; ಆತ್ಮ ಅನಾತ್ಮ ಇವುಗಳ ವಿಚಾರ, ಸ್ವಾತ್ಮಾನುಭವ, ಬ್ರಹ್ಮಾತ್ಮಭಾವದಲ್ಲಿರುವುದು೧ (ಇವು ಕ್ರಮ ವಾಗಿ ಒಂದಕ್ಕಿಂತ ಒಂದು ಅಧಿಕವಾದುವು). (ಇಂಥ) ಮೋಕ್ಷವು ನೂರು ಕೋಟಿ ಜನ್ಮಗಳಲ್ಲಿ ಸಂಪಾದಿಸಿದ ಪುಣ್ಯವಿಲ್ಲದೆ ಲಭಿಸುವುದಿಲ್ಲ೨.

ದುರ್ಲಭಂ ತ್ರಯಮೇವೈತದ್ದೈವಾನುಗ್ರಹ-ಹೇತುಕಮ್ ।
ಮನುಷ್ಯತ್ವಂ ಮುಮುಕ್ಷುತ್ವಂ ಮಹಾಪುರುಷ-ಸಂಶ್ರಯಃ ೩

ದೈವಾನುಗ್ರಹ-ಹೇತುಕ್ಣಂ = ಭಗವಂತನ ಅನುಗ್ರಹವೇ ಕಾರಣವಾಗಿರುವ ಏತತ್ ತ್ರಯ್ಣಂ = ಈ ಮೂರು ದುರ್ಲಭಮ್ ಏವ = ಕಷ್ಟಸಾಧ್ಯವೇ ; ಮನುಷ್ಯತ್ವ್ಣಂ = ಮನುಷ್ಯ ಜನ್ಮ ಮುಮುಕ್ಷುತ್ವ್ಣಂ = ಮೋಕ್ಷೇಚ್ಛೆ ಮಹಾಪುರುಷ-ಸಂಶ್ರಯ್ಣಃ = ಮಹಾತ್ಮರ ಸಂಗ.

೩. ಭಗವಂತನ ಅನುಗ್ರಹವೇ ಕಾರಣವಾಗಿರುವ ಈ ಮೂರು ಕಷ್ಟಸಾಧ್ಯವೇ: ಮನುಷ್ಯಜನ್ಮ, ಮೋಕ್ಷವನ್ನು ಪಡೆಯಬೇಕೆಂಬ ಇಚ್ಛೆ ಮತ್ತು ಮಹಾತ್ಮರ ಸಂಗ (ಅಥವಾ ಆಶ್ರಯ).

[ಭಗವಂತನ ಅನುಗ್ರಹವಿಲ್ಲದಿದ್ದರೆ ಮನುಷ್ಯತ್ವವೇ ಮೊದಲಾದುವನ್ನು ಪಡೆಯುವುದು ಕಷ್ಟ ಎಂದು ಭಾವ.]

ಲಬ್ಧ್ವಾ ಕಥಂಚಿನ್ನರಜನ್ಮ ದುರ್ಲಭಂ
ತತ್ರಾಪಿ ಪುಂಸ್ತ್ವಂ ಶ್ರುತಿ-ಪಾರದರ್ಶನಮ್ ।
ಯಸ್ತ್ವಾತ್ಮಮುಕ್ತೌ ನ ಯತೇತ ಮೂಢಧೀಃ
ಸ ಆತ್ಮಹಾ ಸ್ವಂ ವಿನಿಹಂತ್ಯಸದ್ಗ್ರಹಾತ್ ೪

ಮೂಢಧ್ಣೀಃ = ಮೂಢಬುದ್ಧಿಯುಳ್ಳ ಯಃ ತ್ಣು = ಯಾವನು ದುರ್ಲಭ್ಣಂ = ದುಃಸಾಧ್ಯವಾದ ನರಜನ್ಮ್ಣ = ಮನುಷ್ಯಜನ್ಮವನ್ನು ಕಥಂಚಿತ್ಣ್​ = ಹೇಗೋ ಲಬ್ಧ್ವಾ್ಣ = ಪಡೆದು, ತತ್ರ್ಣ = ಅದರಲ್ಲಿಯೂ ಪುಂಸ್ತ್ವ್ಣಂ = ಪುರುಷತ್ವವನ್ನು ಶ್ರುತಿಪಾರದರ್ಶನ್ಣಂ = ವೇದಪ್ರವೀಣತೆಯನ್ನು [ಲಬ್ಧ್ವಾ] ಅಪ್ಣಿ = ಹೊಂದಿದ್ದರೂ ಆತ್ಮಮುಕ್ತ್ಣೌ = ತನ್ನ ಮೋಕ್ಷದ ವಿಷಯದಲ್ಲಿ ನ ಯತೇತ್ಣ = ಪ್ರಯತ್ನಪಡುವುದಿಲ್ಲವೊ ಸ್ಣಃ = ಅವನು ಆತ್ಮಹ್ಣಾ = ಆತ್ಮಹತ್ಯವನ್ನು ಮಾಡಿಕೊಂಡವನು ; ಹ್ಣಿ = ಏಕೆಂದರೆ ಅಸದ್​-ಗ್ರಹಾತ್ಣ್​ = ಮಿಥ್ಯಾವಸ್ತುಗಳನ್ನು ಹಿಡಿದುಕೊಳ್ಳುವುದರಿಂದ ಸ್ವ್ಣಂ = ತನ್ನನ್ನು ವಿನಿಹಂತ್ಣಿ = ಕೊಂದು ಕೊಳ್ಳುತ್ತಾನೆ.

೪. ದುಃಸಾಧ್ಯವಾದ ಮನುಷ್ಯಜನ್ಮವನ್ನು ಹೇಗೋ ಪಡೆದುಕೊಂಡು, ಅದರ ಲ್ಲಿಯೂ ಪುರುಷತ್ವವನ್ನು ಮತ್ತು ವೇದಪ್ರವೀಣತೆಯನ್ನು ಹೊಂದಿದ್ದರೂ ಮೂಢಬುದ್ಧಿಯುಳ್ಳ ಯಾವ ಮನುಷ್ಯನು ತನ್ನ ಮೋಕ್ಷಕ್ಕಾಗಿ ಪ್ರಯತ್ನಿಸುವು ದಿಲ್ಲವೊ ಅವನು ಆತ್ಮಹತ್ಯಕಾರನು ;೧ ಏಕೆಂದರೆ ಅವನು ಮಿಥ್ಯಾವಸ್ತುಗಳನ್ನು ಹಿಡಿದುಕೊಳ್ಳುವುದರ ದಶೆಯಿಂದ೨ ತನ್ನನ್ನು ಕೊಂದುಕೊಳ್ಳುತ್ತಾನೆ.

ಇತಃ ಕೋ ನ್ವಸ್ತಿ ಮೂಢಾತ್ಮಾ ಯಸ್ತು ಸ್ವಾರ್ಥೇ ಪ್ರಮಾದ್ಯತಿ ।
ದುರ್ಲಭಂ ಮಾನುಷಂ ದೇಹಂ ಪ್ರಾಪ್ಯ ತತ್ರಾಪಿ ಪೌರುಷಮ್ ೫

ಯಃ ತ್ಣು = ಯಾವನು ದುರ್ಲಭ್ಣಂ = ದುಃಖಸಾಧ್ಯವಾದ ಮಾನುಷ್ಣಂ =ಮನುಷ್ಯ ಸಂಬಂಧವಾದ ದೇಹ್ಣಂ = ಶರೀರವನ್ನು ಪ್ರಾಪ್ಯ್ಣ = ಪಡೆದು, ತತ್ರ ಅಪ್ಣಿ = ಅದರಲ್ಲಿಯೂ ಪೌರುಷ್ಣಂ = ಪುರುಷಸಂಬಂಧವಾದುದನ್ನು [ಹೊಂದಿ], ಸ್ವಾರ್ಥೇ್ಣ = ಸ್ವಪ್ರಯೋಜನ ದಲ್ಲಿ, ಮೋಕ್ಷದಲ್ಲಿ ಪ್ರಮಾದ್ಯತ್ಣಿ = ಅಸಡ್ಡೆ ಮಾಡುತ್ತಾನೆಯೊ ಇತ್ಣಃ = ಇವನಿಗಿಂತಲೂ ಮೂಢಾತ್ಮಾ= ಅವಿವೇಕಿಯು ಕಃ ನು ಅಸ್ತ್ಣಿ = ಯಾವನಿರುವನು ?

೫. ದುರ್ಲಭವಾದ ಮನುಷ್ಯಶರೀರವನ್ನು, ಅದರಲ್ಲಿಯೂ ಪುರುಷಶರೀರ ವನ್ನು ಪಡೆದು ಮೋಕ್ಷವಿಷಯದಲ್ಲಿ ಯಾವನು ಅಸಡ್ಡೆಮಾಡುತ್ತಾನೆಯೊ ಅವನಿ ಗಿಂತಲೂ ಅವಿವೇಕಿಯಾದವನು ಯಾವನಿರುವನು ?೧

ವದಂತು ಶಾಸ್ತ್ರಾಣಿ ಯಜಂತು ದೇವಾನ್
ಕುರ್ವಂತು ಕರ್ಮಾಣಿ ಭಜಂತು ದೇವತಾಃ ।
ಆತ್ಮೈಕ್ಯಬೋಧೇನ ವಿನಾಽಪಿ ಮುಕ್ತಿ-
ನR ಸಿಧ್ಯತಿ ಬ್ರಹ್ಮಶತಾಂತರೇಽಪಿ ೬

[ಮನುಷ್ಯರು] ಶಾಸ್ತ್ರಾಣ್ಣಿ = ಶಾಸ್ತ್ರಗಳನ್ನು ವದಂತ್ಣು = ಉಲ್ಲೇಖಿಸಲಿ, ದೇವಾನ್ ಯಜಂತ್ಣು = ದೇವತೆಗಳಿಗೆ ಯಜ್ಞಮಾಡಲಿ, ಕರ್ಮಾಣ್ಣಿ = ಕರ್ಮಗಳನ್ನು ಕುರ್ವಂತ್ಣು = ಅನುಷ್ಠಾನಮಾಡಲಿ, ದೇವತ್ಣಾಃ = ದೇವತೆಗಳನ್ನು ಭಜಂತ್ಣು = ಭಜಿಸಲಿ ; ಆತ್ಮೈಕ್ಯ ಬೋಧೇನ ವಿನಾ ಅಪ್ಣಿ = ಜೀವ-ಪರಮಾತ್ಮರ ಐಕ್ಯಜ್ಞಾನವಿಲ್ಲದಿದ್ದರೆ ಬ್ರಹ್ಮಶತಾಂತರೇ ಅಪ್ಣಿ = ಶತಬ್ರಹ್ಮರು ಪ್ರಲಯವನ್ನು ಪಡೆದರೂ ಮುಕ್ತ್ಣಿಃ = ಮುಕ್ತಿಯು ನ ಸಿಧ್ಯತ್ಣಿ = ಸಿದ್ಧಿಸುವುದಿಲ್ಲ.

೬. (ಮನುಷ್ಯರು) ಶಾಸ್ತ್ರಗಳನ್ನು ಉಲ್ಲೇಖಿಸಲಿ, ದೇವತೆಗಳಿಗೆ ಯಜ್ಞಮಾಡಲಿ, ಕರ್ಮಗಳನ್ನು೧ ಅನುಷ್ಠಿಸಲಿ, ದೇವತೆಗಳನ್ನು ಭಜಿಸಲಿ ; ಜೀವ-ಪರಮಾತ್ಮರ ಐಕ್ಯಜ್ಞಾನವಿಲ್ಲದೆ ನೂರು ಮಂದಿ ಬ್ರಹ್ಮರು ಪ್ರಲಯವನ್ನು ಪಡೆದರೂ೨ ಮುಕ್ತಿಯು ಸಿದ್ಧಿಸುವುದಿಲ್ಲ.

ಅಮೃತತ್ವಸ್ಯ ನಾಶಾಽಸ್ತಿ ವಿತ್ತೇನೇತ್ಯೇವ ಹಿ ಶ್ರುತಿಃ ।
ಬ್ರವೀತಿ ಕರ್ಮಣೋ ಮುಕ್ತೇರಹೇತುತ್ವಂ ಸ್ಫುಟಂ ಯತಃ ೭

ವಿತ್ತೇನ್ಣ = ದ್ರವ್ಯದಿಂದ ಅಮೃತತ್ವಸ್ಯ್ಣ = ಅಮೃತತ್ವದ ಆಶ್ಣಾ = ಆಶೆಯು ನ ಅಸ್ತ್ಣಿ = ಇಲ್ಲ, ಇತಿ ಏವ ಹ್ಣಿ = ಹೀಗೆಂದೇ ಶ್ರುತ್ಣಿಃ = ಶ್ರುತಿಯು ಬ್ರವೀತ್ಣಿ = ಹೇಳುತ್ತದೆ ; ಯತ್ಣಃ = ಏಕೆಂದರೆ ಮುಕ್ತ್ಣೇಃ = ಮುಕ್ತಿಗೆ ಕರ್ಮಣ್ಣಃ =ಕರ್ಮದ ಅಹೇತುತ್ವ್ಣಂ = ಅಕಾರಣತ್ವವು ಸ್ಫುಟ್ಣಂ = ಸ್ಫುಟವಾಗಿದೆ.

೭. ದ್ರವ್ಯದ೧ ಮೂಲಕ ಅಮೃತತ್ವದ ಆಶೆಯಿಲ್ಲ–ಹೀಗೆಂದೇ ಶ್ರುತಿಯು ಹೇಳುತ್ತದೆ೨ ; ಏಕೆಂದರೆ ಮುಕ್ತಿಗೆ ಕರ್ಮವು ಕಾರಣವಲ್ಲವೆಂಬುದು ಸ್ಫುಟ ವಾಗಿದೆ.೩

೩. ಜ್ಞಾನಪ್ರಾಪ್ತಿಗೆ ಸಾಧನಗಳು

ಅತೋ ವಿಮುಕ್ತ್ಯೈ ಪ್ರಯತೇತ ವಿದ್ವಾನ್
ಸಂನ್ಯಸ್ತ-ಬಾಹ್ಯಾರ್ಥ-ಸುಖಸ್ಪೃಹಃ ಸನ್ ।
ಸಂತಂ ಮಹಾಂತಂ ಸಮುಪೇತ್ಯ ದೇಶಿಕಂ
ತೇನೋಪದಿಷ್ಟಾರ್ಥ-ಸಮಾಹಿತಾತ್ಮಾ ೮

ಅತ್ಣಃ = ಆದುದರಿಂದ ವಿದ್ವಾನ್ಣ್​ = ವಿವೇಕಿಯು ಸಂನ್ಯಸ್ತ-ಬಾಹ್ಯಾರ್ಥ-ಸುಖ-ಸ್ಪೃಹಃ ಸನ್ಣ್​ = ಬಾಹ್ಯ ವಿಷಯಗಳಿಂದಾಗುವ ಸುಖದ ಆಶೆಯನ್ನು ತ್ಯಜಿಸಿದವನಾಗಿ ಸಂತ್ಣಂ =ಸತ್ಪುರುಷನಾದ ಮಹಾಂತ್ಣಂ =ಮಹಾತ್ಮನಾದ ದೇಶಿಕ್ಣಂ = ಆಚಾರ್ಯನನ್ನು ಸಮು ಪೇತ್ಯ್ಣ = ಬಳಿಸಾರಿ ತೇನ್ಣ = ಅವನಿಂದ ಉಪದಿಷ್ಟಾರ್ಥ-ಸಮಾಹಿತಾತ್ಮ್ಣಾ = ಉಪದೇಶಿ ಸಲ್ಪಟ್ಟ ತತ್ತ್ವದಲ್ಲಿ ಸಮಾಹಿತವಾದ ಮನುಸ್ಸುಳ್ಳವನಾಗಿ ಮುಕ್ತ್ಯೆ್ಣೈ = ಮುಕ್ತಿಗಾಗಿ ಪ್ರಯತೇತ = ಪ್ರಯತ್ನಿಸಬೇಕು.

೮. ಆದುದರಿಂದ ವಿವೇಕಿಯಾದವನು (ರೂಪ ರಸ ಮೊದಲಾದ) ಬಾಹ್ಯ ವಿಷಯಗಳಿಂದಾಗುವ ಸುಖದ ಆಶೆಯನ್ನು ತ್ಯಜಿಸಿ, ಸತ್ಪುರುಷನೂ ಮಹಾತ್ಮನೂ ಆದ ಆಚಾರ್ಯನನ್ನು೧ ಬಳಿ ಸಾರಿ,೨ ಅವನು ಉಪದೇಶಿಸಿದ ತತ್ತ್ವದಲ್ಲಿ ಸಮಾ ಹಿತವಾದ ಮನಸ್ಸುಳ್ಳವನಾಗಿ ಮುಕ್ತಿಗಾಗಿ ಪ್ರಯತ್ನಿಸಬೇಕು.

ಉದ್ಧರೇದಾತ್ಮನಾಽಽತ್ಮಾನಂ ಮಗ್ನಂ ಸಂಸಾರವಾರಿಧೌ ।
ಯೋಗಾರೂಢತ್ವಮಾಸಾದ್ಯ ಸಮ್ಯಗ್ದರ್ಶನ-ನಿಷ್ಠಯಾ ೯

ಸಮ್ಯಗ್​-ದರ್ಶನ-ನಿಷ್ಠಯ್ಣಾ = ಸಮ್ಯಗ್​ಜ್ಞಾನದಲ್ಲಿರುವ ನಿಷ್ಠೆಯಿಂದ ಯೋಗಾ ರೂಢತ್ವ್ಣಂ = ಯೋಗಾರೂಢತ್ವವನ್ನು ಆಸಾದ್ಯ್ಣ = ಪಡೆದು ಸಂಸಾರ-ವಾರಿಧ್ಣೌ = ಸಂಸಾರ ಸಮುದ್ರದಲ್ಲಿ ಮಗ್ನಂ = ಮುಳುಗಿರುವ ಆತ್ಮನಾ = ತನ್ನನ್ನು ಆತ್ಮನ್ಣಾ = ತನ್ನಿಂದಲೇ ಉದ್ಧರೇತ್ಣ್​ = ಉದ್ಧರಿಸಿಕೊಳ್ಳಬೇಕು.

೯. ಸಮ್ಯಗ್​ಜ್ಞಾನದಲ್ಲಿರುವ ನಿಷ್ಠೆಯಿಂದ ಯೋಗಾರೂಢತ್ವವನ್ನು೧ ಪಡೆದು, ಸಂಸಾರಸಮುದ್ರದಲ್ಲಿ೨ ಮುಳುಗಿರುವ ತನ್ನನ್ನು ತನ್ನಿಂದಲೇ ಉದ್ಧರಿಸಿಕೊಳ್ಳ ಬೇಕು.೩

ಸಂನ್ಯಸ್ಯ ಸರ್ವಕರ್ಮಾಣಿ ಭವಬಂಧ-ವಿಮುಕ್ತಯೇ ।
ಯತ್ಯತಾಂ ಪಂಡಿತೈರ್ಧೀರೈರಾತ್ಮಾಭ್ಯಾಸ ಉಪಸ್ಥಿತೈಃ ॥ ೧೦ ॥

ಸರ್ವಕರ್ಮಾಣ್ಣಿ = ಸರ್ವಕರ್ಮಗಳನ್ನು ಸಂನ್ಯಸ್ಯ್ಣ = ತ್ಯಜಿಸಿ ಭವಬಂಧವಿಮುಕ್ತಯ್ಣೇ = ಸಂಸಾರವೆಂಬ ಕಟ್ಟಿನಿಂದ ಬಿಡಿಸಿಕೊಳ್ಳಲು ಆತ್ಮಾಭ್ಯಾಸ್ಣೇ = ಆತ್ಮಚಿಂತನದಲ್ಲಿ ಉಪಸ್ಥಿತ್ಣೈಃ = ನಿಷ್ಠರಾದ ಧೀರ್ಣೈಃ = ಧೀಮಂತರಾದ ಪಂಡಿತ್ಣೈಃ = ವಿವೇಕಿಗಳಿಂದ ಯತ್ಯತ್ಣಾಂ = ಪ್ರಯತ್ನವು ಮಾಡಲ್ಪಡಲಿ.

೧೦. ಆತ್ಮಚಿಂತನದಲ್ಲಿ೧ ನಿಷ್ಠೆಯುಳ್ಳವರೂ ಧೀಮಂತರೂ೨ ಆದ ವಿವೇಕಿ ಗಳು ಸರ್ವಕರ್ಮಗಳನ್ನೂ೩ ತ್ಯಜಿಸಿ ಸಂಸಾರವೆಂಬ ಕಟ್ಟಿನಿಂದ ಬಿಡಿಸಿಕೊಳ್ಳಲು ಪ್ರಯತ್ನಿಸಲಿ.

ಚಿತ್ತಸ್ಯ ಶುದ್ಧಯೇ ಕರ್ಮ ನ ತು ವಸ್ತೂಪಲಬ್ಧಯೇ ।
ವಸ್ತುಸಿದ್ಧಿರ್ವಿಚಾರೇಣ ನ ಕಿಂಚಿತ್ ಕರ್ಮಕೋಟಿಭಿಃ ೧೧

ಕರ್ಮ್ಣ = ಕರ್ಮವು ಚಿತ್ತಸ್ಯ್ಣ = ಚಿತ್ತದ ಶುದ್ಧಯ್ಣೇ =ಶುದ್ಧಿಗಾಗಿ, ವಸ್ತು ಉಪ ಲಬ್ಧಯ್ಣೇ = ಪರವಸ್ತುವಿನ ಸಕ್ಷಾತ್ಕಾರಕ್ಕಾಗಿ ನ ತ್ಣು = ಅಲ್ಲ ; ವಸ್ತುಸಿದ್ಧ್ಣಿಃ = ವಸ್ತು ಸಾಕ್ಷಾತ್ಕಾರವು ವಿಚಾರೇಣ್ಣ = ವಿಚಾರದಿಂದ (ಉಂಟಾಗುವುದು), ಕರ್ಮಕೋಟಿಭ್ಣಿಃ = ಕೋಟಿಕರ್ಮಗಳಿಂದಲೂ ನ ಕಿಂಚಿತ್ಣ್​ = ಏನೂ ಆಗುವುದಿಲ್ಲ.

೧೧. ಕರ್ಮವು ಚಿತ್ತಶುದ್ಧಿಗಾಗಿ ಇರುವುದೇ ಹೊರತು ಪರವಸ್ತುವಿನ ಸಾಕ್ಷಾ ತ್ಕಾರಕ್ಕಾಗಿ ಅಲ್ಲ. ವಸ್ತುಸಾಕ್ಷಾತ್ಕಾರವು ವಿಚಾರದಿಂದ ಉಂಟಾಗುವುದೇ ವಿನಾ ಕೋಟಿಕರ್ಮಗಳಿಂದಲೂ ಆಗುವುದಿಲ್ಲ.

[ಫಲಾಭಿಸಂಧಿಯಿಲ್ಲದೆ ಮಾಡಿದ ಕರ್ಮವು ಚಿತ್ತಶುದ್ಧಿಯ ಮೂಲಕ ಆತ್ಮಜ್ಞಾನಕ್ಕೆ ಅವಕಾಶವನ್ನುಂಟುಮಾಡುತ್ತದೆ ; ಆತ್ಮಸಾಕ್ಷಾತ್ಕಾರವು ಮಾತ್ರ ನೇರವಾಗಿ ಜ್ಞಾನದಿಂದಲೇ ಉಂಟಾಗುತ್ತದೆ ಎಂಬುದು ಅಭಿಪ್ರಾಯ.]

ಸಮ್ಯಗ್ವಿಚಾರತಃ ಸಿದ್ಧಾ ರಜ್ಜು-ತತ್ತ್ವಾವಧಾರಣಾ ।
ಭ್ರಾಂತ್ಯೋದಿತ-ಮಹಾಸರ್ಪ-ಭಯದುಃಖವಿನಾಶಿನೀ ೧೨

ಭ್ರಾಂತ್ಯಾ =ಭ್ರಾಂತಿಯಿಂದ ಉದಿತ-ಮಹಾಸರ್ಪ-ಭಯದುಃಖ-ವಿನಾಶಿನ್ಣೀ = ಹುಟ್ಟಿದ ಮಹಾಸರ್ಪದಿಂದ ಉಂಟಾದ ಭಯದುಃಖಗಳನ್ನು ಹೋಗಲಾಡಿಸುವ ರಜ್ಜು- ತತ್ತ್ವ-ಅವಧಾರಣ್ಣಾ = ಹಗ್ಗವೆಂಬ ನಿಜಸ್ಥಿತಿಯ ನಿಶ್ಚಯವು ಸಮ್ಯಗ್​-ವಿಚಾರತ್ಣಃ = ಸಮ್ಯಗ್ವಿಚಾರದಿಂದಲೇ ಸಿದ್ಧ್ಣಾ = ಸಿದ್ಧವಾಗುವುದು.

೧೨. ಭ್ರಾಂತಿಜನಿತವಾದ ಮಹಾಸರ್ಪದಿಂದ ಉಂಟಾಗುವ ಭಯದುಃಖ ಗಳನ್ನು ಹೋಗಲಾಡಿಸುವ ಹಗ್ಗವೆಂಬ ನಿಜಸ್ಥಿತಿಯ ನಿಶ್ಚಯವು ಸಮ್ಯಗ್ವಿಚಾರ ದಿಂದಲೇ ಸಿದ್ಧವಾಗುವುದು.

[ನಸುಗತ್ತಲೆಯಲ್ಲಿ ಬಿದ್ದಿರುವ ಹಗ್ಗವನ್ನು ಹಾವೆಂದು ಭಾವಿಸಿಕೊಂಡು ಹೆದರಿದ ಮನುಷ್ಯನು ಒಂದು ದೀಪವನ್ನು ತಂದು ‘ಇದು ಹಗ್ಗ, ಹಾವಲ್ಲ’ ಎಂದು ವಿಚಾರದ ಮೂಲಕ ವಸ್ತುತತ್ತ್ವವನ್ನು ಅರಿತುಕೊಂಡು ಭಯವನ್ನು ಬಿಡುವುದು ಎಲ್ಲರಿಗೂ ತಿಳಿದ ವಿಷಯ. ಹೀಗೆಯೇ ಶರೀರೇಂದ್ರಿಯ-ಸಂಘಾತವನ್ನು ಆತ್ಮನೆಂದು ಭಾವಿಸಿಕೊಂಡು ಆತ್ಮನ ಸ್ವರೂಪವನ್ನು ನಿರ್ಧರಿಸಿದ ಮೇಲೆ ಭ್ರಾಂತಿಯಿಂದ ಊಂಟಾಗಿದ್ದ ಭಯದುಃಖ ಗಳನ್ನು ಬಿಡುತ್ತಾನೆ.]

ಅರ್ಥಸ್ಯ ನಿಶ್ಚಯೋ ದೃಷ್ಟೋ ವಿಚಾರೇಣ ಹಿತೋಕ್ತಿತಃ ।
ನ ಸ್ನಾನೇನ ನ ದಾನೇನ ಪ್ರಾಣಾಯಾಮ-ಶತೇನ ವಾ ೧೩

ಅರ್ಥಸ್ಯ = ತತ್ತ್ವದ ನಿಶ್ಚಯ್ಣಃ = ನಿಶ್ಚಯವು ಹಿತೋಕ್ತಿತ್ಣಃ = ಆಪ್ತವಾಕ್ಯದಿಂದಲೂ ವಿಚಾರೇಣ್ಣ = ವಿಚಾರದಿಂದಲೂ ದೃಷ್ಟ್ಣಃ = ಕಂಡುಬರುತ್ತದೆ ; ಸ್ನಾನೇನ್ಣಅ = ಸ್ನಾನಮಾಡು ವುದರಿಂದಲೂ ನ್ಣ =ಇಲ್ಲ, ದಾನೇನ್ಣ = ದಾನದಿಂದಲೂ ವ್ಣಾ = ಅಥವಾ ಪ್ರಾಣಾಯಾಮ- ಶತೇನ = ನೂರಾರು ಪ್ರಾಣಾಯಾಮಗಳಿಂದಲೂ ನ್ಣ = ಇಲ್ಲ.

೧೩. ತತ್ತ್ವನಿಶ್ಚಯವು ಆಪ್ತವಾಕ್ಯವನ್ನು೧ ಅನುಸರಿಸಿದ ವಿಚಾರದಿಂದ೨ ಸಿದ್ಧಿ ಸುತ್ತದೆಯೇ ವಿನಾ ಸ್ನಾನದಿಂದಾಗಲಿ ದಾನದಿಂದಾಗಲಿ ಅಥವಾ ನೂರಾರು ಪ್ರಾಣಾಯಾಮಗಳಿಂದಾಗಲಿ ಸಿದ್ಧಿಸುವುದಿಲ್ಲ.

[೧ ಆಪ್ತರಾದ ಆಚಾರ್ಯರು ಉಪದೇಶಮಾಡುವ ಯಥಾರ್ಥವಾದ ಮಾತು ಹಿತೋಕ್ತಿ ಎನಿಸುವುದು.

೨ ವಸ್ತುವನ್ನು ಯಥಾರ್ಥವಾಗಿ ನಿರ್ಧರಿಸಲು ಅನುಕೂಲವಾಗಿರುವ ಮಾನಸ ವ್ಯಾಪಾರವು ವಿಚಾರವೆನಿಸುವುದು.]

೪. ಅಧಿಕಾರಿಯ ನಿರೂಪಣ

ಅಧಿಕಾರಿಣಮಾಶಾಸ್ತೇ ಫಲಸಿದ್ಧಿರ್ವಿಶೇಷತಃ ।
ಉಪಾಯಾ ದೇಶ-ಕಾಲಾದ್ಯಾಃ ಸಂತ್ಯಸ್ಮಿನ್ ಸಹಕಾರಿಣಃ ೧೪

ಫಲಸಿದ್ಧ್ಣಿಃ = ಫಲಪ್ರಾಪ್ತಿಯು ಅಧಿಕಾರಣ್ಣಂ = ಅಧಿಕಾರಿಯನ್ನು ವಿಶೇಷತ್ಣಃ = ವಿಶೇಷ ವಾಗಿ ಆಶಾಸ್ತ್ಣೇ = ಬಯಸುತ್ತದೆ ; ಅಸ್ಮಿನ್ಣ್​ = ಈ ವಿಷಯದಲ್ಲಿ ದೇಶಕಾಲಾದ್ಯ್ಣಾಃ = ದೇಶಕಾಲಾದಿಗಳು ಸಹಕಾರಿಣ್ಣಃ =ಸಹಕಾರಿಗಳಾದ ಉಪಾಯಾಃ ಸಂತ್ಣಿ =ಸಾಧನಗಳಾಗಿವೆ.

೧೪. (ಬ್ರಹ್ಮಜ್ಞಾನವೆಂಬ) ಫಲಪ್ರಾಪ್ತಿಯು (ಶಮ-ದಮಾದಿಗಳಿಂದ ಕೂಡಿದ) ಅಧಿಕಾರಿಯನ್ನು ವಿಶೇಷವಾಗಿ ಅಪೇಕ್ಷಿಸುತ್ತದೆ. ಈ ವಿಷಯದಲ್ಲಿ ದೇಶ ಕಾಲ ಮೊದಲಾದುವು ಸಹಾಯಕ ಸಾಧನಗಳಾಗಿವೆ.

[ಮುಮುಕ್ಷುವು ಮುಖ್ಯವಾಗಿ ಸಮರ್ಥನಾಗಿರಬೇಕು ; ಏಕೆಂದರೆ ಒಳಗಿನ ಸಾಮರ್ಥ್ಯವೇ ಫಲವನ್ನು ಕೊಡುವುದೇ ವಿನಾ ಇತರ ಹೊರಗಿನ ಉಪಕರಣಗಳಲ್ಲ. ಈ ಮಾರ್ಗದಲ್ಲಿ ನಿರತರಾಗಿರುವವರು ಯದಾಹವನೀಯೇ ಜುಹೋತಿ, ಸಾಯಂ ಜುಹೋತ್ಣಿ ಎಂದು ಶಾಸ್ತ್ರದಲ್ಲಿ ಹೇಳಿರುವ ಪ್ರಕಾರ ಅಸಾಧಾರಣವಾದ ದೇಶಕಾಲಾದಿಗಳನ್ನು ಅವ ಲಂಬಿಸಬೇಕಾಗಿರುತ್ತದೆ. ಆದರೆ ಬ್ರಹ್ಮಜ್ಞಾನಕ್ಕೆ ಅಂಥ ದೇಶಕಾಲಗಳ ನಿಯಮವಿಲ್ಲ. ಅದನ್ನಪೇಕ್ಷಿಸುವವನಿಗೆ ನಿರ್ಜನಾದಿ ಪ್ರದೇಶಗಳು, ಅನುಕೂಲವಾದ ಮನಸ್ಸಮಾಧಾನದ ಕಾಲ, ಸತ್ಸಂಗ ಇವೇ ಮೊದಲಾದವು ಸಹಕಾರಿ-ಕಾರಣಗಳಾಗಿರುತ್ತವೆ.]

ಣಿ೨ಣಬ೧೩ಶ್ಲಸ್ನ್ಇವ್ಶ್ಶ್ ಪ್ ಅತೋ ವಿಚಾರಃ ಕರ್ತವ್ಯೋ ಜಿಜ್ಞಾಸೋರಾತ್ಮವಸ್ತುನಃ ।
ಸಮಾಸಾದ್ಯ ದಯಾಸಿಂಧುಂ ಗುರುಂ ಬ್ರಹ್ಮವಿದುತ್ತಮಮ್ ॥ ೧೫ ॥

ಅತ್ಣಃ = ಆದುದರಿಂದ ಬ್ರಹ್ಮವಿದುತ್ತಮ್ಣಂ = ಬ್ರಹ್ಮಜ್ಞಾನಿಗಳಲ್ಲಿ ಶ್ರೇಷ್ಠನಾದ ದಯಾಸಿಂಧ್ಣುಂ =ದಯಾಸಾಗರನಾದ ಗುರ್ಣುಂ = ಗುರುವನ್ನು ಸಮಾಸಾದ್ಯ್ಣ = ಬಳಿಸಾರಿ ಜಿಜ್ಞಾಸ್ಣೋಃ = ಜಿಜ್ಞಾಸುವಿಗೆ ಆತ್ಮವಸ್ತುನ್ಣಃ = ಆತ್ಮತತ್ತ್ವದ ವಿಚಾರ್ಣಃ = ವಿಚಾರವು ಕರ್ತವ್ಯ್ಣಃ = ಮಾಡತಕ್ಕದ್ದಾಗಿದೆ.

೧೫. ಆದುದರಿಂದ ಜಿಜ್ಞಾಸುವು ಬ್ರಹ್ಮಜ್ಞಾನಿಗಳಲ್ಲಿ ಶ್ರೇಷ್ಠನೂ ದಯಾ ಸಾಗರನೂ ಆದ ಗುರುವನ್ನು೧ ಬಳಿಸಾರಿ ಆತ್ಮತತ್ತ್ವದ ವಿಚಾರವನ್ನು ಮಾಡ ಬೇಕು.

[೧ ಗು-ಶಬ್ದವು ಅಂಧಕಾರವನ್ನು ರು-ಶಬ್ದವು ಅದರ ನಾಶವನ್ನೂ ಸೂಚಿಸುತ್ತವೆ : ಅಂಧಕಾರವನ್ನು ಎಂದರೆ ಅಜ್ಞಾನವೆಂಬ ಹೃದಯಾಂಧಕಾರವನ್ನು ನಾಶಮಾಡುವವನಿಗೆ ಗುರು ಎಂದು ಹೆಸರು. ಗುಶಬ್ದಸ್ತ್ವಂಧಕಾರಸ್ಸ್ಯಾತ್ ರುಶಬ್ದಸ್ತನ್ನಿರೋಧಕಃ । ಅಂಧಕಾರ ನಿರೋಧಿತ್ವಾತ್ ಗುರುರಿತ್ಯಭಿಧೀಯತೇ ॥ ಎಂದು ಗುರುಗೀತೆಯು ಹೇಳುತ್ತದೆ.]

ಣಿ೨ಣಬ೧೩ಶ್ಲಸ್ನ್ಇವ್ಶ್ಶ್ ಪ್ ಮೇಧಾವೀ ಪುರುಷೋ ವಿದ್ವಾನ್ ಊಹಾಪೋಹ-ವಿಚಕ್ಷಣಃ ।
ಅಧಿಕಾರ್ಯಾತ್ಮವಿದ್ಯಾಯಾಮುಕ್ತಲಕ್ಷಣ-ಲಕ್ಷಿತಃ ॥ ೧೬ ॥

ಮೇಧಾವ್ಣೀ = ಮೇಧಾವಿಯಾದ ವಿದ್ವಾನ್ಣ್​ = ವಿದ್ವಾಂಸನಾದ ಊಹಾಪೋಹ್ಣ ವಿಚಕ್ಷಣ್ಣಃ = ಊಹಾಪೋಹಗಳಲ್ಲಿ ಚತುರನಾದ ಉಕ್ತ-ಲಕ್ಷಣ-ಲಕ್ಷಿತ್ಣಃ = ಇಲ್ಲಿಯವರೆಗೆ ಹೇಳಲ್ಪಟ್ಟ ಲಕ್ಷಣಗಳನ್ನುಳ್ಳ ಪುರುಷ್ಣಃ = ಪುರುಷನು ಆತ್ಮವಿದ್ಯಾಯ್ಣಾಂ = ಆತ್ಮವಿದ್ಯೆ ಯಲ್ಲಿ ಅಧಿಕಾರ್ಣೀ = ಅಧಿಕಾರಿಯು.

೧೬. ಮೇಧಾವಿಯೂ೧ ವಿದ್ವಾಂಸನೂ ಊಹಾಪೋಹಗಳಲ್ಲಿ ಚತುರನೂ೨ ಇಲ್ಲಿಯವರೆಗೆ ಹೇಳಲ್ಪಟ್ಟ ಲಕ್ಷಣಗಳುಳ್ಳವನೂ೩ ಆದ ಪುರುಷನು ಆತ್ಮವಿದ್ಯೆ ಯಲ್ಲಿ ಅಧಿಕಾರಿಯಾಗಿರುತ್ತಾನೆ.೪

[ಬ್ರಹ್ಮಜ್ಞಾನವೆಂಬ ಫಲಕ್ಕೆ ಅಧಿಕಾರಿಯಾದ ಪುರುಷನ ಲಕ್ಷಣಗಳನ್ನು ಇಲ್ಲಿ ಹೇಳಿದೆ.

೧ ಕೇಳಿದ ಅರ್ಥವನ್ನು ನೆನಪಿನಲ್ಲಿಡುವ ಸಾಮರ್ಥ್ಯವುಳ್ಳವನು.

೨ ತರ್ಕ-ಮೀಮಾಂಸೆಗಳಲ್ಲಿ ನುರಿತವನು.

೩ ೮ನೆಯ ಶ್ಲೋಕದಿಂದ ೧೫ನೆಯ ಶ್ಲೋಕದವರೆಗೆ ಸೂಚಿಸಲ್ಪಟ್ಟ ವಿವೇಕ ವೈರಾಗ್ಯ ಮೊದಲಾದುವುಗಳಿಂದ ಕೂಡಿರುವವನು.

೪ ಆತ್ಮವಿದ್ಯಾಫಲವೆಂಬ ಮೋಕ್ಷವನ್ನು ಅನುಭವಿಸಲು ಅರ್ಹನಾಗಿರುತ್ತಾನೆ.]

ವಿವೇಕಿನೋ ವಿರಕ್ತಸ್ಯ ಶಮಾದಿಗುಣ-ಶಾಲಿನಃ ।
ಮುಮುಕ್ಷೋರೇವ ಹಿ ಬ್ರಹ್ಮಜಿಜ್ಞಾಸಾ-ಯೋಗ್ಯತಾ ಮತಾ ೧೭

ವಿವೇಕಿನ್ಣಃ = ವಿವೇಕಿಯಾದ ವಿರಕ್ತಸ್ಯ್ಣ = ವಿರಕ್ತನಾದ ಶಮಾದಿಗುಣಶಾಲಿನ್ಣಃ = ಶಮಾದಿಗುಣಗಳಿಂದ ಕೂಡಿರುವ ಮುಮುಕ್ಷೋಃ ಏವ ಹ್ಣಿ = ಮುಮುಕ್ಷುವಿಗೆ ಮಾತ್ರ ಬ್ರಹ್ಮಜಿಜ್ಞಾಸಾಯೋಗ್ಯತ್ಣಾ = ಬ್ರಹ್ಮವಿಚಾರದಲ್ಲಿ ಅರ್ಹತೆ (ಇರುವುದೆಂಬುದು) ಮತ್ಣಾ = ಸಮ್ಮತವು.

೧೭. ವಿವೇಕಿಯೂ೧ ವಿರಕ್ತನೂ೨ ಶಮ ದಮ ಮೊದಲಾದ ಗುಣಗಳಿಂದ ಕೂಡಿರುವವನೂ ಆದ ಮುಮುಕ್ಷುವಿಗೆ ಮಾತ್ರ ಬ್ರಹ್ಮವಿಚಾರವನ್ನು೩ ಮಾಡುವುದ ರಲ್ಲಿ ಅರ್ಹತೆಯಿರುತ್ತದೆ.

[೧ ನಿತ್ಯಾನಿತ್ಯವಸ್ತುಗಳನ್ನು ವಿವೇಚನೆ ಮಾಡುವವನು.

೨ ವಿಷಯಗಳಲ್ಲಿ ಆಸಕ್ತನಾಗದವನು.

೩ ತಿಳಿದುಕೊಳ್ಳಬೇಕೆಂಬ ಬಯಕೆಗೆ ಜಿಜ್ಞಾಸೆ ಎಂದು ಹೆಸರು ; ಬ್ರಹ್ಮದ ಜಿಜ್ಞಾಸೆ ಬ್ರಹ್ಮಜಿಜ್ಞಾಸೆ.]

೫. ಸಾಧನಚತುಷ್ಟಯ

ಸಾಧನಾನ್ಯತ್ರ ಚತ್ವಾರಿ ಕಥಿತಾನಿ ಮನೀಷಿಭಿಃ ।
ಯೇಷು ಸತ್ಸ್ವೇವ ಸನ್ನಿಷ್ಠಾ ಯದಭಾವೇ ನ ಸಿಧ್ಯತಿ ೧೮

ಅತ್ರ್ಣ = ಈ ವಿಷಯದಲ್ಲಿ ಮನೀಷಿಭ್ಣಿಃ = ಜ್ಞಾನಿಗಳಿಂದ ಚತ್ವಾರ್ಣಿ = ನಾಲ್ಕು ಸಾಧನಾನ್ಣಿ = ಸಾಧನಗಳು ಕಥಿತಾನ್ಣಿ = ಹೇಳಲ್ಪಟ್ಟಿವೆ ; ಯೇಷು ಸತ್ಸು ಏವ್ಣ = ಯಾವ ಸಾಧನಗಳಿದ್ದರೆ ಸನ್ನಿಷ್ಠ್ಣಾ = ಬ್ರಹ್ಮನಿಷ್ಠೆಯು ಸಿಧ್ಯತ್ಣಿ = ಸಿದ್ಧಿಸುವುದು, ಯದಭಾವ್ಣೇ = ಯಾವುದು ಇಲ್ಲದಿದ್ದರೆ ನ್ಣಅ = ಸಿದ್ಧಿಸುವುದಿಲ್ಲ.

೧೮. ಈ ಬ್ರಹ್ಮಜ್ಞಾನವಿಷಯದಲ್ಲಿ ಜ್ಞಾನಿಗಳು ನಾಲ್ಕು ಸಾಧನಗಳನ್ನು ಹೇಳಿರು ತ್ತಾರೆ. ಇವು ಇದ್ದರೇ ಬ್ರಹ್ಮನಿಷ್ಠೆಯು ಸಿದ್ಧಿಸುವುದು, ಇಲ್ಲದಿದ್ದರೆ ಸಿದ್ಧಿಸುವು ದಿಲ್ಲ.

ಣಿ೨ಣಬ೧೩ಶ್ಲಸ್ನ್ಇವ್ಶ್ಶ್ ಪ್ ಆದೌ ನಿತ್ಯಾನಿತ್ಯ-ವಸ್ತು-ವಿವೇಕಃ ಪರಿಗಣ್ಯತೇ ।
ಇಹಾಮುತ್ರ-ಫಲಭೋಗ-ವಿರಾಗಸ್ತದನಂತರಮ್ ।
ಶಮಾದಿ-ಷಟ್ಕ-ಸಂಪತ್ತಿರ್ಮುಮುಕ್ಷುತ್ವಮಿತಿ ಸ್ಫುಟಮ್ ॥ ೧೯ ॥

ಆದ್ಣೌ = ಮೊದಲು ನಿತ್ಯ-ಅನಿತ್ಯ-ವಸ್ತು-ವಿವೇಕ್ಣಃ = ನಿತ್ಯಾನಿತ್ಯ-ವಸ್ತು-ವಿವೇಕವು ಪರಿಗಣ್ಯತ್ಣೇ = ಎಣಿಸಲ್ಪಡುತ್ತದೆ ; ತದ್​-ಅನಂತರ್ಣಂ = ಅದಾದಮೇಲೆ ಇಹ- ಅಮುತ್ರ- ಫಲಭೋಗ-ವಿರಾಗ್ಣಃ = ಇಲ್ಲಿ ಮತ್ತು ಪರಲೋಕದಲ್ಲಿ (ಅನುಭವಿಸುವ) ಕರ್ಮಫಲದ ಭೋಗಗಳಲ್ಲಿ ವೈರಾಗ್ಯವು (ಎಣಿಸಲ್ಪಡುತ್ತದೆ) ; (ಅದಾದಮೇಲೆ) ಶಮಾದಿ-ಷಟ್ಕ- ಸಂಪತ್ತ್ಣಿಃ =ಶಮವೇ ಮೊದಲಾದ ಆರರ ಲಾಭವು (ಎಣಿಸಲ್ಪಡುತ್ತದೆ) ; (ಅದಾದಮೇಲೆ) ಮುಮುಕ್ಷುತ್ವ್ಣಂ = ಮೋಕ್ಷೇಚ್ಛೆಯು ಇತ್ಣಿ = ಹೀಗೆ (ಸಾಧನಚತುಷ್ಟಯವು) ಸ್ಫುಟ್ಣಂ = ಸ್ಫುಟವಾಗಿದೆ.

೧೯. ನಿತ್ಯಾನಿತ್ಯವಸ್ತುವಿವೇಕವು ಮೊದಲನೆಯ ಸಾಧನವೆಂದೂ ಇಲ್ಲಿ ಮತ್ತು ಪರಲೋಕದಲ್ಲಿ ಅನುಭವಿಸುವ ಕರ್ಮಫಲದ ಭೋಗಗಳಲ್ಲಿ ವೈರಾಗ್ಯವು ಎರಡ ನೆಯ ಸಾಧನವೆಂದೂ ಶಮವೇ ಮೊದಲಾದ೧ ಆರರ ಲಾಭವು ಮೂರನೆಯ ಸಾಧನವೆಂದೂ ಮೋಕ್ಷೇಚ್ಛೆಯು ನಾಲ್ಕನೆಯ ಸಾಧನವೆಂದೂ (ಜ್ಞಾನಿಗಳು) ಸ್ಪಷ್ಟ ವಾಗಿ ಎಣಿಸುತ್ತಾರೆ.

[ವಿವೇಕವೇ ಮೊದಲಾದ ನಾಲ್ಕು ಸಾಧನಗಳಲ್ಲಿ ಹಿಂದುಹಿಂದಿನದು ಮುಂದು ಮುಂದಿನದಕ್ಕೆ ಕಾರಣವಾಗಿರುವುದರಿಂದ ಇವುಗಳನ್ನು ಕ್ರಮವಾಗಿ ಹೇಳಿದೆ.

೧ ಶಮ, ದಮ, ಉಪರತಿ, ತಿತಿಕ್ಷೆ, ಶ್ರದ್ಧೆ ಮತ್ತು ಸಮಾಧಾನ.]

ಬ್ರಹ್ಮ ಸತ್ಯಂ ಜಗನ್ಮಿಥ್ಯೇತ್ಯೇವಂರೂಪೋ ವಿನಿಶ್ಚಯಃ ।
ಸೋಽಯಂ ನಿತ್ಯಾನಿತ್ಯ-ವಸ್ತು-ವಿವೇಕಃ ಸಮುದಾಹೃತಃ ॥ ೨ಂ ॥

ಬ್ರಹ್ಮ= ಬ್ರಹ್ಮವು ಸತ್ಯ್ಣಂ = ಸತ್ಯ, ಜಗತ್ಣ್​ = ಜಗತ್ತು ಮಿಥ್ಯ್ಣಾ = ಮಿಥ್ಯೆ ಇತ್ಯೇವಂ- ರೂಪ್ಣಃ = ಈ ರೂಪದಲ್ಲಿರುವ ವಿನಿಶ್ಚಯ್ಣಃ = ವಿನಿರ್ಣಯವೆಂಬ ಸಃ ಅಯ್ಣಂ = ಆ ಇದೇ ನಿತ್ಯಾನಿತ್ಯ-ವಸ್ತು-ವಿವೇಕ್ಣಃ = ನಿತ್ಯಾನಿತ್ಯವಸ್ತುವಿವೇಕವು ಎಂದು ಸಮುದಾಹೃತ್ಣಃ = ಹೇಳ ಲ್ಪಟ್ಟಿದೆ.

೨ಂ. ಪರಬ್ರಹ್ಮವಸ್ತುವೊಂದೇ ಸತ್ಯ, ಈ ಜಗತ್ತೆಲ್ಲ ಮಿಥ್ಯೆ–ಎಂಬೀ ರೂಪದ ವಿನಿರ್ಣಯವೇ ನಿತ್ಯಾನಿತ್ಯವಸ್ತುವಿವೇಕ ಎಂದು ಹೇಳಲ್ಪಟ್ಟಿದೆ.

ತದ್ವೈರಾಗ್ಯಂ ಜಿಹಾಸಾ ಯಾ ದರ್ಶನ-ಶ್ರವಣಾದಿಭಿಃ ।
ದೇಹಾದಿ-ಬ್ರಹ್ಮಪರ್ಯಂತೇ ಹ್ಯನಿತ್ಯೇ ಭೋಗವಸ್ತುನಿ ೨೧

ದೇಹಾದಿ-ಬ್ರಹ್ಮಪರ್ಯಂತ್ಣೇ = ದೇಹದಿಂದ ಪ್ರಾರಂಭಿಸಿ ಬ್ರಹ್ಮನ ವರೆಗಿರುವ ಅನಿತ್ಯ್ಣೇ = ಅನಿತ್ಯವಾದ ಭೋಗವಸ್ತುನ್ಣಿ = ಭೋಗವಸ್ತುವಿನಲ್ಲಿ ದರ್ಶನ-ಶ್ರವಣಾದಿಭ್ಣಿಃ = ನೋಡುವುದು ಕೇಳುವುದು ಮೊದಲಾದುವುಗಳಿಂದ ಯ್ಣಾ = ಯಾವ ಜಿಹಾಸ್ಣಾ = ಜುಗುಪ್ಸೆಯು (ಉಂಟಾಗುವುದೊ) ತತ್ ಹ್ಣಿ = ಅದೇ ವೈರಾಗ್ಯ್ಣಂ = ವೈರಾಗ್ಯವು.

೨೧. ದೇಹದಿಂದ ಪ್ರಾರಂಭಿಸಿ ಬ್ರಹ್ಮನ ವರೆಗಿರುವ ಅನಿತ್ಯವಾದ ಭೋಗವಸ್ತು ವಿನಲ್ಲಿ ದರ್ಶನಶ್ರವಣಾದಿಗಳಿಂದ೧ ಯಾವ ಜುಗುಪ್ಸೆಯುಂಟಾಗುವುದೋ ಅದೇ ವೈರಾಗ್ಯವು.

[೧ ಐಹಿಕ-ದೇಹವೇ ಮೊದಲಾದ ಭೋಗ್ಯವಸ್ತುವಿನ ದರ್ಶನದಿಂದ, ಪರಲೋಕದ ಭೋಗ್ಯವಸ್ತುವಿನ ಶ್ರವಣದಿಂದ.]

ವಿರಜ್ಯ ವಿಷಯವ್ರಾತಾದ್ದೋಷದೃಷ್ಟ್ಯಾ ಮುಹುರ್ಮುಹುಃ ।
ಸ್ವಲಕ್ಷ್ಯೇ ನಿಯತಾವಸ್ಥಾ ಮನಸಃ ಶಮ ಉಚ್ಯತೇ ೨೨

ಮುಹುರ್ಮುಹ್ಣುಃ = ಪುನಃಪುನಃ ದೋಷದೃಷ್ಟ್ಯಾ್ಣ = ದೋಷವನ್ನು ನೋಡುವುದರ ಮೂಲಕ ವಿಷಯವ್ರಾತಾತ್ಣ್​ = ವಿಷಯಸಮೂಹದಿಂದ ವಿರಜ್ಣ್ಯ = ಬೇರ್ಪಡಿಸಿ ಸ್ವಲಕ್ಷ್ಯೇ್ಣ = ತನ್ನ ಗುರಿಯಲ್ಲಿ ಮನಸ್ಣಃ = ಮನಸ್ಸಿನ ನಿಯತಾವಸ್ಥ್ಣಾ = ನಿಯತವಾದ ಸ್ಥಾಪನೆಯು ಶಮ್ಣಃ = ಶಮವು (ಎಂದು) ಉಚ್ಯತ್ಣೇ = ಹೇಳಲ್ಪಡುತ್ತದೆ.

೨೨. ವಿಷಯವಸ್ತುಗಳಲ್ಲಿ ಪುನಃಪುನಃ ದೋಷವನ್ನು ನೋಡುವುದರ ಮೂಲಕ ಅವುಗಳಿಂದ ಮನಸ್ಸನ್ನು ಬೇರ್ಪಡಿಸಿ ತನ್ನ ಗುರಿಯಲ್ಲಿ೧ ಅದನ್ನು ನಿಯಮದಿಂದ ಸ್ಥಾಪಿಸುವುದೇ ಶಮವು ಎನಿಸುತ್ತದೆ.

[೧ ಸಗುಣ ಅಥವಾ ನಿರ್ಗುಣ ಬ್ರಹ್ಮದಲ್ಲಿ.]

ವಿಷಯೇಭ್ಯಃ ಪರಾವರ್ತ್ಯ ಸ್ಥಾಪನಂ ಸ್ವಸ್ವಗೋಲಕೇ ।
ಉಭಯೇಷಾಮಿಂದ್ರಿಯಾಣಾಂ ಸ ದಮಃ ಪರಿಕೀರ್ತಿತಃ ।
ಬಾಹ್ಯಾನಾಲಂಬನಂ ವೃತ್ತೇರೇಷೋಪರತಿರುತ್ತಮಾ ೨೩

ವಿಷಯೇಭ್ಯ್ಣಃ = ವಿಷಯವಸ್ತುಗಳಿಂದ ಪರಾವರ್ತ್ಯ್ಣ = ಹಿಂತಿರುಗಿಸಿ ಉಭಯೇ ಷಾಮ್ ಇಂದ್ರಿಯಾಣ್ಣಾಂ = ಎರಡು ಬಗೆಯ ಇಂದ್ರಿಯಗಳನ್ನೂ ಸ್ವಸ್ವಗೋಲಕ್ಣೇ = ತಮ್ಮ ತಮ್ಮ ಗೋಲಕಗಳಲ್ಲಿ ಸ್ಥಾಪನ್ಣಂ = ಸ್ಥಾಪಿಸುವುದು (ಯಾವುದೋ) ಸ್ಣಃ = ಅದೇ ದಮ್ಣಃ = ದಮವೆಂದು ಪರಿಕೀರ್ತಿತ್ಣಃ = ಹೇಳಲ್ಪಟ್ಟಿದೆ. ವೃತ್ತ್ಣೇಃ = ವೃತ್ತಿಯ ಬಾಹ್ಯ-ಅನಾ ಲಂಬನ್ಣಂ = ಬಾಹ್ಯವಸ್ತುಗಳನ್ನು ಅವಲಂಬಿಸದೆ ಇರುವುದು (ಯಾವುದೋ) ಏಷ್ಣಾ = ಇದೇ ಉತ್ತಮಾ ಉಪರತ್ಣಿಃ = ಉತ್ತಮವಾದ ಉಪರತಿಯು.

೨೩. ಎರಡು ಬಗೆಯ ಇಂದ್ರಿಯಗಳನ್ನೂ೧ ವಿಷಯವಸ್ತುಗಳಿಂದ ಹಿಂತಿರು ಗಿಸಿ ತಮ್ಮ ತಮ್ಮ ಗೋಲಕಗಳಲ್ಲಿ೨ ಸ್ಥಾಪಿಸುವುದೇ ದಮವೆನಿಸುತ್ತದೆ. (ಮನ ಸ್ಸಿನ) ವೃತ್ತಿಯು೩ ಹೊರಗಿನ ಯಾವ ವಸ್ತುವನ್ನೂ ಅವಲಂಬಿಸದಿರುವುದೇ ಉತ್ತಮವಾದ ಉಪರತಿಯು.

[೧ ಜ್ಞಾನೇಂದ್ರಿಯಗಳನ್ನು ಮತ್ತು ಕರ್ಮೇಂದ್ರಿಯಗಳನ್ನು. ೯೨ನೆಯ ಶ್ಲೋಕವನ್ನು ನೋಡಿ.

೨ ಕಣ್ಣು ಕಿವಿ ಮೊದಲಾದ ಇಂದ್ರಿಯಸ್ಥಾನಗಳಲ್ಲಿ.

೩ ಕೆರೆಯಿಂದ ನೀರು ತೂಬಿನ ಮೂಲಕ ಹೊರಹರಿದು ಹೇಗೆ ಗದ್ದೆಯ ಆಕಾರವನ್ನು ಪಡೆಯುತ್ತದೆಯೊ ಹಾಗೆಯೇ ಅಂತಃಸ್ಥವಾದ ಮನಸ್ಸು ಕಿವಿ ಮೊದಲಾದ ಇಂದ್ರಿಯಗಳ ಮೂಲಕ ಹೊರಬಂದು ಶಬ್ದವೇ ಮೊದಲಾದ ವಿಷಯಗಳ ಆಕಾರವನ್ನು ಪಡೆಯುತ್ತದೆ. ಹೊರಗಿನ ಇಂದ್ರಿಯಗಳು ನಿಗ್ರಹಿಸಲ್ಪಟ್ಟರೆ ಒಳಗಿರುವ ಮನಸ್ಸು ಹೊರಗಿನ ಆಕಾರ ವನ್ನು ಪಡೆಯುವುದಿಲ್ಲವೆಂಬುದು ಭಾವ.]

ಸಹನಂ ಸರ್ವದುಃಖಾನಾಮಪ್ರತೀಕಾರಪೂರ್ವಕಮ್ ।
ಚಿಂತಾ-ವಿಲಾಪ-ರಹಿತಂ ಸಾ ತಿತಿಕ್ಷಾ ನಿಗದ್ಯತೇ ೨೪

ಅಪ್ರತೀಕಾರಪೂರ್ವಕ್ಣಂ = ಪ್ರತೀಕಾರಭಾವನೆಯಿಲ್ಲದ ಚಿಂತಾ-ವಿಲಾಪ-ರಹಿತ್ಣಂ = ಚಿಂತಾಶೋಕಗಳಿಂದ ರಹಿತವಾದ ಸರ್ವದುಃಖಾನ್ಣಾಂ = ಸಮಸ್ತ ದುಃಖಗಳ ಸಹನ್ಣಂ = ಸಹನೆಯು (ಯಾವುದೋ) ಸ್ಣಾ = ಅದು ತಿತಿಕ್ಷ್ಣಾ = ತಿತಿಕ್ಷೆಯೆಂದು ನಿಗದ್ಯತ್ಣೇ = ಹೇಳಲ್ಪಡುತ್ತದೆ.

೨೪. ನಾನಾ ಬಗೆಯಾದ ದುಃಖಗಳು ಸಂಭವಿಸಿದ ಸಮಯದಲ್ಲಿ ಅವುಗಳ ಪರಿಹಾರಕ್ಕೆ ಯಾವುದೊಂದು ಪ್ರತೀಕಾರವನ್ನೂ ಮಾಡದೆ, ಮನಸ್ಸಿನಲ್ಲಿ ಕೊರಗದೆ ಮತ್ತು ಶೋಕಿಸದೆ ಸಹಿಸಿಕೊಂಡಿರುವುದನ್ನೇ ತಿತಿಕ್ಷೆ ಎಂದು ಹೇಳುತ್ತಾರೆ.

ಶಾಸ್ತ್ರಸ್ಯ ಗುರುವಾಕ್ಯಸ್ಯ ಸತ್ಯಬುದ್ಧ್ಯವಧಾರಣಮ್ ।
ಸಾ ಶ್ರದ್ಧಾ ಕಥಿತಾ ಸದ್ಭಿರ್ಯಯಾ ವಸ್ತೂಪಲಭ್ಯತೇ ೨೫

ಶಾಸ್ತ್ರಸ್ಯ್ಣ = ಶಾಸ್ತ್ರವನ್ನು ಗುರುವಾಕ್ಯಸ್ಯ್ಣ = ಗುರುವಾಕ್ಯವನ್ನು ಸತ್ಯಬುದ್ಧ್ಯವಧಾರಣ್ಣಂ = ಸತ್ಯವೆಂಬ ಬುದ್ಧಿಯಿಂದ ನಿಶ್ಚಯಿಸುವುದು (ಯಾವುದೋ), ಯಯ್ಣಾ = ಯಾವುದರಿಂದ ವಸ್ಣು್ತ= ಬ್ರಹ್ಮವಸ್ತುವು ಉಪಲಭ್ಯತ್ಣೇ = ಹೊಂದಲ್ಪಡುತ್ತದೆಯೋ ಸ್ಣಾ = ಅದು ಶ್ರದ್ದ್ಣಾ‡ = ಶ್ರದ್ಧೆಯೆಂದು ಸದ್ಭ್ಣಿಃ = ಸಾಧುಗಳಿಂದ ಕಥಿತ್ಣಾ = ಹೇಳಲ್ಪಟ್ಟಿದೆ.

೨೫. ಶಾಸ್ತ್ರ ಮತ್ತು ಗುರು ಉಪದೇಶಿಸಿರುವ ವೇದಾಂತವಾಕ್ಯ–ಇವು ಸತ್ಯ ವೆಂಬ ಬುದ್ಧಿಯಿಂದ ಮನಸ್ಸಿನಲ್ಲಿ ನಿರ್ಧರಿಸಿಕೊಳ್ಳುವುದು ಶ್ರದ್ಧೆ೧ ಎಂದು ಸಾಧುಗಳು ಹೇಳುತ್ತಾರೆ. ಇದು ಬ್ರಹ್ಮಸಾಕ್ಷಾತ್ಕಾರಕ್ಕೆ ಸಾಧನವಾಗಿರುವುದು.

[೧ ಇದು ಅಂಧಶ್ರದ್ಧೆಯಲ್ಲ. ದೃಢವಾದ ವಿಶ್ವಾಸ. ಇಂಥ ದೃಢವಿಶ್ವಾಸವಿಲ್ಲದೆ ಶಾಸ್ತ್ರಾಚಾರ್ಯರ ಉಪದೇಶವನ್ನು ಅನುಷ್ಠಿಸಲು ಸಾಧ್ಯವಾಗುವುದಿಲ್ಲ. ‘ಈ ಧರ್ಮ ದಲ್ಲಿ ಶ್ರದ್ಧೆಯಿಲ್ಲದಿರುವ ಮನುಷ್ಯರು ನನ್ನನ್ನು ಹೊಂದದೆ ಮೃತ್ಯುಸಂಸಾರದ ಮಾರ್ಗಕ್ಕೆ ಹಿಂತಿರುಗುತ್ತಾರೆ’ ಅಶ್ರದ್ದಧಾನಾಃ ಪುರುಷಾ ಧರ್ಮಸ್ಯಾಸ್ಯ ಪರಂತಪ । ಅಪ್ರಾಪ್ಯ ಮಾಂ ನಿವರ್ತಂತೇ ಮೃತ್ಯುಸಂಸಾರವರ್ತ್ಮನಿ ॥ ಎಂದು ಗೀತೆಯಲ್ಲಿ (೯. ೩) ಹೇಳಿದೆ.]

ಣಿ೨ಣಬ೧೩ಶ್ಲಸ್ನ್ಇವ್ಶ್ಶ್ ಪ್ ಸಮ್ಯಗಾಸ್ಥಾಪನಂ ಬುದ್ಧೇಃ ಶುದ್ಧೇ ಬ್ರಹ್ಮಣಿ ಸರ್ವದಾ ।
ತತ್ ಸಮಾಧಾನಮಿತ್ಯುಕ್ತಂ ನ ತು ಚಿತ್ತಸ್ಯ ಲಾಲನಮ್ ॥ ೨೬ ॥

ಶುದ್ಧೇ ಬ್ರಹ್ಮಣ್ಣಿ =ಶುದ್ಧ ಬ್ರಹ್ಮನಲ್ಲಿ ಸರ್ವದ್ಣಾ = ಯಾವಾಗಲೂ ಬುದ್ಧ್ಣೇಃ = ಬುದ್ಧಿಯನ್ನು ಸಮ್ಯಕ್ ಆಸ್ಥಾಪನ್ಣಂ = ಚೆನ್ನಾಗಿ ನೆಲೆಗೊಳಿಸುವುದು (ಯಾವುದೋ) ತತ್ಣ್​ = ಅದು ಸಮಾಧಾನಮ್ ಇತ್ಣಿ = ಸಮಾಧಾನವೆಂದು ಉಕ್ತ್ಣಂ = ಹೇಳಲ್ಪಟ್ಟಿದೆ ; ಚಿತ್ತಸ್ಯ್ಣ = ಮನಸ್ಸಿನ ಲಾಲನ್ಣಂ = ಲಾಲನೆಯು ನ ತ್ಣು = ಅಲ್ಲ.

೨೬. ಶುದ್ಧಬ್ರಹ್ಮದಲ್ಲಿ ಬುದ್ಧಿಯನ್ನು ಯಾವಾಗಲೂ ಚೆನ್ನಾಗಿ ಸ್ಥಿರಗೊಳಿಸು ವುವೇ ಸಮಾಧಾನವೆಂದು ಹೇಳಲ್ಪಟ್ಟಿದೆ೧ ; ಮನಸ್ಸಿನ ಲಾಲನೆಯು೨ ಸಮಾಧಾನ ವಲ್ಲ.

[೧ ‘ಚಂಚಲವೂ ಅಸ್ಥಿರವೂ ಆದ ಮನಸ್ಸು ಯಾವ ಯಾವ ವಿಷಯದಿಂದ ಹೊರಕ್ಕೆ ಬರುವುದೊ ಆಯಾ ವಿಷಯದಿಂದ ಮನಸ್ಸನ್ನು ನಿಗ್ರಹಿಸಿ ಆತ್ಮನಲ್ಲಿಯೇ ವಶಪಡಿಸಿ ಕೊಳ್ಳಬೇಕು’ ಯತೋ ಯತೋ ನಿಶ್ಚರತಿ ಮನಶ್ಚಂಚಲಮಸ್ಥಿರಮ್ । ತತಸ್ತತೋ ನಿಯಮ್ಯೈತದಾತ್ಮನ್ಯೇವ ವಶಂ ನಯೇತ್ ॥ ಎಂದು ಗೀತೆಯಲ್ಲಿ (೬. ೨೬) ಹೇಳಿರುವಂತೆ ಶಮಾವಸ್ಥೆಯಲ್ಲಿಯೇ ಸಂಕಲ್ಪ-ವಿಕಲ್ಪಾತ್ಮಕವಾದ ಮನಸ್ಸನ್ನು ಬ್ರಹ್ಮದಲ್ಲಿ ಸ್ಥಿರಗೊಳಿ ಸಲು ವಿಶೇಷವಾಗಿ ಪ್ರಯತ್ನಪಡಬೇಕು.

೨ ಮನಸ್ಸು ಬಯಸಿದ ವಸ್ತುಗಳನ್ನು ಕೊಟ್ಟು ಲಾಲನೆಮಾಡುವುದು ಸಮಾಧಾನ ವಲ್ಲ.]

ಣಿ೨ಣಬ೧೩ಶ್ಲಸ್ನ್ಇವ್ಶ್ಶ್ ಪ್ ಅಹಂಕಾರಾದಿ-ದೇಹಾಂತಾನ್ ಬಂಧಾನಜ್ಞಾನಕಲ್ಪಿತಾನ್ ।
ಸ್ವಸ್ವರೂಪಾವಬೋಧೇನ ಮೋಕ್ತುಮಿಚ್ಛಾ ಮುಮುಕ್ಷುತಾ ॥ ೨೭ ॥

ಅಜ್ಞಾನಕಲ್ಪಿತಾನ್ಣ್​ = ಅಜ್ಞಾನದಿಂದ ಕಲ್ಪಿತವಾದ ಅಹಂಕಾರಾದಿ-ದೇಹಾಂತಾನ್ಣ್​ = ಅಹಂಕಾರದಿಂದ ಪ್ರಾರಂಭಿಸಿ ಶರೀರದವರೆಗಿರುವ ಬಂಧಾನ್ಣ್​ = ಬಂಧನಗಳನ್ನು ಸ್ವ ಸ್ವರೂಪಾವಬೋಧೇನ್ಣ = ತನ್ನ ನಿಜವಾದ ಆತ್ಮಸ್ವರೂಪದ ಜ್ಞಾನದಿಂದ ಮೋಕ್ತ್ಣುಂ = ಬಿಡಿಸಿಕೊಳ್ಳುವುದಕ್ಕೆ ಇಚ್ಛ್ಣಾ = ಇಚ್ಛೆಯು ಮುಮುಕ್ಷುತ್ಣಾ = ಮುಮುಕ್ಷುತ್ವವು.

೨೭. ಅಹಂಕಾರದಿಂದ ಪ್ರಾರಂಭಿಸಿ ತನ್ನ ಶರೀರದವರೆಗಿರುವ, ಅಜ್ಞಾನದಿಂದ ಕಲ್ಪಿತವಾದ ಕಟ್ಟುಗಳನ್ನು ತನ್ನ ನಿಜವಾದ ಆತ್ಮಸ್ವರೂಪದ ಜ್ಞಾನದಿಂದ ಬಿಡಿಸಿ ಕೊಳ್ಳಬೇಕೆಂಬ ಇಚ್ಛೆಯು ಮುಮುಕ್ಷುತ್ವವೆನಿಸುತ್ತದೆ.

[ಎಲ್ಲವೂ ಆತ್ಮನೇ ಆಗಿರುವಾಗ ಆತ್ಮನಿಗಿಂತ ಭಿನ್ನವಾಗಿ ಕಂಡುಬರುವ ಯಾವ ವಸ್ತುವನ್ನೂ ಬಯಸದಿರುವುದರಿಂದ ಬಂಧನಿವೃತ್ತಿಯು ಆತ್ಯಂತಿಕವು ಎಂದರ್ಥ.]

ಮಂದ-ಮಧ್ಯಮ-ರೂಪಾಽಪಿ ವೈರಾಗ್ಯೇಣ ಶಮಾದಿನಾ ।
ಪ್ರಸಾದೇನ ಗುರೋಸ್ಸೇಯಂ ಪ್ರವೃದ್ಧಾ ಸೂಯತೇ ಫಲಮ್ ೨೮

ಸಾ ಇಯ್ಣಂ = ಆ ಈ (ಮುಮುಕ್ಷುತ್ವವು) ಮಂದ-ಮಧ್ಯಮ-ರೂಪಾ ಅಪ್ಣಿ = ಮಂದ-ಮಧ್ಯಮ-ಸ್ಥಿತಿಯಲ್ಲಿದ್ದರೂ ವೈರಾಗ್ಯೇಣ್ಣ = ವೈರಾಗ್ಯದಿಂದಲೂ ಶಮಾದಿನ್ಣಾ = ಶಮದಮಾದಿ ಸಾಧನದಿಂದಲೂ ಗುರ್ಣೋಃ = ಗುರುವಿನ ಪ್ರಸಾದೇನ್ಣ = ಅನುಗ್ರಹ ದಿಂದಲೂ ಪ್ರವೃದ್ಧ್ಣಾ = ಪ್ರಬಲವಾಗಿ ಫಲ್ಣಂ = ಫಲವನ್ನು ಸೂಯತ್ಣೇ = ಉಂಟು ಮಾಡುತ್ತದೆ.

೨೮. ಆ ಈ ಮುಮುಕ್ಷುತ್ವವು ಮಂದ-ಮಧ್ಯಮ-ಸ್ಥಿತಿಯಲ್ಲಿದ್ದರೂ ವೈರಾಗ್ಯ ದಿಂದಲೂ ಶಮದಮಾದಿ ಸಾಧನದಿಂದಲೂ ಗುರುವಿನ ಅನುಗ್ರಹದಿಂದಲೂ ಪ್ರಬಲವಾಗಿ (ಮೋಕ್ಷರೂಪವಾದ) ಫಲವನ್ನು ಉಂಟುಮಾಡುತ್ತದೆ.

ವೈರಾಗ್ಯಂ ಚ ಮುಮುಕ್ಷುತ್ವಂ ತೀವ್ರಂ ಯಸ್ಯ ತು ವಿದ್ಯತೇ ।
ತಸ್ಮಿನ್ನೇವಾರ್ಥವಂತಃ ಸ್ಯುಃ ಫಲವಂತಃ ಶಮಾದಯಃ ೨೯

ತ್ಣು = ಆದರೆ ಯಸ್ಯ್ಣ = ಯಾವನಿಗೆ ವೈರಾಗ್ಯ್ಣಂ = ವೈರಾಗ್ಯವು ಮುಮುಕ್ಷುತ್ವಂ ಚ್ಣ = ಮತ್ತು ಮುಮುಕ್ಷುತ್ವವು ತೀವ್ರಂ ವಿದ್ಯತ್ಣೇ = ತೀವ್ರವಾಗಿದೆಯೋ ತಸ್ಮಿನ್ ಏವ = ಅವನಲ್ಲಿಯೇ ಶಮಾದಯ್ಣಃ = ಶಮಾದಿಗಳು ಅರ್ಥವಂತ್ಣಃ = ಅರ್ಥವುಳ್ಳವೂ ಫಲವಂತ್ಣಃ = ಫಲವುಳ್ಳವೂ ಸ್ಯ್ಣುಃ = ಆಗುವುವು.

೨೯. ಆದರೆ ಯಾವನಲ್ಲಿ ವೈರಾಗ್ಯವು ಮತ್ತು ಮುಮುಕ್ಷುತ್ವವು ತೀವ್ರವಾಗಿರು ವುವೊ ಅವನಲ್ಲಿಯೆ ಶಮ-ದಮಾದಿಗಳು ಸಾರ್ಥಕವೂ ಸಫಲವೂ ಆಗುವುವು.
